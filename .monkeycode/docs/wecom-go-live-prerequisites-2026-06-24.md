# 企业微信上线前置条件清单

日期：2026-06-24

## 1. 用途

- 本清单用于在真实上线执行前确认外部依赖是否齐备。
- 只有前置条件准备完成，最后 `12%` 的任务才适合进入现场执行。

## 2. 必备访问与权限

| 项目 | 当前状态 | 说明 |
| --- | --- | --- |
| 线上服务器访问权限 | ready | 已确认可通过 SSH 只读访问服务器 |
| 线上数据库执行权限 | pending | 需具备可执行备份、查询、增量变更的权限 |
| 线上配置文件修改权限 | pending | 需可补齐 `WECOM_*` 配置 |
| crontab 修改权限 | pending | 需可挂载同步与提醒相关任务 |

## 3. 企业微信正式配置

配置交接模板：`./wecom-config-handoff-template-2026-06-24.md`

| 配置项 | 当前状态 | 说明 |
| --- | --- | --- |
| `WECOM_ENABLED` | pending | 上线时需开启 |
| `WECOM_CORP_ID` | pending | 企业微信企业 ID |
| `WECOM_AGENT_ID` | pending | 企业微信应用 AgentId |
| `WECOM_APPID` | pending | 企业微信小程序 AppID |
| `WECOM_AGENT_SECRET` | pending | 企业微信应用 Secret |
| `WECOM_MINI_PROGRAM_SECRET` | pending | 企业微信小程序 Secret |
| `WECOM_SYNC_ROOT_DEPARTMENT_ID` | pending | 通讯录同步根部门 ID |

## 4. 真机联调资源

| 资源 | 当前状态 | 说明 |
| --- | --- | --- |
| 企业微信真实员工账号 | pending | 需至少准备已绑定员工、未绑定员工、停用员工三类样本 |
| 联调手机设备 | pending | 需可安装企业微信和企业微信小程序 |
| 门店业务验收人 | pending | 需现场确认登录、提醒、跳转是否符合业务预期 |

## 5. WechatSI 相关前置

| 项目 | 当前状态 | 说明 |
| --- | --- | --- |
| 小程序后台插件授权 | pending | 需在微信后台添加 `WechatSI` 插件 |
| 真机语音权限授权 | pending | 需在设备上允许录音 |
| 通关与销售演练测试样本 | pending | 需准备语音输入测试内容 |

## 6. 回滚与止损前置

| 项目 | 当前状态 | 说明 |
| --- | --- | --- |
| 数据库备份目录 | pending | 需确认备份文件存放位置 |
| 回滚执行人 | pending | 需指定异常时的执行人 |
| 回滚确认人 | pending | 需指定业务确认人 |
| 执行窗口 | pending | 需确定低风险上线时间段 |

## 7. 满足条件后的执行顺序

1. 完成只读预检。
2. 完成结构与数据备份。
3. 执行数据库增量变更。
4. 完成旧链路回归。
5. 补齐企业微信配置。
6. 执行 worker 验证。
7. 完成真机联调。
8. 完成 WechatSI 验证。
9. 回填执行记录并正式收口。

## 8. 上线前最终判断

只有以下四类条件全部满足，才建议开始真实上线执行：

1. 权限到位。
2. 企业微信配置到位。
3. 真机联调资源到位。
4. 回滚和止损准备到位。
