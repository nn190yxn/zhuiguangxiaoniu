# skills 重复资料归档说明

归档时间：2026-06-18

## 背景

仓库内存在两类不同性质的资料：

- `skills/`：面向 AI 调用的技能资产
- `复盘标准/`：面向业务复盘的资料模板与压缩包

在本轮治理前，`skills/` 中混入了三组与 `复盘标准/` 完全重复的追光小牛复盘资料：

- `追光小牛体测解读复盘`
- `追光小牛新签复盘`
- `追光小牛续费复盘`

这些内容已经在 `复盘标准/` 中保留为正式位置，因此将 `skills/` 内重复副本转入本归档目录，仅保留可执行 skill 定义。

## 校验结论

本轮整理前，`skills/` 与 `复盘标准/` 中对应文件已校验为字节级一致，可安全收口为单一来源。

## 当前口径

- `skills/` 仅保留 manifest 管理的 AI skill
- `复盘标准/` 保留业务复盘资料原件
- 本目录保留从 `skills/` 收口出来的历史重复副本，便于追溯
