## 一、欢迎加入教学团队

恭喜你成为追光小牛教练团队的一员。这本手册将陪你完成从入职第一天到顺利转正的全过程。

在正式开始之前，请先了解教练的核心目标：**帮助孩子爱上运动，在安全快乐的环境中进步成长**。所有事情都围绕这个目标展开。

## 二、入职第一周时间表

### Day 0：入职手续

- [ ] 签署《岗位说明书签署确认表》
- [ ] 领取工服、工牌
- [ ] 注册GYMGO系统账号
- [ ] 加入企业微信群（总群 + 校区群）
- [ ] 填写紧急联系人信息
- [ ] 领取本手册

### Day 1-3：试岗培训（3天）

| 日期 | 上午（9:00-12:00） | 下午（13:30-17:30） |
|------|-------------------|---------------------|
| Day1 | 公司介绍、组织架构、行为规范 | 参观校区、认识团队、了解课程体系 |
| Day2 | 观摩正式教练上课（至少3节） | 学习GYMGO系统操作（点名、排课、课时记录） |
| Day3 | 了解课程教案模板与备课流程 | 模拟试讲（15分钟，面向店长+带教教练） |

试岗期关键任务：**观察、模仿、提问**。不要急着上手，先把流程看清楚。

### Day 4-6：跟岗实习（3天）

- [ ] 全程跟随带教教练，观察课前准备→课中执行→课后反馈全流程
- [ ] 协助带教教练完成场地布置和器材准备
- [ ] 在带教教练监督下，试带课堂中的某个环节（热身/游戏/拉伸）
- [ ] 写好第一份完整教案，交给带教教练审核
- [ ] 学会使用GYMGO进行出勤标记

### Day 7：试岗考核

| 考核项目 | 考核人 | 通过标准 |
|----------|--------|---------|
| 试讲（15分钟） | 店长+带教教练 | 评分≥B级 |
| GYMGO基本操作 | 店长 | 能独立完成点名和课时记录 |
| 安全知识测试 | 带教教练 | 满分通过 |
| 综合印象 | 店长 | 态度积极，无投诉 |

通过试岗考核 → 进入试用期。未通过 → 延长试岗3天，仍不通过 → 终止。

## 三、试用期内必须掌握的8项技能

### 3.1 核心教学技能

#### 技能1：课前准备（第1-2周完成）

- [ ] 能独立备课，写出完整教案（按公司标准模板）
- [ ] 能提前15分钟到场地检查环境安全
- [ ] 能完成器材摆放（不遗漏、不错误）
- [ ] 能确认器材性能安全（无破损、无异响）

**通过标准**：连续4次课前准备零失误。

#### 技能2：课堂执行（第1-4周完成）

- [ ] 能完整执行一节45-60分钟的课程
- [ ] 能独立完成热身→技能训练→游戏→拉伸全流程
- [ ] 课堂中能保持学员参与度≥80%
- [ ] 能处理课堂中的常见突发情况（孩子哭闹、器材问题等）
- [ ] 能做到无安全事故

**通过标准**：带教教练旁听评分≥B级，连续4节。

#### 技能3：学员安全管理（第1周开始，持续）

- [ ] 能独立进行课前安全检查（器材、场地、学员状态）
- [ ] 上课时能时刻关注学员安全
- [ ] 知道受伤应急处理流程
- [ ] 知道如何预防常见运动损伤

**通过标准**：安全知识笔试满分。

#### 技能4：课后反馈（第2-4周完成）

- [ ] 能向家长做口头反馈（孩子今天学了什么、表现如何、需要练什么）
- [ ] 能在GYMGO系统中填写课后记录
- [ ] 能独立完成教学总结（每节课的自我反思）

**通过标准**：带教教练陪同至少3次家长反馈，评分≥B级。

### 3.2 服务与沟通技能

#### 技能5：续费认知（第3-4周了解）

- [ ] 了解续费的基本流程和7个核心节点
- [ ] 了解续费在教练工作中的主责地位
- [ ] 了解月度续费业绩目标

**通过标准**：能口述续费流程。

#### 技能6：体测解读（第2-4周完成）

- [ ] 能独立带领学员完成基础体测
- [ ] 能向家长口头解读体测结果
- [ ] 能根据体测结果给出训练建议

**通过标准**：独立完成3次体测解读，家长满意度评分≥4分（5分制）。

#### 技能7：GYMGO系统（第1周开始，持续）

- [ ] 能独立完成出勤点名
- [ ] 能记录学员课堂表现
- [ ] 能查看学员档案和课包信息
- [ ] 能操作排课和调课

**通过标准**：抽查4项操作全部正确。

#### 技能8：团队协作（第3-4周）

- [ ] 能与带教教练顺畅配合
- [ ] 能在团队内部培训中积极发言提问
- [ ] 能主动承担校区公共事务（器材整理、卫生清洁）

**通过标准**：带教教练和店长综合评分≥B级。

## 五、转正考核

### 5.1 转正条件

| 维度 | 标准 |
|------|------|
| 入职天数 | ≤48天 |
| 试讲评分 | ≥B级 |
| 安全知识测试 | 满分 |
| 8项技能 | 全部通过 |
| 学员出勤率 | 所带学员平均出勤率≥60% |
| 家长投诉 | 零投诉 |
| 个人出勤 | 100%（无迟到/旷工） |
| 耗课量 | 累计≥50节 |

### 5.2 转正材料清单

1. 转正申请表
2. 试岗培训考核表（已签字）
3. 8项技能通过确认表（带教教练签字）
4. 至少4份完整的教案（已审核通过）
5. 自我总结（500字以上，含：收获、不足、下阶段目标）

### 5.3 转正流程

```
提交转正申请 → 店长初审 → 安排考核（实操+理论）
  → 店长评分 → 总经理审批 → 签订岗位说明书 → 转正生效
```

### 5.4 转正后

转正即获一星教练资格。底薪调整为2,300元，享课时费+业绩提成+续费激励。转正当月起参与季度星级评定。

## 六、带教教练职责

你的带教教练会：

- 每周至少旁听你2节课并给出书面反馈
- 审核你写的每一份教案（第1-4周）
- 带你完成首次体测解读
- 带你完成首次课后家长反馈
- 每个月向店长提交你的成长评估报告

**带教教练是你最重要的学习资源，主动提问，主动请教。**

## 七、常见问题

**Q：试岗期感觉跟不上怎么办？**

A：正常。第一周的任务是观察和学习，不是独立教学。跟不上的地方及时告诉带教教练，不要自己硬撑。试岗期可以延长一次（再给3天）。

**Q：教案写不出来怎么办？**

A：先看带教教练的教案范例，模仿格式和结构。公司有标准教案模板，把每个环节填进去就好。写完后请带教教练审核。

**Q：家长问的问题我不知道怎么回答？**

A：诚实告诉家长"这个问题我需要确认一下"，然后找带教教练或店长确认后再回复。**宁可晚回，不要乱回。**

**Q：孩子课上哭闹怎么办？**

A：先安抚，带到旁边休息。不是每个孩子都一次适应。课后第一时间告知家长当时发生了什么。不要强硬要求孩子完成训练。

**Q：转正后底薪怎么定？**

A：转正即为一星，底薪2,300元。后续星级晋升见《教练手册》。
