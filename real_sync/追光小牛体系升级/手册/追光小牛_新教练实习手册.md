## 一、欢迎加入教学团队

恭喜你成为追光小牛教练团队的一员。这本手册将陪你完成从入职第一天到顺利转正的全过程。

在正式开始之前，请先了解教练的核心目标：**帮助孩子爱上运动，在安全快乐的环境中进步成长。**

在追光小牛，新教练的成长路径是：

```
入职 → 试岗培训(3天) → 跟岗实习(3天) → 助教角色训练(2-4周)
  → 局部主教上手(1-2周) → 转正考核 → 一星教练
```

新人进入课堂的三步走原则：**先观察 → 再协助 → 最后才能独立带课。**

---

## 二、课堂角色认知

进入课堂前，先搞清楚三种角色：

### 2.1 跟岗观察角色

你只管：看流程、学站位、学口令、学节奏。不参与教学，不主动干预。

### 2.2 助教行为角色

你是主教的帮手，负责补位和协助。核心任务：**抓安全、抓秩序、抓个别学员、抓器材**。

记住一句话：**助教的职责不是教课，是让主教能安心教课。**

### 2.3 主教行为角色

你是当堂课堂的主责人，对教学结果、学员安全和家长感知负责。这是你转正后要承担的角色。

**在转正之前，你只承担跟岗和助教两种角色。未经放行，不得独立承担主教行为。**

## 四、助教工作标准（必须逐条掌握）

以下标准来源于《追光小牛各项标准分类汇总》和 `05D_教练与助教上课执行标准`。从第2周开始，你将以助教身份进入课堂，以下每条都是你的执行标准。

### 4.1 助教课前标准

| 序号 | 工作内容 | 标准要求 |
|------|---------|---------|
| 1 | 沟通教案 | 主动与主教沟通教案，协助准备教具 |
| 2 | 熟悉学员 | 熟悉学员名字，了解学员运动情况及性格 |
| 3 | 提前到馆 | 陪提前到场馆的小朋友玩游戏，维持秩序与安全 |
| 4 | 器材准备 | 按要求布置器材和课堂动线 |
| 5 | 点名整队 | 协助点名、迎接、鞋服和入场秩序检查 |
| 6 | 状态反馈 | 向主教反馈学员异常状态（情绪不稳、身体不适、家长临时说明） |

### 4.2 助教课中标准

#### 4.2.1 动作教学辅助

| 场景 | 标准要求 |
|------|---------|
| 动作示范 | 位于主教旁进行侧面示范，或跪于学员旁进行辅助动作讲解 |
| 动作纠错 | 口头提醒未改正的，需让学员返回重做 |
| 动作引导 | 循环训练中帮助主教完成动作引导，不应出现停顿时间过长 |
| 动作保护 | 危险动作及危险器材使用进行对应保护（体操课程按保护手法规定执行） |

#### 4.2.2 秩序维持标准

| 场景 | 标准要求 |
|------|---------|
| 落座休息时 | 及时让学员落座，与学员进行互动小游戏 |
| 观看示范时 | 侧身跪蹲在学员身边，注意学员目光是否集中在主教身上 |
| 排队出发时 | 辅助主教维持队形 |
| 循环训练时 | 与主教站位成对角，站在循环训练动作外缘，面向内部 |
| 游戏对抗时 | 及时清场，根据游戏需要配合主教 |

#### 4.2.3 特殊情况处理标准

| 情况 | 处理方案 |
|------|---------|
| 学员发呆走神 | 口头提醒 + 接触引导 |
| 动作卡顿 | 让后面动作流畅的学员做示范先出发，再单独教学不会的学员 |
| 学员产生推搡、插队等矛盾 | 互相道歉 / 引导比比谁做动作更厉害 |
| 学员受伤等突发事件 | 检查伤情，不严重则鼓励，严重则寻求场外空闲教练帮助 |

#### 4.2.4 学龄前动作保护及辅助标准

| 动作 | 保护/辅助方法 |
|------|-------------|
| 攀爬跳箱 | 可托举腋下辅助上箱；在箱上时前后手护着学员；下箱时提醒趴着下 |
| 双脚跳跳箱 | 双手前后护着学员 |
| 前滚翻 | 提醒低头，右手护住脖颈，左手托腿给予助力 |
| 平衡木行走 | 站于学员身边陪伴行走，防止掉落磕碰 |

#### 4.2.5 五个核心职责（05D标准）

在课堂上，助教的五项核心职责：

1. **抓队形**：维持排队、轮换和等待秩序
2. **抓安全**：站在风险点、掉队点和容易失控的位置
3. **抓个别**：对慢半拍、情绪波动、动作偏差较大的学员做近身辅助
4. **抓器材**：及时完成器材切换、回收和重置
5. **抓补位**：主教示范时协助照看全体学员

### 4.3 助教课后标准

| 序号 | 工作内容 | 标准要求 |
|------|---------|---------|
| 1 | 合照准备 | 准备牛牛币和手机拍合照 |
| 2 | 整理教室 | 整理教室、收拾器材 |
| 3 | 复盘反馈 | 主动询问主教本节课助教表现及提升点 |
| 4 | 观课记录 | 完成当节观课记录（转正前须完成至少8篇，见附表A） |

### 4.4 助教家长沟通边界

| 助教可以做 | 助教不能做 |
|-----------|-----------|
| 做事实型反馈（出勤、配合度、情绪表现） | 擅自承诺升班、疗效、训练结果 |
| 在主教在场或已授权前提下补充观察信息 | 擅自评价孩子问题成因或专业诊断 |
| — | 在未与主教同步前直接输出异常事件结论 |

**沟通顺序**：先内部同步 → 再对家长表达 → 先讲事实 → 再讲建议。

### 4.5 助教禁止行为（红线）

- 擅自改变主教节奏
- 同时发主口令、另起一套规则
- 在风险动作中背身、离岗、搬器材
- 对学员异常不汇报，私自处理后不留信息
- 只搬器材，不看学员

## 五、教练工作标准（必须了解）

以下标准来源于《追光小牛各项工作标准分类汇总》。转正后你将按这些标准执行，转正前需要了解并能在带教指导下完成。

### 5.1 仪容仪表标准

| 项目 | 标准要求 |
|------|---------|
| 服装 | 工服 + 运动裤（长款） |
| 鞋袜 | 运动鞋 |
| 配饰 | 哨子、秒表；无其他配饰 |
| 发型 | 整洁，无碎发 |
| 妆容 | 口气清新、淡妆（女） |
| 体态 | 站姿/坐姿/跪姿端正 |
| 精神面貌 | 精神饱满、阳光开朗、笑容甜美、亲和力强 |
| 语言表达 | 语速平稳、音量合适、表达清晰、逻辑思维强 |
| BMI要求 | 男生19-24，女生17-22 |
| 吸烟规定 | 课前1小时禁止吸烟 |

### 5.2 教案准备标准

| 项目 | 标准要求 |
|------|---------|
| 准备时间 | 提前一天准备教案 |
| 教案模板 | 训练目标 + 训练内容 + 结束拉伸/游戏 |
| 磨课要求 | 主教/助教至少提前2小时磨课 |
| 上传要求 | 当日主课需上传教案（算1个工作量） |

### 5.3 器材准备标准

| 项目 | 标准要求 |
|------|---------|
| 检查要求 | 器材干净整洁、无破损 |
| 准备时间 | 上课前30分钟准备必需器材 |
| 摆放要求 | 摆放整齐 |

### 5.4 学员互动标准（课前）

| 项目 | 标准要求 |
|------|---------|
| 到店学员 | 相互自我介绍、建立信任、简单互动游戏 |
| 体测准备 | 准备体测评估表、板夹、笔、秒表、身高体重数值表 |
| 体测频率 | 一周一次课→16节课一次体测；一周两次课→24节课一次体测 |

### 5.5 课程结构标准

**通用60分钟课程结构**：

| 环节 | 时长 | 内容 |
|------|------|------|
| 热身与激活 | 10分钟 | 关节活动、动态拉伸 |
| 技能/动作发展 | 20分钟 | 核心技能教学 |
| 游戏 | 15分钟 | 合作、规则、竞争 |
| 拉伸/恢复再生 | 10分钟 | 静态拉伸、放松 |
| 过渡 | 5分钟 | 点评、总结 |

### 5.6 师生比标准

| 年龄段 | 师生比 |
|--------|-------|
| ≤3岁 | 1:4 |
| 3-6岁 | 1:6 |
| 6岁以上 | 1:8 |

### 5.7 课中质量标准

**安全标准**：清场、突发事件控场、器材正确使用、危险器材分类、器材间距和空间、运动空间起始终点路径闭环、危险动作保护（颈部/关节/距离）、不得长时间大重量重复练习某一部位。

**趣味性标准**：激情鼓励、语言表达清晰生动、肢体语言适当夸张、背景音乐、情景故事带入、角色扮演、道具配合、多感官引导（声音/视觉/触觉）、器材颜色和造型运用。

**教学质量标准**：教学内容丰富专业严谨、运动参数合理（强度/时间/组数/内容）、队形队列正确使用、动作示范标准、纠错及时有效、全场控制使学员全身心投入、主助教协作完成。

### 5.8 课后标准

| 项目 | 标准要求 |
|------|---------|
| 器材收拾 | 使用完器材及时归回位置 |
| 学员看护 | 无人接送时实时陪同或游戏陪同 |
| 家长沟通 | 一对一体测报告解读 |
| 课后作业 | 根据不同学员需求布置家庭作业（配视频示范） |
| 课后点评 | 每节课按模板点评，不超过当天23:00 |
| 缺勤跟踪 | 缺勤后24小时内联系家长 |

### 5.9 授课能力三层模型

| 层级 | 能力描述 | 对应角色 |
|------|---------|---------|
| 第一层 | 课堂辅助、特殊情况控制、场地提前布置、动作纠错、情景故事描述 | 助教能力（你现在的位置） |
| 第二层 | 课堂气氛营造、课堂流畅性把控、课堂运动效果、安全隐患辨别、口令指导、动作示范引导 | 主教能力（转正后的目标） |
| 第三层 | 教学内容展示、主助教气氛烘托、站位区分、教态区分 | 主助配合（进阶目标） |

### 5.10 对家长的沟通标准（8步法）

| 步骤 | 内容 |
|------|------|
| 1 | 先夸奖后点评 |
| 2 | 拉近距离，了解生活 |
| 3 | 专业角度分析 |
| 4 | 挖掘家长痛点，扩大需求 |
| 5 | 强调专业科学的锻炼 |
| 6 | 提高体能对其他学科的占比 |
| 7 | 说明健全的体系 |
| 8 | 跟家长成为朋友 |

## 六、课堂角色授权与跟岗标准

### 6.1 角色授权总原则

1. 角色由当堂授权决定，不由岗位名称自动决定
2. 放行必须基于真实课堂表现，不是基于资历想当然
3. 先看能力，再给角色，不因人员紧张直接放宽授权
4. 先能稳定协助，再能独立承担局部动作，最后才考虑承担主责

### 6.2 跟岗推进路径

```
跟岗观察 → 助教行为（稳定） → 局部主教行为（带教下） → 独立主教行为（放行后）
```

### 6.3 助教行为放行条件

你能稳定承担助教行为前，必须满足：

- [ ] 清楚课堂基本流程
- [ ] 清楚主教口令体系和配合方式
- [ ] 清楚高风险动作区不能乱站、乱动、乱离岗
- [ ] 清楚异常发生时先保护、先汇报
- [ ] 已经在课堂中稳定执行，不靠临场提醒才能完成

### 6.4 主教行为放行条件（转正后）

你能承担主教行为前，必须满足：

- [ ] 能稳定带完整课堂流程
- [ ] 能在真实课堂中守住节奏和安全
- [ ] 能明确分工并带动助教协同
- [ ] 能完成课后反馈和基本家长沟通

### 6.5 角色切换规则

同一名员工在不同课堂承担不同角色时：

1. 每堂课前先明确自己本堂角色
2. 本堂不是主教时，不得按自己习惯主导课堂
3. 本堂是助教时，应以补位、看护、协同为主
4. 即便个人资历更高，只要不是本堂主责人，也应服从当堂主教安排
5. 若发现主教遗漏明显安全风险，应先补保护，再即时提醒，不可旁观

### 6.6 管理红线

- 不得因排班压力让未放行员工事实性独立顶岗
- 放行后若出现明显失稳，应及时收回授权并补训

## 七、课堂安全保护标准（必须满分通过）

以下内容来自 `05E_课堂安全保护与异常处理标准`。安全是追光小牛的第一底线，此章节转正考核必须满分。

### 7.1 安全三大原则

1. **安全优先于进度**
2. **保护优先于教学**
3. **控风险优先于控场面**

发生异常时：**先处理人 → 再处理流程 → 再处理记录**

### 7.2 风险等级划分

| 等级 | 定义 | 典型动作 |
|------|------|---------|
| L1 低风险 | 地面基础动作，低速度低高度低并发 | 原地热身、低强度爬行、低高度传接球 |
| L2 中风险 | 存在失衡、碰撞或轻度跌落可能 | 单脚站、低高度平衡木、基础连续跳跃 |
| L3 高风险 | 存在翻滚失控、离地跌落或冲击风险 | 前滚翻/后滚翻、跳箱、攀爬、较高平衡木 |

### 7.3 各风险等级执行要求

| 等级 | 执行要求 |
|------|---------|
| L1 | 主教统一组织，助教以秩序维护和掉队补位为主 |
| L2 | 必须明确保护位和进出顺序；不得边讲边散放学员；新人可参与辅助但不得作为唯一看护人 |
| L3 | 必须先讲规则、做示范、清保护位、控并发；必须明确主责、保护位和并发控制；不满足条件不得开展 |

### 7.4 高风险场景清单

重点看护场景：翻滚类动作、平衡木/单脚平衡等离地动作、跳箱/跨越等冲击性动作、攀爬/悬垂/快速轮换器材。

### 7.5 课前安全要求

**场地检查**：
- 地面干燥、无障碍物、无明显滑倒风险
- 器材稳定，无破损、松动、锐角外露
- 高风险区域有足够缓冲垫和看护位置
- 学员进场前完成风险点复查

**人员检查**：
- 主教必须清楚当堂高风险动作
- 助教必须清楚自己站位和保护点
- 实习员工不得被安排独立看守高风险动作区
- 学员存在身体不适、情绪异常、家长特别提醒时，必须课前确认

### 7.6 课中安全站位

- 主教站在能看全场、能做示范、能控制节奏的位置
- 助教站在高风险区域、后排学员、器材切换口或掉队点
- 高风险动作或多学员并发练习时，不得出现风险区域无人看护
- 一名员工无法同时覆盖多个高风险点时，必须减并发或降低难度

### 7.7 异常处理口诀

```
发现异常 → 先保护 → 再汇报主教 → 再按流程处理 → 课后记录
```

- 情绪波动学员：助教优先近身稳定，主教保持课堂总节奏
- 动作落后学员：助教先做局部辅助，必要时主教决定是否降阶
- 过度兴奋学员：主教负责重建规则，助教负责贴身控位
- 突发受伤：检查伤情 → 不严重鼓励 → 严重寻求场外帮助 → 课后上报记录

### 7.8 主辅配合安全要求（三个一）

1. **一个口令源**：关键指令以主教口令为准
2. **一个课堂节奏**：助教补位，不额外开新节奏
3. **一个安全标准**：风险动作前，主教确认，助教到位

## 八、教练专业技能能力模型

以下内容来自 `02I_教练专业技能能力模型`。这是你转正后持续进阶的框架，但转正前需要了解标准。

### 8.1 四大核心能力

1. **课堂执行能力**：组织课堂、控节奏、主助分工、目标落地
2. **教学表达能力**：口令清晰、示范标准、讲解简洁有效、按年龄调整表达
3. **安全与风险控制能力**：提前识别风险、守住保护站位、异常的正确处理顺序
4. **家长沟通与服务承接能力**：说事实、说重点、说建议、守住专业边界

### 8.2 三级能力标准

| 能力维度 | 基础可用（你的转正目标） | 稳定独立（转正后1-2年） | 示范带教（三星以上） |
|---------|----------------------|----------------------|-------------------|
| 课堂执行 | 能按既定流程完成课堂，需少量提醒即可守住秩序 | 能根据学员状态调整节奏，稳定处理常见课堂波动 | 能带动助教和新人把课上稳 |
| 教学表达 | 完成基础动作讲解和示范，口令基本清楚 | 动作讲解易懂，根据孩子反应调整表达 | 用稳定富感染力的表达带动课堂 |
| 安全控制 | 知道常见风险点，不会明显失位和越权 | 主动预判风险并调整课堂，处理一般异常 | 复杂场景下提前化解风险，指导他人 |
| 家长沟通 | 做简要、事实型反馈，不乱承诺不乱诊断 | 结合课堂表现做完整反馈，稳定处理家长追问 | 专业反馈+情绪安抚+服务承接结合 |

### 8.3 对不同年龄段孩子的沟通方式

| 年龄段 | 关键词 | 沟通方式 |
|--------|-------|---------|
| 3-5岁 | 陪伴、玩耍 | 和学员一起参与课堂游戏 |
| 6-8岁 | 带领、引导 | 做学员榜样，引导互相竞争、相互协作 |
| 9岁+ | 专业、学习 | 需有足够专业性让学员信服 |

## 十一、投诉处理基本认知

### 11.1 你需要知道的

作为新教练，投诉处理不是你的主责，但你需要知道基本规则：

- 课堂上发现家长不满，第一时间告诉主教
- 如果家长直接向你投诉，先倾听、不辩解，然后引介给主教或店长
- 严禁对投诉家长态度恶劣或推诿
- 严禁隐瞒投诉不报

### 11.2 投诉分三级

| 等级 | 定义 | 处理人 |
|------|------|--------|
| 绿色 | 教学质量、服务态度等日常问题 | 店长 |
| 黄色 | 安全、收费争议、多次投诉 | 店长+督导 |
| 红色 | 人身安全、媒体曝光、法律纠纷 | 总经理+督导 |

### 11.3 你的角色

- 在课堂上：发现不满→内部同步→交给主教处理
- 课后：如果家长向你表达不满→先听→记录→上报带教教练或店长
- 严禁：擅自给出承诺、擅自评价是非、擅自做补偿决定

## 十三、带教教练职责

你的带教教练会：

- 每周至少旁听你2节课堂并给出书面反馈
- 审核你写的每一份教案（第1-4周）

**带教教练是你最重要的学习资源。主动提问，主动请求反馈，不要等。**

## 十四、常见问题

**Q：试岗期感觉跟不上怎么办？**

A：正常。第一周的任务是观察和学习，不是独立教学。跟不上的地方及时告诉带教教练，不要自己硬撑。试岗期可以延长一次（再给3天）。

**Q：课堂上我不知道该站哪里？**

A：助教站位原则：高风险区域、后排学员、器材切换口或掉队点。如果不确定，课前问主教"这堂课我站哪里"。不要自己猜。

**Q：家长问的问题我不知道怎么回答？**

A：诚实告诉家长"这个问题我需要确认一下"，然后找主教或店长确认后再回复。宁可晚回，不要乱回。助教不要独立向家长输出带结论的专业反馈。

**Q：孩子课上哭闹怎么办？**

A：先安抚，带到旁边休息。不是每个孩子都一次适应。课后第一时间告知家长当时发生了什么。不要强硬要求孩子完成训练。

**Q：主教让我做的事情我不同意怎么办？**

A：在课堂上，先服从主教安排。课后私下找主教或带教人提出你的想法。不要在课堂上和主教产生分歧或另发口令。

**Q：观课记录怎么写？**

A：见附表A模板。核心是记录：课堂流程是什么、助教做了什么、哪做得好、哪可以改进、学到了什么。不是流水账。

**Q：转正后底薪怎么定？**

A：转正即为一星，底薪2,300元。后续星级晋升见《教练手册》。
