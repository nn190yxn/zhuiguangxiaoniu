# 追光小牛 · 员工手册

> 版本：v2.0 | 2026年6月 | 适用于全部校区 | 人手一册，入职当天签收

## 一、欢迎加入追光小牛

追光小牛是一家专注于3-14岁儿童体适能训练的连锁培训机构，目前在营5家校区。

**我们的使命**：让每个孩子爱上运动，在运动中成长。

**核心价值观**：专业、真诚、负责、成长。

这本手册汇总了你在追光小牛工作期间需要了解的基本制度、行为规范、薪酬福利和成长路径。请认真阅读并妥善保管。

## 二、组织架构

```
             总经理
            /      \
         督导      财务
          |
      ┌───┼───┐
   店长1 店长2 店长3 ...
    / \
  教练 顾问
```

- **总经理**：全面管理，重大决策审批
- **督导**：受总经理授权，对店长执行情况进行检查和评估
- **店长**：门店业绩达标、团队管理、日常运营
- **教练**：教学执行、学员服务、续费主责
- **顾问**：线索跟进、体验邀约、签单转化
- **财务**：全门店统一核算

**岗位原则**：教销分离，一岗一责，店长兼管行政。

## 三、人事制度

### 3.1 试用期

| 岗位 | 试用期 | 试岗期 |
|------|--------|--------|
| 教练 | 1-3个月 | 前3天试岗+3天跟岗 |
| 课程顾问 | 1-3个月 | 前3天试岗+3天跟岗 |
| 店长 | 3个月 | 60天带教计划 |

试用期合格标准按各岗位转正要求执行。不合格者终止试用。

### 3.2 转正

教练试岗期满 + 试用期考核通过（试讲评分≥B级、家长零投诉、出勤率100%），填写《岗位说明书签署确认表》，正式转正。转正考核在入职48天内完成申请。

### 3.3 离职

| 类型 | 流程 | 要求 |
|------|------|------|
| 主动离职 | 提前30天书面申请 → 店长审批 → 总经理审批 | 完成工作交接 |
| 被动离职 | 店长发起 → 总经理审批 | 按劳动法执行 |
| 试用期离职 | 提前3天通知 | 完成简单交接 |

离职前须：归还工服工牌、完成GYMGO账号交接、签署离职确认书、结算薪资。

## 四、薪酬体系

### 4.1 薪酬结构

教练收入 = 底薪 + 课时费 + 业绩提成 + 续费激励 + 门店分红（14万以上触发）+ 季度激励

薪资每月15日发放上月薪资（遇节假日提前），以GYMGO系统数据+考勤记录为依据。

### 4.2 底薪参考

| 星级 | 实习 | 一星 | 二星 | 三星 | 四星 | 五星 |
|------|------|------|------|------|------|------|
| 底薪 | 1,800 | 2,300 | 2,500 | 2,800 | 3,000 | 3,000 |

### 4.3 薪资保密

薪资属于个人隐私，员工之间不得互相打听。违反薪资保密规定者书面警告。

## 五、考勤制度

### 5.1 工作时间

实行排班制，具体班次由店长根据课表安排。每月休息4天，由店长统一排休。

### 5.2 考勤要求

| 项目 | 标准 |
|------|------|
| 准时到岗 | 提前10分钟到店 |
| 请假 | 提前24小时向店长申请 |
| 迟到/早退 | 当月累计3次以上，扣除全勤奖 |
| 旷工 | 连续旷工3天或累计5天，按自动离职处理 |

### 5.3 假期

| 假期类型 | 天数 | 说明 |
|----------|------|------|
| 法定节假日 | 按国家规定 | 排班调休 |
| 年假 | 满1年享5天 | 逐年递增 |
| 婚假 | 3天 | 需提前1个月申请 |
| 丧假 | 3天 | 直系亲属 |

## 六、成长基金制度

成长基金是追光小牛的公司级奖罚机制，通过乐捐（违规扣款）形成奖金池，再通过奖励回馈优秀员工，形成"违规者出资、优秀者受益"的正向循环。

### 6.1 乐捐标准（统一）

| 违规情形 | 乐捐金额 | 说明 |
|---------|---------|------|
| 工作量连续2天未达成 | 20元/个差额 | 按累计差额计算 |
| 红线违规（一般） | 员工10元/次 | 书面警告+乐捐 |
| 红线违规（店长） | 店长20元/次 | 书面警告+乐捐 |
| 数据造假 | 员工10元/店长20元 | 首次书面警告+乐捐，二次辞退 |

**乐捐原则**：
- 及时记录：乐捐发生后24小时内记录到台账
- 告知当事人：乐捐前必须告知违规事实和金额
- 不得代缴：乐捐必须由当事人自行承担
- 所有乐捐金进入成长基金奖金池，专款专用

### 6.2 奖励标准

| 奖励项目 | 频率 | 规则 |
|---------|------|------|
| 周工作量最佳 | 每周 | 第1名60%、第2名30%、第3名10%（奖金池分配） |
| 月度之星（6项） | 每月 | 销售/续费/教学/服务/进步/明星老师 |
| 团队荣誉（3项） | 每月 | 冠军团队、服务标杆店、进步最大店 |
| 专项荣誉 | 不定期 | 最佳分享者、创新之星、安全卫士 |

### 6.3 奖金池管理

- 每月1日归零，重新累计
- 周奖励每周一公布，周内发放
- 奖金池不足时，奖励顺延，不补发
- 月末结余转入下月，不退还个人

## 七、行为规范

### 7.1 教练行为准则

1. 穿统一工服，佩戴工牌，保持专业形象
2. 上课期间严禁玩手机或做与教学无关的事
3. 不得体罚、冷暴力或言语侮辱学员
4. 不得私下收取家长费用或承诺优惠
5. 不得在家长群发布未经审核的内容
6. 不得与家长发生争吵或态度恶劣
7. 不得向家长传递与机构政策不符的信息
8. 严禁数据造假（出勤、课时、业绩等）

### 7.2 总红线（不可触碰）

以下行为直接辞退：

- 体罚或冷暴力对待学员
- 数据造假（经查实二次）
- 发生重大安全事故且为主责方
- 私自收费或侵占公司财物
- 泄露公司核心商业信息

## 八、投诉处理

### 8.1 投诉分级

| 等级 | 定义 | 响应时限 | 解决时限 | 处理人 |
|------|------|----------|----------|--------|
| 绿色（一般） | 教学质量、服务态度、环境设施等日常问题 | 1小时内 | 24小时内 | 店长 |
| 黄色（较重） | 涉及安全、收费争议、多次投诉未解决 | 30分钟内 | 48小时内 | 店长+督导 |
| 红色（严重） | 人身安全、媒体曝光风险、法律纠纷、群体投诉 | 15分钟内 | 当天 | 总经理+督导 |

### 8.2 处理原则

| 原则 | 说明 |
|------|------|
| 先处理情绪，再处理问题 | 不急于辩解，先让家长说完 |
| 首问负责 | 谁接到投诉谁负责跟进到底 |
| 升级及时 | 超出处理权限立即升级 |
| 有记录 | 所有投诉必须登记，不得遗漏 |
| 有回访 | 解决后48小时内必须回访 |

### 8.3 补偿标准

| 投诉等级 | 补偿方式 | 审批权限 |
|----------|----------|----------|
| 绿色 | 口头道歉+1节免费补课 | 店长 |
| 黄色 | 书面道歉+2节免费补课或等值优惠 | 督导 |
| 红色 | 书面道歉+3节免费补课+经济补偿 | 总经理 |

### 8.4 投诉红线

- 收到投诉后未在规定时效内响应 → 书面警告+乐捐进成长基金
- 对投诉家长态度恶劣或推诿 → 书面警告+乐捐进成长基金
- 隐瞒投诉不报 → 书面警告+乐捐进成长基金

## 九、卫生与安全

### 9.1 日常要求

- 每日开课前和关店后各清洁一次教学区域
- 每节课后对教学器材进行酒精消毒并记录
- 每周三全员大扫除，同步进行安全排查
- 安全检查：器材螺丝松动、地垫破损、尖锐边角等

### 9.2 课堂安全底线

- 师生比：3岁以下≤1:4、3-6岁≤1:6、6岁以上≤1:8
- 每节课前须进行安全检查
- 发现安全隐患立即上报，不得隐瞒
- 安全优先于进度，保护优先于教学

## 十、突发事件应急处理

### 10.1 三级应急响应

| 级别 | 定义 | 处理要求 |
|------|------|----------|
| 绿色（一般） | 学员轻微擦伤、碰伤，无需就医 | 教练当场处理，记录，课后告知家长 |
| 黄色（较重） | 学员受伤需要就医、家长严重投诉 | 教练+店长处理，1小时内上报总经理，24小时回访 |
| 红色（严重） | 学员严重受伤、安全事故、媒体曝光 | 店长+总经理到场，立即启动应急预案 |

### 10.2 伤害处理标准

| 程度 | 处理方式 |
|------|----------|
| 轻微（擦伤、碰伤） | 急救箱处理，冰敷/消毒/贴创可贴 |
| 中度（扭伤、淤青） | 冰敷+固定，建议就医，联系家长 |
| 重度（骨折、昏迷） | 拨打120，不移动学员，联系家长，上报总经理 |

### 10.3 应急红线

- 突发事件隐瞒不报 → 立即辞退
- 未按应急流程处理导致事态扩大 → 书面警告+乐捐进成长基金

### 10.4 处理口诀

```
先保障学员安全 → 分级处理 → 联系家长 → 上报记录 → 事后回访复盘
```

## 十一、培训与成长

### 11.1 新员工培训

所有新员工须完成试岗期培训（教练/顾问 3天）和跟岗实习（3天），考核通过后方可进入试用期。

### 11.2 教练星级晋升

教练职业发展路径：实习 → 一星 → 二星 → 三星 → 四星 → 五星。

每年3次评定窗口（4月/9月/12月），每3个月一次考核周期。续费率为硬门槛（不达标其他满分也不升星）。四五星教练须承担低星教练带教工作。

### 11.3 月度荣誉体系

追光小牛每月评选个人之星（销售/续费/教学/服务/进步/明星老师）、团队荣誉（冠军团队/服务标杆店/进步最大店）和专项荣誉，获得者上形象墙、全员通报表扬。具体标准见《06B 激励方案》。

### 11.4 学习资源

- 公司内网制度中心：全部制度文件和知识库
- GYMGO系统：教务、客户管理、工作记录
- 月度培训：由教学主管或店长组织
- 教练星评考核：理论+体能+实操+演讲

## 十二、制度申诉

如果你对处理决定有异议，有权按以下流程申诉：

1. 向店长提交书面申诉（3个工作日内）
2. 店长无法解决的，提交总经理裁定
3. 总经理裁定为最终决定
