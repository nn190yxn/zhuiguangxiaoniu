# 追光小牛 · 下半年组织架构与责权利调整方案

> **版本**：v1.1 | **日期**：2026年6月
> **适用**：5家校区 | **编制**：创始人

---

## 一、现状诊断

### 1.1 核心问题

追光小牛5家店处于"连而不锁"状态——各有各的做法，标准不统一，执行看心情。

**三大病灶**：

| 病灶 | 表现 | 根因 |
|------|------|------|
| 店长执行力弱 | 任务推不动、管理靠自觉 | 学历不高、综合素质低，但更关键的是"管得太杂"——什么都让他管，什么都管不深 |
| 教练偏懒缺思 | 不主动备课、不主动服务 | 旧模式（公司发教案）剥夺了思考义务，也剥夺了被考核的依据 |
| 销售散漫无固 | 想干就干、不想干就混 | 没有固化的日行为标准，连"做没做"都没人盯 |

### 1.2 组织层面根因

```
旧架构：
        总经理
       /      \
    督导(虚)  财务
      |
  5位店长(什么都管)
    / \
 教练  顾问
```

**督导被架空**：平级管店长、没有实权来源，店长不配合，只能干助理活——不是岗的问题，是权力来源没理顺。
**财务超荷**：收款退费薪资合同一肩挑，没有精力做数据核查。
**店长过载**：又要管人又要看课又要跑市场又要做财报，全要＝全不行。
**创始人缺位**：定完制度没人盯执行，制度落不下去。

## 二、组织架构调整

### 2.1 新架构

```
                创始人
              （定标准/看红线/定方向）
                   │
         ┌─────────┼─────────┐
         │         │          │
      总经理      财务       督导
  （运营+品质控制）（数据中心） （总经理品质执行臂）
         │
  ┌───┬──┼──┬───┐
 店长1 店长2 店长3 店长4 店长5
 [体验线+人员线 日控]
   │
[四五星教练 = 交付线第一道品控]
```

### 2.2 关键变化

| 变化 | 旧 | 新 |
|------|----|----|
| 督导岗 | 平级查店长、无实权来源、压不住 | **保留，重新定位**——总经理的品质执行臂，代总经理巡店/核查/追踪整改，背靠总经理权威执行 |
| 总经理 | 审批日常、上传下达 | **兼品质控制官**，直接对三条线品质负总责，日常巡审任务分配给督导执行 |
| 创始人 | 管日常+管战略，混在一起 | **只管标准、方向和红线**，不介入日常 |
| 四五星教练 | 教得好的人，无管理职能 | **教学品控执行者**，审低星教案+交叉听课 |

### 2.3 编制

| 岗位 | 人数 | 说明 |
|------|------|------|
| 创始人 | 1 | 定标准、看红线、管方向 |
| 总经理 | 1 | 日常运营+品质控制 |
| 督导 | 1 | 总经理的品质执行臂，代总经理巡店/核查/追踪 |
| 财务 | 1 | 数据中心+财务合规 |
| 店长 | 5 | 每店1人，体验线+人员线日控 |
| 教练 | 按课表 | 四五星教练兼教学品控 |
| 顾问 | 按店规模 | 销售执行 |

**不增编**。

## 三、创始人-总经理分工区隔

### 3.1 核心原则

**创始人定"系统长什么样"，总经理跑"系统跑没跑"。**

创始人不变身最贵的店长，总经理不操心战略的事。

### 3.2 详细分工

| 维度 | 创始人 | 总经理 |
|------|--------|--------|
| **定位** | 定标准、定方向、看红线 | 执行标准、日常运营、品质控制 |
| **三条线** | 定义什么是控制节点、什么是达标线 | 每天盯控制节点到了没、谁没做到 |
| **看什么** | 只看系统亮红灯的异常指标 | 看每天的绿灯清单，红灯出现才深挖 |
| **做什么** | 定规则、调组织、外部资源/品牌/融资/扩张 | 盯执行、审周/月度品质报告、管人 |
| **参加什么会** | 不参加周例会、不审日常审批 | 主持周例会、主持月度经营复盘 |
| **审批什么** | 重大政策变更、重大人事任免、大额支出（≥5000元） | 日常审批、周/月度KPI核定、采购500-5000元 |
| **干预时机** | 月度品质报告出现连续红色指标时介入 | 每天看红灯清单、每周诊断、每月打分 |
| **不能做什么** | 不直接指挥一线员工、不跳过总经理做门店决策 | 不擅自修改创始人定的标准、不瞒报红色预警 |

### 3.3 汇报机制

```
创始人 ← 总经理（月度汇报：品质报告+异常清单+下月重点）
           │
     ┌─────┼─────┐
     │     │     │
   督导  财务   5位店长
  [执行臂][数据] [一线日控]

财务 → 总经理（每日红灯清单）
财务 → 创始人（月度数据月报，抄送）
督导 → 总经理（巡店/核查/追踪结果，实时）
店长 → 总经理（每日红灯反馈，有红才报）
```

**创始人不看日报**。财务出的数据月报总经理先审，创始人只收最终品质报告副本。

## 四、各岗位责权利重新定义

### 4.1 总经理（兼品质控制官）

**一句话定岗**：对三条线的执行品质负总责。5家店的体验线、交付线、人员线达标，就是你的成绩单。

| 项目 | 内容 |
|------|------|
| **直接上级** | 创始人 |
| **直接下级** | 督导、财务、5位店长 |
| **核心职责** | 1. 每日核查各店红灯清单，红→追，绿→过<br>2. 安排督导日常巡店、核查、追踪任务，验收督导产出<br>3. 每月每店深度审计1次（抽查教案5份+听课1节+查工作量5天）——可由督导代为执行<br>4. 主持周例会、月度经营复盘<br>5. 月度品质审计报告提交创始人<br>6. 日常人事管理（招聘/培训/考核/调动） |
| **日工作** | 晨间5分钟翻财务出的"红灯清单"→红色项安排督导电话追问→重大红自己操刀 |
| **周工作** | 周例会主持（复盘红灯+部署下周重点）→ 验收督导周度巡店报告 |
| **月工作** | 5家店深度审计→月度品质报告→月度经营复盘会 |
| **禁止事项** | 不得修改创始人制定的标准、不得跳过创始人做重大人事决策、不得隐瞒红色预警 |
| **KPI** | 体验线节点执行率≥85%、交付线安全通过率100%、人员工作量达成率≥80% |

### 4.2 财务（数据中心+财务合规）

**一句话定岗**：出数据、核财务、管合同。三条线的数据月报从你这里出，总经理和创始人的决策依据从你这里来。

| 项目 | 内容 |
|------|------|
| **直接上级** | 总经理（汇报）、创始人（数据月报同时抄送） |
| **直接下级** | 无 |
| **核心职责** | 1. 每日从GYMGO拉"各店红灯清单"，发总经理<br>2. 每月出具三条线数据月报<br>3. 收款/退费/合同/薪资核算（保持现有）<br>4. 成长基金台账管理<br>5. GYMGO系统数据核验（每月抽对合同vs系统） |
| **日工作** | 晨间拉GYMGO数据→生成红灯清单→发总经理（10分钟） |
| **周工作** | 成长基金台账更新、乐捐/奖励发放 |
| **月工作** | 数据月报、薪酬核算、合同台账抽检 |
| **禁止事项** | 不得修改数据、不得延迟发放薪资超过15日 |
| **KPI** | 数据月报准时提交率100%、薪资发放准时率100%、合同台账抽检准确率≥95% |

**财务数据月报模板**（每月5日前出具）：

| 数据维度 | 具体指标 | 店1 | 店2 | 店3 | 店4 | 店5 |
|---------|---------|-----|-----|-----|-----|-----|
| 体验线 | 预约确认率 | | | | | |
| 体验线 | 首月回访完成率 | | | | | |
| 体验线 | 续费触达启动率 | | | | | |
| 体验线 | 转介绍邀请率 | | | | | |
| 交付线 | 教案安全预案填写率 | | | | | |
| 交付线 | ACE评估完成率 | | | | | |
| 人员线 | 工作量达成率 | | | | | |
| 人员线 | 乐捐人次/金额 | | | | | |

### 4.3 督导

**一句话定岗**：总经理的品质执行臂。代总经理巡店、核查、追踪整改闭环。背靠总经理权威执行，不是独立检查者。

| 项目 | 内容 |
|------|------|
| **直接上级** | 总经理 |
| **直接下级** | 无 |
| **核心职责** | 1. 代总经理执行日常巡店核查（按总经理指派）<br>2. 追踪各店红灯整改闭环（谁欠了什么、做没做）<br>3. 收各店长周报/月报/整改反馈，汇总交总经理<br>4. 整理品质月报素材<br>5. 会议记录和待办追踪 |
| **日工作** | 1. 追踪前一日的红灯整改是否闭环→汇报总经理<br>2. 按总经理当天指派，核查指定门店指定项目 |
| **周工作** | 收齐各店周报→汇总→交总经理；输出周度巡店问题清单 |
| **权威来源** | 督导的所有检查行为均受总经理授权，店长必须配合。督导本人不做独立决策和判断，只做事实记录和上报 |
| **禁止事项** | 不得独立做出经营判断、不得越过总经理直接指挥店长、不得对外代表公司、不得自行发起不在总经理指派范围内的检查 |
| **KPI** | 整改跟踪闭环率≥90%、报告收齐率100%、巡店任务完成率100% |

### 4.4 店长

**一句话定岗**：只管两件事——体验线的5个勾打没打、人员线的工作量够不够。其他不用管。

| 项目 | 内容 |
|------|------|
| **直接上级** | 总经理 |
| **直接下级** | 本店教练、顾问 |
| **核心职责** | 1. 每日5个勾（体验线节点核查，3分钟）<br>2. 每日工作量核查（每人4个够不够）<br>3. 晨间红灯清单反馈（总经理追问时，5分钟内回复）<br>4. 开关店检查、班前/班后会（保持）<br>5. 教案安全预案"填没填"检查（不看质量） |
| **不再负责** | 不再审教案ACE深度（四五星教练审）<br>不再做课堂品质判断（督导/总经理抽查）<br>不再手动统计经营数据（财务出） |
| **日工作** | 开关店→班前会→5个勾打一遍→工作量表翻一遍→红灯反馈→班后会（总耗时约30分钟管理动作） |
| **周工作** | 周报（数据财务已出，店长只写"做了什么、碰到了什么问题"）+周例会 |
| **月工作** | 月度述职、工资表核算 |
| **禁止事项** | 不得拖欠工作量核查超过次日10点、不得在红灯追问时推诿 |
| **KPI** | 5个勾日清率100%、工作量日查率100%、安全预案填写率100% |

**店长的5个勾（体验线日控清单）**：

```
□ 昨天新增线索，2h内联系了没有？    Y/N    （看GYMGO通话/微信记录）
□ 昨天体验课的家长，到店签到了没有？  Y/N    （看签到表）
□ 昨天体验课后，24h内回访了没有？    Y/N    （看GYMGO回访记录）
□ 到期前30天学员，启动续费了没有？    Y/N    （看GYMGO续费标签）
□ 成交后7天，转介绍邀请发了没有？    Y/N    （看微信聊天截图）
```

### 4.5 教练（含四五星教学品控）

**一句话定岗**：教好课、写好教案、盯好续费。四五星教练额外审低星教案。

| 项目 | 内容 |
|------|------|
| **直接上级** | 店长 |
| **核心职责** | 1. 按ACE框架编写每节课教案（安全预案必填）<br>2. 课中执行：安全保护+课程对齐+ACE三维观察<br>3. 课后：GYMGO出勤标记+家长反馈+教案反思<br>4. 续费主责（7节点全流程）<br>5. 每日工作量4个 |
| **四五星教练额外职责** | 审一二三星教案（只看安全+对齐，不管深度）<br>月度交叉听课1节（用傻瓜式匹配度"是/否"表打分） |
| **禁止事项** | 不得跳过安全预案、不得体罚学员、不得私下收费 |
| **KPI** | 续费率+安全零事故+工作量达成 |

**四五星教练交叉听课"是/否"清单**：

```
□ 安全保护动作按标准执行了没有？                 Y/N
□ 课程内容与教案描述一致吗？                       Y/N
□ 整节课的教学主题匹配当天规定的课程类型吗？         Y/N
□ 师生比在规定范围内吗？（≤1:8）                   Y/N
```

4个Y=匹配，2个及以下Y=不匹配→上报总经理。

### 4.6 课程顾问

**一句话定岗**：找线索、约体验、签单。职责不变，但新增"行为红线清单"日控。

| 项目 | 内容 |
|------|------|
| **直接上级** | 店长 |
| **核心职责** | 1. 线索跟进（2h内联系新线索）<br>2. 体验邀约+到店接待<br>3. 签单+GYMGO录入<br>4. 首月交接（成交后3天交教练）<br>5. 每日工作量4个 |
| **禁止事项** | 不得虚假承诺、不得私自降价、不得越位主导续费 |
| **KPI** | 新签转化率+邀约率+工作量达成 |

## 六、傻瓜式日控清单（店长版）

店长每天管理动作不超过30分钟，全部Y/N判断。

### 6.1 体验线日控表（晨间3分钟）

```
追光小牛 · 体验线日控清单       店名：____    日期：____   检查人：____

□ 1. 昨天新增线索，2h内联系了吗？       线索__条，已联系__条    Y / N
□ 2. 昨天体验课，签到做了吗？          体验__人，已签到__人    Y / N
□ 3. 前天体验课后，24h回访做了吗？      体验__人，已回访__人    Y / N
□ 4. 有学员剩≤20节未启动续费的吗？      剩≤20节__人，已启动__人  Y / N
□ 5. 成交满7天的，转介绍邀请发了吗？     成交__人，已发__人     Y / N

所有Y→过。有N→写谁没做→发总经理（微信一句话，不写理由）
```


## 七、渐进实施节奏

### 7.1 阶段规划

| 阶段 | 时间 | 主题 | 控制范围 | 激励搭配 |
|------|------|------|---------|---------|
| **僵化期** | 7-8月 | 只抓"做没做" | 体验线5个勾+人员工作量+教案安全 | 周工作量前三名群内表扬 |
| **固化期** | 9-10月 | 上线"安不安全、对不对" | 僵化期全部+四五星审教案+交叉听课 | 月度之星上墙 |
| **优化期** | 11-12月 | 数据开始说话 | 固化期全部+ACE匹配率+续费率进星级 | 星级晋升兑现 |

### 7.2 僵化期（7-8月）只抓4件事

| 序号 | 控制项 | 谁做 | 怎么查 | 达标线 | 不达标后果 |
|------|--------|------|--------|--------|-----------|
| 1 | 店长每天打5个勾 | 店长 | 总经理每天看红灯清单 | 日清率≥90% | 总经理电话追问 |
| 2 | 每人每天4个工作量 | 全体一线 | 店长次晨查 | 达成率≥80% | 晨会点名→连2天不达标上报 |
| 3 | 教案安全预案必填 | 教练 | 店长上课前查 | 100% | 缺安全预案→换人上课 |
| 4 | 财务每天出红灯清单 | 财务 | 总经理每天收 | 准时率100% | 迟出→总经理约谈 |

**这4件事之外的任何考核，僵化期一律不启动。**

### 7.3 固化期（9-10月）新增2件事

| 序号 | 控制项 | 谁做 | 怎么查 |
|------|--------|------|--------|
| 5 | 四五星教练审一二三星教案（安全+对齐） | 四五星教练 | 周审1次，结果报总经理 |
| 6 | 四五星教练交叉听课"是/否"4项 | 四五星教练 | 月听1节，结果报总经理 |

### 7.4 优化期（11-12月）新增2件事

| 序号 | 控制项 | 谁做 | 怎么查 |
|------|--------|------|--------|
| 7 | ACE评估完成率纳入月报 | 财务出数 | 每月5日数据月报 |
| 8 | 教案评分+课堂匹配度纳入星级评定 | 总经理打分 | 教学质量维度（教案30%+匹配30%+ACE合格20%+续费20%） |

## 九、风险预案

| 风险 | 表现 | 应对 |
|------|------|------|
| 店长打勾造假 | 勾了但实际没做 | 总经理月度审计抽查5天，造假=书面警告+乐捐 |
| 总经理扛不住 | 5店审计+周例会让总经理超负荷 | 僵化期只做4件事，督导分摊巡店核查任务 |
| 四五星教练不配合 | 不愿审教案听课 | 审教案/听课计为工作量+纳入星级晋升考核 |
| 团队反弹过大 | 觉得管太严 | 僵化期只宣导"不考核、只提醒"，不扣钱 |
| 财务数据月报出不来 | GYMGO不支持 | 第一个月手工统计Excel→第二个月要求GYMGO方增加报表 |
