# 追光小牛 · 教练教案与教学计划编写规范

> 版本：v1.0 | 2026年7月起执行 | 适用对象：全体教练
> 本规范配合"公司下发标准+教练自主编写"的新教案模式

---

## 一、新模式说明

### 1.1 变更原因

旧模式：公司统一发教案 → 教练照着执行。
问题：教练逐渐变成"执行机器"，不再思考为什么这样教、能不能教得更好。

### 1.2 新模式（2026年7月起）

```
公司每月发布        教练自主编写         公司审核         课堂执行
─────────────────────────────────────────────────────────────────
教学原则           教学计划(月/周)        店长初审         按审核后教案
教学主题/目标      教案(每节课)          教学主管终审     上课
各课程大纲框架     在标准框架内发挥       退回修改重提     课后反思更新
审核评分标准       体现个人教学思考
```

**核心理念**：公司定方向和底线，教练做设计和执行。教练从"被动执行者"升级为"课程设计者"。

## 二、教案编写六大原则（所有教案必须遵守）

以下六条是追光小牛教案的硬性底线，审核时每条都是否决项。

### 原则一：安全性（一票否决）

| 检查项 | 要求 |
|--------|------|
| 风险识别 | 教案中明确标注本课高风险动作和风险区域 |
| 保护方案 | 每个L2/L3风险动作必须写明保护站位和保护手法 |
| 并发控制 | 多学员同时操作高风险动作时，必须说明并发上限和轮换方式 |
| 应急预案 | 教案末尾注明：本课可能出现的常见意外及第一反应动作 |
| 器材检查 | 列出本课所需器材的安全检查要点 |

**不合格示例**：教案只有动作名称，没有保护说明。
**合格示例**："前滚翻练习——保护位：教练蹲于学员右侧，右手护颈，左手托大腿后侧；并发控制：一次2人，其余学员坐垫观察。"

### 原则二：渐进性（必须体现进阶逻辑）

| 检查项 | 要求 |
|--------|------|
| 难度递进 | 热身→基础→进阶→挑战，难度逐步提升，不跳跃 |
| 降阶方案 | 每个核心动作必须写明降阶方案（做不了这个的学员练什么） |
| 升阶路径 | 每个核心动作必须写明升阶方向（做得好的学员下一步练什么） |
| 周间衔接 | 本周教案和上周教案之间有明确的递进关系（新学了什么、复习了什么） |

**不合格示例**：热身完直接上高难度跳箱，没有中间过渡。
**合格示例**："跳箱练习分为三级：L1地面双脚跳→L2低箱双脚跳上→L3标准跳箱。根据学员能力分层执行。"

### 原则三：系统性（必须在课程大框架内）

| 检查项 | 要求 |
|--------|------|
| 与月度目标对齐 | 教案标注"本月教学主题：XXX"，本课内容服务于月度目标 |
| 与课程线主目标对齐 | 不跑题不串课（篮球课不能上成感统课） |
| 体能要素覆盖 | 单节课至少覆盖3项身体素质（见各课程线要求） |
| 长期发展考虑 | 教案标注本课属于该学员训练计划的哪个阶段（基础期/提升期/巩固期） |

**不合格示例**：篮球课整节课都在做跑跳游戏，没有球感训练。
**合格示例**：篮球课60分钟分配——控球基础15min→运球节奏15min→传接配合15min→游戏10min→拉伸5min。

### 原则四：趣味性（必须有参与度保障）

| 检查项 | 要求 |
|--------|------|
| 情境设计 | 至少1个环节有情景带入（故事、角色、挑战任务等） |
| 游戏化元素 | 至少1个环节包含游戏/竞赛/闯关元素 |
| 互动设计 | 学员之间必须有协作或竞争互动，不是全程单人练习 |
| 激励设计 | 教案中注明本课的激励方式（积分、勋章、称号等） |

**不合格示例**：60分钟全程排队轮换做动作，无游戏无互动。
**合格示例**："体能闯关环节（15min）：设置4个关卡（平衡关/力量关/速度关/协调关），分组PK，完成全部关卡获得'全能勇士'勋章。"

### 原则五：个性化（必须体现分层教学）

| 检查项 | 要求 |
|--------|------|
| 学员分析 | 教案标注：本课重点关注的3-5名学员及各自关注点 |
| 分层设计 | 每个核心环节至少有2个难度层级（基础版/挑战版） |
| 个别辅导预案 | 标注：如有学员严重掉队/过度兴奋/情绪波动的处理方案 |

**不合格示例**：全班统一动作统一标准，没有分层。
**合格示例**："核心训练分层：基础组（3名新学员）—跪姿俯卧撑8个×2组；进阶组（5名老学员）—标准俯卧撑10个×3组；挑战组（2名尖子）—窄距俯卧撑8个×3组。"

### 原则六：可执行性（必须能落地）

| 检查项 | 要求 |
|--------|------|
| 时间分配合理 | 各环节时间总和不超课程时长，过渡时间已预留 |
| 器材可行 | 所列器材均为校区已有器材，数量和规格明确 |
| 人员可行 | 助教/主教分工明确，不假设不存在的人力 |
| 场地可行 | 场地使用方案不与同时段其他班级冲突 |

## 五、各课程线教学大纲与编写指南

### 5.1 感统体操

**课程主目标**：基础动作模式、身体控制、平衡协调、感觉输入整合

**月度主题示例**：
| 月份 | 主题 | 核心训练内容 |
|------|------|-------------|
| 第1月 | 建立课堂安全感 | 跟随指令、基础爬行、简单翻滚 |
| 第2月 | 基础平衡能力 | 单脚站、平衡木初级、直线行走 |
| 第3月 | 身体协调性 | 手脚并用爬行、跨障碍、简单节奏练习 |

**教案编写重点**：
- 优先建立"敢参与、能跟随"的课堂状态
- 降阶方案比升阶方案更重要（本条课程线尤其如此）
- 游戏化程度要求最高（至少2个游戏化环节）
- 重点关注学员基础控制能力和指令反应
- 对掉队学员做降阶而非拔高

**体能要素侧重**（每节课至少覆盖3项）：平衡、协调、柔韧、基础力量

### 5.2 篮球

**课程主目标**：运控、节奏、手眼协调、空间判断、团队规则

**月度主题示例**：
| 月份 | 主题 | 核心训练内容 |
|------|------|-------------|
| 第1月 | 球感建立 | 原地运球、身体绕球、基础传球 |
| 第2月 | 运球控制 | 行进间运球、变向、护球 |
| 第3月 | 传接配合 | 两人传接、三人传切、基础规则 |

**教案编写重点**：
- 先建立控球和运球节奏，再谈投篮结果
- 每课必须有球感训练，不能只有跑跳没有球
- 规则理解和团队配合纳入每课目标
- 教案必须注明：本课技能在比赛场景中怎么用

**体能要素侧重**（每节课至少覆盖3项）：协调、速度、灵敏、耐力

### 5.3 跳绳

**课程主目标**：节奏、下肢协调、耐力、连续动作能力

**月度主题示例**：
| 月份 | 主题 | 核心训练内容 |
|------|------|-------------|
| 第1月 | 节奏入门 | 无绳节奏练习、单次跳过、手脚配合 |
| 第2月 | 连续跳绳 | 连续跳10-30个、匀速节奏 |
| 第3月 | 速度与变化 | 加速跳、单脚交替、双摇入门 |

**教案编写重点**：
- 先练起跳节奏和手脚配合，再追连续个数
- 每课必须有节奏练习（不拿绳也练节奏）
- 对急躁学员优先稳成功率
- 教案必须区分：节奏问题、手脚配合问题、耐力问题，分别处理
- 课堂不追求一次性练到力竭（避免过度冲击膝盖）

**体能要素侧重**（每节课至少覆盖3项）：协调、耐力、节奏、下肢力量

### 5.4 中考体训

**课程主目标**：明确项目成绩、体能储备、专项训练计划

**月度主题示例**：
| 月份 | 主题 | 核心训练内容 |
|------|------|-------------|
| 第1月 | 基础体能储备 | 心肺基础、核心力量、柔韧基础 |
| 第2月 | 专项技术分解 | 各考试项目技术动作分解训练 |
| 第3月 | 成绩冲刺 | 模拟测试、弱项强化、考试技巧 |

**教案编写重点**：
- 每节课必须标注本课训练对应哪个考试项目
- 必须有训练数据记录（组数、成绩、心率等）
- 必须说明本课在学员阶段计划中的位置（打基础/练技术/冲成绩）
- 训练安排围绕阶段目标，不随意跳项
- 课后反馈必须包含：当前成绩、提升点、下周目标

**体能要素侧重**（每节课至少覆盖3项）：力量、耐力、速度、柔韧

### 5.5 搏击防身

**课程主目标**：反应、协调、规则意识、自我保护、力量传导

**月度主题示例**：
| 月份 | 主题 | 核心训练内容 |
|------|------|-------------|
| 第1月 | 站架与距离 | 基础站架、安全距离、前后移动 |
| 第2月 | 直拳与防守 | 直拳技术、格挡、闪躲基础 |
| 第3月 | 组合与反应 | 拳法组合、条件对练、情境反应 |

**教案编写重点**：
- 强调规则、距离、控制和停手意识
- 让学员先学会规范动作和保护边界
- 对兴奋度高的学员优先控节奏和控场面
- 教案必须标注：本课的对练规则和安全边界
- 不以"打得猛不猛"为评价标准

**体能要素侧重**（每节课至少覆盖3项）：反应、协调、核心力量、灵敏

### 5.6 体态矫正

**课程主目标**：姿态觉察、基础稳定、肌群平衡、动作控制

**月度主题示例**：
| 月份 | 主题 | 核心训练内容 |
|------|------|-------------|
| 第1月 | 姿态感知 | 感知正确站姿坐姿、核心激活、基础呼吸 |
| 第2月 | 稳定性训练 | 核心稳定、肩胛稳定、骨盆控制 |
| 第3月 | 肌群平衡 | 前后侧平衡、左右对称训练、动态控制 |

**教案编写重点**：
- 先让学员感知姿态和发力位置，再推进训练量
- 先稳基础控制，再上复杂组合
- 对代偿明显的动作及时降阶和纠错
- 教案必须标注：本课训练对应的体态问题是什么
- 不追求短期大幅变化，讲质量不讲数量
- 不向家长承诺"纠正完成"

**体能要素侧重**（每节课至少覆盖3项）：柔韧、核心力量、控制、稳定

### 5.7 跑酷

**课程主目标**：越障、支撑、落地、空间判断、勇气建立

**月度主题示例**：
| 月份 | 主题 | 核心训练内容 |
|------|------|-------------|
| 第1月 | 落地与安全 | 安全落地技术、翻滚卸力、基础支撑 |
| 第2月 | 跨越与平衡 | 低障碍跨越、平衡通过、猫爬 |
| 第3月 | 组合穿越 | 多障碍组合、连续动作、速度控制 |

**教案编写重点**：
- 先做动作分解和落地控制，再做连续连接
- 先保证保护位和并发控制，再提高刺激性
- 对胆怯学员优先建信心，对冒进学员优先控风险
- 教案必须标注每个越障动作的安全保护方案
- 不以"多刺激"为卖点，以"有控制的挑战"为核心

**体能要素侧重**（每节课至少覆盖3项）：力量、协调、灵敏、爆发力

## 六、教案审核评分标准

### 6.1 评分维度与权重

| 维度 | 权重 | 评分标准 |
|------|------|---------|
| 六大原则符合度 | 40% | 安全性(10分)+渐进性(8分)+系统性(7分)+趣味性(6分)+个性化(5分)+可执行性(4分) |
| 教学目标清晰度 | 20% | 主目标明确、与月度主题对齐、可衡量 |
| 课程结构合理性 | 20% | 时间分配、环节过渡、器材安排 |
| 分层设计质量 | 10% | 降阶方案有效性、升阶方案合理性 |
| 课前准备充分度 | 10% | 安全预案完整性、学员分析准确性 |

### 6.2 评分等级

| 等级 | 分数 | 处理方式 |
|------|------|---------|
| A（优秀） | ≥90分 | 通过，纳入优秀教案库，月度分享 |
| B（合格） | 75-89分 | 通过 |
| C（需修改） | 60-74分 | 退回修改，24小时内重提 |
| D（不合格） | <60分 | 退回重写，48小时内重提，连续2次D级→教练约谈 |

### 6.3 一票否决项

以下任一条不满足，直接退回，不评分：

1. 无安全预案或无保护说明
2. 教案模板不完整（缺少必修栏位）
3. 明显跑题串课（篮球课上成感统课）
4. 无分层设计（全班统一动作统一标准）
5. 时间分配严重失衡（某个环节超1/2课时）

### 6.4 审核通过率纳入星级考核

| 考核项 | 计算方式 | 考核影响 |
|--------|---------|---------|
| 教案一次通过率 | 当月一次评分≥B的教案数÷总提教案数 | 纳入教学质量维度评分 |
| 平均教案评分 | 当月所有教案评分的平均值 | 四星/五星教练升星参考 |
| 退修次数 | 当月退回修改次数 | 退修≥3次/月→教练约谈 |

## 七、公司每月下发内容

每月25日前，教学主管/店长向全体教练下发以下内容：

### 7.1 每月下发的标准包

| 下发内容 | 说明 | 示例 |
|---------|------|------|
| 月度教学主题 | 本月统一的教学方向 | "3月主题：下肢力量与爆发力" |
| 月度核心训练动作 | 本月各课程线须覆盖的关键动作（3-5个） | "深蹲变式、跳箱入门、弓步蹲" |
| 月度安全重点 | 本月需重点关注的安全事项 | "春季关节热身要充分，跳箱保护必修课" |
| 月度考核重点 | 本月教案审核重点关注的维度 | "本月重点检查分层教学设计" |

### 7.2 教练的自主空间

公司在以下方面不做限制，教练自主发挥：

- 具体训练动作的组合和排列顺序
- 游戏/竞赛的创意设计
- 训练强度的具体数值（在安全范围内）
- 激励方式和情景设定
- 对个别学员的个性化训练安排
- 辅助训练和课后作业的设计

**一句话概括**：公司管"方向对不对、安全不安全、专业不专业"，教练管"怎么教得更好、更有趣、更有针对性"。

## 八、执行流程

### 8.1 月度循环

```
每月25日    公司下发：下月教学主题、核心动作、安全重点、考核重点
每月25-28日 教练编写：下月教学计划
每月28-30日 店长初审→教学主管终审
每月1日前   教练拿到审核后的月度教学计划

每周日前    教练提交：下周全部教案
每周日      店长/四星教练审核教案
上课前1天  教练根据审核意见修改，定稿

每节课后   教练填写课后反思
月末       教学主管汇总教案评分，纳入星级考核
```

### 8.2 分层审核机制

| 教练星级 | 教案审核人 | 审核频率 |
|---------|-----------|---------|
| 实习 | 带教教练 | 每节必审 |
| 一星 | 店长/四星教练 | 每周全审 |
| 二星 | 四星教练 | 每两周全审 |
| 三星 | 四星/五星教练 | 月度抽查 |
| 四星 | 教学主管 | 季度抽查 |
| 五星 | 教学主管 | 季度抽查 |

## 九、防止教练变懒的机制

### 9.1 教案去重检查

- 同一教练同一班级连续4节课教案不得完全相同
- 不同教练同课程线教案不得全文雷同
- 月度教学计划不得逐月照搬
- 违反以上→退回重写+书面警告

### 9.2 教案库激励机制

- 每月评选"最佳教案"（按课程线各1份）
- 入选教案库的作者获得"创新之星"专项荣誉+100元
- 累计3次入选→教学分享会主讲资格
- 教案被多次引用的教练，在星级评定时加分

### 9.3 课堂抽查与教案复盘

- 店长/主管每月随机抽查至少2节课
- 抽查内容：教案与课堂实际执行是否一致
- 教案写得漂亮但课堂执行差→教案评分降级+教练约谈
- 课堂执行好但教案写得敷衍→退回补写
