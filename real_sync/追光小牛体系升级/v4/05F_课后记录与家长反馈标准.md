# 05F_课后记录与家长反馈标准

## 1. 文件定位

本标准用于统一课后记录、课堂反馈和家长沟通收口动作。

本文件重点解决：

1. 一节课结束后必须留下什么信息。
2. 主教和助教分别负责哪些课后动作。
3. 对家长应该怎么反馈，哪些话不能越权说。

## 2. 基本原则

1. 先记录事实，再输出判断。
2. 先内部同步，再对外表达。
3. 课后反馈要稳定、清楚、不过度承诺。
4. 异常事件必须有交接，不得只停留在口头说过。

## 3. 课后必须完成的动作

1. 课堂结果收口。
2. 重点学员和异常学员信息确认。
3. 课后记录补全。
4. 家长核心反馈输出。
5. 必要时向店长或相关人员交接。

## 4. 主教课后职责

1. 确认本堂课程目标完成情况。
2. 确认重点学员表现和异常情况。
3. 输出对家长的主反馈。
4. 对异常事件作正式说明和后续交接判断。

## 5. 助教课后职责

1. 协助补充出勤、课堂表现和异常信息。
2. 协助整理器材和恢复场地。
3. 把自己观察到的重点情况反馈给主教。
4. 不得跳过主教直接形成带结论的对外反馈。

## 6. 课后记录内容

课后至少应记录清楚：

1. 出勤情况。
2. 本堂重点训练内容。
3. 学员整体参与状态。
4. 重点学员表现。
5. 异常事件或特殊情况。
6. 需要交接的后续事项。

## 7. 家长反馈原则

1. 先说本堂做了什么。
2. 再说孩子表现在哪些地方有亮点或需要关注。
3. 最后说后续建议或需要家长配合的事项。

## 8. 主教可以说什么

1. 本堂课程目标和训练内容。
2. 孩子的课堂表现和变化。
3. 下一步建议和家长配合点。
4. 已发生异常事件的事实和处理结果。

## 9. 助教可以说什么

1. 事实型信息，如出勤、情绪、参与度、配合度。
2. 经主教同步后的补充观察。

## 10. 助教不能说什么

1. 擅自承诺训练结果、升级结果或专业结论。
2. 擅自解释异常事件责任归属。
3. 在主教未同步前直接对家长作结论性说明。

## 11. 异常交接要求

出现以下情况时，课后必须交接：

1. 受伤。
2. 多次哭闹或失控。
3. 冲撞、离场、明显课堂失序。
4. 家长明显投诉或质疑。
5. 需要后续跟进的学习、情绪或服务问题。

## 12. 沟通边界要求

1. 不把课堂问题讲成医学诊断。
2. 不把阶段表现讲成确定结果。
3. 不把临时观察讲成长期结论。
4. 不为了安抚家长而做超权限承诺。

## 13. 一句话结论

课后记录与家长反馈标准的核心，不是把话说多，而是把事实留清楚、把边界守住、把后续交接做完整。
