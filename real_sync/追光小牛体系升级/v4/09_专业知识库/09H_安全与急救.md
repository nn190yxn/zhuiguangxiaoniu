# 09H_安全与急救

## 1. 文件定位

本文件用于统一追光小牛课堂中的安全认知、风险判断和基础应急处理边界，是所有教练必须掌握的知识底线。

## 2. 基本原则

1. 安全优先于进度。
2. 保护优先于完成度。
3. 无法保证安全条件时，宁可降阶，不可硬上。
4. 先处理人，再处理课堂，再处理记录。

## 3. 教练必须掌握的安全判断

1. 哪些动作有离地、翻滚、冲撞和跌落风险。
2. 哪些学员需要更高保护等级。
3. 哪些场景下必须降低并发或替换动作。
4. 哪些异常情况必须立即中止当前环节。

重点判断顺序：

1. 先看动作风险。
2. 再看学员状态。
3. 再看人员配置。
4. 最后看是否具备继续开展条件。

## 4. 常见风险场景

1. 翻滚、跳箱、攀爬、较高平衡木等高风险动作。
2. 情绪波动大、频繁离场、注意极不稳定的课堂状态。
3. 器材切换频繁、空间拥挤、多人并发的场景。
4. 学员身体不适、旧伤未清、家长有特别提醒的情况。

## 5. 必须立即升级处理的情况

1. 学员出现明显受伤、持续疼痛或无法继续活动。
2. 学员出现意识异常、持续哭闹无法稳定、明显惊吓失控。
3. 课堂环境已无法保证当前动作继续开展。
4. 现场人员配置无法覆盖风险点。

## 6. 异常处理的统一顺序

1. 立即停下相关动作。
2. 先确认学员状态和受伤风险。
3. 必要时分离围观、稳定课堂秩序。
4. 按门店流程通知相关负责人和家长。
5. 课后完成记录、交接和复盘。

## 7. 课堂预防要求

1. 课前必须确认高风险点和保护位。
2. 高风险动作前必须统一规则和进出顺序。
3. 新人、实习员工不得被默认放在高风险主保护位。
4. 只要保护不到位，就必须主动降阶或换项。

## 8. 急救边界

1. 教练必须掌握基础现场处理常识和升级汇报顺序。
2. 教练不是医生，不做超出能力范围的处置。
3. 遇到明显受伤、持续疼痛、意识异常等情况，必须立即升级处理。
4. 任何涉及较大风险的异常，都必须留痕。

## 9. 对家长的沟通要求

1. 讲事实、讲处理动作、讲后续安排。
2. 不隐瞒，不模糊，不抢结论。
3. 不在信息未清前做绝对承诺。
4. 统一口径，避免多人说法不一致。

## 10. 常见误区

1. 为了不影响课堂节奏，延迟处理风险。
2. 以为“问题不大”就不留痕。
3. 对家长先做安抚承诺，再补事实。
4. 把保护理解成站在旁边看，而不是主动预判和控制风险。
