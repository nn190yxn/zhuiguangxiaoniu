# 02J_课堂角色授权与跟岗标准

## 1. 文件定位

本标准用于承接“助教优先是课堂角色行为，不是固定岗位”的判断，统一课堂角色授权、跟岗和放行规则。

本文件重点解决：

1. 谁在什么阶段可以承担什么课堂角色。
2. 新员工如何从跟岗走向可承担助教行为，再走向可承担主教行为。
3. 主教在别的课堂承担助教行为时，边界是什么。

## 2. 基本原则

1. 角色由当堂授权决定，不由岗位名称自动决定。
2. 放行必须基于真实课堂表现，而不是基于资历想当然。
3. 跟岗的目的是稳定可用，不是形式性旁听。

## 3. 课堂角色定义

当前按 3 类角色理解：

1. 跟岗观察角色。
2. 助教行为角色。
3. 主教行为角色。

## 4. 跟岗观察角色

可承担：

1. 观察课堂流程。
2. 学习站位、口令、节奏和安全顺序。
3. 在明确安排下做简单辅助。

不得承担：

1. 发主口令。
2. 带节奏。
3. 独立处理异常。

## 5. 助教行为角色

可承担：

1. 课前准备、器材布置、点名整队。
2. 课堂秩序维护和轮换推进。
3. 在授权范围内做低风险辅助和保护补位。
4. 课后收尾和基础记录协助。

不得承担：

1. 抢主口令、改主节奏。
2. 单独带高风险动作区。
3. 独立向家长输出带结论的异常说明。

## 6. 主教行为角色

必须承担：

1. 当堂目标与节奏主导。
2. 重点动作讲解与示范。
3. 风险动作前的安全确认。
4. 异常事件决策和升级。
5. 家长主反馈和课堂收口。

## 7. 跟岗推进路径

建议按以下顺序推进：

1. 先完成观察和规则理解。
2. 再承担稳定的助教行为。
3. 再在带教下承担局部主教行为。
4. 最后再放行承担相应课堂主责。

## 8. 助教行为放行条件

员工被允许稳定承担助教行为前，至少应满足：

1. 清楚课堂基本流程。
2. 清楚主教口令体系和配合方式。
3. 清楚高风险动作区不能乱站、乱动、乱离岗。
4. 清楚异常发生时先保护、先汇报。

## 9. 主教行为放行条件

员工被允许承担主教行为前，至少应满足：

1. 能稳定带完整课堂流程。
2. 能在真实课堂中守住节奏和安全。
3. 能明确分工并带动助教协同。
4. 能完成课后反馈和基本家长沟通。

## 10. 角色切换规则

同一名员工在不同课堂承担不同角色时，必须遵守：

1. 每堂课前先明确自己本堂角色。
2. 本堂不是主教时，不得按自己习惯主导课堂。
3. 本堂是助教时，应以补位、看护、协同为主，不抢戏、不抢责。

## 11. 管理要求

1. 店长和带教人对放行结果负责。
2. 不得因排班压力让未放行员工事实性独立顶岗。
3. 放行后若出现明显失稳，应及时收回授权并补训。

## 12. 一句话结论

课堂角色授权与跟岗标准的目标，是把“谁能做什么”讲清楚，让课堂角色不靠默认、不靠经验、不靠临时顶替，而靠清晰授权和真实能力。
