# 07 品牌一致性标准

**版本：** v2.1
**适用对象：** 全体员工
**归口部门：** 总经办

---

## 一、制度

### 1.1 目的

统一品牌对外形象和传播内容，确保家长在任何触点都能感受到一致的品牌调性，建立专业、可信赖的品牌形象。

### 1.2 适用范围

所有对外品牌传播，包括门店形象、员工话术、朋友圈、美团/大众点评、抖音蓝V等线上触点。

### 1.3 品牌核心信息

| 项目 | 内容 |
|------|------|
| 品牌名称 | 追光小牛 |
| 品牌定位 | 专业儿童运动培训机构 |
| 品牌使命 | 让每个孩子爱上运动 |
| 品牌slogan | 追光小牛，运动成就未来 |
| 成立时间 | 2019年 |

### 1.4 红线

1. **数据造假** → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
2. 未经审批发布品牌相关内容 → 口头警告
3. 发布与品牌调性不符的内容 → 书面警告+乐捐进成长基金（员工10元/店长20元）
4. 使用非官方品牌素材 → 口头警告

## 二、要求

### 2.1 品牌视觉规范

| 项目 | 标准 |
|------|------|
| 品牌Logo | 使用官方标准版，不得自行修改颜色、比例 |
| 品牌色 | 主色：追光蓝（#XXXXXX）；辅色：活力橙（#XXXXXX） |
| 品牌字体 | 标题使用XX字体，正文使用XX字体 |
| 品牌图片 | 使用官方图库，不得使用未经授权的网图 |

### 2.2 品牌话术规范

| 要求 | 说明 |
|------|------|
| 统一介绍 | 所有员工使用统一的30秒品牌介绍语 |
| 统一用语 | 不使用"最好""第一""保证"等绝对化用语 |
| 统一数据 | 所有对外数据必须经总经办统一确认 |
| 统一承诺 | 不做无法兑现的承诺 |

### 2.3 门店品牌执行边界

| 角色 | 主责 |
|------|------|
| 总经办 | 统一品牌口径、统一素材模板、审核重要对外内容 |
| 店长 | 主责门店品牌落地、线上触点排期、数据复盘和整改闭环 |
| 顾问/教练 | 按店长安排提供课堂素材、服务案例、家长反馈与好评邀约，不擅自发布优惠 |
| 督导 | 检查门店品牌执行和形象达标情况，不代替店长做线上运营 |

## 三、标准

### 3.1 30秒品牌介绍语

**标准版：**

```
追光小牛成立于2019年，是一家专业的儿童运动培训机构。
我们专注于3-12岁儿童的运动能力发展，提供体适能、篮球、足球、跳绳、平衡车、体态矫正六大课程。
我们的教学理念是"让每个孩子爱上运动"，通过独创的ACE评估体系，
从运动能力、认知能力、参与动能三个维度全面关注孩子的成长。
目前在XX城市拥有XX家门店，服务超过XX组家庭。
```

**简短版（15秒）：**

```
追光小牛成立于2019年，专注3-12岁儿童运动培训，
提供体适能、篮球、足球等六大课程，
通过ACE评估体系全面关注孩子成长，目前在XX城市拥有XX家门店。
```

### 3.2 门店形象标准

| 项目 | 标准 |
|------|------|
| 门头 | 统一品牌Logo+名称，灯箱正常亮灯 |
| 前台 | 干净整洁，放置品牌宣传册、名片 |
| 教学区 | 墙面贴有品牌海报+学员风采展示 |
| 家长休息区 | 有品牌杂志、饮水机、充电设施 |
| 卫生间 | 清洁无异味，有品牌标识 |
| 员工着装 | 统一工服+工牌，工服干净整洁 |
| 音乐 | 播放品牌指定歌单，音量适中 |

### 3.3 朋友圈规范

#### 发布频率

| 类型 | 频率 | 说明 |
|------|------|------|
| 教学内容 | 每周2-3条 | 课堂花絮、学员进步、教学理念 |
| 活动宣传 | 按活动节点 | 使用总经办统一素材 |
| 个人生活 | 适量 | 不影响专业形象即可 |

#### 发布要求

| 要求 | 说明 |
|------|------|
| 使用官方素材 | 活动宣传必须使用总经办提供的素材 |
| 统一话题标签 | #追光小牛 #儿童运动 #让每个孩子爱上运动 |
| 不发负面内容 | 不抱怨工作、不批评学员/家长 |
| 不发竞品信息 | 不转发或提及竞品相关内容 |
| 不发未经审核的优惠 | 所有优惠信息必须经总经办审核 |

#### 朋友圈模板

```
【教学日常】
今天XX班的孩子们完成了XX挑战！
每个孩子都在进步，追光小牛为你们骄傲！
#追光小牛 #儿童运动 #让每个孩子爱上运动
```

```
【活动宣传】（使用总经办统一素材+文案）
追光小牛XX活动火热报名中！
XX时间、XX地点，期待你的加入！
#追光小牛 #XX活动
```

### 3.4 美团/大众点评/抖音规范

| 项目 | 标准 |
|------|------|
| 门店信息 | 名称、地址、电话、营业时间准确完整 |
| 门店头图 | 使用总经办统一头图 |
| 相册管理 | 每周更新3-5张课堂照片 |
| 团购套餐 | 使用总经办统一套餐设计 |
| 美团经营分 | 不低于80分 |
| 评价评分 | 美团/大众点评评分不低于4.8分 |
| 月度好评数 | 美团+抖音合计新增好评不少于15条 |
| 抖音蓝V | 每天发布1条门店抖音蓝V视频，内容使用总部指定主题 |
| 评价回复 | 48小时内回复所有评价 |
| 好评回复模板 | "感谢您的认可和好评！追光小牛会继续努力，让每个孩子爱上运动！" |
| 差评回复模板 | "非常抱歉给您带来了不好的体验，我们已私信您了解详情，会尽快为您解决。" |

## 四、流程SOP

### 4.1 品牌内容发布流程

```
顾问/教练按安排提供素材 → 店长初审并排期 → 重要内容提交总经办审核
    → 审核通过 → 按计划发布 → 记录归档并复盘
```

### 4.2 线上平台运营流程

```
每日 → 检查美团/大众点评新评价并48h内回复 → 发布1条抖音蓝V视频
    → 每周 → 更新相册3-5张 → 复盘经营分/评分/视频数据
    → 每月 → 汇总好评数（美团+抖音）/团购销量/经营分变化 → 提交运营报告
```

### 4.3 品牌物料管理流程

```
总经办统一设计 → 总经理审批 → 批量制作 → 分发到店
    → 门店按规范使用 → 定期检查更新 → 旧物料回收销毁
```

## 五、节点管理与跟踪考核

### 5.1 检查机制

| 检查项 | 频次 | 检查人 | 检查方式 |
|--------|------|--------|----------|
| 门店形象 | 每月 | 督导 | 巡店检查+拍照记录 |
| 朋友圈规范 | 每周 | 店长 | 抽查员工朋友圈 |
| 线上平台运营 | 每周 | 店长 | 平台后台检查+周报复盘 |
| 品牌介绍语 | 每季度 | 督导 | 神秘访客+抽查 |
| 员工着装 | 每日 | 店长 | 日常检查 |

### 5.2 考核标准

| 指标 | 达标 | 未达标处理 |
|------|------|------------|
| 门店形象检查 | 全部达标 | 1项不达标→口头警告；3项不达标→书面警告 |
| 朋友圈规范 | 符合发布要求 | 违规1次→口头警告；3次→书面警告 |
| 线上平台运营 | 美团经营分≥80分、评分≥4.8分、月好评≥15条、抖音蓝V周发布≥5条 | 未达标→当周整改并复盘 |
| 品牌介绍语 | 统一使用标准版 | 发现1次不使用→口头警告 |
| 员工着装 | 100%规范 | 1次不规范→口头警告 |

### 5.3 数据造假处理

- 伪造美团/大众点评/抖音好评 → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
- 伪造品牌数据（门店数、学员数等） → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
