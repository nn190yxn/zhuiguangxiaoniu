# 追光小牛 · 岗位说明书

> **版本**：v4.1 | **日期**：2026年4月
> **适用**：5家校区 | **主责**：总经理

---

## 一、制度

### 1.1 红线

| 红线 | 违反后果 |
|------|---------|
| 任何人不得超越岗位职责擅自决策 | 越权→书面警告 |
| 岗位职责明确到人，不得互相推诿 | 推诿→书面警告 |
| 岗位变动必须经总经理审批 | 未经审批→无效 |
| 数据造假 | 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退 |

### 1.2 岗位架构

```
            总经理
           /     \
        督导     财务
         |
      ┌──┼──┐
   店长1 店长2 店长3...（5家店）
    / \
  教练  顾问
```

**说明**：督导受总经理授权，对店长执行情况进行检查和评估，店长须配合督导工作。

---

## 二、要求

### 2.1 岗位设置原则

| 原则 | 说明 |
|------|------|
| 教销分离 | 教练管教学，顾问管销售，不得混岗 |
| 一岗一责 | 每个岗位有明确的职责边界 |
| 店长无行政 | 门店不设行政岗，店长兼管行政 |
| 兼岗报批 | 兼岗必须经总经理审批 |

### 2.2 编制标准

| 岗位 | 每店编制 | 说明 |
|------|---------|------|
| 店长 | 1人/店 | 管理双店可设副店长 |
| 教练 | 按课表排班 | 全职+兼职搭配 |
| 课程顾问 | 1-2人/店 | 按门店规模定 |
| 督导 | 1人 | 受总经理授权检查评估店长 |
| 财务 | 1人 | 全门店统一 |

## 三、标准

### 3.1 店长岗位说明书

| 项目 | 内容 |
|------|------|
| **岗位名称** | 店长 |
| **直接上级** | 总经理 |
| **平级** | 督导、其他店长 |
| **直接下级** | 本店教练、顾问 |
| **核心职责** | 门店业绩达标、团队管理、服务交付、上传下达、**获客资源管理、会员活动组织、转介绍推动** |
| **团队管理** | **员工工作量督导（每日核查4个/人）、经营面谈与提升辅导、工作量未达成时的帮扶与跟进、团队氛围建设** |
| **线上运营** | **门店线上平台运营：美团/大众点评经营分≥80分、评分≥4.8分、门店抖音蓝V号每日1条视频、月好评数（美团+抖音合计）≥15条** |
| **日工作** | 开关店检查、班前会/班后会、6个核心经营信号检查、巡店、**一访到店数量检查、资源补充安排、工作量核查（次日10:00前）、抖音视频发布** |
| **周工作** | 周报、管理周例会、周数据汇总、**转介绍数据盘点、市场活动筹备进度检查、周计划制定（四板块：销售/市场/教学/管理）、美团/抖音经营数据复盘、线上平台运营周报复盘、团队工作量周统计与排名通报** |
| **月工作** | 月度述职、月度复盘、成本管控、**会员活动策划与执行、资源获取计划制定、周边资源拓展（每月至少1次外联）、市场调查更新（每季度1次）、顾客满意度调查（每月至少5份）、异业合作洽谈（每月至少1次）、工资表核算、员工经营面谈（每人至少1次）、团队工作量月度总结与改进、线上平台运营月报复盘** |
| **任职要求** | 1年以上管理经验、能独立运营门店、数据敏感 |
| **授权边界** | 详见03_店长管理机制 |

### 3.2 教练岗位说明书

| 项目 | 内容 |
|------|------|
| **岗位名称** | 教练 |
| **直接上级** | 店长 |
| **核心职责** | 教学执行、学员安全、课后反馈、体验课转化 |
| **日工作** | 课前准备→课中执行→课后反馈→GYMGO出勤标记→场地复位 |
| **周工作** | 教案提交、周度教学总结 |
| **月工作** | 学员升班考核、教练星级评定 |
| **任职要求** | 体育/教育相关专业优先、有爱心、有耐心、能吃苦 |
| **禁止事项** | 不得私自收家长费用、不得私下承诺优惠、不得体罚学员 |

### 3.3 课程顾问岗位说明书

| 项目 | 内容 |
|------|------|
| **岗位名称** | 课程顾问 |
| **直接上级** | 店长 |
| **核心职责** | 线索跟进、体验邀约、签单转化 |
| **日工作** | 线索回访→体验邀约→到店接待→签单→GYMGO录入 |
| **周工作** | 周度线索盘点 |
| **月工作** | 月度业绩汇总 |
| **任职要求** | 沟通能力强、有销售经验优先、抗压能力强 |
| **禁止事项** | 不得虚假承诺、不得贬低竞品、不得私自降价、不得越位主导续费（仅可在店长安排下协助转化） |

### 3.4 督导岗位说明书

| 项目 | 内容 |
|------|------|
| **岗位名称** | 督导 |
| **直接上级** | 总经理 |
| **平级** | 店长 |
| **核心职责** | 受总经理授权，对店长执行情况进行检查和评估；新店长带教；巡店检查 |
| **日工作** | 巡店（按计划）、问题记录、带教指导 |
| **周工作** | 管理周例会参与、周度巡店报告 |
| **月工作** | 新店长验收、落后店长帮扶执行、KPI打分 |
| **任职要求** | 熟悉门店全流程、有带教能力、与店长平级不是上级 |
| **禁止事项** | 不得越权指挥店长、不得单独做门店人事决策 |

## 四、流程SOP

### 4.1 岗位调整SOP

```
Step 1 → 提出需求（店长/总经理）
  └→ 店长提出本店人力需求
  └→ 或总经理根据业务需要调整

Step 2 → 总经理审批
  └→ 审批通过→执行
  └→ 审批不通过→维持现状

Step 3 → 执行调整
  └→ 新增岗位→招聘（见02B）
  └→ 岗位变更→重新签岗位说明书
  └→ 岗位取消→按离职流程处理（见02F）

Step 4 → GYMGO系统更新
  └→ 更新员工角色权限
  └→ 更新排班信息
```

## 五、节点管理与跟踪考核

### 5.1 节点管理

| 节点 | 时间要求 | 责任人 | 跟踪方式 |
|------|---------|--------|---------|
| 岗位说明书签署 | 入职当天 | 店长 | 签署确认 |
| 岗位变更审批 | 变更前 | 总经理 | 审批记录 |
| GYMGO权限更新 | 岗位变更后1天内 | 店长 | 系统检查 |

### 5.2 考核机制

| 考核项 | 考核对象 | 考核频率 | 考核人 | 考核标准 |
|--------|---------|---------|--------|---------|
| 岗位职责执行 | 全体员工 | 每月 | 店长 | 无越权、无推诿 |
| 督导检查配合 | 店长 | 每次巡店 | 督导 | 配合度评分 |
| 编制合规 | 店长 | 每月 | 总经理 | 不超编、不混岗 |
