# 追光小牛 · 薪酬结构

> **版本**：v5.1 | **日期**：2026年4月
> **适用**：5家校区全体员工 | **主责**：总经理（审批）、财务（核算）
> **数据来源**：2026年工资体系表 + 2026年薪酬提点方案说明 + 2025执行版教练薪资体系

---

## 一、制度

### 1.1 红线

| 红线 | 违反后果 |
|------|---------|
| 薪资必须按时发放，不得拖欠 | 拖欠→总经理负责 |
| 薪资结构必须透明，不得暗箱操作 | 数据造假→首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退 |
| 任何人不得私下承诺薪资外的额外收入 | 违反→书面警告 |
| 薪资调整必须经总经理审批 | 未经审批→无效 |

### 1.2 薪资保密原则

- 薪资属于个人隐私，员工之间不得互相打听
- 店长不得公开他人薪资信息
- 违反薪资保密→书面警告

## 二、要求

### 2.1 薪酬发放要求

| 项目 | 要求 |
|------|------|
| 发放时间 | 每月15日发上月薪资（遇节假日提前） |
| 核算依据 | GYMGO系统数据+考勤记录 |
| 薪资条 | 每月随薪资发放，明细清晰 |
| 异议处理 | 收到薪资3日内可提出异议 |

### 2.2 薪酬结构原则

| 原则 | 说明 |
|------|------|
| 底薪+提成 | 固定部分保底，提成部分激励（教练/顾问/店长取消月度绩效） |
| 多劳多得 | 课时费/签单提成上不封顶 |
| 星级挂钩 | 教练底薪+课时费均与星级挂钩 |
| 业绩导向 | 顾问/教练收入与业绩强相关 |
| 工作量管理 | 每日4个工作量，过程管理+成长基金奖励（详见02G） |

> **重要变更（2026年4月起执行）**：教练、顾问、店长取消月度绩效，改为每日工作量管理（详见02G_工作量成长基金方案）。工作量与薪酬独立，不影响底薪/课时费/提成。督导保留绩效。

### 2.3 薪酬口径说明

| 类别 | 是否由本文件负责 | 说明 |
|------|----------------|------|
| 底薪/课时费/提成 | 是 | 本文件定义收入结构 |
| 年度KPI奖金 | 是 | 属于年度结果奖励 |
| 工作量管理 | 否 | 由 02G 负责过程动作管理 |
| 荣誉激励 | 否 | 由 06B 和《成长基金管理办法》负责 |
| 违规处罚 | 否 | 由总纲和《成长基金管理办法》统一口径 |

## 四、流程SOP

### 4.1 月度薪资核算SOP

```
Step 1 → 数据导出（每月1日）
  └→ GYMGO导出：课时/签单/续费/出勤
  └→ 考勤系统导出：迟到/早退/请假/旷工

Step 2 → 财务核算（每月5日前）
  └→ 教练：底薪 + 课时费（查星级+出勤人数表）+ 业绩提成（查阶梯表）
  └→ 顾问：底薪 + 签单提成（查阶梯表）
  └→ 店长：底薪 + 业绩提成（查阶梯表）
  └→ 生成薪资明细表

Step 3 → 店长审核（每月8日前）
  └→ 核对本店员工数据是否准确
  └→ 有异议→标注→财务核实

Step 4 → 总经理审批（每月10日前）
  └→ 审核薪资总额
  └→ 审批通过→财务执行

Step 5 → 发放（每月15日）
  └→ 银行转账
  └→ 发放薪资条

Step 6 → 异议处理
  └→ 员工3日内提出异议
  └→ 财务核实→如有误→次月补发
```

|------|---------|--------|---------|

| 薪资发放 | 每月15日 | 财务 | 银行转账记录 |
| 异议处理 | 发放后3日内 | 财务 | 异议记录 |

### 5.2 考核机制

| 考核项 | 考核对象 | 考核频率 | 考核人 | 考核标准 | 做不到怎么办 |
|--------|---------|---------|--------|---------|------------|
| 薪资核算准确率 | 财务 | 每月 | 总经理 | 零差错 | 出错→书面警告+限期修正 |
| 发放时效 | 财务 | 每月 | 总经理 | 15日前发放 | 逾期→总经理约谈 |
| 薪资保密 | 全体员工 | 持续 | 店长 | 无打听/泄露 | 违反→书面警告 |
