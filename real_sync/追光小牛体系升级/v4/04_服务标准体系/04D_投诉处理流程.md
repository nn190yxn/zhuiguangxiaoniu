# 04D 投诉处理流程

**版本：** v2.1
**适用对象：** 门店全员、店长、督导、总经理
**归口部门：** 运营部

---

## 一、制度

### 1.1 目的

建立分级投诉处理机制，快速响应、妥善解决，降低投诉升级风险，维护品牌声誉。

### 1.2 适用范围

所有来自家长、学员、合作方的投诉和建议。

### 1.3 红线

1. **数据造假** → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
2. 收到投诉后未在规定时效内响应 → 书面警告+乐捐进成长基金（员工10元/店长20元）
3. 对投诉家长态度恶劣或推诿 → 书面警告+乐捐进成长基金（员工10元/店长20元）
4. 隐瞒投诉不报 → 书面警告+乐捐进成长基金（员工10元/店长20元）

## 二、要求

### 2.1 投诉分级标准

| 等级 | 定义 | 响应时限 | 解决时限 | 处理人 |
|------|------|----------|----------|--------|
| 绿色（一般） | 教学质量、服务态度、环境设施等日常问题 | 1小时内 | 24小时内 | 店长 |
| 黄色（较重） | 涉及安全、收费争议、多次投诉未解决 | 30分钟内 | 48小时内 | 店长+督导 |
| 红色（严重） | 人身安全、媒体曝光风险、法律纠纷、群体投诉 | 15分钟内 | 当天 | 总经理+督导 |

### 2.2 投诉处理原则

| 原则 | 说明 |
|------|------|
| 先处理情绪，再处理问题 | 不急于辩解，先让家长说完 |
| 首问负责 | 谁接到投诉谁负责跟进到底 |
| 升级及时 | 超出处理权限立即升级 |
| 有记录 | 所有投诉必须登记，不得遗漏 |
| 有回访 | 解决后48小时内必须回访 |

### 2.3 补偿标准

| 投诉等级 | 补偿方式 | 审批权限 |
|----------|----------|----------|
| 绿色 | 口头道歉+1节免费补课 | 店长 |
| 黄色 | 书面道歉+2节免费补课或等值优惠 | 督导 |
| 红色 | 书面道歉+3节免费补课+经济补偿（金额由总经理审批确定） | 总经理 |

## 三、标准

### 3.1 投诉处理话术标准

#### 首次响应话术

```
【绿色】XX妈妈您好，非常抱歉给您带来了不好的体验。
您反映的问题我已经记录下来了，我会在24小时内给您一个满意的答复。

【黄色】XX妈妈您好，非常抱歉给您带来了不好的体验，这是我的失职。
您反映的问题我已经记录，我会和督导一起在48小时内给您解决方案。

【红色】XX妈妈您好，我是总经理XXX，非常抱歉发生了这样的事情。
我非常重视这个问题，今天一定会给您一个处理结果。
```

#### 解决方案话术

```
针对您反映的XX问题，我们的处理方案是：
1. XXX（具体措施）
2. XXX（补偿方案）
3. XXX（后续预防措施）
您看这样处理可以吗？如果还有其他要求，我们再协商。
```

#### 回访话术

```
XX妈妈您好，上次您反映的XX问题，我们处理得您还满意吗？
感谢您的反馈，我们会持续改进。
如果以后有任何问题，随时联系我。
```

### 3.2 退费相关投诉标准

| 情形 | 处理方式 |
|------|----------|
| 家长主动要求退费 | 按04C退费流程执行 |
| 因投诉引发退费 | 优先协商解决，协商不成按退费流程执行 |
| 退费过程中产生投诉 | 店长介入，3天内给出方案，确认后30个工作日内退款 |

## 四、流程SOP

### 4.1 投诉处理全流程

```
收到投诉 → 1h/30min/15min内首次响应 → 安抚情绪+记录问题
    → 分析原因 → 制定解决方案 → 审批补偿方案
    → 与家长沟通方案 → 执行方案 → 24h/48h/当天内解决
    → 解决后48h内回访 → 记录归档 → 月度分析改进
```

### 4.2 投诉升级流程

```
店长处理（绿色）→ 24h未解决 → 升级为黄色 → 督导介入
    → 48h未解决 → 升级为红色 → 总经理介入
    → 涉及安全/法律 → 直接红色 → 总经理+法务
```

### 4.3 退费投诉流程

```
家长提出退费 → 前台登记 → 店长3天内核算方案 → 与家长确认
    → 家长签署退费确认书 → 财务30个工作日内打款 → 顾问3天内回访
```

## 五、节点管理与跟踪考核

### 5.1 检查机制

| 检查项 | 频次 | 检查人 | 检查方式 |
|--------|------|--------|----------|
| 投诉响应时效 | 每月 | 督导 | 投诉登记表 |
| 投诉解决时效 | 每月 | 督导 | 投诉登记表 |
| 回访完成率 | 每月 | 店长 | 投诉登记表 |
| 投诉满意度 | 每月 | 督导 | 回访记录 |
| 投诉率 | 每月 | 总经理 | 投诉登记表 |

### 5.2 考核标准

| 指标 | 达标 | 未达标处理 |
|------|------|------------|
| 投诉响应时效 | 按分级标准执行 | 超时1次→口头警告；超时3次→书面警告 |
| 投诉解决时效 | 按分级标准执行 | 超时1次→书面警告 |
| 回访完成率 | 100% | 漏1次→口头警告 |
| 投诉率 | ≤5% | >5%→店长书面警告+专项整改 |
| 投诉满意度 | ≥80% | <80%→店长约谈，制定改进计划 |

### 5.3 数据造假处理

- 隐瞒投诉不登记 → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
- 伪造投诉处理记录 → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
