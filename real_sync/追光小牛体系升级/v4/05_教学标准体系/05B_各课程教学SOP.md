# 05B 各课程教学SOP

**版本：** v4.0
**适用对象：** 全体教练、教学主管
**归口部门：** 教学部

---

## 一、制度

### 1.1 目的

规范各课程教学流程，确保教学质量稳定、可复制，为学员提供一致的高品质教学体验。

### 1.2 适用范围

全部门店所有课程的教学执行。

### 1.3 课程体系概览

| 编号 | 课程名称 | 适合年龄 | 核心能力 |
|------|----------|----------|----------|
| 1 | 感统体操 | 3-12岁 | 感统训练、基础运动能力、协调性 |
| 2 | 篮球 | 3-12岁 | 球感、团队配合、竞技能力 |
| 3 | 跳绳 | 4-12岁 | 协调性、节奏感、心肺耐力 |
| 4 | 中考体训 | 8-14岁 | 中考项目训练、体能提升 |
| 5 | 搏击防身 | 4-12岁 | 自我保护、体能、专注力 |
| 6 | 体态矫正 | 6-12岁 | 体态改善、核心力量、柔韧性 |
| 7 | 跑酷 | 4-12岁 | 平衡能力、空间感知、胆量 |

### 1.4 红线

1. **数据造假** → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
2. 上课期间玩手机或做与教学无关的事 → 书面警告+乐捐进成长基金（员工10元/店长20元）
3. 未按安全规范操作导致学员受伤 → 书面警告+乐捐进成长基金（员工10元/店长20元），严重者辞退
4. 未经审批擅自更改课程内容 → 口头警告
5. 对学员体罚或冷暴力 → 立即辞退

## 二、要求

### 2.1 课前准备

| 项目 | 要求 |
|------|------|
| 教案准备 | 每节课前完成教案，明确教学目标、内容、时间分配 |
| 场地检查 | 课前30分钟检查场地安全、器材完好、卫生达标 |
| 学员出勤确认 | 课前1天确认出勤人数，准备相应器材 |
| 着装要求 | 教练穿统一工服、运动鞋，佩戴工牌 |

### 2.2 课中执行

| 项目 | 要求 |
|------|------|
| 准时开课 | 严格按照课表时间开课，不迟到不早退 |
| 师生比 | 3岁以下≤1:4、3-6岁≤1:6、6岁以上≤1:8 |
| 安全第一 | 每节课前进行安全检查，课中全程关注学员安全 |
| 关注每个学员 | 确保每个学员都得到关注和指导，不冷落任何人 |
| 活泼有感染力 | 教学语言生动、示范动作标准、氛围积极向上 |

### 2.3 课后总结

| 项目 | 要求 |
|------|------|
| 课后反馈 | 课后2小时内完成学员反馈（微信/面谈） |
| 教学反思 | 每周提交1篇教学反思记录 |
| 器材归位 | 课后10分钟内完成器材整理和场地恢复 |
| 异常记录 | 如有学员受伤或异常情况，立即记录并上报 |

## 三、标准

### 3.1 通用教学结构（每节课60分钟）

| 环节 | 时长 | 内容 | 教练动作 |
|------|------|------|----------|
| 热身 | 10分钟 | 趣味跑跳、关节活动、动态拉伸 | 带领热身，观察学员状态 |
| 技能教学 | 20分钟 | 本节课核心技能教学与练习 | 示范→学员练习→纠正→再练习 |
| 游戏互动 | 15分钟 | 技能融入游戏，强化练习 | 组织游戏，关注参与度 |
| 放松拉伸 | 10分钟 | 静态拉伸、呼吸放松、课堂总结 | 带领拉伸，点评学员表现 |
| 过渡环节 | 5分钟 | 整理器材、签到下课 | 与家长简短沟通 |

| 禁止事项 | 严禁让学员做超出能力范围的动作 |

## 四、流程SOP

### 4.1 日常教学流程

```
课前30min → 场地检查+器材准备
    → 课前10min → 学员签到+出勤确认
    → 准时开课 → 热身(10min) → 技能教学(20min) → 游戏互动(15min) → 放松拉伸(10min)
    → 课后 → 与家长简短沟通 → 2h内发送课后反馈
    → 器材归位 → 场地恢复
```

### 4.2 新学员第一节课流程

```
课前 → 查阅学员信息（年龄/性格/运动基础）
    → 课前5min → 与家长沟通了解孩子特点
    → 课中 → 安排助教重点关注新学员
    → 课后 → 详细反馈（适应情况/表现亮点/建议）
    → 课后1天 → 电话回访家长
```

### 4.3 教学异常处理流程

```
学员受伤 → 停止活动 → 评估伤情 → 轻微伤→冰敷+通知家长
    → 严重伤→送医+立即上报店长+教学主管
学员情绪崩溃 → 安抚情绪 → 安排休息 → 联系家长 → 课后沟通
器材故障 → 停止使用 → 替换器材 → 课后报修
```

## 五、节点管理与跟踪考核

### 5.1 检查机制

| 检查项 | 频次 | 检查人 | 检查方式 |
|--------|------|--------|----------|
| 教案完成率 | 每周 | 教学主管 | 抽查教案本/电子教案 |
| 课堂执行规范 | 每周 | 教学主管 | 随堂听课（每月每人至少1次） |
| 课后反馈及时率 | 每周 | 店长 | CRM系统抽查 |
| 场地安全检查 | 每日 | 当班教练 | 检查清单签字 |
| 教学反思提交 | 每周 | 教学主管 | 系统提交记录 |

### 5.2 考核标准

| 指标 | 达标 | 未达标处理 |
|------|------|------------|
| 教案完成率 | 100% | 漏1次→口头警告 |
| 课后反馈及时率 | ≥90% | 连续2周<90%→书面警告 |
| 随堂听课评分 | ≥80分 | <80分→教学主管约谈，制定改进计划 |
| 场地安全检查 | 100% | 漏1次→口头警告；导致事故→书面警告+扣绩效 |
| 教学反思提交 | 每周1篇 | 漏1篇→口头警告 |

### 5.3 数据造假处理

- 伪造教案或教学记录 → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
- 伪造随堂听课记录 → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
