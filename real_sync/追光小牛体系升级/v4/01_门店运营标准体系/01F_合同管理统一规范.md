## 一、制度（红线 + 规则）

### 1.1 红线

| 序号 | 红线内容 | 处罚 |
|------|----------|------|
| 1 | 不使用公司统一合同版本 | 书面警告 + 乐捐进成长基金（员工10元/店长20元） |
| 2 | 私自承诺优惠/赠送 | 书面警告 + 乐捐进成长基金（员工10元/店长20元） + 差额自补 |
| 3 | 退费不按统一标准执行 | 书面警告 + 差额自补 |
| 4 | 合同信息填写不完整/不真实 | 书面警告 |
| 5 | 数据造假（虚报合同金额、伪造签名等） | 首次书面警告 + 乐捐进成长基金（员工10元/店长20元），二次辞退 |

### 1.2 规则

1. 所有合同必须使用公司统一版本，不允许自行修改
2. 合同填写必须完整、准确，不能有空项
3. 优惠活动须经店长审批，不得私自承诺
4. 退费必须按统一标准执行，任何人不得擅自决定
5. 合同签订后当天录入GYMGO系统

## 二、要求（做什么）

### 2.1 合同填写要求

每份合同必须填写以下内容，确保完整准确。

### 2.2 退费管理要求

退费必须按标准流程执行，确保合规。

### 2.3 转卡/停卡管理要求

转卡和停卡必须按规定办理，做好记录。

### 2.4 分期管理要求

分期付款必须按政策执行，做好还款跟踪。

## 三、标准（做到什么程度）

### 3.1 合同填写规范

| 填写项 | 标准 | 说明 |
|--------|------|------|
| 单次训练单价 | 150元 | 统一单价，所有合同一致 |
| 会员管理费 | 训练总价 x 30% | 训练总价 = 课时数 x 单次训练单价 |
| 实际总金额 | 顾客实际支付金额 | 优惠后金额 |
| 购买课时数 | 按实际购买填写 | 课时卡填写课时数，周期卡填写周期 |
| 有效期 | 课时卡 = 购买课时数 x 5天 | 从首次上课日开始计算 |
| 首次开卡 | 周期卡最晚购买后60天内 | 超过60天未开卡须申请延期 |
| 缺席次数 | 课时卡 = 购买课时数 / 3 | 取整数，如30节可缺席10次 |
| 家长签名 | 必须本人签字 | 不接受代签 |

#### 合同金额计算示例

| 项目 | 计算 | 金额 |
|------|------|------|
| 购买课时 | 48节 | — |
| 单次训练单价 | — | 150元 |
| 训练总价 | 48 x 150 | 7,200元 |
| 会员管理费 | 7,200 x 30% | 2,160元 |
| 合同总金额 | 7,200 + 2,160 | 9,360元 |
| 实际支付（如打9折） | 9,360 x 0.9 | 8,424元 |
| 有效期 | 48 x 5 | 240天 |
| 缺席次数 | 48 / 3 | 16次 |

### 3.2 停卡规定

| 课时数量 | 可停卡次数 | 每次最长停卡时间 | 办理要求 |
|----------|-----------|-----------------|----------|
| 50节以下 | 1次 | 30天 | 须提前申请，填写停卡申请 |
| 50-100节 | 2次 | 30天/次 | 须提前申请，填写停卡申请 |
| 100节以上 | 3次 | 30天/次 | 须提前申请，填写停卡申请 |

> **注意：** 停卡期间有效期顺延。停卡须在GYMGO系统中操作，不能只口头说。

### 3.3 转卡规定

| 项目 | 标准 |
|------|------|
| 转卡条件 | 已上课时不超过总课时的50% |
| 办理流程 | 填写转卡申请单 + 原会员/新会员/店长三方确认签字 |
| 转卡费用 | 按公司统一标准收取（如有） |
| 系统操作 | 财务在GYMGO系统中操作转卡（检查全部文件） |

### 3.4 退费标准

| 项目 | 计算方式 |
|------|----------|
| 剩余退费金额 | 实际支付总额 - 已消费课程数 x 单次购买课程单价 - 合理成本（15%） |
| 单次购买课程单价 | 实际支付总额 / 购买课时总数 |
| 合理成本 | 实际支付总额 x 15% |

#### 退费计算示例

| 项目 | 计算 | 金额 |
|------|------|------|
| 实际支付总额 | — | 8,424元 |
| 购买课时 | — | 48节 |
| 已上课时 | — | 20节 |
| 单次购买课程单价 | 8,424 / 48 | 175.5元 |
| 已消费课程费用 | 20 x 175.5 | 3,510元 |
| 合理成本（15%） | 8,424 x 15% | 1,263.6元 |
| 剩余退费金额 | 8,424 - 3,510 - 1,263.6 | **3,650.4元** |

### 3.5 退费责任划分

| 情形 | 责任人 | 处罚 |
|------|--------|------|
| 学员出勤率 < 70% 且要求退费 | 主课教练 + 店长 | 教练扣300元 + 店长罚500元 |
| 前三节课内无理由退费 | 顾问 + 教练 | 顾问不计算该订单提成，教练不计算已上课程的课时费 |
| 合同信息填写不完整导致退费 | 顾问 + 店长 | 顾问承担30%退费损失，店长承担50%退费损失 |
| 因教学质量问题退费 | 教练 + 店长 | 按实际情况追责 |

### 3.6 分期政策

| 卡种 | 最多分期数 | 每期金额 | 还款方式 |
|------|-----------|----------|----------|
| 季卡 | 最多2期 | 总价 / 2 | 按月扣款 |
| 半年卡 | 最多2期 | 总价 / 2 | 按月扣款 |
| 年卡 | 最多3期 | 总价 / 3 | 按月扣款 |

> **注意：** 分期付款的学员，退费时只退已收金额，未收部分不再追缴。

### 3.7 未签标准合同处罚

| 情形 | 处罚 |
|------|------|
| 未签订标准合同 | 相关顾问提成为0 |
| 合同信息不完整 | 限期补全，逾期提成为0 |
| 合同未录入GYMGO | 限期补录，逾期提成为0 |

## 四、流程SOP（怎么做）

### 4.1 合同签订流程

```
顾问介绍课程和价格
    │
    ▼
家长确认购买
    │
    ▼
顾问填写合同（使用公司统一版本）
    │
    ├── 检查填写是否完整
    │
    ├── 检查金额计算是否正确
    │
    └── 确认家长签名
    │
    ▼
店长审核签字
    │
    ├── 合格 → 收款 → 财务录入GYMGO（检查全部文件）→ 合同归档
    │
    └── 不合格 → 退回修改 → 重新审核
    │
    ▼
合同原件每周回收财务，照片存在企业微信微盘中
```

### 4.2 退费流程

```
家长提出退费要求
    │
    ▼
顾问记录退费原因
    │
    ▼
教练沟通（第一轮，填写沟通记录表）
    │
    ├── 挽留成功 → 继续上课
    │
    └── 挽留失败
         │
         ▼
        顾问沟通（第二轮，填写沟通记录表）
         │
         ├── 挽留成功 → 继续上课
         │
         └── 挽留失败
              │
              ▼
             店长沟通（第三轮，填写沟通记录表）
              │
              ├── 挽留成功 → 继续上课
              │
              └── 挽留失败
                   │
                   ▼
                  顾问填写退费申请表
                   │
                   ▼
                  店长审核（检查合同、计算退费金额）
                   │
                   ▼
                  总经理审批（3天内完成）
                   │
                  ├── 同意 → 财务执行退费（30个工作日内）
                  │
                  └── 不同意 → 退回，说明原因
                   │
                   ▼
                  财务执行退费
                   │
                   ▼
                  财务在GYMGO更新学员状态（检查全部文件）
                   │
                   ▼
                  退费完成，归档记录（包括三轮沟通记录表）
```

### 4.3 停卡流程

```
家长提出停卡申请 / 顾问/教练发起停卡
    │
    ▼
顾问检查停卡资格（次数/时长）
    │
    ├── 符合 → 填写停卡申请单 → 家长签字 → 店长审批 → 财务在GYMGO操作停卡（检查全部文件）
    │
    └── 不符合 → 向家长说明原因
    │
    ▼
停卡开始，有效期顺延
    │
    ▼
停卡到期 → GYMGO自动恢复 → 通知家长复课
```

### 4.4 转卡流程

```
原会员提出转卡
    │
    ▼
顾问检查转卡条件（已上课时≤50%）
    │
    ├── 符合
    │    │
    │    ▼
    │   填写转卡申请单
    │    │
    │    ▼
    │   原会员签字 + 新会员签字 + 店长签字（三方确认）
    │    │
    │    ▼
    │   财务在GYMGO操作转卡（检查全部文件）
    │    │
    │    ▼
    │   转卡完成
    │
    └── 不符合 → 向原会员说明原因
```

### 4.5 话术模板

#### 挽留话术

> **顾问：** "XX妈妈，了解到您想退费，我非常理解。方便问一下是什么原因吗？是时间安排不过来，还是对课程有什么不满意的地方？"
> （根据原因针对性挽留）

## 五、节点管理与跟踪考核（谁检查、怎么查、做不到怎么办）

### 5.1 检查机制

| 检查项 | 谁检查 | 怎么查 | 频率 | 做不到怎么办 |
|--------|--------|--------|------|-------------|
| 合同完整性 | 财务 | 每月抽查合同台账 vs GYMGO | 每月 | 合同不完整→限期补全 |
| 合同金额准确性 | 财务 | 核对合同金额 vs 实际收款 | 每月 | 金额不一致→追查原因 |
| 退费合规性 | 财务 | 核对退费金额 vs 退费标准 | 每笔退费 | 不合规→追回差额+追责 |
| GYMGO录入及时性 | 店长 | 检查合同签订日期 vs GYMGO录入日期 | 每周 | 未及时录入→顾问书面警告 |
| 分期还款跟踪 | 财务 | 检查还款记录 | 每月 | 逾期未还→顾问跟进 |
| 督导抽查 | 督导 | 合同台账 vs GYMGO vs 实际收款 | 每月1次 | 发现问题→限期整改 |

### 5.2 处罚细则

| 违规行为 | 责任人 | 处罚 |
|----------|--------|------|
| 不使用统一合同版本 | 顾问 | 书面警告 + 乐捐进成长基金（员工10元/店长20元） |
| 私自承诺优惠 | 顾问 | 书面警告 + 乐捐进成长基金（员工10元/店长20元） + 差额自补 |
| 合同信息不完整 | 顾问 | 限期补全，逾期提成为0 |
| 合同未录入GYMGO | 顾问 | 限期补录，逾期提成为0 |
| 退费不按标准执行 | 责任人 | 书面警告 + 差额自补 |
| 合同不完善导致退费 | 顾问 + 店长 | 顾问承担30%退费损失，店长承担50%退费损失 |
| 虚报合同金额（数据造假） | 当事人 | 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退 |
| 伪造家长签名（数据造假） | 当事人 | 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退 |
