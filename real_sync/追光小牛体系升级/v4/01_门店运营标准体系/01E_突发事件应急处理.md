## 一、制度（红线 + 规则）

### 1.1 红线

| 序号 | 红线内容 | 处罚 |
|------|----------|------|
| 1 | 突发事件隐瞒不报 | 立即辞退 |
| 2 | 数据造假（虚报事件经过、伪造记录等） | 首次书面警告 + 乐捐进成长基金（员工10元/店长20元），二次辞退 |
| 3 | 未按应急流程处理导致事态扩大 | 书面警告 + 乐捐进成长基金（员工10元/店长20元），情节严重辞退 |

### 1.2 规则

1. 突发事件发生后，第一时间保障学员安全
2. 按照三级应急响应标准处理，不能擅自降低响应级别
3. 所有突发事件必须记录，不能隐瞒
4. 事件处理完成后必须做复盘，总结经验
5. 涉及学员伤害的事件，24小时内必须回访家长

---

## 二、要求（做什么）

### 2.1 三级应急响应要求

| 级别 | 定义 | 响应要求 |
|------|------|----------|
| 绿色（一般） | 学员轻微擦伤、碰伤，无需就医 | 教练当场处理，记录，课后告知家长 |
| 黄色（较重） | 学员受伤需要就医、家长严重投诉 | 教练+店长处理，1小时内上报总经理，24小时回访 |
| 红色（严重） | 学员严重受伤、安全事故、媒体曝光 | 店长+总经理到场处理，立即启动应急预案 |

### 2.2 伤害处理要求

- 轻微伤害：当场用急救箱处理，课后告知家长
- 需就医伤害：立即联系家长，陪同就医，保留就医记录
- 严重伤害：拨打120，联系家长，上报总经理，保留所有记录

### 2.3 家长沟通要求

- 事件发生后第一时间联系家长，说明情况
- 态度诚恳，不推卸责任
- 24小时内回访，确认学员恢复情况

## 三、标准（做到什么程度）

### 3.1 应急响应时效标准

| 事件级别 | 发现后响应 | 上报时效 | 回访时效 | 复盘时效 |
|----------|-----------|----------|----------|----------|
| 绿色 | 立即 | 当日报店长 | 当日 | 3个工作日内 |
| 黄色 | 立即 | 1小时内报总经理 | 24小时内 | 3个工作日内 |
| 红色 | 立即 | 立即报总经理 | 24小时内 | 1个工作日内 |

### 3.2 伤害处理标准

| 伤害程度 | 处理标准 |
|----------|----------|
| 轻微（擦伤、碰伤） | 急救箱处理，冰敷/消毒/贴创可贴 |
| 中度（扭伤、淤青） | 冰敷+固定，建议就医，联系家长 |
| 重度（骨折、昏迷） | 拨打120，不移动学员，联系家长，上报总经理 |

### 3.3 家长沟通标准

| 沟通环节 | 标准 |
|----------|------|
| 首次沟通 | 事件发生后10分钟内联系家长，说明情况 |
| 沟通态度 | 诚恳道歉，不推卸责任，说明处理措施 |
| 跟进沟通 | 24小时内回访，确认学员恢复情况 |
| 后续跟进 | 伤愈后再次沟通，表达关心 |

### 3.4 常见突发事件处理标准

| 事件类型 | 处理标准 |
|----------|----------|
| 学员受伤 | 按伤害程度分级处理 |
| 学员走失 | 立即封锁出口，全员寻找，10分钟内通知家长 |
| 家长冲突 | 店长出面，带至单独区域沟通，避免影响其他学员 |
| 设备故障 | 停止使用，报修，调整课程安排 |
| 停电/停水 | 启动应急照明，安抚学员，联系物业 |
| 火灾/地震 | 按消防预案疏散，确保全员安全撤离 |

## 四、流程SOP（怎么做）

### 4.1 学员受伤处理流程

```
发现学员受伤
    │
    ▼
立即停止该学员活动
    │
    ▼
判断伤害程度
    │
    ├── 轻微（擦伤/碰伤）
    │    │
    │    ▼
    │   急救箱处理
    │    │
    │    ▼
    │   课后告知家长
    │    │
    │    ▼
    │   记录到突发事件记录表（绿色）
    │
    ├── 中度（扭伤/淤青/需就医）
    │    │
    │    ▼
    │   立即联系家长
    │    │
    │    ▼
    │   陪同就医（如家长未到）
    │    │
    │    ▼
    │   保留就医记录
    │    │
    │    ▼
    │   1小时内上报总经理
    │    │
    │    ▼
    │   24小时内回访家长
    │    │
    │    ▼
    │   记录到突发事件记录表（黄色）
    │
    └── 重度（骨折/昏迷）
         │
         ▼
        拨打120
         │
         ▼
        不移动学员，等待急救
         │
         ▼
        立即联系家长
         │
         ▼
        立即上报总经理
         │
         ▼
        保留所有记录（照片、视频、目击者信息）
         │
         ▼
        记录到突发事件记录表（红色）
```

### 4.2 家长冲突处理流程

```
家长情绪激动/发生冲突
    │
    ▼
店长立即出面
    │
    ▼
带家长至单独区域（不影响其他学员）
    │
    ▼
倾听家长诉求，不打断
    │
    ▼
诚恳道歉，不推卸责任
    │
    ▼
提出解决方案
    │
    ├── 家长接受 → 记录 → 跟进落实
    │
    └── 家长不接受
         │
         ▼
        上报总经理
         │
         ▼
        总经理介入处理
```

### 4.3 事件复盘流程

```
事件处理完成
    │
    ▼
3个工作日内组织复盘（黄色以上事件）
    │
    ▼
复盘内容：
├── 事件经过回顾
├── 处理过程评估
├── 做得好的地方
├── 需要改进的地方
└── 改进措施和责任人
    │
    ▼
形成复盘报告
    │
    ▼
全员学习，防止再次发生
```

### 4.4 话术模板

#### 轻微伤害告知家长话术

> **教练：** "XX妈妈，今天上课的时候XX不小心擦了一下手，我已经用碘伏消毒贴了创可贴，伤口很小，您不用担心。回去注意不要沾水，明天应该就好了。"

#### 中度伤害联系家长话术

> **教练：** "XX妈妈，XX今天上课的时候脚扭了一下，有些肿。我们做了冰敷处理，建议您带他去医院拍个片子确认一下。您现在方便过来吗？我陪您一起去。"

#### 家长冲突处理话术

> **店长：** "XX妈妈，非常抱歉给您带来了不好的体验。我理解您的心情，您先坐下，慢慢说，我仔细听。这件事确实是我们的问题，我们会这样处理...您看这样行吗？"

## 五、节点管理与跟踪考核（谁检查、怎么查、做不到怎么办）

### 5.1 检查机制

| 检查项 | 谁检查 | 怎么查 | 频率 | 做不到怎么办 |
|--------|--------|--------|------|-------------|
| 事件记录完整性 | 店长 | 查看突发事件记录表 | 每次事件后 | 记录不全→补全+口头提醒 |
| 黄色以上事件复盘 | 店长 | 查看复盘报告 | 每次黄色以上事件 | 未做复盘→店长书面警告 |
| 24小时回访 | 店长 | 查看回访记录 | 每次黄色以上事件 | 未做回访→书面警告 |
| 上报时效 | 督导 | 查看上报记录 | 每次黄色以上事件 | 未按时上报总经理→店长书面警告 |
| 隐瞒不报 | 督导/财务 | 多渠道核实 | 每月 | 隐瞒不报→立即辞退 |
| 急救箱检查 | 店长 | 开箱检查 | 每周 | 药品不足→立即补充 |

### 5.2 处罚细则

| 违规行为 | 责任人 | 处罚 |
|----------|--------|------|
| 隐瞒不报突发事件 | 当事人 | 立即辞退 |
| 未按时上报总经理（黄色以上） | 店长 | 书面警告 |
| 未做24小时回访 | 责任人 | 书面警告 |
| 未做事件复盘（黄色以上） | 店长 | 书面警告 |
| 未按应急流程处理 | 责任人 | 书面警告 + 乐捐进成长基金（员工10元/店长20元） |
| 虚报事件经过（数据造假） | 当事人 | 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退 |
