## 一、制度（红线 + 规则）

### 1.1 红线

| 序号 | 红线内容 | 处罚 |
|------|----------|------|
| 1 | 私拿公司器材/物料 | 赔偿 + 书面警告 |
| 2 | 故意损坏器材 | 赔偿 + 书面警告，情节严重辞退 |
| 3 | 数据造假（虚报盘点数据、虚报损耗等） | 首次书面警告 + 乐捐进成长基金（员工10元/店长20元），二次辞退 |
| 4 | 器材损坏不报修导致安全事故 | 书面警告 + 乐捐进成长基金（员工10元/店长20元） + 追究责任 |

### 1.2 规则

1. 所有器材和物料归公司所有，个人不得私拿
2. 器材使用后必须归位，保持整齐
3. 发现器材损坏必须当天报修，不能拖延
4. 耗材使用按需领取，不能浪费
5. 每月25日进行月度盘点，确保账物一致

---

## 二、要求（做什么）

### 2.1 资产管理要求

- 建立门店资产台账，记录所有器材和物料
- 新购器材/物料入库时及时登记
- 报废器材/物料须经店长审批

### 2.2 器材管理要求

- 每日使用前检查器材是否完好
- 使用后归位到指定位置
- 损坏器材当天报修

### 2.3 耗材管理要求

- 耗材按需领取，登记使用量
- 库存不足时提前3天申请采购
- 不得私拿或浪费

## 三、标准（做到什么程度）

### 3.1 资产分类标准

| 类别 | 包含内容 | 管理方式 |
|------|----------|----------|
| 大型器材 | 跑步机、平衡木、跳箱等 | 固定位置，每月检查 |
| 小型器材 | 呼啦圈、小球、标志桶等 | 使用后归位，每周检查 |
| 电子设备 | 电脑、音响、投影仪等 | 专人管理，每月检查 |
| 办公耗材 | 纸巾、洗手液、打印纸等 | 按需领取，库存预警 |
| 教学耗材 | 奖品、贴纸、证书等 | 按需领取，库存预警 |
| 清洁用品 | 消毒液、清洁剂、垃圾袋等 | 按需领取，库存预警 |

### 3.2 器材管理标准

| 管理项 | 标准 |
|--------|------|
| 摆放位置 | 按器材清单指定位置摆放 |
| 使用检查 | 每次使用前检查是否完好 |
| 归位要求 | 使用后10分钟内归位 |
| 报修时效 | 发现损坏当天报修 |
| 维修时效 | 一般维修3个工作日内完成 |
| 报废标准 | 无法维修或维修费用超过原价50% |

### 3.3 耗材管理标准

| 管理项 | 标准 |
|--------|------|
| 领取方式 | 登记领取，记录使用人和用途 |
| 库存预警 | 低于最低库存量时提前3天申请 |
| 存放要求 | 分类存放，标识清晰 |
| 盘点频率 | 每月25日盘点 |

### 3.4 最低库存量参考

| 耗材名称 | 最低库存 | 采购周期 |
|----------|----------|----------|
| 纸巾 | 10包 | 3天 |
| 洗手液 | 5瓶 | 3天 |
| 垃圾袋 | 2卷 | 3天 |
| 75%酒精 | 2瓶 | 5天 |
| 84消毒液 | 1瓶 | 5天 |
| 打印纸 | 2包 | 5天 |
| 奖品/贴纸 | 按需 | 7天 |

## 四、流程SOP（怎么做）

### 4.1 器材报修流程

```
发现器材损坏
    │
    ▼
停止使用，移至维修区
    │
    ▼
填写报修登记表
    │
    ▼
报店长审批
    │
    ├── 能自行维修 → 店长安排维修 → 维修完成 → 恢复使用
    │
    └── 需外部维修
         │
         ├── 店长联系供应商/维修方
         │
         └── 维修完成 → 验收 → 恢复使用
    │
    ▼
更新资产台账
```

### 4.2 耗材领取流程

```
需要耗材
    │
    ▼
检查库存是否充足
    │
    ├── 充足 → 登记领取 → 使用
    │
    └── 不足
         │
         ├── 低于最低库存 → 填写采购申请
         │
         └── 报店长审批 → 采购 → 入库登记
    │
    ▼
使用后记录消耗量
```

### 4.3 月度盘点流程

```
每月25日
    │
    ▼
店长组织盘点
    │
    ▼
按资产台账逐项清点实物
    │
    ├── 账物一致 → 盘点完成 → 签字确认
    │
    └── 账物不一致
         │
         ├── 查明原因（报修未登记/损耗/丢失）
         │
         └── 处理（补登记/报损/追责）
    │
    ▼
盘点报告报督导/财务
```

### 4.4 话术模板

#### 报修话术

> **教练：** "店长，我在课前检查时发现XX器材的XX部分坏了，已经停止使用，填了报修登记表，请审批。"
> **店长：** "好的，我看一下。这个需要外部维修，我联系供应商，预计3天内修好。这期间用XX替代。"

#### 耗材申请话术

> **教练：** "店长，洗手液只剩1瓶了，低于最低库存，需要申请采购。"
> **店长：** "好的，我记录一下，今天统一采购。"

## 五、节点管理与跟踪考核（谁检查、怎么查、做不到怎么办）

### 5.1 检查机制

| 检查项 | 谁检查 | 怎么查 | 频率 | 做不到怎么办 |
|--------|--------|--------|------|-------------|
| 器材归位 | 店长 | 巡店检查 | 每日 | 未归位→教练口头提醒 |
| 报修及时性 | 店长 | 查看报修登记表 | 每周 | 未按时报修→教练书面警告 |
| 耗材使用 | 店长 | 查看领取记录 | 每周 | 私拿物料→赔偿+书面警告 |
| 月度盘点 | 店长 | 库存台账 vs 实物 | 每月25日 | 未盘点→店长书面警告 |
| 盘点差异 | 店长+财务 | 查明原因 | 每月 | 有差异→查原因+追责 |
| 督导抽查 | 督导 | 现场抽查 | 每月1次 | 不合格→限期整改 |

### 5.2 处罚细则

| 违规行为 | 责任人 | 处罚 |
|----------|--------|------|
| 未按时报修 | 教练 | 书面警告 |
| 私拿公司物料 | 当事人 | 赔偿 + 书面警告 |
| 故意损坏器材 | 当事人 | 赔偿 + 书面警告 |
| 未做月度盘点 | 店长 | 书面警告 |
| 盘点差异未查明 | 店长 | 书面警告 |
| 虚报盘点数据（数据造假） | 当事人 | 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退 |
