# 追光小牛体系文件 -- 执行控制表单包

| 项目 | 内容 |
|------|------|
| 文件编号 | 表单包 |
| 版本 | v1.0 |
| 生效日期 | 2026年4月 |
| 适用范围 | 总经理、督导、店长、教练、顾问 |
| 编制部门 | 总经办 |

> 目的：将执行、监督、升级、整改、验收相关表单单独成册，便于门店、督导、总经理直接查找和使用。
> 使用原则：主文件只保留制度和规则，本文件集中放控制类表单，并尽量采用可直接打印填写的版式。

## 十、使用要求

1. 主文件负责制度和规则，本文件负责控制表单，不混放。
2. 店长每周至少更新一次红黄绿看板、风险会员升级台账、关键节点漏执行升级表。
3. 督导每次巡店后必须更新整改销项复查表和督导复查验收表。
4. 总经理每周固定查看周推进验收表，不只听汇报，要看表单证据。
5. 培训未抽测、抽测未通过、未验收通过的人员，不得视为已完成制度上线。
