# 追光小牛 · 成长基金管理办法

> **版本**：v1.1 | **日期**：2026年4月
> **适用**：全体员工 | **主责**：财务（管理）、店长（执行）、总经理（审批）

---

## 一、制度

### 1.1 目的

成长基金是追光小牛的**公司级奖罚机制**，通过乐捐（违规扣款）形成奖金池，再通过奖励回馈优秀员工，形成"违规者出资、优秀者受益"的正向循环。

### 1.2 基金性质

| 项目 | 说明 |
|------|------|
| 基金来源 | 全体员工的各类乐捐（违规扣款） |
| 基金用途 | 周奖励、月度明星奖励、其他公司认可的奖励 |
| 基金管理 | 财务统一管理，专款专用 |
| 基金归属 | 公司所有，不退还个人 |

### 1.3 管理边界

| 类别 | 是否由成长基金负责 | 说明 |
|------|------------------|------|
| 违规乐捐 | 是 | 统一承接教练/顾问/店长的轻度违规处罚 |
| 正向奖励 | 是 | 承接周奖励、月度荣誉等奖金池奖励 |
| 业绩提成 | 否 | 由薪酬结构和激励方案负责 |
| 工作量达标 | 否 | 由 02G 负责动作管理，成长基金只承接对应乐捐 |
| 严重红线辞退 | 否 | 由总纲直接处理，不进入成长基金 |

### 1.4 红线

| 红线 | 违反后果 |
|------|---------|
| 乐捐必须如实记录，不得隐瞒 | 隐瞒→店长书面警告 |
| 奖金发放必须公开透明 | 暗箱操作→店长书面警告 |
| 基金不得挪作他用 | 挪用→总经理约谈+追回 |

## 二、要求

### 2.1 乐捐标准（统一）

| 违规情形 | 乐捐金额 | 乐捐对象 | 说明 |
|---------|---------|---------|------|
| 工作量连续2天未达成 | 20元/个差额 | 员工 | 按累计差额计算（详见02G） |
| 工作量连续3天未达成（店长责任） | 50元/人 | 店长 | 店长跟进赋能不及时 |
| 红线违规（一般） | 10元/次 | 员工 | 书面警告+乐捐 |
| 红线违规（店长） | 20元/次 | 店长 | 书面警告+乐捐 |
| 数据造假 | 按情节严重程度 | 员工/店长 | 首次书面警告+乐捐，二次辞退 |

> **说明**：员工统一乐捐10元/次，店长统一乐捐20元/次。工作量相关乐捐按02G标准执行。

### 2.2 奖励标准

| 奖励项目 | 频率 | 奖金来源 | 奖励规则 |
|---------|------|---------|---------|
| 周工作量最佳 | 每周 | 成长基金奖金池 | 第1名60%/第2名30%/第3名10% |
| 月度荣誉-个人之星 | 每月 | 成长基金奖金池 | 销售/续费/教学之星与周奖励合并；服务/进步之星各100元 |
| 月度荣誉-团队 | 每月 | 成长基金奖金池 | 冠军团队500元/服务标杆店300元/进步最大店300元 |
| 月度荣誉-专项 | 每月 | 成长基金奖金池 | 最佳分享者50元/创新之星100元/安全卫士50元 |
| 月度明星老师 | 每月 | 公司额外预算 | 校区明星老师称号+形象墙+惊喜礼品 |
| 其他奖励 | 不定期 | 成长基金奖金池或公司预算 | 由总经理审批 |

### 2.3 乐捐执行要求

| 要求 | 说明 |
|------|------|
| 及时记录 | 乐捐发生后24小时内记录到成长基金台账 |
| 告知当事人 | 乐捐前必须告知当事人违规事实和乐捐金额 |
| 不得代缴 | 乐捐必须由当事人自行承担，不得由他人代缴 |
| 不得预支 | 奖金池余额不足时，奖励顺延至下周/下月 |

### 3.2 奖金池管理标准

| 项目 | 标准 |
|------|------|
| 奖金池起始金额 | 每月1日归零，重新累计 |
| 周奖励发放时间 | 每周一公布，周内发放 |
| 周奖励计算 | 上周乐捐总额×分配比例（60%/30%/10%） |
| 奖金池不足时 | 奖励顺延，不补发 |
| 奖金池结余 | 月末结余转入下月，不退还个人 |

## 四、流程SOP

### 4.1 乐捐执行流程

```
Step 1 → 发现违规（店长/督导/财务）
  └→ 确认违规事实
  └→ 对照乐捐标准确定金额

Step 2 → 告知当事人（店长）
  └→ 口头告知违规事实和乐捐金额
  └→ 当事人确认

Step 3 → 记录台账（店长）
  └→ 24小时内记录到成长基金台账
  └→ 包含：日期、姓名、违规事项、乐捐金额

Step 4 → 汇总上报（店长→财务）
  └→ 每周五汇总本周乐捐总额
  └→ 上报财务
```

### 4.2 周奖励发放流程

```
每周一 → 店长统计上周工作量完成情况
  └→ 确定前三名
  └→ 计算奖金池金额和分配
  └→ 公布获奖名单
  └→ 提交财务
  └→ 财务周内发放
```

### 4.3 月度明星评选流程

```
每月最后一天 → 店长统计本月工作量完成总量
  └→ 评选月度最高工作量完成者
  └→ 提交总经理审批
  └→ 校区形象墙更新
  └→ 准备月度礼品（公司预算）
```

## 五、节点管理与跟踪考核

### 5.1 节点管理

| 检查项 | 谁检查 | 怎么查 | 频率 | 做不到怎么办 |
|--------|--------|--------|------|-------------|
| 乐捐记录及时性 | 督导 | 台账核查 | 每周 | 未及时记录→店长口头提醒 |
| 乐捐金额准确性 | 财务 | 台账核对 | 每周 | 金额错误→更正+店长口头提醒 |
| 奖金发放及时性 | 督导 | 发放记录 | 每周 | 未按时发放→财务书面警告 |
| 奖金池透明度 | 总经理 | 月度报告 | 每月 | 不透明→财务书面警告 |
