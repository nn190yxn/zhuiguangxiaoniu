# 08 体系推进计划

**版本：** v2.4（5周冲刺版）
**适用对象：** 总经理、督导、店长、全体员工
**归口部门：** 总经办
**截止日期：** 2026年5月18日（周年庆5月25日前1周）

---

## 一、推进原则

### 1.1 核心原则

| 原则 | 说明 |
|------|------|
| 简单先行 | 先做最基础的、最容易落地的，不要一开始就追求完美 |
| 每周有结果 | 每周必须看到具体的执行成果，哪怕很小 |
| 有人负责 | 每件事必须有明确的责任人，不能"大家负责=没人负责" |
| 有人检查 | 布置了不检查等于没布置，必须有检查和反馈 |
| 周年庆优先 | 所有调整必须围绕周年庆（5月25日）展开，确保周年庆前核心流程跑通 |

### 1.2 推进心态

- 不追求一步到位，允许试错和调整
- 先让制度跑起来，再在执行中优化
- 遇到问题及时反馈，不要等到月底才说做不了
- 店长是第一责任人，必须带头执行
- 团队基础薄弱，培训要手把手、反复讲

### 1.3 时间约束

> **关键背景**：2026年销售目标近800万，压力非常大。4月13日距5月15日仅5周，必须在这5周内完成周年庆前所有核心调整。ACE评估、教学SOP、品牌标准等非紧急内容延至周年庆后（6-8月）推进。

## 四、每周推进标准

### 4.1 每周节奏

| 时间 | 动作 | 责任人 | 产出 |
|------|------|--------|------|
| 周一 | 总经理发布本周任务清单（具体到人、具体到事） | 总经理 | 本周任务清单 |
| 周三 | 督导检查执行进度，群内通报 | 督导 | 进度通报 |
| 周五 | 店长提交本周执行情况（3句话） | 店长 | 周执行汇报 |
| 周日 | 总经理+督导复盘本周，调整下周计划 | 总经理+督导 | 下周调整计划 |

### 4.2 周执行汇报模板（店长填写）

```
1. 做了什么：（本周完成的3件最重要的事）
2. 遇到什么问题：（1-2个主要困难）
3. 需要什么支持：（需要总部/督导提供什么帮助）
```

### 4.3 推进沟通机制

| 沟通方式 | 频次 | 参与人 | 内容 |
|----------|------|--------|------|
| 体系推进群 | 每日 | 全体管理层 | 每日进展+问题反馈 |
| 周度推进会 | 每周一 | 总经理+督导+店长 | 本周任务+上周复盘 |
| 5周冲刺总结会 | 5月15日 | 总经理+督导+店长 | 阶段验收+周年庆准备 |
