# 05D_教练与助教上课执行标准

## 1. 文件定位

本标准用于统一追光小牛课堂中主教行为、助教行为和主助协同方式。

本文件重点解决：

1. 一堂课里谁做什么才算合规。
2. 新员工、实习员工、正式员工在课堂中的授权边界是什么。
3. 主教和助教如何配合，避免职责重叠、遗漏或失控。

## 2. 适用范围

本标准适用于：

1. 所有正式课程。
2. 体验课、展示课、复训课。
3. 新员工跟岗、实习和放行前课堂。

## 3. 角色定义

### 3.1 主教行为

主教行为是指对当堂课程结果负责的课堂主责行为，包括：

1. 明确课堂目标和节奏。
2. 主导教学组织、动作示范、口令发起和课堂转换。
3. 对学员安全、课堂秩序和家长感知负责。
4. 对课后记录、问题反馈和教学结果负责。

### 3.2 助教行为

助教行为是指围绕主教教学目标，协助完成组织、安全、照护、器材和个别支持的课堂协同行为，包括：

1. 协助课堂组织。
2. 协助安全保护。
3. 协助器材准备与切换。
4. 协助个别学员情绪、秩序和动作纠偏。

说明：助教优先定义为课堂角色行为，不默认等同于固定岗位。

## 4. 课堂授权边界

### 4.0 授权原则

1. 先看能力，再给角色，不因人员紧张直接放宽授权。
2. 先能稳定协助，再能独立承担局部动作，最后才考虑承担主责。
3. 同一名员工是否能承担主教行为，以当堂放行状态为准，不以岗位名称自动推定。

### 4.1 实习员工 / 跟岗员工

允许承担：

1. 课前器材准备、场地整理。
2. 学员点名、排队、引导集合。
3. 在主教明确安排下做低风险动作辅助。
4. 课后收尾和基础记录协助。

不得单独承担：

1. 独立带完整课堂流程。
2. 独立执行高风险动作保护。
3. 独立处理受伤、失控、离场等异常事件。
4. 独立向家长输出带结论的专业反馈。

### 4.2 正式员工但非当堂主责人

可在其他课堂承担助教行为，但应遵守：

1. 以当堂主教安排为准。
2. 不抢主口令，不抢节奏，不多头指挥。
3. 发现风险时先保护、再提示主教，不得放任。

### 4.3 当堂主教

必须承担：

1. 目标说明和课堂开场。
2. 动作示范与口令主导。
3. 风险动作前的安全确认。
4. 异常情况决策与升级。
5. 课后结果确认和家长主反馈。

### 4.4 放行判断逻辑

员工在被允许承担更高一级课堂职责前，至少应满足：

1. 知道当堂目标和流程，不会跟丢节奏。
2. 知道自己该站哪里、什么时候补位、什么时候不能插口。
3. 遇到异常时知道先保护、先汇报、先控风险。
4. 已经在低风险和中风险课堂中稳定执行，不靠临场提醒才能完成。

## 5. 课堂阶段分工标准

### 5.1 开场集合阶段

主教负责：

1. 建立课堂注意力。
2. 明确当堂规则、目标和开始口令。
3. 观察整体状态，决定是否调整节奏。

助教负责：

1. 补队形、整队列、控掉队学员。
2. 协助点名和入场秩序。
3. 对迟到、情绪波动或明显分心学员做近身稳定。

### 5.2 热身阶段

主教负责：

1. 统一发热身口令和节奏。
2. 控制热身强度和转换速度。
3. 判断是否有学员不适合进入后续主体环节。

助教负责：

1. 纠正明显掉拍、掉队或危险站位。
2. 处理边缘学员和慢反应学员。
3. 协助维持动作间距，避免碰撞。

### 5.3 主体教学阶段

主教负责：

1. 做重点动作示范和规则说明。
2. 决定动作分组、轮换和进阶降阶。
3. 盯高风险动作、高价值纠错和课堂完成度。

助教负责：

1. 负责轮换秩序和队列推进。
2. 在指定位置做保护和近身辅助。
3. 对单个学员做重复提醒和局部纠偏。

### 5.4 游戏 / 对抗阶段

主教负责：

1. 讲清规则、边界和输赢标准。
2. 控整体兴奋度和公平性。
3. 必要时中断并重建规则。

助教负责：

1. 看边线、器材区和冲撞高发点。
2. 快速处理插队、争抢、脱队等局部问题。
3. 在不打断主节奏的前提下做秩序补位。

### 5.5 收尾整理阶段

主教负责：

1. 总结当堂表现和重点进步点。
2. 明确课后交接和家长沟通重点。

助教负责：

1. 协助学员归队、整理器材、恢复秩序。
2. 补充异常信息和观察信息给主教。

## 6. 课前执行标准

### 6.1 主教课前动作

1. 明确当堂课程目标、重点动作和节奏切换点。
2. 明确分组方式、风险点和重点学员。
3. 明确助教站位和分工。
4. 课前检查器材、场地和学员状态。

### 6.2 助教课前动作

1. 按要求布置器材和课堂动线。
2. 协助点名、迎接、鞋服和入场秩序检查。
3. 向主教反馈学员异常状态，如情绪不稳、身体不适、家长临时说明。

### 6.3 课前禁止行为

1. 未经确认擅自改动作顺序。
2. 器材未摆完就让学员自行进入高风险区域。
3. 主教和助教都不清楚本堂分工就直接开课。

## 7. 课中执行标准

### 7.1 主教核心职责

1. 讲目标：让学员知道当前任务是什么。
2. 发口令：所有关键转换口令由主教统一发起。
3. 做示范：动作标准、节奏标准、规则标准由主教统一示范。
4. 控节奏：热身、主体、游戏、整理环节切换明确。
5. 抓重点：把注意力放在高风险动作、关键学员和课堂效果上。

### 7.2 助教核心职责

1. 抓队形：维持排队、轮换和等待秩序。
2. 抓安全：站在风险点、掉队点和容易失控的位置。
3. 抓个别：对慢半拍、情绪波动、动作偏差较大的学员做近身辅助。
4. 抓器材：及时完成器材切换、回收和重置。
5. 抓补位：主教示范时协助照看全体学员。

### 7.3 主助协同要求

1. 一个口令源：关键指令以主教口令为准。
2. 一个课堂节奏：助教补位，不额外开新节奏。
3. 一个安全标准：风险动作前，主教确认，助教到位。
4. 一个反馈链路：助教先向主教反馈，再统一处理。

### 7.4 角色切换要求

同一名员工在不同课堂承担不同角色时，应遵守：

1. 进入新课堂前先清楚自己本堂是主教还是助教。
2. 一旦本堂被定义为助教，就不能按自己习惯改主节奏。
3. 即便个人资历更高，只要不是本堂主责人，也应服从当堂主教安排。
4. 若发现主教遗漏明显安全风险，应先补保护，再即时提醒，不可旁观。

### 7.5 课堂站位要求

1. 主教站在能看全场、能做示范、能控制节奏的位置。
2. 助教站在高风险区域、后排学员、器材切换口或掉队点。
3. 高风险动作或多学员并发练习时，不得出现风险区域无人看护。

### 7.6 个别学员支持要求

1. 对情绪波动学员，助教优先近身稳定，主教保持课堂总节奏。
2. 对动作明显落后学员，助教先做局部辅助，必要时主教决定是否降阶。
3. 对过度兴奋学员，主教负责重建规则，助教负责贴身控位。

## 8. 课后执行标准

### 8.1 主教课后动作

1. 做课堂简要复盘，确认本堂目标完成情况。
2. 完成主记录或确认课后记录内容。
3. 向家长输出本堂核心反馈和后续建议。
4. 对异常学员、重点学员做必要交接。

### 8.2 助教课后动作

1. 协助器材归位、场地恢复。
2. 协助整理出勤、课堂表现、异常事件基础信息。
3. 向主教反馈自己观察到的学员状态和问题点。

## 9. 家长沟通边界

### 9.1 主教负责

1. 对课堂效果作结论性反馈。
2. 对动作表现、学习进度、后续建议做主说明。
3. 对异常事件做正式说明和安抚。

### 9.2 助教可做

1. 做事实型反馈，如出勤、配合度、情绪表现。
2. 在主教在场或已授权前提下补充观察信息。

### 9.3 助教不得做

1. 擅自承诺升班、疗效、训练结果。
2. 擅自评价孩子问题成因或专业诊断。
3. 在未与主教同步前直接输出异常事件结论。

### 9.4 沟通顺序要求

1. 先内部同步，再对家长表达。
2. 先讲事实，再讲建议，不先讲判断。
3. 遇到异常、投诉、质疑时，先由主教主说，助教不抢答。

## 10. 禁止行为与常见违规行为

### 10.1 主教禁止行为

1. 把全场交给助教，自己只做讲解不看风险点。
2. 未明确分工就开课。
3. 把高风险动作默认交给实习员工协助兜底。
4. 对异常事件不决策、不上报、只想先把课上完。

### 10.2 助教禁止行为

1. 擅自改变主教节奏。
2. 同时发主口令、另起一套规则。
3. 在风险动作中背身、离岗、搬器材。
4. 对学员异常不汇报，私自处理后不留信息。

### 10.3 常见违规行为

1. 主教只讲不看场，风险区域失管。
2. 助教只搬器材，不看学员。
3. 两个人同时发口令，导致课堂混乱。
4. 实习员工被默认独立顶岗。
5. 高风险动作无人保护或保护站位错误。
6. 课后无人记录异常事件。

## 11. 日常检查重点

店长、主管或带教人在检查本文件执行情况时，至少看：

1. 当堂主教是谁，是否真正承担主责。
2. 助教是否真正在补安全、补秩序、补个别支持。
3. 实习员工是否越权。
4. 课前分工是否明确。
5. 课后记录和家长反馈是否清晰。

## 12. 检查与追责

1. 店长或主管按巡课标准检查本文件执行情况。
2. 首次偏差以现场纠偏和复训为主。
3. 再次发生或涉及安全问题的，纳入专项整改。
4. 因违反本标准造成课堂失控或安全事故的，按门店管理机制追责。

## 13. 一句话结论

本标准不是把“助教”固化成岗位，而是把课堂中的主教行为、助教行为、授权边界和协同动作标准化，确保每堂课有人主责、有人补位、有人看安全、有人做收口。
