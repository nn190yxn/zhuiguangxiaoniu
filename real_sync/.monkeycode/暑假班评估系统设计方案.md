# 暑假班评估系统设计方案

## 一、需求概述

### 1.1 背景
- 暑假班6-8月运营，需要针对性评估支持教练复盘和家长沟通
- 现有体测评估系统（fitness-assessment）成熟，可复用架构
- 新增5个主题营：中考体训、体能达标、跳绳达标、篮球体能、驼背体态调整

### 1.2 目标
- 教练录入学员测试数据 → 生成评估报告 → 家长理解问题和阶段训练目标
- 复用体测评估的技术架构，降低开发成本
- 5个营各有不同的测试项目、评分标准、提示词模板

---

## 四、后端API设计

### 4.1 扩展AI服务（api/ai-services.php）

**新增action：summer_camp_report**

```php
if ($action === 'summer_camp_report') {
    $campType = trim((string) ($payload['camp_type'] ?? ''));
    $testData = (array) ($payload['test_data'] ?? []);
    $studentInfo = (array) ($payload['student_info'] ?? []);
    $coachDiagnosis = (array) ($payload['coach_diagnosis'] ?? []);
    
    if ($campType === '' || empty($testData)) {
        throw new InvalidArgumentException('缺少营类型或测试数据');
    }
    
    // 加载对应营的提示词模板
    $promptTemplate = ai_load_camp_prompt_template($campType);
    
    // 替换占位符（{name}, {grade}, {jump_rope}等）
    $filledPrompt = ai_fill_prompt_template($promptTemplate, $studentInfo, $testData, $coachDiagnosis);
    
    // 调用DeepSeek生成报告
    $systemPrompt = ai_get_camp_system_prompt($campType);
    $content = ai_deepseek_chat($filledPrompt, $systemPrompt, 3000, 0.7);
    
    echo json_encode(array('content' => $content), JSON_UNESCAPED_UNICODE | JSON_UNESCAPED_SLASHES);
    exit;
}
```

**提示词模板管理**：
```php
function ai_load_camp_prompt_template(string $campType): string {
    // 从数据库或文件加载对应营的提示词模板
    $templates = [
        'zhongkao' => '中考体训达标营提示词（见AI评估报告提示词模板.md）',
        'tineng' => '体能达标营提示词',
        'tiaosheng' => '跳绳达标营提示词',
        'lanqiu' => '篮球体能营提示词',
        'tuobei' => '驼背体态调整营提示词'
    ];
    return $templates[$campType] ?? '';
}

function ai_fill_prompt_template(string $template, array $studentInfo, array $testData, array $coachDiagnosis): string {
    // 替换占位符：{name}, {grade}, {gender}, {height}, {weight}等
    $replacements = array_merge($studentInfo, $testData, $coachDiagnosis);
    foreach ($replacements as $key => $value) {
        $template = str_replace('{' . $key . '}', (string) $value, $template);
    }
    return $template;
}
```

    INDEX idx_store_id (store_id),

    coach_name VARCHAR(100) COMMENT '教练姓名',
    coach_phone VARCHAR(20) COMMENT '教练电话',
    coach_store VARCHAR(100) COMMENT '教练门店',
    report_date DATE COMMENT '报告生成日期',
    created_at DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_assessment_id (assessment_id),
    FOREIGN KEY (assessment_id) REFERENCES summer_camp_assessments(id) ON DELETE CASCADE

**评分函数**：
```javascript
function getRating(campType, metricCode, grade, gender, value) {
  const rules = campConfigs[campType].gradingRules[metricCode];
  const gradeRules = rules.grade_levels[grade];
  
  for (const [ratingName, criteria] of Object.entries(gradeRules)) {
    if (criteria.min !== undefined && value >= criteria.min) {
      return { label: ratingName, color: criteria.color || '' };
    }
    if (criteria.max !== undefined && value <= criteria.max) {
      return { label: ratingName, color: criteria.color || '' };
    }
    if (criteria.min !== undefined && criteria.max !== undefined && 
        value >= criteria.min && value <= criteria.max) {
      return { label: ratingName, color: criteria.color || '' };
    }
  }
  return { label: '未评级', color: '' };
}
```

## 八、风险与注意事项

### 8.1 技术风险
- **AI提示词质量**：需要测试每个营的提示词，确保生成报告符合预期
- **评分标准准确性**：需要核对评估表中的评分标准，避免评级错误
- **OCR识别准确性**：不同营的测试项目不同，OCR需要识别不同的数据字段

### 8.2 业务风险
- **教练使用门槛**：需要简化操作流程，避免教练不愿意使用
- **家长接受度**：报告内容需要通俗易懂，避免家长看不懂专业术语
- **沟通闭环效果**：需要设计明确的复盘路径，避免评估后缺少后续训练建议

### 8.3 数据风险
- **评估记录隐私**：学员数据需要权限控制，避免泄露
- **AI报告质量**：需要人工审核机制，避免AI生成错误建议
