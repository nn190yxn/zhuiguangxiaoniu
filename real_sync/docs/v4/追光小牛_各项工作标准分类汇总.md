### 2.4 教练专业技能标准（三层能力模型）

| 层级 | 能力描述 |

## 三、助教工作标准

> **来源**: 喂养资料(追光小牛助教上课标准.docx)
> **状态**: 体系v4完全缺失助教岗位定义，喂养资料已有详细标准，**急需整合进体系**

### 3.1 助教课前标准

| 序号 | 工作内容 | 标准要求 |
|------|---------|---------|
| 1 | 沟通教案 | 主动与主教沟通教案，协助准备教具 |
| 2 | 熟悉学员 | 熟悉学员名字，了解学员运动情况及性格 |
| 3 | 提前到馆 | 陪提前到场馆的小朋友玩游戏，维持秩序与安全 |

### 3.2 助教课中标准

#### 3.2.1 动作教学辅助

| 场景 | 标准要求 |
|------|---------|
| 动作示范 | 位于主教旁进行侧面示范，或跪于学员旁进行辅助动作讲解 |
| 动作纠错 | 口头提醒未改正的，需让学员返回重做 |
| 动作引导 | 循环训练中帮助主教完成动作引导，不应出现停顿时间过长 |
| 动作保护 | 危险动作及危险器材使用进行对应保护（体操课程按保护手法规定执行） |

#### 3.2.2 秩序维持标准

| 场景 | 标准要求 |
|------|---------|
| 落座休息时 | 及时让学员落座，与学员进行互动小游戏 |
| 观看示范时 | 侧身跪蹲在学员身边，注意学员目光是否集中在主教身上 |
| 排队出发时 | 辅助主教维持队形 |
| 循环训练时 | 与主教站位成对角，站在循环训练动作外缘，面向内部 |
| 游戏耐力时 | 及时清场，根据游戏需要配合主教 |

#### 3.2.3 特殊情况处理标准

| 情况 | 处理方案 |
|------|---------|
| 学员发呆走神 | 口头提醒 + 接触引导 |
| 动作卡顿 | 让后面动作流畅的学员做示范先出发，再单独教学不会的学员 |
| 学员产生推搡、插队等矛盾 | 互相道歉 / 引导比比谁做动作更厉害 |
| 学员受伤等突发事件 | 检查伤情，不严重则鼓励，严重则寻求场外空闲教练帮助 |

#### 3.2.4 学龄前动作保护及辅助标准

| 动作 | 保护/辅助方法 |
|------|-------------|
| 攀爬跳箱 | 可托举腋下辅助上箱；在箱上时前后手护着学员；下箱时提醒趴着下 |
| 双脚跳跳箱 | 双手前后护着学员 |
| 前滚翻 | 提醒低头，右手护住脖颈，左手托腿给予助力 |
| 平衡木行走 | 站于学员身边陪伴行走，防止掉落磕碰 |

### 3.3 助教课后标准

| 序号 | 工作内容 | 标准要求 |
|------|---------|---------|
| 1 | 合照准备 | 准备牛牛币和手机拍合照 |
| 2 | 整理教室 | 整理教室、收拾器材 |
| 3 | 复盘反馈 | 主动询问主教本节课助教表现及提升点 |
| 4 | 观课记录 | 8篇助教观课记录（转正必备） |

### 3.4 助教岗位缺失分析

**体系v4中的空白**:
- 02A岗位说明书仅定义4个岗位（店长/教练/顾问/督导），**无助教岗位**
- 02C新员工入职培训仅3天试岗，**无助教专项培训**
- 02D教练星级晋升从"实习"开始，**无助教→实习教练的晋升通道**
- 02G工作量管理**无助教工作量标准**

**建议补充**:
1. 在02A中增加助教岗位说明书
2. 在02C中增加助教专项培训内容
3. 建立助教→实习教练的晋升通道
4. 制定助教工作量标准
