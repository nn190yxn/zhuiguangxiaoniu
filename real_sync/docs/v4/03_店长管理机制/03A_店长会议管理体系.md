# 追光小牛 · 店长会议管理体系

> **版本**：v3.2 | **日期**：2026年4月
> **适用范围**：追光小牛全体店长
> **督导定位**：督导受总经理授权，对店长执行情况进行检查和评估，店长须配合督导工作
> **闭环定位**：会议在店长经营链中的作用是“通报数据、明确问题、分配动作”，不替代现场经营。闭环总则见 `03H`。

---

## 一、制度

### 1.1 红线

| 红线 | 违反后果 |
|------|---------|
| 会议必须准时开始，迟到者群内自罚红包（10元-20元） | 屡次迟到→书面警告 |
| 会议数据必须来自GYMGO系统和财务系统，禁止估算 | 数据造假→首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退 |
| 会议结论必须落到人，有负责人和完成时间 | 无结论→视为无效会议 |
| 禁止开无效会（闲聊式/表演式）| 浪费时间→店长书面警告 |

### 1.2 会议体系总览

| 会议 | 时间 | 参与人 | 主持人 | 记录人 |
|------|------|--------|--------|--------|
| 班前会 | 工作日13:00 / 周末08:45 | 店长+当日团队 | 店长 | 轮值教练/顾问 |
| 班后会 | 工作日20:30（周末无）| 店长+当日团队 | 店长 | 轮值教练/顾问 |
| 管理周例会 | 每周二10:00，120分钟 | 店长+总经理+督导 | 店长 | 轮值教练/顾问 |
| 月度经营会 | 每月第一周周一（全天）| 总经理+全体店长 | 总经理 | 督导（协助）|

## 二、要求

### 2.1 开会的五条原则

1. **准时开始，准时结束**——等一个人浪费所有人时间
2. **有数据才开会**——没有数据就没有发言权
3. **结论要落到人**——每次会议必须有结论、负责人和完成时间
4. **负面信息当面说**——不在会议上抱怨背后说
5. **禁止开无效会**——闲聊式开会是最大的时间浪费

### 2.2 每次会议必须产出的三样东西

| 产出 | 说明 | 谁负责 |
|-----|------|-------|
| **数据通报** | 当天/周/月的6个核心经营信号 | 店长提前5分钟整理 |
| **问题清单** | 发现的待解决问题 | 记录人 |
| **任务分配** | 今日/本周待办，含负责人+截止时间 | 店长宣布 |

## 三、标准

### 3.1 班前会标准

#### 周中班前会（周二至周五 13:00，10-15分钟）

| 步骤 | 时间 | 内容 |
|------|------|------|
| 集合签到 | 1分钟 | 全体到岗，点名 |
| 数据通报 | 3分钟 | 今日体验/新签进度/续费窗口/风险会员 |
| 顾问场 | 3-4分钟 | 今日到店/跟进进展/主攻目标/卡点 |
| 教练场 | 2-3分钟 | 第一次来的孩子/未出勤孩子/特别关注 |
| 今日重点 | 1分钟 | 今日目标喊话 |
| 解散 | 30秒 | 各就各位 |

#### 周末班前会（周六/周日 08:45，10分钟内）

| 步骤 | 时间 | 内容 |
|------|------|------|
| 集合点名 | 1分钟 | 店长点名 |
| 预约通报 | 2分钟 | 今日到店体验/预约 |
| 顾问场 | 3分钟 | 今天主攻目标1个 |
| 教练场 | 2分钟 | 班级特别事项 |
| 目标+解散 | 2分钟 | 今日目标+加油+解散 |

#### 班前会合格标准

| 检查项 | 合格标准 | 不合格表现 |
|-------|---------|----------|
| 时长 | 周中10-15分钟，周末10分钟内 | 开到20分钟以上 |
| 签到 | 100%到岗 | 有人不通知就不来 |
| 数据通报 | 有具体数字 | "昨天还不错" |
| 顾问发言 | 每人发言1-2次 | 顾问全程沉默 |
| 教练发言 | 每人发言1次 | 教练不知道今日重点 |
| 任务分配 | 今日最重要的1件事说清楚了 | 开完会不知道做什么 |

### 3.2 班后会标准

#### 周中班后会（周二至周五 20:30，10-15分钟）

| 步骤 | 时间 | 内容 |
|------|------|------|
| 数据复盘 | 3分钟 | 今日新签/续费/耗课/风险会员处理结果 |
| 逐一复盘 | 5-6分钟 | 顾问场+教练场 |
| 问题升级 | 1-2分钟 | 当天解决/明天解决/上报 |
| 明日预告 | 1分钟 | 明日体验/重点跟进 |

#### 班后会合格标准

| 检查项 | 合格标准 | 不合格表现 |
|-------|---------|----------|
| 时长 | 10-15分钟 | 开到半小时以上 |
| 数据复盘 | 有今日数字，有差距分析 | 只说"今天还不错" |
| 顾问复盘 | 每个顾问发言1-2次 | 顾问说"没什么特别的" |
| 明日计划 | 顾问说出明天第一件事 | 明天的事明天再说 |

### 3.3 管理周例会标准

#### 周例会流程（每周二 10:00，120分钟）

| 环节 | 时长 | 内容 | 谁主持 |
|-----|-----|------|-------|
| 数据同步 | 15分钟 | 全门店业绩/资源/消课看板 | 总部 |
| 门店业绩汇报 | 25分钟 | 各店新签/续费/退费，对照目标找差额 | 各店店长 |
| 消课/开课分析 | 20分钟 | 各店周消课、团课/私教、班均人次 | 各店店长 |
| 销售漏斗分析 | 20分钟 | 资源数量、到访量（一访、二访）、成交量、邀约率、到访率、成交率、成交客单价 | 各店店长 |
| 线索跟进情况 | 20分钟 | 教练新增资源、渠道线索总数和成交 | 各店店长 |
| 重点工作回顾 | 15分钟 | 上周完成情况、本周计划 | 各店店长 |
| 运营同步 | 10分钟 | 流量端/教学端/服务端计划 | 总部 |
| 其他事项 | 10分钟 | 教学/市场/销售/教务同步 | 总部 |
| 会议记录 | 5分钟 | 总结，群内同步 | 轮值记录 |

#### 周例会合格标准

| 检查项 | 合格标准 |
|-------|---------|
| 会前准备 | 店长周报表在周一12:00前发出 |
| 数据质量 | 有对比（vs上周、vs目标）、有去比分析 |
| 问题数量 | 每店最多提2个问题 |
| 决策效率 | 80%的问题当场决策 |
| 执行跟踪 | 上周问题本周通报解决结果 |

### 3.4 月度经营会标准

#### 月度经营会（每月第一周周一，全天）

**上午场（10:00-12:00）：数据复盘**

| 环节 | 内容 |
|-----|------|
| 督导数据通报 | 各店上月核心数据（KPI达成率排名）|
| 各店问题汇总 | 上月问题解决情况通报 |
| 总经理总结 | 上月整体表现，表扬+批评 |

**下午场（14:00-17:00）：目标设定与计划**

| 环节 | 内容 |
|-----|------|
| 本月目标分解 | 各店本月新签/续费/耗课目标 |
| 各店策略汇报 | 每个店长3分钟 |
| 总部支持通报 | 本月活动/支持/资源 |
| 帮带与帮扶确认 | 谁需要重点帮带 |

## 四、流程SOP

### 4.1 班前会SOP

```
Step 1 → 店长提前5分钟从GYMGO导出今日数据
Step 2 → 集合签到（1分钟）
Step 3 → 数据通报（3分钟）
Step 4 → 顾问场：逐个问今日计划（3-4分钟）
Step 5 → 教练场：逐个问重点孩子（2-3分钟）
Step 6 → 今日目标喊话（1分钟）
Step 7 → 解散，各就各位（30秒）
```

### 4.2 班后会SOP

```
Step 1 → 店长汇总今日最终数字
Step 2 → 数据复盘（3分钟）
Step 3 → 顾问场：逐个复盘（3-4分钟）
Step 4 → 教练场：表扬+问题+未出勤（2-3分钟）
Step 5 → 问题升级（1-2分钟）
Step 6 → 明日预告（1分钟）
Step 7 → 结束语
```

### 4.3 周例会SOP

```
Step 1 → 店长周一完成周报并发群
Step 2 → 周二10:00准时开会
Step 3 → 按流程逐项进行（120分钟）
Step 4 → 形成会议记录，群内同步
Step 5 → 上周待办事项通报结果
```
