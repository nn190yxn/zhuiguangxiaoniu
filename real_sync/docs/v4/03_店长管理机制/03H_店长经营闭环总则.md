# 03H 店长经营闭环总则

**版本：** v1.1
**适用对象：** 店长、督导、总经理
**归口部门：** 运营部

---

## 一、目的

将店长管理统一为一条“看数-开会-派工-巡店-升级-帮扶”的经营闭环，避免店长陷入碎事务、越位替代一线执行或督导越权代经营。

---

## 二、经营闭环

### 2.1 六步闭环

| 步骤 | 核心问题 | 主责 | 输出物 |
|------|----------|------|--------|
| 1. 数据检查 | 今天哪里异常 | 店长 | 核心数据异常清单 |
| 2. 会议分配 | 今天谁处理什么 | 店长 | 班前会/周例会任务清单 |
| 3. 动作执行 | 顾问、教练是否按分工推进 | 顾问/教练 | 过程记录 |
| 4. 巡店检查 | 现场执行有没有断点 | 店长 | 巡检记录/整改项 |
| 5. 升级处理 | 风险会员、投诉、续费卡点是否升级 | 店长 | 升级处理记录 |
| 6. 帮扶验收 | 连续不达标是否进入帮扶并验收 | 督导/总经理 | 帮扶记录/验收结果 |

### 2.2 角色边界

1. 店长主责经营结果、节点覆盖、问题升级和整改闭环，不替代顾问做新签，不替代教练做续费。
2. 顾问主责新签、到店转化、转介绍转化。
3. 教练主责首月服务、教学成果可视化和续费推进。
4. 督导主责检查、评估、帮扶验收，不直接下场代替店长经营门店。

### 2.3 店长每天必须盯的6个经营信号

| 信号 | 关注重点 | 异常后的第一动作 |
|------|----------|------------------|
| 到店与体验 | 到店率、体验预约、体验转化 | 当天补排邀约与接待分工 |
| 耗课与课表 | 耗课、空堂、教练出勤 | 当天补排课、补代课 |
| 新签进度 | 顾问线索、一访、成交 | 班前会明确顾问今日主攻名单 |
| 续费窗口 | 未来30天到期、续费覆盖率 | 当天确认教练跟进名单和卡点会员 |
| 风险会员 | 投诉、退费、低出勤、流失预警 | 店长当日升级介入 |
| 资源池 | 新增资源、一访到店、池子存量 | 48小时内启动补量动作 |

| 文件 | 负责环节 | 作用定位 |
|------|---------|---------|
| 03B 店长数据管理体系 | 看数 | 统一异常识别和数据动作 |

| 03D 店长巡店检查体系 | 巡店 | 统一现场检查和整改 |
| 03E 店长帮带与帮扶体系 | 帮扶 | 统一帮扶触发、周期和验收 |
| 03G 督导考核标准 | 评估 | 统一督导检查与验收边界 |

---

## 四、检查原则

1. 店长每天先看数据，再开会分工，不允许先忙事情后补看数据。
2. 店长每周至少完成一次“续费窗口排查 + 风险会员升级”。
3. 巡店发现的问题必须明确整改人和完成时间。
4. 连续两周未改善的问题，必须升级到督导/总经理帮扶。
