# 追光小牛 · 店长自我成长工具

> **版本**：v2.2 | **日期**：2026年4月
> **适用对象**：全体店长
> **核心原则**：先做到，再做好。公司要求是底线，做到才有资格谈成长。店长经营闭环总则见 `03H`。

---

## 一、制度

### 1.1 红线

| 红线 | 违反后果 |
|------|---------|
| 月度自检必须按时完成 | 未按时提交→月度考核扣分 |
| 月度复盘必须按时完成 | 无复盘不计述职，影响月度评定 |
| 经验分享每月至少1次 | 连续2个月未分享→总经理约谈 |
| 新店长60天内按帮带SOP执行 | 拒绝帮带=不适合当店长 |
| 数据造假 | 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退 |

### 1.2 成长红线

| 红线 | 说明 |
|------|------|
| 禁止拒绝学习 | 公司安排的培训必须参加，不得以"忙"为借口 |
| 禁止只学不练 | 每次培训后1周内必须有1个具体行动落地 |
| 禁止自满 | 自检评分与总经理评分差距>2分，需要面谈说明 |

## 二、要求

### 2.1 三阶段成长路径

#### 第一阶段：听话照做（第1-2个月）

**目标**：公司教什么，你做什么，先做到不走样。

| 要求项 | 具体标准 |
|--------|---------|
| 开关店 | 100%按03D巡店检查SOP执行 |
| 班前会/班后会 | 100%按03A会议SOP执行 |
| 查数据 | 每天查6个核心经营信号，异常立刻报总经理/督导 |
| 填周报 | 每周一按时发门店运营群+企业微信 |
| 带团队 | 当日出勤当日管，不拖延 |

**验收标准**：总经理20天/40天/60天三次验收（督导协助执行），全部通过才算合格。

#### 第二阶段：独立运营（第3-6个月）

**目标**：不再需要人盯着，自己能把门店管好。

| 要求项 | 具体标准 |
|--------|---------|
| 独立开会 | 班前会/班后会/周会汇报，不用总经理/督导旁听 |
| 独立看数据 | 能从数据中找出问题，主动采取措施 |
| 独立管人 | 教练/顾问冲突自己处理，处理不了再上报 |
| 独立管客户 | 续费窗口自己盯覆盖和卡点，投诉自己处理 |
| 月度复盘 | 每月5日前完成复盘，分析原因+改进动作 |

**验收标准**：连续3个月核心指标达标（新签/续费/消课/出勤率）。

#### 第三阶段：主动优化（第7个月起）

**目标**：不只是守住门店，还能主动让门店更好。

| 要求项 | 具体标准 |
|--------|---------|
| 主动发现问题 | 不是等数据异常才发现，是提前预判 |
| 提出优化建议 | 每月至少提1条门店运营优化建议 |
| 带新店长 | 帮带新店长（公司安排时） |
| 经验分享 | 每月在店长群分享实战经验 |

## 三、标准

### 3.1 店长必备技能清单（10项）

| # | 技能项 | 合格标准 | 怎么验收 |
|---|--------|---------|---------|
| 1 | 查GYMGO数据 | 能独立查6个核心经营信号，说清异常原因 | 总经理/督导抽查 |
| 2 | 开班前会 | 按03A流程开，10分钟内完成 | 总经理/督导旁听评分 |
| 3 | 开班后会 | 按03A流程开，15分钟内完成 | 总经理/督导旁听评分 |
| 4 | 填周报 | 按模板填写，数据准确，按时提交 | 总经理/督导检查 |
| 5 | 周会汇报 | 3分钟讲清本店数据+问题+计划 | 总部评分 |
| 6 | 开关店巡检 | 按03D检查表逐项执行，无遗漏 | 不定期抽查 |
| 7 | 处理投诉 | 24小时内响应，72小时内解决 | 客诉记录 |
| 8 | 续费窗口管理 | 续费窗口到期前30天全部启动跟进，卡点会员有升级记录 | GYMGO记录 |
| 9 | 体验转化 | 体验后3天内跟进，转化率>40% | GYMGO数据 |
| 10 | 团队出勤 | 当日出勤率>90% | GYMGO数据 |

### 3.2 评分规则

| 评分 | 含义 | 后续动作 |
|------|------|---------|
| 10项全合格 | **达标店长** | 正常运营，可以进入第三阶段 |
| 7-9项合格 | **待提升店长** | 总经理/督导针对性帮带，1个月内补齐 |
| ≤6项合格 | **不合格店长** | 启动30天帮扶，无效则换人 |

## 四、流程SOP

### 4.1 月度自检SOP（每月1日）

```
Step 1 → 对照10项技能清单逐项自评
Step 2 → 写本月做得最好的1件事
Step 3 → 写下月最需要补的1个短板
Step 4 → 写需要总部支持的事项
Step 5 → 发总经理/督导
```

### 4.2 月度复盘SOP（每月5日前）

```
Step 1 → 从GYMGO导出本月数据
Step 2 → 填月度复盘表（数据总览+达成率最低指标分析+下月3个重点工作）
Step 3 → 纳入述职
```

## 五、节点管理与跟踪考核

### 5.1 节点管理

| 节点 | 时间要求 | 责任人 | 跟踪方式 |
|------|---------|--------|---------|
| 月度自检 | 每月1日 | 店长 | 企业微信 |
| 月度复盘 | 每月5日前 | 店长 | 企业微信 |
| 经验分享 | 每月至少1次 | 店长 | 店长群 |
| 技能抽查 | 不定期 | 总经理/督导 | 现场检查 |
| 阶段验收 | 按阶段节点 | 总经理 | 验收评分 |

### 5.2 考核机制

| 考核项 | 考核对象 | 考核频率 | 考核人 | 考核标准 |
|--------|---------|---------|--------|---------|
| 自检提交 | 店长 | 每月 | 总经理 | 每月1日提交 |
| 复盘质量 | 店长 | 每月 | 总经理 | 有数据有分析有计划 |
| 经验分享 | 店长 | 每月 | 总经理 | 每月至少1次 |
| 技能达标 | 店长 | 每季度 | 总经理 | 10项技能≥7项合格 |
