# 追光小牛 · 店长巡店检查体系

> **版本**：v2.3 | **日期**：2026年4月
> **适用对象**：全体店长
> **核心原则**：巡店不是走马观花，是带着检查清单走、带着问题回、带着方案改。巡店在店长经营闭环中承担“检查与整改”环节，闭环总则见 `03H`。

---

## 一、制度

### 1.1 红线

| 红线 | 违反后果 |
|------|---------|
| 开店检查必须在开店前完成，值班教练执行、店长复核 | 未完成→店长书面警告，执行人连带 |
| 关店检查必须由最后离店人员执行，店长确认 | 未完成→店长书面警告，执行人连带 |
| 安全类问题必须当场处理 | 不处理→店长书面警告 |
| 巡检记录必须如实填写 | 数据造假→首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退 |

### 1.2 巡店频率

| 巡店类型 | 频率 | 时长 | 重点 |
|---------|------|------|------|
| **开店复核** | 每天1次 | 5-10分钟 | 核对01A执行情况 + 关键安全/设备/接待点位 |
| **课中巡检** | 每天3次以上 | 每次5-10分钟 | 教学质量/课堂安全/家长感受 |
| **关店确认** | 每天1次 | 5-10分钟 | 核对01A执行情况 + 水电/门窗/数据收口 |
| **周度深度巡检** | 每周1次 | 30分钟 | 全面排查+拍照记录+整改清单 |
| **月度设备盘点** | 每月1次 | 60分钟 | 设备损耗/物料库存/报采购 |

## 二、要求

### 2.1 巡店的四个目的

| 目的 | 说明 |
|------|------|
| 发现问题 | 安全漏洞、设备损坏、卫生死角、服务不规范 |
| 现场纠偏 | 看到问题当场纠正 |
| 传递重视 | 店长在场和不在场，员工状态完全不同 |
| 收集信息 | 家长真实反馈、孩子真实状态、设备真实状况 |

### 2.2 巡检记录要求

| 要求 | 标准 |
|------|------|
| 开店检查 | 每天开店前完成，由值班教练执行、店长复核，记录存档 |
| 课中巡检 | 每天至少3次，发现问题记录 |
| 关店检查 | 每天关门前完成，由最后离店人员执行、店长确认，记录存档 |
| 周度巡检 | 每周1次，输出巡检报告和整改清单 |
| 月度盘点 | 每月1次，输出采购清单 |

## 三、标准

### 3.1 开店复核重点（12项）

| 序号 | 检查项 | 合格标准 | 不合格处理 |
|------|--------|---------|-----------|
| 1 | 门窗 | 已开、无损坏 | 报修 |
| 2 | 照明 | 全部亮、无闪烁 | 换灯泡/报修 |
| 3 | 场地地面 | 干净、无水渍、无破损 | 拖地/报修 |
| 4 | 器材摆放 | 归位、按区域摆放、无散落 | 整理 |
| 5 | 安全出口 | 畅通、无遮挡 | 立即清理 |
| 6 | 急救箱 | 在位、药品在有效期内 | 补充 |
| 7 | 前台/接待区 | 整洁、宣传物料摆放整齐 | 整理 |
| 8 | 饮水机 | 有水、杯子充足 | 补充 |
| 9 | 课表/价目表 | 已更新、无过期信息 | 更新 |
| 10 | 消毒记录 | 昨日已填写 | 补填 |
| 11 | WiFi/网络 | 正常 | 报修 |
| 12 | 体验课准备 | 确认当日体验课学员名单、准备体测器材和体验课教案 | 补充准备 |

### 3.2 课中巡检标准

| 巡检维度 | 看什么 | 合格标准 | 不合格处理 |
|---------|--------|---------|-----------|
| **教练状态** | 是否按教案上课、是否关注每个孩子 | 全程专注、无手机 | 课后沟通 |
| **课堂安全** | 高空动作有无保护、器材稳固 | 无安全隐患 | 立即干预 |
| **助教配合** | 助教是否在位、是否协助保护 | 配合到位 | 课后沟通 |
| **家长感受** | 观察区家长是否舒适、有无投诉苗头 | 无负面反馈 | 主动沟通 |
| **出勤情况** | 应到几人、实到几人 | 点名已完成 | 查GYMGO |
| **卫生状态** | 课中是否及时清理水渍/汗渍 | 无湿滑地面 | 安排清洁 |

### 3.3 关店确认重点（10项）

| 序号 | 检查项 | 合格标准 |
|------|--------|---------|
| 1 | 器材归位 | 所有器材按区域摆放回原位 |
| 2 | 地面清洁 | 无垃圾、无水渍、无杂物 |
| 3 | 照明 | 全部关闭 |
| 4 | 门窗 | 全部锁好 |
| 5 | 水电 | 水龙头关好、电源关闭 |
| 6 | 前台 | 收银/资料收好、电脑关机 |
| 7 | 更衣室/卫生间 | 无遗留物品、水电关好 |
| 8 | 安全出口 | 门已锁、无异常 |
| 9 | 监控 | 确认正常运转 |
| 10 | GYMGO | 当日点名记录已更新、数据已记录 |

## 四、流程SOP

### 4.1 开店复核SOP（5-10分钟）

```
Step 1 → 查看01A《开店检查表》，确认值班教练已逐项执行
Step 2 → 现场抽查关键点位（安全/设备/接待区/体验课准备）
Step 3 → 安全类问题当场处理，设备类问题记入整改清单
Step 4 → 复核无误后进入班前会与当日排班
```

### 4.2 课中巡检SOP

```
Step 1 → 站在教室外/观察区观察30秒（不打扰课堂）
Step 2 → 发现问题→判断：紧急还是非紧急？
  └→ 紧急（安全问题）→立即进场处理
  └→ 非紧急→记下来，课后沟通
Step 3 → 主动和1-2个家长聊1句
Step 4 → 体验家长重点关照
```

### 4.3 关店确认SOP（5-10分钟）

```
Step 1 → 查看01A《关店检查表》，确认最后离店人员已逐项执行
Step 2 → 重点复核门窗、水电、监控、GYMGO数据
Step 3 → 发现异常当天处理，确认无误后锁门离店
```

### 4.4 周度深度巡检SOP（30分钟）

```
Step 1 → 全面排查（安全/卫生/品牌/设备/库存）
Step 2 → 拍照记录问题
Step 3 → 输出《周度巡检报告》
Step 4 → 每个问题指定整改人和完成时间
```

## 五、节点管理与跟踪考核

### 5.1 节点管理

| 节点 | 时间要求 | 责任人 | 跟踪方式 |
|------|---------|--------|---------|
| 开店检查 | 开店前完成 | 值班教练执行，店长复核 | 开店检查表 |
| 课中巡检 | 每天3次以上 | 店长 | 巡检记录 |
| 关店检查 | 关门前 | 最后离店人员执行，店长确认 | 关店检查表 |
| 周度巡检报告 | 每周五 | 店长 | 巡检报告 |
| 月度设备盘点 | 每月最后一天 | 店长 | 采购清单 |

### 5.2 考核机制

| 考核项 | 考核对象 | 考核频率 | 考核人 | 考核标准 |
|--------|---------|---------|--------|---------|
| 开店复核完成率 | 店长 | 每日 | 督导 | 100%复核 |
| 关店确认完成率 | 店长 | 每日 | 督导 | 100%确认 |
| 课中巡检次数 | 店长 | 每日 | 督导 | 每天至少3次 |
| 周度巡检报告 | 店长 | 每周 | 督导 | 按时提交 |
| 问题整改率 | 店长 | 每周 | 督导 | 100%整改 |
