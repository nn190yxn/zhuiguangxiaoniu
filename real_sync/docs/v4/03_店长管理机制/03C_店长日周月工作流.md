# 追光小牛 · 店长日周月工作流

> **版本**：v5.5 | **日期**：2026年4月
> **适用对象**：全体店长
> **核心原则**：店长不是等事情发生的人，是让事情发生的人。店长经营闭环总则见 `03H`。

### 1.2 角色边界

1. 店长负责看数、派工、巡店、升级和闭环，不替顾问做新签，不替教练做续费。
2. 顾问主责新签、体验转化和转介绍转化。
3. 教练主责首月服务、教学成果可视化和续费推进。

---

## 一、制度

### 1.1 红线

| 红线 | 违反后果 |
|------|---------|
| 日数据检查不可跳过，再忙也要做 | 不做→店长书面警告 |
| 班前会/班后会必须按时开 | 不开→店长书面警告 |
| 续费窗口到期前30天必须启动跟进 | 未启动跟进导致流失→店长书面警告 |
| 数据必须来自GYMGO系统和财务系统 | 数据造假→首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退 |

## 二、要求

### 2.1 日工作流要求

#### 周中（周二至周五 13:30-20:30）

| 时间 | 动作 | 具体内容 |
|------|------|---------|
| 13:00 | 到店准备 | 开灯/检查场地安全/设备归位 |
| 13:00-13:15 | 查数据（10分钟） | 打开GYMGO查6个核心经营信号 |
| 13:15-13:25 | 班前会（10分钟） | 通报数据→分配今日任务→提醒重点客户 |
| 13:25-13:30 | 会后确认 | 确认顾问跟进计划、教练课表 |
| 13:30-17:00 | 走动管理 | 课中巡检/接待体验家长/处理突发事件 |
| 17:00-17:15 | 下午数据速查 | 查今日到店/耗课/体验情况 |
| 17:15-20:30 | 走动管理 | 晚间课程巡检/家长沟通/续费面谈 |
| 20:30-20:35 | 班后会（5分钟） | 今日数据复盘+明日重点提醒 |
| 20:35-20:45 | 日数据记录 | GYMGO或纸质表格记录今日数据 |
| 20:45-20:55 | **工作量核查** | **核查每位伙伴当日工作量完成情况，未完成记录原因** |
| 全天 | **抖音视频发布** | **发布1条门店抖音蓝V号视频（公司指定主题，时间不限）** |
| 20:55-21:00 | 关店检查 | 关灯/锁门/器材归位/安全检查 |

#### 周末（周六至周日 09:00-19:00）

| 时间 | 动作 | 具体内容 |
|------|------|---------|
| 08:30 | 到店准备 | 同周中 |
| 08:30-08:45 | 查数据+班前会 | 快速版：只通报3个数字（5分钟内）|
| 08:45-09:00 | 会后确认 | 确认体验接待人、重点续费客户 |
| 09:00-12:00 | 走动管理 | 周末客流大，重点盯体验接待 |
| 12:00-12:30 | 午餐+午间数据速查 | 查上午到店/耗课/体验转化 |
| 12:30-19:00 | 走动管理 | 下午课中巡检/续费面谈 |
| 19:00-19:15 | 日数据记录+关店 | 周末无班后会，直接关门 |

#### 周一（公休日）

| 动作 | 具体内容 | 截止时间 |
|------|---------|---------|
| 填写周报 | GYMGO导出上周数据，填周报模板 | 12:00前 |
| 发周报 | 发门店运营群+企业微信 | 12:00前 |
| 下周排课确认 | 确认下周课表已排好 | 下午 |

### 2.2 周工作流要求

| 日期 | 必做事项 | 具体内容 | 产出 |
|------|---------|---------|------|
| 周一 | 填周报 | GYMGO导出数据+填周报+发群 | 周报文档 |
| 周一 | 排课确认 | 确认下周课表已排好 | 下周课表 |
| 周二 | 管理周例会 | 10:00开会，120分钟 | 会议记录 |
| 周三 | 续费窗口排查 | GYMGO查未来30天到期会员，分配教练跟进（主责），店长标记卡点会员，可安排顾问协助转化 | 续费跟进名单 |
| 周三 | **资源与一访数量检查** | **本周新增资源数、一访到店数是否达标，不足则安排顾问补量** | **资源补充动作** |
| 周四 | 教练1对1沟通 | 和本周表现异常的教练聊15分钟 | 沟通记录 |
| 周四 | **转介绍数据盘点** | **本周转介绍新增/跟进/成交情况，推动顾问执行转介绍请求，店长只盯结果与卡点** | **转介绍周报** |
| 周四 | **美团/抖音经营数据复盘** | **检查美团经营分（≥80分）、好评评分（≥4.8分）、抖音好评（≥4.8分）、本月月度好评数（目标≥15条）、抖音视频数据，制定改进措施** | **线上平台运营周报** |
| 周五 | 下周课表终检 | 最终确认下周排课 | 下周课表终版 |
| 周五 | 周度数据小结 | 本周数据汇总，和周初目标对比 | 心中有数 |
| 周五 | **团队工作量周统计** | **统计本周全员工作量完成率，群内通报排名，连续未达标者安排面谈** | **工作量周报** |
| 周五 | **市场活动筹备检查** | **如有营销节点活动（参照06C），检查筹备进度** | **筹备进度确认** |
| 周五 | **下周周计划制定** | **按四板块（销售/市场/教学/管理）规划下周工作，排入班表，周日22:00前发群** | **下周周计划表** |

**周工作节奏一句话**：周一写周报→周二开会→周三盯续费+查资源→周四聊教练+盘转介绍+查线上→周五锁课表+查活动+做下周计划+统工作量

### 2.3 月工作流要求

| 时间节点 | 必做事项 | 具体内容 | 产出 |
|---------|---------|---------|------|
| 月初第1天 | 月度复盘 | GYMGO导出上月数据，填写总册第10表 | 月度复盘文档 |
| 月初第1天 | 发月度复盘 | 发企业微信给总经理/督导 | — |
| 月初第1周周二 | 月度经营会 | 全天开会 | 会议记录 |
| 月初 | 本月目标分解 | 把月目标拆到每周、每顾问、每教练 | 目标分解表 |
| 月中（15号前后）| 月中进度检查 | 进度过半，数据是否跟上？落后则补救 | 进度报告 |
| 月中 | 续费高峰排查 | 查本月+下月到期会员，到期前30天触达 | 续费触达名单 |
| 月末 | 下月排课规划 | 根据本月出勤数据，调整下月课表 | 下月课表草案 |
| 月末 | 设备/物料盘点 | 检查器材损耗、物料库存、报采购 | 采购清单 |
| **月末** | **会员活动策划** | **制定下月会员活动方案（亲子运动会/户外拓展/节日主题活动等），提前2周完成筹备** | **会员活动方案** |
| **月末** | **资源获取计划** | **制定下月资源获取目标（新增资源数、渠道来源），明确负责人和检查节奏，确保一访量充足** | **资源获取计划表** |
| **月末** | **周边资源拓展** | **每月至少1次外联（周边幼儿园/社区/商户/学校），建立合作渠道** | **外联记录** |
| **月末** | **转介绍推动计划** | **制定下月转介绍推动方案（目标设定、执行分工、检查节点），由顾问执行转化** | **转介绍推动方案** |
| **月末** | **市场调查更新** | **每季度更新1次周边3公里市场调查（小区/幼儿园/商场/竞品排摸）** | **市场调查报告** |
| **月末** | **顾客满意度调查** | **每月面谈或线上问卷完成至少5份顾客满意度调查** | **满意度调查记录** |
| **月末** | **异业合作洽谈** | **每月至少1次异业合作洽谈（周边商户/品牌/商场中庭），达成leads引流合作** | **异业合作记录** |
| **月末** | **工资表核算** | **按薪酬标准核算本店员工工资，提交财务审核** | **工资表** |
| **月末** | **公共关系维护** | **维护商场/物业/社区关系，确保门店运营顺畅** | **关系维护记录** |
| **月中** | **员工经营面谈** | **每人至少1次经营面谈（15分钟），回顾本月工作量完成情况、结果达成、能力提升，制定下月改进计划** | **面谈记录** |
| **月末** | **团队工作量月度总结** | **汇总本月全员工作量达标率，分析未达标原因，制定下月团队提升方案** | **工作量月度总结** |
| **月末** | **美团/抖音月度运营总结** | **汇总美团经营分变化、本月好评数（美团+抖音合计≥15条）、好评评分（≥4.8分）、抖音视频发布量及数据表现，制定下月运营计划** | **线上平台运营月报** |

**月度节奏一句话**：月初复盘定目标→月中检查纠偏差+经营面谈→月末盘点备下月+策划活动+资源计划+市场调查+满意度调查+工作量总结

## 四、流程SOP

### 4.1 店长每日标准动作流程

```
Step 1 → 到店准备（开店检查12项）
Step 2 → 查GYMGO 6个核心经营信号（10分钟）
Step 3 → 班前会（10-15分钟）
Step 4 → 会后确认（顾问跟进计划+教练课表）
Step 5 → 走动管理（课中巡检/接待/处理问题）
Step 6 → 下午数据速查
Step 7 → 班后会（10-15分钟）
Step 8 → 日数据记录
Step 9 → 关店检查（10项）
```

### 4.2 续费窗口排查SOP（每周三）

```
Step 1 → 打开GYMGO→会员列表
Step 2 → 筛选未来30天到期会员
Step 3 → 逐个确认续费意向
Step 4 → 分配教练跟进（主责），可安排顾问协助（到期前30天开始触达）
Step 5 → 形成续费跟进名单
Step 6 → 班前会/班后会通报跟进进度
```

### 4.3 周计划制定SOP（每周五，40-60分钟）

```
Step 1 → 自我准备（5分钟）
  └→ 回顾本周目标完成情况
  └→ 了解当月业绩进度/耗课金额/人员情况/区域重点事务/节假日安排

Step 2 → 规划销售板块（10分钟）
  └→ 顾问周进度跟进（当访/销售目标完成情况）
  └→ 续费面谈排期表、续费学员意向跟踪表
  └→ 转介绍leads获取与跟进安排
  └→ 新单/续费/转介绍业绩拆解到每人

Step 3 → 规划市场板块（10分钟）
  └→ 市场活动规划（目标、人手、时间段）
  └→ 线上平台运营规划（每日责任人分配）
  └→ 异业合作/外联安排

Step 4 → 规划教学板块（10分钟）
  └→ 教练续费沟通排期
  └→ 教练排课和活动人手安排
  └→ 安排每周过课质检

Step 5 → 规划管理板块（10分钟）
  └→ 周例会/对单会安排
  └→ 跨部门问题协调
  └→ 校区清洁卫生安排

Step 6 → 排入班表并发群（5分钟）
  └→ 按人名、事务、时间段编写入班表
  └→ 右侧标注每天关键词和数据目标（如"2单续费、5个小课包、过课质检"）
  └→ 周日22:00前发校区微信大群
```

### 4.4 市场调查SOP（每季度1次）

```
Step 1 → 确定调查范围：以校区为中心，辐射周边3公里
Step 2 → 周边小区排摸：入住率、人群、小区价格、适龄段孩子比例、物业情况
Step 3 → 幼儿园排摸：排名及性质、班级数、在读学员数、是否有同类课程
Step 4 → 大型商场/超市排摸：规模、辐射人群、是否可定点或举办活动
Step 5 → 同行异业机构排摸：品牌、规模、所授课程、价格、学员数
Step 6 → 竞品分析（SWOT）：品牌、规模、价格、学员数、特色/劣势、环境、师资、教案教材
Step 7 → 形成市场调查报告，提交总经理
```
