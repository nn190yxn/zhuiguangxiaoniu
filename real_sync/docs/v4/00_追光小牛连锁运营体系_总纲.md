## 一、制度（红线 + 规则）

### 1.1 通用红线

以下红线适用于所有岗位，任何人违反即按对应处罚执行：

| 序号 | 红线内容 | 处罚 |
|------|----------|------|
| 1 | 体罚或变相体罚学员 | 立即辞退，追究法律责任 |
| 2 | 收受家长红包/礼物 | 立即辞退 |
| 3 | 私自收费、飞单（绕开公司收费） | 立即辞退，追缴全部款项 |
| 4 | 泄露学员/家长隐私信息 | 书面警告 + 乐捐进成长基金（员工10元/店长20元），情节严重辞退 |
| 5 | 在校区内吸烟、饮酒 | 书面警告 + 乐捐进成长基金（员工10元/店长20元） |
| 6 | 数据造假（虚报出勤、虚报续费、篡改数据等） | 首次书面警告 + 乐捐进成长基金（员工10元/店长20元），二次辞退 |
| 7 | 与家长发生严重冲突 | 书面警告 + 乐捐进成长基金（员工10元/店长20元），情节严重辞退 |

> **说明：** 数据造假包括但不限于：虚报学员出勤、虚报续费金额、篡改GYMGO系统数据、伪造家长签名等。一经查实，按上述标准执行。

### 1.2 管理规则

1. **教销分离**：顾问主责新签，教练主责教学与续费，店长主责经营结果，各司其职，不得越权
2. **系统为王**：所有数据以GYMGO系统为准，不允许线下私自记录
3. **逐级汇报**：教练/顾问 → 店长 → 督导/财务 → 总经理，不允许越级（紧急情况除外）
4. **督导授权**：督导受总经理授权，对店长执行情况进行检查和评估，店长须配合督导工作

### 1.3 全体系统一原则

| 原则 | 说明 |
|------|------|
| 一事一主责 | 一件事只能有一个主责人，协同人不替代主责 |
| 结果与动作分开 | KDI管结果，SOP/工作量管动作，激励管奖励，成长基金管处罚 |
| 处罚口径统一 | 教练/顾问/店长违规统一使用“书面警告 + 乐捐进成长基金”口径 |
| 表单必须闭环 | 每张表单必须明确谁填、何时填、谁看、如何处理 |

> 详细说明见 `00A_全体系统一原则.md`。

### 2.2 体系模块关系

| 模块编号 | 模块名称 | 核心作用 |
|----------|----------|----------|
| 00 | 总纲 | 全局规则、版本管理、实施路径 |
| 01 | 门店运营标准体系 | 日常运营基础（开关店、课程、卫生、设备、应急、合同） |
| 02 | 人员管理体系 | 招聘、培训、认证、授权、薪酬、晋升、工作量 |
| 03 | 店长管理机制 | 数据、会议、日周月管理、巡店、巡课、帮带 |
| 04 | 服务标准体系 | 接待、沟通、首月服务、续费、投诉 |
| 05 | 教学标准体系 | ACE、教学SOP、课堂执行、安全保护、课后反馈、升班考核 |

| 08 | 体系推进计划 | 体系发布节奏、推进安排、验收标准 |

| 岗位 | 核心职责 | 汇报对象 |

| 总经理 | 战略决策、审批、最终裁决 | — |

```
├── 01G体测与05教学标准联动
├── 06业绩管理体系与续费目标联动（续费率Q2≥35%）
└── 目标：打通“新签→首月服务→成果可视化→续费/转介绍/挽回”

阶段三：经营管理成熟（9-12月）
├── 03店长管理机制收口数据/会议/巡店/帮带
├── 07品牌一致性标准与会员运营协同落地
├── 续费率Q3≥40%，Q4≥50%
└── 目标：续费收入占比≥35%，体系稳定运转
```

## 附录A：版本管理规范

### A.1 文件命名规则

```
编号_文件名称_版本号.md

示例：
00_追光小牛连锁运营体系_总纲_v3.1.md
01A_开关店SOP_v2.0.md
```

### A.2 更新流程

```
提出修改需求 → 总经理审批 → 修改文件 → 更新版本号 → 旧版归档 → 全员通知
```

| 步骤 | 负责人 | 要求 |
|------|--------|------|
| 1.提出修改需求 | 任何人 | 书面说明修改原因和内容 |
| 2.审批 | 总经理 | 总经理签字/确认后才能修改 |
| 3.修改文件 | 编制部门 | 修改后更新版本号和修改记录 |
| 4.旧版归档 | 编制部门 | 旧版文件移入"历史版本"文件夹，保留备查 |
| 5.全员通知 | 店长 | 新版发布后3个工作日内通知到每位员工 |

### A.3 回顾机制

| 回顾类型 | 频率 | 参与人 | 产出 |
|----------|------|--------|------|
| 季度回顾 | 每季度末 | 总经理+督导+店长 | 回顾记录+修改清单 |
| 年度回顾 | 每年12月 | 全体管理层 | 下一年度优化计划 |

## 附录B：通用红线详解

### B.1 数据造假处罚细则

| 造假类型 | 具体行为 | 首次处罚 | 二次处罚 |
|----------|----------|----------|----------|
| 出勤造假 | 虚报学员出勤、代签到 | 书面警告 + 乐捐进成长基金（员工10元/店长20元） | 辞退 |
| 续费造假 | 虚报续费金额、伪造续费记录 | 书面警告 + 乐捐进成长基金（员工10元/店长20元） | 辞退 |
| 数据篡改 | 篡改GYMGO系统数据 | 书面警告 + 乐捐进成长基金（员工10元/店长20元） | 辞退 |
| 伪造签名 | 伪造家长/学员签名 | 书面警告 + 乐捐进成长基金（员工10元/店长20元） | 辞退 |
| 其他造假 | 其他虚报、瞒报行为 | 书面警告 + 乐捐进成长基金（员工10元/店长20元） | 辞退 |

> **注意：** 数据造假由财务部门负责核查，发现线索后3个工作日内完成调查并出具结论。

### B.2 "情节严重"定义

> **"情节严重"指：** 造成重大经济损失、媒体曝光、法律诉讼、人身伤害等情形。
>
> 本定义适用于本体系所有文件中涉及"情节严重"的处罚条款。凡出现"情节严重"表述的，均按此定义执行。
