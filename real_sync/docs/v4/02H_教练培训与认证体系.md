# 02H_教练培训与认证体系

## 1. 文件定位

本体系用于补齐 `02C_新员工入职培训` 在教练培养路径上的颗粒度不足问题。

本文件重点解决：

1. 新教练从入职到转正，应该经历哪些训练阶段。
2. 每个阶段要学什么、练什么、过什么。
3. 什么时候可以从“跟岗协助”进入“可承担主教行为”。

## 2. 基本原则

1. 培训不是走流程，而是确保课堂可用。
2. 认证不是只看上课次数，而是看是否稳定达标。
3. 先训标准，再训技巧，再训经营延伸。
4. 未完成认证，不得默认独立承担对应课堂主责。

## 3. 与 02C 的关系

1. `02C_新员工入职培训` 负责入职框架、试用期管理和基础适应。
2. `02H_教练培训与认证体系` 负责教练训练路径、阶段考核和放行认证。

## 4. 适用对象

适用于：

1. 新入职教练。
2. 从其他岗位转教练的员工。
3. 需要重新认证或补训的教练。

## 5. 培训总路径

建议按 4 个阶段推进：

1. 入职适应期。
2. 基础跟岗期。
3. 带教上手期。
4. 转正认证期。

## 6. 第一阶段：入职适应期（建议 1-7 天）

目标：

1. 认识品牌、规则、岗位要求和课堂基本秩序。
2. 知道门店基本流程、仪容要求、安全底线和课堂红线。

重点内容：

1. V4 总体结构和教学标准边界。
2. 门店日常流程和基础纪律。
3. 课堂安全基本原则。
4. 主教 / 助教角色认知。
5. 基础服务和家长沟通底线。

阶段结果要求：

1. 能理解门店运行节奏。
2. 知道哪些行为不能做。
3. 能以跟岗身份进入课堂，不制造额外混乱。

## 7. 第二阶段：基础跟岗期（建议 2-3 周）

目标：

1. 在真实课堂中完成助教行为训练。
2. 建立基本的器材、秩序、安全、补位能力。

重点内容：

1. 课前准备、点名、整队、器材布置。
2. 课堂站位、秩序维护和掉队学员补位。
3. 低风险动作辅助。
4. 异常事件第一反应动作。
5. 课后收尾和基础记录协助。

阶段结果要求：

1. 能稳定承担助教行为。
2. 不会越权、不抢口令、不抢节奏。
3. 能识别风险点并及时汇报主教。

## 8. 第三阶段：带教上手期（建议 3-6 周）

目标：

1. 在带教人陪同下逐步承担局部主教行为。
2. 从“会协助”走向“能组织一段、讲一段、带一段”。

重点内容：

1. 开场组织与规则说明。
2. 热身带领和基础节奏控制。
3. 简单动作示范与分组说明。
4. 主体环节中的局部带练。
5. 家长面前的基础表达方式。

阶段结果要求：

1. 能在带教人可控范围内承担局部主教行为。
2. 讲解不乱，节奏不断，安全不失位。
3. 对常见异常有基本处理顺序。

## 9. 第四阶段：转正认证期

目标：

1. 判断员工是否具备稳定承担正式课堂职责的能力。
2. 决定其可承担的课堂类型和授权边界。

认证重点：

1. 是否理解课程目标和课堂结构。
2. 是否能稳定执行 `05D` 标准。
3. 是否能守住 `05E` 安全底线。
4. 是否具备基础家长沟通能力。
5. 是否能在真实课堂中维持秩序、节奏和完成度。

认证结果分为：

1. 通过：可按门店安排承担相应课堂职责。
2. 限通过：仅可承担部分课堂或部分角色，需补训后再评。
3. 不通过：继续跟岗和专项补训，不得独立承担对应职责。

## 10. 每周训练与检查要求

在培养周期内，建议每周至少完成：

1. 一次带教复盘。
2. 一次课堂实操观察。
3. 一次问题纠偏反馈。
4. 一次阶段性判断，确认是否可进入下一阶段。

## 11. 补训机制

出现以下情况时，应进入补训：

1. 课堂节奏明显失控。
2. 反复越权或职责不清。
3. 风险识别和保护动作明显不足。
4. 家长沟通出现明显偏差。

补训要求：

1. 先明确问题归因。
2. 再安排针对性跟岗和复练。
3. 补训完成后重新判断是否放行。

## 12. 管理责任

1. 店长负责训练安排和放行结果把关。
2. 带教人负责日常跟岗指导和问题反馈。
3. 教练本人对训练投入、课堂表现和整改结果负责。

## 13. 一句话结论

教练培训与认证体系的目的，不是把员工“上满课时”，而是让其在真实课堂中稳定、安全、可检查地承担对应职责。
