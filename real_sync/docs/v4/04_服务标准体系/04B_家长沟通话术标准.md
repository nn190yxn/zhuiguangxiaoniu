# 04B 家长沟通话术标准

**版本：** v3.1
**适用对象：** 门店顾问、教练、店长
**归口部门：** 运营部

---

## 一、制度

### 1.1 目的

统一全链路家长沟通表达方式，确保“首次到店、首月服务、体测反馈、续费触达、投诉处理”说法一致、价值清晰。

### 1.2 适用范围

本文件只提供标准话术模板，不单独定义流程主责。流程主责以 `04A / 04E / 01G / 04C / 04F` 为准。

### 1.3 红线

1. **数据造假** → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
2. 向家长传递与机构政策不符的信息 → 书面警告+乐捐进成长基金（员工10元/店长20元）
3. 在家长群或朋友圈发布未经审核的内容 → 口头警告
4. 与家长发生争吵或态度恶劣 → 书面警告+乐捐进成长基金（员工10元/店长20元）

## 二、要求

### 2.1 沟通原则

| 原则 | 说明 |
|------|------|
| 真诚 | 不夸大效果，不隐瞒问题 |
| 及时 | 在规定时效内完成沟通 |
| 专业 | 使用规范话术，避免口语化表达 |
| 一致 | 同一问题，不同员工给出一致答案 |
| 有记录 | 所有重要沟通必须记录在CRM系统 |

### 2.2 沟通时效要求

| 场景 | 时效 | 方式 | 责任人 |
|------|------|------|--------|
| 课后反馈 | 首课后2h内，常规课当日23:00前 | 微信/面谈 | 教练 |
| 家长提问 | 收到后4h内回复 | 微信/电话 | 顾问/教练 |
| 投诉响应 | 收到后1h内首次回复 | 电话 | 店长 |
| 首月适应回访 | 首课后7天内 | 电话/微信 | 教练 |
| 体测/阶段反馈 | 体测后48h内 | 电话/微信 | 教练 |
| 续费触达（到期前30天） | 到期前30天启动 | 电话+微信 | 教练 |
| 续费触达（到期前14天） | 到期前14天 | 电话 | 教练 |
| 续费触达（到期前7天） | 到期前7天 | 电话+面谈 | 教练+店长 |
| 续费触达（到期前3天） | 到期前3天 | 电话/微信 | 教练 |
| 请假/调课 | 提前24h | 微信/电话 | 家长→前台 |
| 活动通知 | 活动前7天 | 微信群+私聊 | 顾问 |

### 2.3 沟通角色分工

| 场景 | 主说话人 | 说明 |
|------|----------|------|
| 首次到店与成交 | 顾问 | 主讲课程匹配、安排与报名事项 |
| 首课后反馈 | 教练 | 主讲课堂观察、孩子表现与家庭练习 |
| 首月阶段反馈 | 教练 | 主讲变化、问题、训练建议 |
| 正式续费触达 | 教练 | 主讲学习连续性、阶段成果和后续规划 |
| 价格异议/方案协调 | 店长/顾问 | 在教练给出价值基础上协助方案表达 |
| 投诉处理 | 店长 | 统一对外口径，避免多人回应 |

## 三、标准

### 3.1 话术结构（万能公式）

```
共情（理解家长感受）→ 事实（客观描述情况）→ 方案（给出解决建议）→ 行动（明确下一步）
```

### 3.2 各场景话术标准

#### 场景一：课后反馈

```
【共情】XX妈妈您好，今天XX在课上表现得很棒/有一些小状况
【事实】今天练习了XX技能，XX完成了X次，其中X次动作标准
【方案】建议在家可以陪XX练习XX，每天5分钟就能巩固
【行动】下周我会继续关注XX的XX方面，有进展随时和您沟通
```

#### 场景二：家长询问进度

```
【共情】理解您关心XX的学习进度，这是非常正常的
【事实】XX目前处于XX阶段，按照评估标准，XX已经掌握了XX
【方案】接下来会重点提升XX，预计X周后可以达到XX水平
【行动】下周我给您发一份详细的进度报告，您看可以吗
```

#### 场景三：首月适应回访（首课后7天内）

```
【共情】XX妈妈，想和您确认一下XX这一周上课适应得怎么样
【事实】这周XX在课堂上已经能做到XX，但在XX方面还需要我们继续带
【方案】接下来我会重点带XX练习XX，您在家里可以配合做一个小动作
【行动】我会在下次课后继续给您反馈变化，您有感受也随时告诉我
```

#### 场景四：续费触达（到期前30天首次触达）

```
【共情】XX妈妈，时间过得真快，XX来追光小牛已经X个月了
【事实】这段时间XX在XX方面进步很大，上次ACE评估得分XX分
【方案】XX的课包还有X节课，大约X月X日到期，建议提前规划续费
【行动】我整理了一份下阶段的学习规划，方便这周和您详细聊聊吗
```

#### 场景五：续费触达（到期前7天冲刺）

```
【共情】XX妈妈，XX的课包快到期了，想和您确认一下后续安排
【事实】目前还剩X节课，到期日是X月X日
【方案】现在续费可以享受XX优惠，而且能保证XX的学习连续性
【行动】如果您今天决定，我可以帮您锁定这个优惠，您看怎么样
```

#### 场景六：家长表达不满

```
【共情】非常理解您的心情，给您带来了不好的体验，我代表门店向您道歉
【事实】您反映的XX问题我已经详细记录下来了
【方案】我会立即和XX沟通，X小时内给您一个解决方案
【行动】解决方案出来后我第一时间联系您，您看这样处理可以吗
```

### 3.3 禁用语清单

| 禁止说 | 替代说 |
|--------|--------|
| "这个我不清楚" | "我确认一下，稍后回复您" |
| "别的家长都没意见" | "我理解您的顾虑，我们来看看怎么解决" |
| "这是公司规定" | "这个政策的考虑是XX，我来帮您看看有没有更好的方案" |
| "你孩子基础差" | "XX在某些方面还有提升空间，我们一起来帮TA" |
| "别人家孩子都报了" | "很多家长选择续费是因为看到了孩子的进步" |

## 四、流程SOP

### 4.1 日常沟通流程

```
课后2h内 → 教练发送课后反馈（微信）
    → 家长回复 → 教练互动答疑
    → 家长未回复 → 次日电话跟进
    → 重要问题 → 记录CRM → 通知店长
```

### 4.2 续费触达流程（与04C一致）

```
剩余课时≤20节/到期前30天 → 启动续费周期（发送学习规划）
    → 到期前14天 → 第二次触达（电话，了解决策状态）
    → 到期前7天 → 冲刺触达（电话+面谈，教练主讲，店长协助）
    → 到期前3天 → 最终触达（明确方案+截止提醒）
    → 到期 → 未续费转入流失挽回流程
```

### 4.3 投诉沟通流程

```
收到投诉 → 1h内首次响应（电话）→ 安抚情绪+记录问题
    → 4h内给出初步方案 → 24h内（绿色）/48h内（黄色）/当天（红色）解决
    → 解决后48h内回访 → 记录归档
```

## 五、节点管理与跟踪考核

### 5.1 检查机制

| 检查项 | 频次 | 检查人 | 检查方式 |
|--------|------|--------|----------|
| 课后反馈及时率 | 每周 | 店长 | 抽查CRM记录 |
| 续费触达节点执行率 | 每月 | 督导 | GYMGO系统数据 |
| 家长回复率 | 每月 | 店长 | CRM数据统计 |
| 投诉首次响应时效 | 每月 | 督导 | 投诉记录表 |
| 禁用语抽查 | 每季度 | 督导 | 神秘访客+录音抽查 |

### 5.2 考核标准

| 指标 | 达标 | 未达标处理 |
|------|------|------------|
| 课后反馈及时率 | ≥90% | 连续2周<90%→教练口头警告 |
| 续费触达节点执行率 | 100% | 漏1个节点→口头警告；漏3个→书面警告 |
| 家长满意度 | ≥4.5/5.0 | <4.0→店长约谈，制定改进计划 |

### 5.3 数据造假处理

- 伪造课后反馈记录 → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
- 伪造续费触达记录 → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
