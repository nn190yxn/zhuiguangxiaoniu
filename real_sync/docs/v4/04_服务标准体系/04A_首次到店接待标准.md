# 04A 首次到店接待标准

**版本：** v4.1
**适用对象：** 门店顾问、教练、前台、店长
**归口部门：** 运营部

---

## 一、制度

### 1.1 目的

规范首次到店接待、体验转化和成交交接流程，确保从“到店体验”顺利进入“服务启动”。

### 1.2 适用范围

所有门店的首次到店体验接待，包括线上预约、自然到访、体验成交及当日交接。

### 1.3 核心指标

| 指标 | 达标 | 优秀 |
|------|------|------|
| 体验到店率 | ≥70% | ≥80% |
| 体验转化率 | ≥30% | ≥50% |

### 1.4 红线

1. **数据造假** → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
2. 体验预约后未提前1天确认 → 口头警告
3. 体验当日无人接待 → 书面警告+乐捐进成长基金（员工10元/店长20元）

## 二、要求

### 2.1 预约阶段

- 预约后2小时内发送确认信息（微信/短信）
- 预约后4小时内建立三方服务群（顾问+教练+家长）
- 服务群内告知门店地址，做提前欢迎
- 鼓励家长提前20-30分钟到店，为小朋友做体测
- 体验日前1天电话确认到访时间，提醒穿着运动服
- 体验日前2小时再次提醒（微信）

### 2.2 到店接待

- 门店门口迎接，主动叫出孩子名字
- 5分钟内完成签到、发放体验名牌
- 介绍门店环境（洗手间、家长休息区、饮水区）

### 2.3 体验过程

- 顾问全程陪同家长，讲解课程体系和教学理念
- 教练专注带课，课后3分钟内给家长口头反馈
- 体验课时长严格控制在45-60分钟

### 2.4 成交交接

- 当日成交后，顾问须在离店前完成成交交接
- 交接内容包括：报名原因、孩子特点、家长顾虑、已承诺事项、首课安排
- 教练须在签约当天知道该学员的基本情况和首月关注重点
- 已成交会员当日转入 `04E 首月服务跟进标准`

### 2.5 体验后跟进

- 体验结束当天内发送感谢信息+课程照片
- 24小时内电话沟通体验感受，了解决策顾虑

## 三、标准

### 3.1 接待与交接8步法

| 步骤 | 动作 | 时限 | 责任人 |
|------|------|------|--------|
| 第1步 | 预约确认 | 预约后2h内 | 顾问 |
| 第2步 | 建立三方服务群（顾问+教练+家长） | 预约后4h内 | 顾问 |
| 第3步 | 到访前1天确认 | 体验日前1天 | 顾问 |
| 第4步 | 到店迎接+签到 | 到店即时 | 前台/顾问 |
| 第5步 | 环境介绍+需求沟通 | 签到后5min内 | 顾问 |
| 第6步 | 体验课进行 | 45-60min | 教练 |
| 第7步 | 课后反馈+方案推荐 | 课后10min内 | 顾问+教练 |
| 第8步 | 未签单跟进或已签单交接 | 离店前完成 | 顾问 |

### 3.2 未签单跟进节奏

| 时间节点 | 跟进动作 | 方式 |
|----------|----------|------|
| 体验当天 | 发送感谢信息+课程照片 | 微信 |
| 体验后24h | 电话沟通感受，了解顾虑 | 电话 |
| 体验后3天 | 发送匹配课程方案+优惠信息 | 微信 |
| 体验后7天 | 邀请参加免费公开课/活动 | 电话+微信 |
| 体验后15天 | 最后一次跟进，了解决策状态 | 电话 |
| 体验后30天 | 转入长期维护池，季度触达 | 微信 |

### 3.3 接待环境标准

| 项目 | 标准 |
|------|------|
| 门店卫生 | 地面无杂物、镜面无指纹、洗手间无异味 |
| 温度 | 夏季24-26°C，冬季20-22°C |
| 音乐 | 轻快儿童音乐，音量适中 |
| 饮水 | 提供常温水和温水 |
| 家长休息区 | 座椅整洁、有品牌宣传物料 |

## 四、流程SOP

### 4.1 首次到店接待流程

```
预约登记 → 预约确认(2h内) → 建立三方服务群(4h内) → 前一天电话确认 → 当天到店迎接
    → 签到+名牌 → 环境介绍 → 需求沟通(5min) → 体验课(45-60min)
    → 课后反馈(10min) → 方案推荐 → 当日成交交接/未成交分流 → 当天感谢信息
    → 24h电话跟进 → 3天方案推送 → 7天活动邀请 → 15天最后跟进
```

### 4.2 异常处理流程

```
家长未按时到店 → 15min后电话确认 → 30min未到→改约
体验课中孩子哭闹 → 教练安抚 → 顾问与家长沟通 → 调整方案
家长对价格有异议 → 了解预算 → 提供分期/优惠方案 → 店长介入
```

## 五、节点管理与跟踪考核

### 5.1 检查机制

| 检查项 | 频次 | 检查人 | 检查方式 |
|--------|------|--------|----------|
| 预约确认及时率 | 每周 | 督导 | 抽查GYMGO系统记录 |
| 服务群建立及时率 | 每周 | 店长 | 抽查微信服务群记录 |
| 8步法执行率 | 每月 | 店长 | 神秘访客+监控抽查 |
| 体验到店率 | 每月 | 督导 | GYMGO数据报表 |
| 体验转化率 | 每月 | 店长 | GYMGO数据报表 |
| 未签单跟进完成率 | 每月 | 店长 | CRM跟进记录 |

### 5.2 考核标准

| 指标 | 达标 | 未达标处理 |
|------|------|------------|
| 体验到店率 | ≥70% | 连续2月<70%→顾问进入培训辅导期 |
| 体验转化率 | ≥30% | 连续2月<30%→顾问进入培训辅导期 |
| 8步法执行率 | 100% | 发现1次未执行→口头警告；3次→书面警告 |

### 5.3 数据造假处理

- 体验到店率、体验转化率数据造假 → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
- 伪造家长签到记录 → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
