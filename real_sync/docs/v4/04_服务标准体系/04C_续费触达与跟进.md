# 04C 续费触达与跟进

**版本：** v4.1
**适用对象：** 教练（主责）、店长、顾问（协同）、督导
**归口部门：** 运营部

> **重要说明**：续费不是到期前才开始，而是建立在首月服务和阶段反馈基础上的正式成交动作。首月服务见 `04E`，阶段成果展示见 `01G`，主链总则见 `04F`。

---

## 一、制度

### 1.1 目的

建立系统化的正式续费、流失挽回和退费处理机制，提升续费率和续费收入占比，保障门店持续经营。

### 1.2 适用范围

所有在籍学员的续费触达、跟进、流失挽回及退费处理。

### 1.3 核心指标

#### 续费率目标（月度考核）

| 季度 | 达标标准 | 优秀标准 |
|------|----------|----------|
| Q2（4-6月） | ≥35% | ≥50% |
| Q3（7-9月） | ≥40% | ≥50% |
| Q4（10-12月） | ≥50% | ≥50% |

#### 续费率分级

| 等级 | 续费率 | 处理方式 |
|------|--------|----------|
| 优秀 | ≥50% | 通报表扬，纳入年度评优 |
| 合格 | 35%-49% | 正常运营，持续优化 |
| 危险 | <35% | 总经理介入，启动专项帮扶 |

#### 续费收入占比

| 时间节点 | 目标 |
|----------|------|
| Q2（6月底前） | ≥25% |
| 年度（12月底） | ≥35% |

### 1.4 红线

1. **数据造假** → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
2. 隐瞒学员流失不报 → 书面警告+乐捐进成长基金（员工10元/店长20元）
3. 退费流程超时未处理 → 书面警告+乐捐进成长基金（员工10元/店长20元）

## 二、要求

### 2.1 正式续费7个核心节点

| 节点 | 时间 | 主责 | 动作 |
|------|------|------|------|
| 节点1 | 剩余课时≤20节 | 教练 | 启动续费周期，输出当前进度+学习规划 |
| 节点2 | 到期前30天 | 教练 | 正式续费触达，明确课包到期日和下阶段目标 |
| 节点3 | 到期前14天 | 教练 | 了解决策状态，解决顾虑，记录卡点 |
| 节点4 | 到期前7天 | 教练 | 发起冲刺沟通，必要时安排面谈 |
| 节点5 | 到期前3天 | 教练 | 最终确认，说明截止节点和后续安排 |
| 节点6 | 到期日 | 店长 | 未续费会员升级介入，决定挽回动作 |
| 节点7 | 流失后30天内 | 店长 | 挽回触达，分析流失原因，决定是否回流 |

> 说明：
> 1. 首月30天、首次体测和阶段反馈属于续费前置动作，不在本文件重复定义。
> 2. 顾问不主责续费；如教练确实忙不过来，店长可安排顾问协助节点 2-5 的方案表达或成交推进。

### 2.2 退费处理要求

| 阶段 | 时限 | 责任人 | 动作 |
|------|------|--------|------|
| 接收退费申请 | 即时 | 前台 | 登记退费信息，安抚家长情绪 |
| 给出退费方案 | 3天内 | 店长 | 核算金额，与家长确认方案 |
| 家长确认方案 | 确认后即时 | 店长 | 签署退费确认书 |
| 完成退款 | 确认后30个工作日内 | 财务 | 打款到账 |
| 退款回访 | 到账后3天内 | 店长 | 确认到账，了解流失原因 |

### 2.3 续费收入占比要求

- 每月统计续费收入占总营收的比例
- Q2底线：续费收入占比≥25%
- 年度目标：续费收入占比≥35%
- 未达标门店需提交分析报告和改进计划

## 三、标准

### 3.1 续费触达话术标准

#### 节点1（剩余课时≤20节）— 启动续费周期

```
XX妈妈，XX的课包还剩下不到20节课了，我们已经为XX启动了续费周期。
这段时间XX在XX方面进步很大，教练对XX的表现很满意。
为了保证XX学习的连续性，我们准备了个性化的学习规划和续费优惠方案，方便这周和您详细聊聊吗？
```

#### 节点2（到期前30天）— 正式续费触达

```
XX妈妈，XX的课包大约还有30天到期，预计是X月X日。
这段时间XX在XX方面进步很大，ACE评估XX分。
我整理了一份下阶段的学习规划，方便这周和您聊聊续费方案吗？
```

#### 节点3（到期前14天）— 决策跟进

```
XX妈妈，上次和您聊了续费方案，想了解一下您的想法。
您还有什么顾虑吗？我可以帮您解答。
现在续费还有XX优惠，名额有限。
```

#### 节点4（到期前7天）— 冲刺面谈

```
XX妈妈，XX的课包下周就到期了。
教练对XX的评价很高，下阶段会重点提升XX。
为了不影响XX的学习进度，建议尽快确认续费。
```

#### 节点7（流失后30天内）— 挽回触达

```
XX妈妈，XX离开追光小牛一个月了，我们都很想念TA。
最近我们推出了XX新课程/活动，很适合XX。
如果XX有意愿回来，我们可以安排一节免费体验课。
```

### 3.2 退费金额核算标准

| 退费情形 | 核算方式 |
|----------|----------|
| 未上课 | 全额退款 |
| 已上课 | 按剩余课时×单节课时费退款 |
| 优惠课包 | 按实际支付金额÷总课时×剩余课时退款 |
| 赠送课时 | 赠送课时不退款 |
| 物料费 | 已发放物料不退款 |

## 四、流程SOP

### 4.1 续费触达全流程

```
首月服务与阶段反馈完成 → 剩余课时≤20节启动续费周期
    → 到期前30天正式触达 → 到期前14天决策跟进 → 到期前7天冲刺面谈
    → 到期前3天最终确认 → 到期日店长升级介入 → 续费成功/转入流失挽回
```

### 4.2 退费处理流程

```
家长提出退费 → 前台登记 → 店长3天内核算方案 → 与家长沟通确认
    → 家长签署退费确认书 → 财务30个工作日内打款 → 店长3天内回访
```

### 4.3 流失预警流程

```
连续2次未出勤 → 店长电话了解原因 → 连续3次未出勤 → 店长介入
    → 到期前30天未续费 → 启动专项帮扶 → 到期未续费 → 转入流失挽回
```

## 五、节点管理与跟踪考核

### 5.1 检查机制

| 检查项 | 频次 | 检查人 | 检查方式 |
|--------|------|--------|----------|
| 续费触达节点执行率 | 每月 | 督导 | GYMGO续费数据 |
| 续费率 | 每月 | 督导 | GYMGO数据报表 |
| 续费收入占比 | 每月 | 总经理 | 财务报表 |
| 退费处理时效 | 每月 | 督导 | 退费登记表 |
| 流失挽回率 | 每月 | 店长 | CRM数据 |

### 5.2 考核标准

| 指标 | 达标 | 未达标处理 |
|------|------|------------|
| 续费率 | 按季度分级达标 | <35%（危险）→总经理介入，启动专项帮扶 |
| 续费收入占比 | Q2≥25%，年度≥35% | 连续2月不达标→提交分析报告+改进计划 |
| 退费处理时效 | 3天给方案，30工作日退款 | 超时1次→口头警告；超时3次→书面警告 |
| 续费触达节点执行率 | 100% | 漏1个节点→口头警告；漏3个→书面警告 |

### 5.3 专项帮扶机制

- **续费率<35%连续2月** → 店长+教练专项帮扶
  - 督导带队到店分析原因
  - 制定2周专项提升计划
  - 每日跟进执行进度
  - 2周后复盘效果

### 5.4 数据造假处理

- 续费率数据造假 → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
- 伪造退费记录 → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
