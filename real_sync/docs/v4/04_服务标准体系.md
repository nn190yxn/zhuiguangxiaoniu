**版本：** v2.1
**生效日期：** 2026年4月
**适用范围：** 全部门店
**归口部门：** 运营部

## 体系定位

服务标准体系是追光小牛"以学员和家长体验为中心"的核心落地文件，覆盖从首次到店到长期续费的全服务链路。

## 核心红线

以下红线全员必须遵守，违反即追责：

1. **数据造假** → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
2. **向家长承诺无法兑现的事项** → 书面警告+乐捐进成长基金（员工10元/店长20元）
3. **泄露学员/家长个人信息** → 立即辞退
4. **对学员体罚或冷暴力** → 立即辞退
5. **私下收费或引导家长绕过机构交易** → 立即辞退

## 文件关系

```
04A 首次到店接待与交接 → 04E 首月服务跟进 → 01G/阶段反馈 → 04C 续费触达与跟进
            │                         │                                  ↕
            └────────────→ 04B 家长沟通话术标准 ────────────┘
                                              ↕
                                        04D 投诉处理流程
                                              ↑
                                              │
                                  04F 会员服务与续费主链路总则
```

- 04A是服务入口，决定了家长第一印象、成交体验和交接质量
- 04E承接新签后的首月服务，是续费前置信任建立的主文件
- 04B为全链路提供标准表达，不替代主流程定义
- 04C是正式续费和流失挽回主文件，与06业绩管理体系直接关联
- 04D是服务兜底，确保问题及时解决、不升级
- 04F负责把首次到店、首月服务、成果展示和续费收口成一条统一经营主链
