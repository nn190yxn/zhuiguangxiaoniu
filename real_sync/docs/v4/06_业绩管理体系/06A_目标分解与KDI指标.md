# 06A 目标分解与KDI指标

**版本：** v6.2
**适用对象：** 总经理、督导、店长、顾问、教练
**归口部门：** 运营部

---

## 一、制度

### 1.1 目的

将年度业绩目标科学分解到门店、岗位、季度、月度，确保全团队目标清晰、责任到人。

### 1.2 适用范围

全部门店的业绩目标分解与KDI指标管理。

### 1.3 KDI定义

KDI = Key Driving Indicators（关键驱动指标），是影响业绩结果的核心过程指标。

### 1.4 红线

1. **数据造假** → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
2. 虚报业绩数据 → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
3. 隐瞒真实数据 → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退

### 1.5 KDI管理边界

| 类别 | 是否由本文件负责 | 说明 |
|------|----------------|------|
| 年度/季度/月度目标 | 是 | 本文件负责结果分解 |
| 关键驱动指标 | 是 | 本文件只管结果和关键驱动 |
| 工作量过程动作 | 否 | 由 02G 和各SOP文件负责 |
| 违规处罚 | 否 | 由总纲和成长基金统一处理 |
| 荣誉激励 | 否 | 由 06B 负责 |

## 二、要求

### 2.1 目标分解原则

| 原则 | 说明 |
|------|------|
| 自上而下 | 总目标→门店目标→岗位目标→个人目标 |
| 可量化 | 每个目标必须有明确的数字 |
| 可追踪 | 每个目标必须有周/月追踪机制 |
| 有挑战但可达 | 目标高于历史数据10%-20%，但不是遥不可及 |

### 2.2 季度分配

| 季度 | 占比 | 业绩重点 |
|------|------|----------|
| Q1（1-3月） | 15% | 春季招生、续费启动、寒假收尾 |
| Q2（4-6月） | 30% | 周年庆大促、六一活动、暑假预热 |
| Q3（7-9月） | 25% | 暑假运营、开学季招生 |
| Q4（10-12月） | 30% | 双十一活动、年末收官、续费冲刺 |

### 2.3 各岗位目标分解

| 岗位 | 核心目标 | KDI指标 |
|------|----------|---------|
| 店长 | 门店总业绩达标 | 续费率、体验到店率、团队人效、**月度新增资源数、转介绍成交占比** |
| 顾问 | 新签业绩 | 体验到店率、体验转化率、月新签业绩 |
| 教练 | 续费业绩+教学质量 | 续费业绩、出勤率、ACE合格率、升班率 |

> **重要变更（2026年4月起）**：续费由教练为主责，顾问主要负责新签。教练KDI增加续费业绩指标，顾问KDI移除续费相关指标。

## 三、标准

### 3.1 KDI指标体系

#### 门店级KDI

| 指标 | 定义 | 底线标准 | 优秀标准 | 数据来源 |
|------|------|----------|----------|----------|
| 体验到店率 | 到店体验人数/预约体验人数 | ≥70% | ≥80% | GYMGO |
| 体验转化率 | 体验后7天内签单人数/体验人数 | ≥30% | ≥50% | GYMGO |
| 续费率 | 当月续费人数/当月到期人数 | ≥35% | ≥50% | GYMGO |
| 续费收入占比 | 续费收入/总营收 | ≥25%（Q2）→≥35%（年度） | ≥40% | 财务报表 |
| 出勤率 | 当月出勤人次/应出勤人次 | ≥80% | ≥90% | GYMGO |
| 投诉率 | 当月投诉数/当月总学员数 | ≤5% | ≤2% | 投诉登记表 |
| 升班率 | 当季升班人数/参加考核人数 | ≥70% | ≥85% | GYMGO |
| 一访到店率 | 本月一访到店组数/本月邀约组数 | ≥60% | ≥75% | GYMGO |
| 转介绍成交占比 | 转介绍成交单数/总成交单数 | ≥15% | ≥25% | GYMGO |
| 月度会员活动次数 | 本月实际组织会员活动次数 | ≥1次 | ≥2次 | 活动记录 |
| 顾客满意度 | 月度顾客满意度调查平均分 | ≥4.0/5.0 | ≥4.5/5.0 | 满意度调查表 |
| 美团经营分 | 美团/大众点评门店经营分 | ≥80分 | ≥90分 | 美团后台 |
| 美团评价评分 | 美团/大众点评用户评价平均分 | ≥4.8分 | ≥4.9分 | 美团后台 |
| 月度好评数 | 美团+抖音合计新增好评数 | ≥15条/月 | ≥25条/月 | 美团+抖音后台 |
| 团队工作量达标率 | 全员月度工作量日均达标率 | ≥80% | ≥90% | 工作量统计表 |

#### 顾问级KDI

| 指标 | 底线标准 | 优秀标准 |
|------|----------|----------|
| 月新增预约数 | ≥44个（约11个/周×4周） | ≥55个（约14个/周×4周） |
| 体验到店率 | ≥70% | ≥80% |
| 体验转化率 | ≥30% | ≥50% |
| 月新签业绩 | ≥门店月度目标的60% | ≥门店月度目标的80% |

> **说明**：顾问不再考核续费相关指标，续费由教练为主责。顾问月新签业绩的具体数值需根据各店实际情况和年度目标设定。

#### 教练级KDI

| 指标 | 底线标准 | 优秀标准 | 权重 |
|------|----------|----------|------|
| 月续费业绩 | ≥XX元（根据门店目标分配） | ≥XX元（根据门店目标分配） | 30% |
| 学员出勤率 | ≥80% | ≥90% | 20% |
| ACE评估完成率 | 100% | 100% | 10% |
| ACE合格率 | ≥85% | ≥95% | 15% |
| 升班率 | ≥70% | ≥85% | 15% |
| 教学满意度 | ≥4.5/5.0 | ≥4.8/5.0 | 10% |

> **说明**：教练为续费主责人，月续费业绩权重30%，与季度星级评定中续费贡献权重一致。具体数值需根据各店实际情况和年度目标设定。

### 3.2 续费率分级标准

| 等级 | 续费率 | 处理方式 |
|------|--------|----------|
| 优秀 | ≥50% | 通报表扬，纳入年度评优 |
| 合格 | 35%-49% | 正常运营，持续优化 |
| 危险 | <35% | 总经理介入，启动专项帮扶 |

## 四、流程SOP

### 4.1 年度目标分解流程

```
总经理制定年度总目标（12月底前）
    → 分解到各门店（1月初）→ 分解到各岗位（1月中旬）
    → 分解到各季度（1月底）→ 分解到各月度（每月初）
```

### 4.2 月度目标追踪流程

```
每月1日 → 店长发布本月目标（分解到周、到人）
    → 每周一 → 班前会通报上周进度
    → 每周三 → 督导检查各店进度
    → 每月底 → 店长提交月度业绩报表
    → 次月3日前 → 总经理召开月度复盘会
```

### 4.3 目标调整流程

```
连续2周进度落后>20% → 店长分析原因 → 制定追赶计划
    → 连续4周进度落后>20% → 督导介入帮扶
    → 季度目标完成率<70% → 总经理介入评估
```

## 五、节点管理与跟踪考核

### 5.1 检查机制

| 检查项 | 频次 | 检查人 | 检查方式 |
|--------|------|--------|----------|
| 月度目标发布 | 每月1日 | 店长 | 班前会记录 |
| 周进度追踪 | 每周一 | 店长 | 班前会 |
| 月度业绩报表 | 每月底 | 店长 | GYMGO数据导出 |
| KDI达标率 | 每月 | 督导 | GYMGO数据报表 |
| 季度复盘 | 每季度 | 总经理 | 复盘会议 |

### 5.2 考核标准

| 指标 | 达标 | 未达标处理 |
|------|------|------------|
| 体验到店率 | ≥70% | 连续2月<70%→顾问进入培训辅导期 |
| 体验转化率 | ≥30% | 连续2月<30%→顾问进入培训辅导期 |
| 续费率 | 按季度分级达标 | <35%→总经理介入 |
| 续费收入占比 | Q2≥25%，年度≥35% | 连续2月不达标→提交改进计划 |
| 出勤率 | ≥80% | 连续2月<70%→教练降级评估 |
| 一访到店率 | ≥60% | 连续2月<50%→店长提交资源获取改进计划 |
| 转介绍成交占比 | ≥15% | 连续2月为0→排查转介绍流程，制定激励方案 |
| 月度会员活动 | ≥1次 | 当月未组织→下月补办+说明原因 |

### 5.3 数据造假处理

- 业绩数据造假 → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
- KDI指标数据造假 → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
