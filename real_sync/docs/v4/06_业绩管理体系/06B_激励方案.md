# 06B 激励方案

**版本：** v6.2
**适用对象：** 教练、顾问、店长、团队
**归口部门：** 运营部

---

## 一、制度

### 1.1 目的

建立"多劳多得、优劳优得"的激励机制，同时明确惩罚底线，激发团队积极性，保障大活动期间业绩目标达成。

### 1.2 适用范围

全部门店的大活动激励、月度荣誉体系、年度KPI奖金、惩罚机制。

### 1.3 红线

1. **数据造假** → 立即辞退
2. 私下收费或引导家长绕过机构交易 → 立即辞退
3. 恶意抢夺其他门店/顾问的客户 → 书面警告+乐捐进成长基金（员工10元/店长20元）

### 1.4 激励管理边界

| 类别 | 是否由本文件负责 | 说明 |
|------|----------------|------|
| 大活动奖励 | 是 | 面向节点经营结果和专项活动 |
| 月度荣誉 | 是 | 面向正向表扬和文化建设 |
| 年度KPI奖金 | 是 | 面向年度结果奖励 |
| 工作量达标 | 否 | 由 02G 负责过程管理 |
| 日常违规处罚 | 否 | 由总纲和成长基金统一处理 |

## 二、要求

### 2.1 激励原则

| 原则 | 说明 |
|------|------|
| 公开透明 | 所有激励规则全员公示，不暗箱操作 |
| 及时兑现 | 活动结束后15日内发放 |
| 结果导向 | 以数据和结果说话，不搞人情 |
| 团队+个人 | 个人激励与团队激励结合，避免内耗 |

### 2.2 激励发放条件

| 条件 | 说明 |
|------|------|
| 活动期间出勤达标 | 活动期间出勤率≥80%方可享受激励 |
| 无红线违规 | 活动期间无红线违规记录 |
| 数据真实 | 所有业绩数据真实，无造假 |

## 三、标准

### 3.1 大活动激励（随大活动出台，具体方案以活动通知为准）

| 活动类型 | 激励对象 | 激励内容 | 说明 |
|----------|----------|----------|------|
| 周年庆（5-6月） | 全体员工 | 业绩达标奖、超额奖、签单奖 | 具体金额和规则以活动方案为准 |
| 开学季（8-9月） | 全体员工 | 业绩达标奖、新生转化奖、续费奖 | 具体金额和规则以活动方案为准 |
| 双十一（11月） | 全体员工 | 业绩达标奖、限时折扣奖、团队奖 | 具体金额和规则以活动方案为准 |
| 年末收官（12月） | 全体员工 | 业绩冲刺奖、续费完成奖、年度贡献奖 | 具体金额和规则以活动方案为准 |

### 3.2 月度荣誉体系（精神鼓励为主，成长基金奖励为辅）

> **定位**：荣誉体系是追光小牛企业文化建设的核心载体，以精神鼓励为主，让每一位努力付出的伙伴被看见、被认可。月度荣誉奖励资金从成长基金奖金池中支出（详见《成长基金管理办法》）。

| 任意荣誉累计10次 | **追光元勋** | 终身荣誉，享受年度特别福利 |

### 3.3 年度KPI奖金

| 岗位 | 年度KPI奖金基数 | 计算方式 |
|------|-----------------|----------|
| 店长 | 8万元 | 8万 × 年度目标完成率 |
| 顾问 | 2万元 | 2万 × 年度目标完成率 |
| 教练 | 1万元 | 1万 × 年度目标完成率 |

**年度目标完成率计算：**
- 完成率<60%：无年度奖金
- 完成率60%-100%：按比例发放
- 完成率>100%：超出部分按1.5倍计算（上限150%）

### 3.4 惩罚机制

| 情形 | 处理方式 | 触发条件 |
|------|----------|----------|
| 体验转化率低 | 顾问进入培训辅导期 | 连续2月<30% |
| 续费率低 | 店长+教练启动专项帮扶 | 连续2月<35% |
| 出勤率低 | 教练降级评估 | 连续2月<70% |
| 投诉率高 | 店长书面警告+专项整改 | 投诉率>5% |
| 数据造假 | 立即辞退 | 任何数据造假行为 |
| 红线违规 | 按红线标准处理 | 触犯任一红线 |

**培训辅导期规定：**
- 为期4周
- 每周由督导或资深同事进行1对1辅导
- 辅导期结束考核通过→恢复正常
- 辅导期结束考核未通过→延长2周或调岗

**专项帮扶规定：**
- 由督导带队到店
- 分析问题、制定2周提升计划
- 每日跟进执行进度
- 2周后复盘，未改善→升级处理

## 四、流程SOP

### 4.1 大活动激励核算流程

```
活动结束后1-3日 → 各店提交活动期间业绩数据
    → 4-5日 → 督导审核数据真实性
    → 6-8日 → 运营部核算激励金额
    → 9-10日 → 总经理审批
    → 活动结束后15日内 → 财务发放
```

### 4.2 月度荣誉体系评选流程

```
每月最后1个工作日 → 各店提交月度数据（GYMGO导出）
    → 次月1-2日 → 运营部统计数据+店长提名
    → 3日 → 确定各荣誉获得者
    → 5日 → 总经理审批
    → 次月10日前 → 颁发荣誉证书+更新形象墙+全员群发布荣誉榜
    → 次月15日前 → 成长基金奖励发放（财务）
```

### 4.2.1 团队荣誉评选流程

```
每月最后1个工作日 → 督导汇总各店数据
    → 次月1-2日 → 对比各店业绩/投诉/满意度
    → 3日 → 确定团队荣誉获得者
    → 5日 → 总经理审批
    → 次月10日前 → 全员群发布+流动红旗交接
```

### 4.3 惩罚触发流程

```
月度数据统计 → 发现触发惩罚条件
    → 督导核实 → 总经理审批 → 下发处罚通知
    → 启动帮扶/培训 → 跟踪改善情况 → 复盘评估
```

### 4.4 年度KPI核算流程

```
12月底 → 统计全年业绩数据
    → 1月初 → 核算各岗位年度目标完成率
    → 1月中旬 → 总经理审批
    → 春节前 → 发放年度KPI奖金
```

## 五、节点管理与跟踪考核

### 5.1 检查机制

| 检查项 | 频次 | 检查人 | 检查方式 |
|--------|------|--------|----------|
| 活动期间数据真实性 | 每次活动 | 督导 | 抽查GYMGO系统+财务数据 |
| 激励发放及时性 | 每次活动 | 运营部 | 发放记录 |
| 月度荣誉评选及时性 | 每月 | 运营部 | 评选记录 |
| 惩罚执行情况 | 每月 | 督导 | 处罚通知+帮扶记录 |
| 培训辅导期效果 | 按需 | 督导 | 考核成绩+业绩变化 |

### 5.2 考核标准

| 指标 | 达标 | 未达标处理 |
|------|------|------------|
| 激励核算准确率 | 100% | 出错1次→口头警告 |
| 激励发放时效 | 活动结束后15日内 | 超时→运营部书面检讨 |
| 月度荣誉评选时效 | 次月10日前 | 超时→运营部书面检讨 |
| 惩罚执行率 | 100% | 应罚未罚→督导书面警告 |
| 培训辅导通过率 | ≥70% | <70%→审查培训方案 |

### 5.3 数据造假处理

- 任何业绩数据造假 → 立即辞退
- 协助他人造假 → 立即辞退
