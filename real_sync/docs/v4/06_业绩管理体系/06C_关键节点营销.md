# 追光小牛 · 关键节点营销

> **版本**：v3.1 | **日期**：2026年4月
> **适用**：全体门店 | **主责**：总经理（审批）、店长（执行）

---

## 一、制度

### 1.1 红线

| 红线 | 违反后果 |
|------|---------|
| 营销数据造假（虚报体验数、转化数等） | 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退 |
| 未经审批擅自开展营销活动 | 口头警告 |
| 营销承诺与实际不符 | 书面警告+乐捐进成长基金（员工10元/店长20元） |
| 活动方案未按时提交 | 口头警告，影响店长考核 |

### 1.2 营销节点定位

营销节点是追光小牛全年业绩的关键支撑，通过集中资源、统一执行，实现获客和转化的最大化。

### 1.3 全年业绩节奏

| 季度 | 业绩占比 | 关键节点 |
|------|---------|---------|
| Q1（1-3月） | 20% | 3月开学季 |
| Q2（4-6月） | 25% | 5-6月周年庆 |
| Q3（7-9月） | 25% | 9月开学季 |
| Q4（10-12月） | 30% | 双十一+年底收官 |

## 二、要求

### 2.1 营销活动要求

| 要求项 | 标准 |
|--------|------|
| 提前规划 | 每个营销节点提前4周完成方案策划 |
| 统一执行 | 全部门店统一主题、统一话术、统一优惠 |
| 预算控制 | 每次活动预算需提前审批，不得超支 |
| 效果追踪 | 每次活动结束后3天内提交效果报告 |
| 物料管理 | 活动物料提前2周到位，活动后及时回收 |

### 2.2 营销活动审批流程

```
店长提交活动方案（提前4周）
    → 督导审核 → 总经理审批（提前3周）
    → 物料准备（提前2周）→ 预热宣传（提前1周）
    → 活动执行 → 效果复盘（活动后3天内）
```

## 四、流程SOP

### 4.1 营销活动全流程

```
节点前4周 → 方案策划+目标设定
    → 节点前3周 → 方案审批+物料设计
    → 节点前2周 → 物料制作+到店
    → 节点前1周 → 预热宣传+体验排期
    → 活动期间 → 每日追踪+每日复盘
    → 活动结束 → 3天内提交效果报告
```

### 4.2 活动期间每日流程

```
早上班前会 → 通报昨日业绩+今日目标
    → 白天执行 → 顾问邀约+教练带课
    → 晚下班后会 → 统计当日业绩+问题复盘
    → 店长每日22:00前 → 在工作群报当日业绩
```

### 4.3 活动效果复盘流程

```
活动结束 → 3天内提交效果报告
    → 报告内容：目标vs实际、获客成本、转化率、问题总结、改进建议
    → 督导审核 → 总经理复盘会
    → 优秀经验全店推广
```

## 五、节点管理与跟踪考核

### 5.1 检查机制

| 检查项 | 频次 | 检查人 | 检查方式 |
|--------|------|--------|----------|
| 活动方案提交时效 | 每次活动 | 督导 | 方案提交记录 |
| 物料到位情况 | 每次活动 | 运营部 | 物料清单 |
| 预热宣传执行 | 每次活动 | 督导 | 线上+线下检查 |
| 活动期间每日业绩 | 活动期间每日 | 店长 | 工作群日报 |
| 效果报告提交 | 每次活动后 | 督导 | 报告提交记录 |

### 5.2 考核标准

| 指标 | 达标 | 未达标处理 |
|------|------|------------|
| 活动方案按时提交 | 提前4周 | 超时→口头警告 |
| 活动目标完成率 | ≥80% | <80%→提交分析报告 |
| 效果报告按时提交 | 活动后3天内 | 超时→口头警告 |
| 每日业绩上报 | 每日22:00前 | 漏报1次→口头警告 |

### 5.3 数据造假处理

营销数据造假（虚报体验数、转化数等）→ 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
