## 一、店长角色定位

### 1.1 核心职责

**一句话定位：店长是门店的第一责任人，对业绩、团队、服务负全部责任。**

| 核心职责 | 说明 |
|---------|------|
| **业绩达标** | 完成总部下达的现金流、续费率、利润目标 |
| **团队管理** | 管好教练、顾问，让门店正常运转 |
| **服务交付** | 确保教学质量和家长满意度 |
| **上传下达** | 执行总部政策，向上反馈真实情况 |

### 1.2 店长角色禁忌

| 禁止行为 | 原因 |
|---------|------|
| 店长替顾问做新签/替教练做续费 | 店长是经营闭环负责人，不是一线执行人 |
| 越级沟通 | 汇报线是店长→总经理，不允许跳过 |
| 随意承诺家长优惠 | 超过授权范围的优惠，必须上报 |
| 自己决定招聘/辞退 | 店长有建议权，决策权在总经理 |

### 1.3 门店架构

5家校区：小河 / 小十字 / 青少店 / 玖福城 / 蘑菇城

教销分离：教练管教学，顾问管销售。

门店无行政岗位，店长兼管行政事务。

## 二、要求

### 2.1 店长KPI体系

> 店长年度奖金 = 8万总额 × 各维度完成率
> 三项维度独立计算，完成率按比例发放，不倒扣。

| KPI维度 | 权重 | 年度金额 | 考核指标 | 数据来源 |
|--------|------|---------|---------|---------|
| **现金流达标** | 35% | **2.8万** | 季度现金流到账金额÷目标金额 | GYMGO系统 |
| **续费率** | 35% | **2.8万** | 实际续费会员数÷应续费会员数 | GYMGO系统 |
| **成本控制** | 30% | **2.4万** | 实际费用÷预算费用 | 财务核算 |

### 2.2 各维度计算方式

#### 维度一：现金流达标（35%）

**现金流 = 新签收款 + 续费收款**（实际到账金额）

| 完成率 | 奖金发放比例 |
|--------|------------|
| 100%及以上 | 100%发放（2.8万）|
| 80%-99% | 按比例发放 |
| 60%-79% | 按比例发放 |
| 低于60% | 该项不得 |

#### 维度二：续费率（35%）

**续费率 = 实际续费会员数 ÷ 应续费会员数**

| 完成率 | 奖金发放比例 |
|--------|------------|
| 100%及以上 | 100%+额外奖励 |
| 80%-99% | 按比例发放 |
| 60%-79% | 按比例发放 |
| 低于60% | 该项不得 |

> 应续费会员 = 合同到期前30天内+合同已到期未续的会员

#### 维度三：成本控制（30%）

**成本完成率 = 预算费用 ÷ 实际费用**

| 完成率 | 奖金发放比例 |
|--------|------------|
| 实际费用≤预算 | 按节省比例额外奖励 |
| 实际费用=预算 | 100%发放（2.4万）|
| 实际费用超预算≤10% | 100%发放（不倒扣）|
| 实际费用超预算>10% | 按比例扣减 |

### 2.3 双店校长特别KPI

管辖两家门店的校长，KPI设计：

| 考核方式 | 说明 |
|---------|------|
| **合并计算** | 两店现金流、续费率合并考核，权重不变 |
| **薄弱门店专项奖** | 薄弱门店续费率单独达标，额外奖励 |
| **薄弱门店底线** | 薄弱门店续费率不得低于15%，低于则合并奖金打8折 |

### 2.4 店长授权边界

> **原则**：清单内的事，店长可以自主决定；清单外的事，必须上报。

**店长可以自主决定的事：**

| 事项 | 授权范围 | 备注 |
|------|---------|------|
| 日常排班 | 教练/顾问排班调整 | 需提前3天通知总部 |
| 课程调课 | 同一课程调换时间/教练 | 需提前1天通知家长 |
| 小额优惠 | 最高200元以内优惠 | 需记录存档 |
| 物料采购 | 单次500元以内教具耗材 | 需保留发票 |
| 学员请假 | 3次以内的请假审批 | 超过需上报 |
| 日常奖惩 | 口头表扬/警告 | 书面处分需上报 |

**店长必须上报的事：**

| 事项 | 上报时限 | 上报对象 |
|------|---------|---------|
| 家长投诉升级 | 当天 | 总经理 |
| 员工申请离职 | 得知当天 | 总经理 |
| 安全事故 | 发生当时立刻电话 | 总经理 |
| 超过200元的优惠 | 承诺前上报 | 总经理 |
| 超过500元的采购 | 采购前上报 | 总经理+财务 |
| 竞争机构重大动作 | 得知当天 | 总经理 |
| 媒体/监管部门来访 | 接待前上报 | 总经理 |

**店长无权决定的事：**

| 禁止事项 | 说明 |
|---------|------|
| 招聘新员工 | 只有建议权，面试+录用由总经理决定 |
| 开除/辞退员工 | 必须经总经理审批 |
| 超过500元优惠 | 必须经总经理审批 |
| 对外接受采访/发布信息 | 必须经总经理审批 |
| 变更课程价格 | 必须经总经理+财务审批 |
| 对外签合同/合作协议 | 必须经总经理审批 |

## 三、标准

### 3.1 店长汇报体系

**日数据汇报**：店长每天查6个核心经营信号（到店与体验/耗课与课表/新签进度/续费窗口/风险会员/资源池），异常立即报总经理。详见 **03B_店长数据管理体系**。

**周报（每周一完成）**：发门店运营群 + 上传企业微信。

**月度述职**：详见 **03B_店长数据管理体系**，月度复盘统一填写 `追光小牛体系_合并版表单总册.md` 第 10 表。

**紧急事项汇报**：

| 情况 | 汇报方式 | 时限 |
|------|---------|------|
| 学员受伤 | 电话+照片 | 发生当时立刻 |
| 家长激烈投诉 | 电话+简述 | 当天2小时内 |
| 员工突发情况 | 微信+电话 | 当天 |
| 竞争机构重大动作 | 微信群 | 得知当天 |
| 系统/设备重大故障 | 电话 | 发生当时立刻 |

### 3.2 会议机制总览

| 会议 | 时间 | 参与人 | 详情 |
|------|------|--------|------|
| 班前会 | 工作日13:00 / 周末08:45 | 店长+当日团队 | 详见03A |
| 班后会 | 工作日20:30（周末无）| 店长+当日团队 | 详见03A |
| 管理周例会 | 每周二10:00，120分钟 | 店长+总经理+督导 | 详见03A |
| 月度经营会 | 每月第一周周一（全天）| 总经理+全体店长 | 详见03A |

### 3.3 帮带与成长总览

| 机制 | 适用对象 | 周期 | 详情 |
|------|---------|------|------|
| 新店长带教 | 新上任店长 | 60天（20/40/60天三次验收）| 详见03E |
| 落后店长帮扶 | 连续2月KPI不达标 | 30天 | 详见03E |
| 三阶段成长 | 全体店长 | 听话照做→独立运营→主动优化 | 详见03F |
| 经营闭环总则 | 全体店长/督导 | 持续执行 | 详见03H |
