## 一、制度（红线 + 规则）

### 1.1 红线

| 序号 | 红线内容 | 处罚 |
|------|----------|------|
| 1 | 严重超员上课（超过额定班级人数2人以上） | 教练书面警告 + 店长连带 |
| 2 | 课程中发生安全事故隐瞒不报 | 立即辞退 |
| 3 | 体罚或变相体罚学员 | 立即辞退，追究法律责任 |
| 4 | 数据造假（虚报出勤、伪造家长反馈等） | 首次书面警告 + 乐捐进成长基金（员工10元/店长20元），二次辞退 |
| 5 | 无故缺课、迟到15分钟以上 | 书面警告 + 扣当节课时费 |

### 1.2 规则

1. 每节课必须按标准流程执行：课前准备→课中教学→课后反馈
2. 严禁严重超员上课（超过额定班级人数2人以上），按课程类型严格执行容量标准
3. 所有出勤必须在GYMGO系统中及时标记，课程结束后30分钟内完成
4. 每节课后必须在GYMGO系统内及家庭群给家长反馈（点评），课程结束后60分钟内完成，当天23:00前完成
5. 体验课必须由顾问全程跟课，课后及时跟进
6. 未写课后点评的，不给予教练结算课时费

## 二、要求（做什么）

### 2.1 课前准备要求

- 提前10分钟到岗，检查器材和场地
- 点名确认学员出勤，记录缺勤学员
- 准备教案和教学计划
- 与缺勤学员家长联系，了解原因

### 2.2 课中教学要求

- 按教案执行教学，不得随意更改
- 全程关注学员安全，及时制止危险行为
- 鼓励学员，营造积极氛围
- 适时与家长互动，让家长看到进步

### 2.3 课后服务要求

- 课程结束后整理器材归位
- 在GYMGO标记出勤（30分钟内）
- 给家长发送课后反馈（2小时内，当天23:00前完成）
- 记录学员表现和进步

## 三、标准（做到什么程度）

### 3.1 课程时间标准

| 课程类型 | 时长 | 课前准备 | 课后整理 |
|----------|------|----------|----------|
| 常规课 | 60分钟 | 提前10分钟 | 课后10分钟 |
| 体验课 | 45分钟 | 提前10分钟 | 课后15分钟 |
| 私教课 | 60分钟 | 提前10分钟 | 课后10分钟 |
| 进阶课 | 90分钟 | 提前15分钟 | 课后15分钟 |

### 3.2 课程分类与容量标准

| 课程类型 | 年龄段 | 最大人数 | 教练配置 |
|----------|--------|----------|----------|
| 启蒙班 | 2-3岁 | 4人 | 1主教 |
| 基础班 | 4-6岁 | 12人 | 1主教（1:6配置） |
| 进阶班 | 6-8岁 | 12人 | 1主教（1:6配置） |
| 精英班 | 8-12岁 | 12人 | 1主教（1:6配置） |
| 私教课 | 不限 | 1人 | 1主教 |

### 3.3 课前准备检查标准

| 检查项 | 标准 |
|--------|------|
| 教练到岗 | 提前10分钟到岗，着装整齐 |
| 器材检查 | 器材齐全，摆放整齐，无安全隐患 |
| 场地检查 | 地面干净，无杂物，温度适宜 |
| 点名确认 | 与GYMGO一致，缺勤已记录 |
| 教案准备 | 教案已准备，教学目标明确 |

### 3.4 课中教学标准

| 检查项 | 标准 |
|--------|------|
| 热身环节 | 5-10分钟，充分热身，防止受伤 |
| 教学环节 | 按教案执行，动作示范清晰 |
| 安全关注 | 全程关注学员，及时制止危险 |
| 互动鼓励 | 鼓励学员，及时表扬进步 |
| 家长沟通 | 适时与家长互动，让家长看到效果 |

### 3.5 课后服务标准

| 检查项 | 标准 |
|--------|------|
| 器材归位 | 10分钟内完成，按标准摆放 |
| 出勤标记 | 课程结束后30分钟内在GYMGO标记 |
| 家长反馈 | 课程结束后60分钟内发送反馈，当天23:00前完成 |
| 记录更新 | 记录学员表现和进步 |

## 四、流程SOP（怎么做）

### 4.1 课前准备流程

```
提前10分钟到岗
    │
    ▼
检查器材和场地
    │
    ▼
点名确认出勤
    │
    ├── 全员到齐 → 开始热身
    │
    └── 有缺勤
         │
         ├── 联系家长了解原因
         │
         └── 记录缺勤原因到GYMGO
    │
    ▼
准备教案，明确教学目标
```

### 4.2 课中教学流程

```
热身环节（5-10分钟）
    │
    ▼
教学环节（40-50分钟）
    │
    ├── 动作示范
    │
    ├── 学员练习
    │
    ├── 纠正动作
    │
    └── 鼓励表扬
    │
    ▼
放松总结（5分钟）
    │
    ▼
与家长简短沟通
```

### 4.3 课后服务流程

```
课程结束
    │
    ▼
整理器材归位（10分钟内）
    │
    ▼
GYMGO标记出勤（30分钟内）
    │
    ▼
准备家长反馈内容
    │
    ▼
发送家长反馈（60分钟内，当天23:00前完成）
    │
    ▼
记录学员表现和进步
```

### 4.4 话术模板

#### 课前提醒模板（提前1天发送）

> **追光小牛运动馆温馨提示🌸：**
> 家长您好，宝贝的课程时间为星期六 明天17:10—18：40（体操课），记着准时带宝贝来上课哦[耶]同时以下几点请注意：
> 1、运动课穿着运动服或宽松衣服，备一块汗巾。
> 2、请如需就餐，请在开课前半小时。
> 3、运动课会消耗很大体力，请给宝贝带一些水果或酸奶补充能量
> 感谢您对追光小牛的信任，我们上课见[愉快]

#### 课前点名话术

> **教练：** "小朋友们好！今天我们开始上课啦。我来点一下名：XX、XX、XX...好的，今天有X位小朋友到课。XX小朋友今天没来，老师知道为什么吗？"

#### 课中鼓励话术

> **教练：** "XX小朋友做得真棒！动作很标准，大家给他鼓掌！"
> "XX小朋友比上次进步了，继续加油！"

#### 课后反馈话术

> **教练：** "XX妈妈，XX今天表现很棒！今天我们学习了XX动作，他掌握得很好，特别是XX部分做得特别标准。回家可以让他多练习一下XX，下周我们继续学习新内容。"

## 五、节点管理与跟踪考核（谁检查、怎么查、做不到怎么办）

### 5.1 检查机制

| 检查项 | 谁检查 | 怎么查 | 频率 | 做不到怎么办 |
|--------|--------|--------|------|-------------|
| 课前准备 | 店长 | 巡店检查 | 每日 | 未做→教练书面警告 |
| 课中教学 | 店长 | 巡店检查（每天至少1次课中巡检） | 每日 | 违反教学规范→书面警告 |
| 出勤标记 | 店长 | GYMGO系统检查 | 每日 | 未及时标记→书面警告 |
| 家长反馈 | 店长 | 查看反馈记录 | 每日 | 未做反馈→书面警告 |
| 严重超员上课 | 店长 | 现场核对实际人数 | 每日 | 超员→教练书面警告+店长连带 |
| 数据真实性 | 督导 | 合同台账 vs GYMGO vs 实际 | 每月抽查 | 数据造假→按红线处罚 |

### 5.2 处罚细则

| 违规行为 | 责任人 | 处罚 |
|----------|--------|------|
| 未做课前准备 | 教练 | 书面警告 |
| 严重超员上课 | 教练 | 书面警告 |
| 严重超员上课（店长知情不报） | 店长 | 书面警告（连带） |
| 未及时标记出勤 | 教练 | 书面警告 |
| 未做家长反馈 | 教练 | 书面警告 |
| 无故缺课 | 教练 | 书面警告 + 扣当节课时费 |
| 迟到15分钟以上 | 教练 | 书面警告 + 扣当节课时费 |
| 虚报出勤（数据造假） | 教练 | 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退 |
