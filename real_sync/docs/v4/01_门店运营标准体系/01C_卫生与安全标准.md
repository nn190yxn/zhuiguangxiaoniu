## 一、制度（红线 + 规则）

### 1.1 红线

| 序号 | 红线内容 | 处罚 |
|------|----------|------|
| 1 | 发现安全隐患隐瞒不报 | 严肃处理，书面警告 + 乐捐进成长基金（员工10元/店长20元） |
| 2 | 未做消毒就开课 | 教练书面警告 + 店长连带 |
| 3 | 数据造假（虚报消毒记录、虚报清洁等） | 首次书面警告 + 乐捐进成长基金（员工10元/店长20元），二次辞退 |
| 4 | 因清洁不到位导致学员受伤 | 教练书面警告 + 店长连带 + 赔偿 |

### 1.2 规则

1. 卫生和安全是门店运营的底线，必须严格执行
2. 每日清洁是硬性要求，不能跳过
3. 每周三进行周大扫除，全员参与
4. 消毒工作必须记录，不能只做不记
5. 安全隐患发现后必须立即处理或上报，不能拖延

---

## 二、要求（做什么）

### 2.1 日清洁要求

每天营业前和营业后必须完成以下清洁工作。

### 2.2 周大扫除要求

每周三进行一次全面大扫除，全员参与，重点强调办公室区域的打扫以及整洁度整理。

### 2.3 消毒要求

对高频接触区域定期消毒，并做好记录。

### 2.4 安全排查要求

每周三进行一次安全排查（与周大扫除同步），发现隐患及时处理。

---

## 三、标准（做到什么程度）

### 3.1 日清洁标准

| 区域 | 清洁内容 | 标准 | 频率 |
|------|----------|------|------|
| 教学区域 | 地面拖洗、器材擦拭 | 无灰尘、无水渍、无污渍 | 每日2次（开课前+关店前） |
| 前台/接待区 | 桌面擦拭、地面清扫 | 整洁无杂物 | 每日1次 |
| 卫生间 | 地面拖洗、马桶清洁、镜面擦拭 | 干净无异味 | 每日2次 |
| 更衣室 | 地面清扫、柜子擦拭 | 干净整洁 | 每日1次 |
| 门口/走廊 | 地面清扫、垃圾清理 | 干净无垃圾 | 每日1次 |

### 3.2 周大扫除标准

| 区域 | 清洁内容 | 标准 |
|------|----------|------|
| 教学区域 | 地面深度清洁、墙面擦拭、器材深度清洁 | 无死角灰尘、无污渍 |
| 卫生间 | 全面消毒、管道疏通检查 | 无异味、无堵塞 |
| 更衣室 | 柜子内部清洁、通风除味 | 无异味、无遗留物品 |
| 前台/接待区 | 资料整理归档、储物柜清理 | 整洁有序 |
| 办公室区域 | 桌面整理、文件归档、地面清洁、设备擦拭 | 整洁有序、无杂物堆积 |

### 3.3 消毒标准

| 消毒区域 | 消毒方式 | 频率 | 记录要求 |
|----------|----------|------|----------|
| 教学器材 | 75%酒精擦拭 | 每节课后 | 记录到消毒记录表 |
| 门把手/开关 | 75%酒精擦拭 | 每日2次 | 记录到消毒记录表 |
| 卫生间 | 84消毒液喷洒 | 每日1次 | 记录到消毒记录表 |
| 更衣室 | 84消毒液喷洒 | 每周1次 | 记录到消毒记录表 |
| 地垫/软垫 | 专用消毒液喷洒 + 消毒灯照射 | 每周1次 | 记录到消毒记录表 |

### 3.4 安全排查标准

| 排查项 | 检查内容 | 标准 |
|--------|----------|------|
| 器材安全 | 器材是否有松动、损坏、锐角 | 无安全隐患 |
| 地面安全 | 地面是否有破损、翘起、湿滑 | 平整防滑 |
| 电器安全 | 插座、电线是否正常 | 无裸露电线、无漏电 |
| 消防安全 | 灭火器、安全出口、应急灯 | 灭火器在有效期、出口畅通 |
| 急救箱 | 药品是否齐全、是否过期 | 齐全且在有效期内 |
| 门窗安全 | 门窗是否牢固、锁具是否正常 | 牢固、可正常开关 |

## 四、流程SOP（怎么做）

### 4.1 日清洁流程

```
开店前（开课前30分钟）
    │
    ▼
教学区域：拖地 + 擦拭器材
    │
    ▼
前台/接待区：桌面擦拭 + 地面清扫
    │
    ▼
卫生间：拖地 + 清洁马桶 + 补充纸巾/洗手液
    │
    ▼
更衣室：清扫 + 检查遗留物品
    │
    ▼
门口/走廊：清扫 + 清理垃圾
    │
    ▼
关店前（重复一次）
```

### 4.2 周大扫除流程

```
周三开店前
    │
    ▼
全员集合，店长分配任务
    │
    ├── A组：教学区域深度清洁
    ├── B组：卫生间全面消毒
    ├── C组：更衣室+前台清理
    ├── D组：办公室区域整理清洁
    └── E组：安全排查
    │
    ▼
各区域完成后，店长逐区域验收
    │
    ├── 合格 → 记录签字
    │
    └── 不合格 → 返工至合格
    │
    ▼
拍照存档
```

### 4.3 安全排查流程

```
每周三（配合周大扫除）
    │
    ▼
按安全排查表逐项检查
    │
    ├── 无隐患 → 记录签字
    │
    └── 发现隐患
         │
         ├── 能当场解决 → 立即解决 → 记录
         │
         └── 不能当场解决 → 报店长 → 报修/采购
              │
              └── 跟踪至解决 → 记录
    │
    ▼
排查结果报店长存档
```

### 4.4 话术模板

#### 周大扫除分配话术

> **店长：** "今天是周三，我们进行周大扫除。A组（XX、XX）负责教学区域深度清洁，B组（XX、XX）负责卫生间全面消毒，C组（XX）负责更衣室和前台，D组（XX）负责办公室区域整理清洁，E组（XX）负责安全排查。30分钟后我逐区域验收，合格了才能开始今天的课程。"

#### 安全隐患上报话术

> **教练：** "店长，我在检查时发现XX器材的螺丝松了，有安全隐患，需要报修。"
> **店长：** "好的，我马上报修。这节课这个器材先不要用，用XX替代。"

## 五、节点管理与跟踪考核（谁检查、怎么查、做不到怎么办）

### 5.1 检查机制

| 检查项 | 谁检查 | 怎么查 | 频率 | 做不到怎么办 |
|--------|--------|--------|------|-------------|
| 日清洁执行 | 店长 | 现场检查 | 每日 | 不合格→值日教练书面警告 |
| 周大扫除执行 | 店长 | 现场验收+拍照 | 每周三 | 取消→店长书面警告 |
| 消毒记录 | 店长 | 查看消毒记录表 | 每日 | 未做/未记录→教练书面警告 |
| 安全排查 | 店长 | 查看排查记录+现场抽查 | 每周 | 未做→店长书面警告 |
| 安全隐患处理 | 店长 | 跟踪报修记录 | 随时 | 隐瞒不报→严肃处理 |
| 督导巡检 | 督导 | 现场检查 | 每月1次 | 不合格→限期整改 |

### 5.2 处罚细则

| 违规行为 | 责任人 | 处罚 |
|----------|--------|------|
| 日清洁不合格 | 值日教练 | 书面警告 |
| 未做消毒就开课 | 教练 | 书面警告 |
| 未做消毒记录 | 教练 | 书面警告 |
| 周大扫除取消 | 店长 | 书面警告 |
| 周大扫除不合格 | 责任组 | 返工至合格 |
| 安全隐患隐瞒不报 | 发现人 | 严肃处理（书面警告+乐捐进成长基金（员工10元/店长20元）） |
| 因清洁不到位导致学员受伤 | 值日教练+店长 | 书面警告+连带+赔偿 |
| 虚报消毒记录（数据造假） | 教练 | 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退 |
