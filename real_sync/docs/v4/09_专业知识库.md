### 3.3 基础知识应知应会

#### 3.2.1 儿童体适能定义

身体有足够的精力和体力进行日常事务，而不会有过度疲倦，并且有足够的精力享受休闲活动和应付突发事件的能力。

#### 3.2.2 体适能分类

| 类别 | 内容 |
|------|------|
| 代谢性体适能 | 血糖、血脂、血胰岛素、骨密度等，反映机能状态 |
| 健康体适能 | 心肺耐力、肌力与肌耐力、身体成分、柔韧性、神经肌肉松弛 |
| 运动体适能 | 灵敏度、平衡性、协调性、爆发力、反应时与速度 |

#### 3.2.3 儿童体智能

体（体能）+ 智（智能）+ 能（人能/品德修养与社会适应力）

#### 3.2.4 儿童运动能力分类

| 类别 | 说明 |
|------|------|
| 粗大动作类 | 躯体运动 |
| 精细动作类 | 小肌肉动作/手-眼-脑协调 |
| 平衡感类 | — |
| 固定类 | — |

#### 3.2.5 体质五方面

1. 身体形态发育水平
2. 生理功能水平
3. 身体素质和运动能力的发展水平
4. 心理素质发展水平
5. 对内外环境的适应能力

### 3.4 感觉统合核心知识

#### 3.3.1 定义

大脑接收信息 → 处理信息 → 做出反应

#### 3.3.2 七大感觉系统详解

| 感觉系统 | 核心功能 | 失调表现（幼儿期） | 失调表现（儿童期） |
|---------|---------|-------------------|-------------------|
| 视觉 | 视觉对焦、追视、区辨力、理解力、记忆力 | — | — |
| 听觉 | 听觉理解力、记忆力、抑制能力、区辨力 | 听觉敏感/听觉不足 | — |
| 嗅觉 | 个别差异大、极大适应性 | — | — |
| 味觉 | 影响生活品质、幼儿语言发展、饮食习惯 | — | — |
| 触觉 | 接收外界压力、温度、湿度、物体质感 | 触觉反应不足/触觉防御型 | — |
| 前庭觉 | 位于内耳中，保持身体平衡 | 前庭反应不足/前庭反应过度敏感 | — |
| 本体觉 | 肌腱关节等运动器官产生的感觉，自我身体操控能力 | — | — |

#### 3.3.3 大脑发展关键时期

| 年龄 | 大脑发育完成度 |
|------|-------------|
| 新生儿 | 25% |
| 0-3岁 | 60% |
| 4-6岁 | 80% |
| 7-9岁 | 90% |
| 10-12岁 | 100% |

### 3.5 人体三大供能体系

| 供能体系 | 说明 |
|---------|------|
| ATP-CP供能系统 | 爆发力活动（10秒内） |
| 糖酵解供能系统 | 中等强度活动（10秒-3分钟） |
| 有氧供能系统（脂肪供能） | 长时间低强度活动（3分钟以上） |

### 3.6 青少年力量训练原则

#### 3.5.1 各年龄段训练原则

| 年龄段 | 训练原则 |
|--------|---------|
| 7岁及以下 | 不负重练习，教会动作及要领 |
| 8-10岁 | 增加练习手段数量，逐渐增加负重 |
| 11-13岁 | 教会所有基本动作，逐渐增加练习负重 |
| 14-15岁 | 开始正规阶段性训练，增加专项内容 |
| 16岁及以上 | 建立坚实基础，逐步开始成人化训练 |

#### 3.5.2 敏感期

- 女孩肌肉力量发展敏感期：11-13岁
- 男孩肌肉力量发展敏感期：12-14岁
- 女孩绝对力量10-13岁可提高46%
- 男孩12-16岁平均增长57%

### 3.7 体测标准

#### 3.6.1 依据

《国民体质测定标准（幼儿部分）》

#### 3.6.2 分组

3-6周岁，3-5岁每0.5岁一组，6岁为一组，男女共14个组别

#### 3.6.3 测试指标

| 类别 | 项目 |
|------|------|
| 形态 | 身高、体重 |
| 素质 | 10米折返跑、立定跳远、网球掷远、双脚连续跳、坐位体前屈、走平衡木 |

#### 3.6.4 评分制

- 单项5分制
- 综合评级四个等级：一级（优秀）、二级（良好）、三级（合格）、四级（不合格）

## 四、流程SOP

### 4.1 专业知识学习路径

```
新教练入职 → 先学 09A / 09B / 09H 基础底线 → 再学 09C / 09D / 09G 课堂核心 → 再学 09E / 09F 课程与评估应用 → 转正理论考试 → 季度知识更新培训 → 年度知识考核
```

| 阶段 | 时间 | 内容 | 考核 |
|------|------|------|------|
| 基础培训 | 入职第1周 | 09A、09B、09H 基础学习 | ACE理念笔试（Day7） |
| 成长计划 | 入职第2-6周 | 09C、09D、09G、09E、09F 学习与跟岗实践 | 转正理论考试（第6周） |
| 持续学习 | 转正后每季度 | 知识更新培训、新课程技能 | 季度知识更新考核 |
| 综合提升 | 每年度 | 全领域知识回顾与深化 | 年度综合知识考核 |

## 五、节点管理与跟踪考核

### 5.1 知识考核节点

| 节点 | 考核内容 | 合格标准 | 未通过处理 |
|------|---------|---------|-----------|
| 第1周末 | ACE理念笔试 | ≥80分 | 补考一次，仍未通过延长试用期 |

| 季度 | 专业知识更新考核 | ≥80分 | 补考一次，两次未通过进入帮扶期 |
| 年度 | 综合知识考核 | ≥85分 | 补考一次，两次未通影响星级评定 |
