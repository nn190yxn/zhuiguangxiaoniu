# 追光小牛 · 招聘流程与标准

> **版本**：v2.1 | **日期**：2026年4月
> **适用**：5家校区 | **主责**：总经理（审批）、店长（需求+初筛）

---

## 一、制度

### 1.1 红线

| 红线 | 违反后果 |
|------|---------|
| 任何岗位招聘必须经总经理审批 | 未经审批→无效 |
| 不得录用有暴力/性侵/毒品前科人员 | 违反→立即辞退+追究店长责任 |
| 不得录用未满18周岁人员 | 违反→立即辞退+追究店长责任 |
| 不得在招聘中虚假承诺薪资/职位 | 违反→书面警告 |
| 数据造假 | 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退 |

### 1.2 招聘权限

| 角色 | 权限 |
|------|------|
| 店长 | 提出招聘需求、初筛简历、推荐候选人 |
| 总经理 | 审批招聘需求、终面、录用决策 |
| 督导 | 协助总经理面试（受总经理指派时） |

## 二、要求

### 2.1 招聘时效要求

| 环节 | 时限 |
|------|------|
| 店长提交招聘需求 | 岗位空缺3天内 |
| 总经理审批 | 收到需求2天内 |
| 发布招聘 | 审批通过当天 |
| 简历初筛 | 收到简历3天内 |
| 面试安排 | 初筛通过5天内 |
| 录用决策 | 面试后3天内 |
| 入职办理 | 录用通知后7天内 |

### 2.2 招聘渠道

| 岗位 | 主要渠道 | 备选渠道 |
|------|---------|---------|
| 教练 | 58同城/BOSS直聘 | 体育院校校招/内部推荐 |
| 课程顾问 | BOSS直聘/58同城 | 内部推荐 |
| 店长 | BOSS直聘/猎聘 | 内部晋升 |

## 三、标准

### 3.1 教练招聘标准

| 维度 | 标准 | 权重 |
|------|------|------|
| 亲和力 | 喜欢小孩、有耐心、有爱心 | 30% |
| 专业背景 | 体育/教育相关专业优先 | 25% |
| 运动能力 | 能完成基础体操/跑酷动作示范 | 20% |
| 沟通表达 | 能清晰讲解动作、能和家长交流 | 15% |
| 形象气质 | 阳光健康、着装整洁 | 10% |

### 3.2 课程顾问招聘标准

| 维度 | 标准 | 权重 |
|------|------|------|
| 销售能力 | 有销售经验、能达成目标 | 35% |
| 沟通表达 | 口齿清晰、有感染力 | 25% |
| 抗压能力 | 能接受业绩压力、不轻易放弃 | 20% |
| 学习能力 | 能快速掌握课程知识 | 10% |
| 形象气质 | 亲和力强、着装整洁 | 10% |

### 3.3 面试评分标准

| 评分 | 含义 | 建议 |
|------|------|------|
| A（90-100）| 优秀，强烈推荐录用 | 优先录用 |
| B（75-89）| 良好，可以录用 | 正常录用 |
| C（60-74）| 一般，勉强达标 | 谨慎考虑 |
| D（60以下）| 不合格 | 不录用 |

## 四、流程SOP

### 4.1 招聘全流程SOP

```
Step 1 → 店长提交需求
  └→ 填写《招聘需求表》（岗位/人数/到岗时间/特殊要求）
  └→ 发总经理审批

Step 2 → 发布招聘
  └→ 审批通过→督导发布招聘信息
  └→ 督导初筛后发给店长
  └→ 招聘信息统一模板（总部提供）

Step 3 → 简历初筛（店长）
  └→ 按招聘标准筛选
  └→ 3天内完成初筛，通知合格者面试

Step 4 → 第一轮面试（店长）
  └→ 教练：试讲10分钟+动作示范
  └→ 顾问：模拟销售场景5分钟
  └→ 填写《面试评分表》

Step 5 → 第二轮面试（总经理）
  └→ 店长推荐B级以上候选人
  └→ 总经理终面
  └→ 面试后3天内决策

Step 6 → 发录用通知
  └→ 总经理确认薪资和入职日期
  └→ 店长通知候选人

Step 7 → 三天试岗
  └→ 店长安排三天试岗
  └→ 每天填写试岗跟进记录
  └→ 试岗期培训（见02C）
  └→ 试岗结束后，店长向总经理反馈是否通过

Step 8 → 试用期入职（试岗通过后）
  └→ 签劳动合同
  └→ 签岗位说明书
  └→ 签员工手册
  └→ 签相关制度
  └→ GYMGO创建账号
  └→ 进入试用期（见02C）
```

## 五、节点管理与跟踪考核

### 5.1 节点管理

| 节点 | 时间要求 | 责任人 | 跟踪方式 |
|------|---------|--------|---------|
| 招聘需求提交 | 空缺3天内 | 店长 | 招聘需求表 |
| 审批完成 | 收到后2天内 | 总经理 | 审批记录 |
| 面试完成 | 初筛通过后5天内 | 店长 | 面试评分表 |
| 录用决策 | 面试后3天内 | 总经理 | 录用通知 |
| 试岗期入职 | 录用通知后7天内 | 店长 | 试岗确认 |
| 试用期入职 | 试岗通过后 | 店长 | 入职确认 |

### 5.2 考核机制

| 考核项 | 考核对象 | 考核频率 | 考核人 | 考核标准 |
|--------|---------|---------|--------|---------|
| 招聘时效 | 店长 | 每次招聘 | 总经理 | 各环节是否按时完成 |
| 招聘质量 | 店长 | 每次招聘 | 总经理 | 试用期通过率 |
| 试岗跟进 | 店长 | 每次试岗 | 总经理 | 三天跟进记录是否完整 |
