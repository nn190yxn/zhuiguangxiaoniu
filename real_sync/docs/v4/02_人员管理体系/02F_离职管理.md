# 追光小牛 · 离职管理

> **版本**：v2.0 | **日期**：2026年4月
> **适用**：5家校区全体员工 | **主责**：店长（执行）、总经理（审批）

---

## 一、制度

### 1.1 红线

| 红线 | 违反后果 |
|------|---------|
| 试用期离职需提前3天通知 | 不通知直接走→扣发当月绩效 |
| 正式员工离职需提前30天书面通知 | 不通知直接走→扣发当月薪资 |
| 离职必须完成交接，不得带走公司资料 | 带走资料→追究法律责任 |
| 竞业禁止期内不得在竞业范围内任职 | 违反→追究竞业责任 |
| 数据造假 | 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退 |

### 1.2 离职类型

| 类型 | 说明 | 审批 |
|------|------|------|
| 辞职 | 员工主动提出 | 店长→总经理 |
| 辞退 | 公司解除劳动合同 | 总经理直接审批 |
| 协商离职 | 双方协商一致 | 总经理直接审批 |
| 自动离职 | 连续旷工3天 | 视为自动离职 |

## 二、要求

### 2.1 离职交接要求

| 交接项 | 要求 | 责任人 |
|--------|------|--------|
| 工作 | 手头工作全部交接给接替人 | 离职人 |
| 客户 | 所负责客户资料移交 | 离职人 |
| 物品 | 工装/工牌/钥匙/器材归还 | 离职人 |
| 系统 | GYMGO账号权限关闭 | 财务 |
| 财务 | 借款/欠款结清 | 财务 |

### 2.2 离职面谈要求

| 要求 | 标准 |
|------|------|
| 面谈人 | 店长面谈（辞职），总经理面谈（核心员工辞职） |
| 面谈时机 | 收到离职申请后3天内 |
| 面谈内容 | 离职原因、对公司建议、是否有挽留可能 |
| 面谈记录 | 填写《离职面谈记录表》 |

---

## 三、标准

### 3.1 离职薪资结算标准

| 情况 | 薪资结算 |
|------|---------|
| 正常辞职（提前通知）| 结算至最后工作日，含当月课时费/提成 |
| 即时辞职（未提前通知）| 结算至最后工作日，扣发当月绩效奖金 |
| 辞退（公司原因）| 结算至最后工作日+按劳动合同补偿 |
| 辞退（员工违规）| 结算至最后工作日，不额外补偿 |
| 自动离职 | 结算至最后出勤日，扣发当月绩效 |

**特殊说明**：销售顾问和教练离职工资延迟一个月发放，以避免出现客户退费。

### 3.2 竞业禁止标准

| 项目 | 统一标准 |
|------|---------|
| 竞业期限 | 统一6个月 |
| 竞业范围 | 5公里内同类型儿童运动机构 |
| 竞业补偿金 | 离职前12个月平均工资的30%/月 |

> **重要说明**：以上竞业禁止条款为管理告知，具体以单独签署的《竞业限制协议》为准。本条款不替代书面协议，未签署《竞业限制协议》的员工，本条款不产生法律约束力。

## 四、流程SOP

### 4.1 辞职处理SOP

```
Step 1 → 收到辞职申请
  └→ 员工提交书面辞职信
  └→ 店长签收确认

Step 2 → 离职面谈（3天内）
  └→ 店长面谈，了解原因
  └→ 核心员工→总经理面谈
  └→ 填写《离职面谈记录表》

Step 3 → 挽留评估
  └→ 值得挽留→总经理定方案
  └→ 不挽留或挽留失败→进入交接流程

Step 4 → 交接（最后工作日前完成）
  └→ 填写《离职交接表》
  └→ 逐项交接：工作→客户→物品→系统
  └→ 店长签字确认交接完成

Step 5 → 权限关闭
  └→ GYMGO账号关闭（财务操作）
  └→ 工作群移出
  └→ 门禁/钥匙收回

Step 6 → 薪资结算
  └→ 财务核算最后薪资
  └→ 下月15日发放
```

### 4.2 辞退处理SOP

```
Step 1 → 确认辞退理由
  └→ 店长提供事实依据
  └→ 总经理审批

Step 2 → 辞退通知
  └→ 店长1对1通知员工
  └→ 出具书面辞退通知

Step 3 → 交接（3天内完成）
  └→ 同辞职交接流程

Step 4 → 薪资结算
  └→ 按辞退标准结算
  └→ 3日内结清
```

### 4.3 自动离职处理SOP

```
Step 1 → 连续旷工3天
  └→ 店长电话联系确认
  └→ 无法联系→发书面通知到预留地址

Step 2 → 视为自动离职
  └→ 通知发出后3天未到岗→自动离职生效

Step 3 → 后续处理
  └→ 按自动离职标准结算薪资
  └→ 关闭所有权限
  └→ 保留追讨公司财物的权利
```

## 五、节点管理与跟踪考核

### 5.1 节点管理

| 节点 | 时间要求 | 责任人 | 跟踪方式 |
|------|---------|--------|---------|
| 离职面谈 | 收到申请后3天内 | 店长 | 面谈记录表 |
| 交接完成 | 最后工作日前 | 离职人+店长 | 离职交接表 |
| 权限关闭 | 最后工作日当天 | 财务 | 系统确认 |
| 薪资结算 | 最后工作日后3天内 | 财务 | 结算记录 |
| 竞业协议签署 | 离职当天（如适用）| 店长 | 协议存档 |

### 5.2 考核机制

| 考核项 | 考核对象 | 考核频率 | 考核人 | 考核标准 |
|--------|---------|---------|--------|---------|
| 交接完整性 | 店长 | 每次离职 | 总经理 | 交接表全部打勾 |
| 权限关闭及时性 | 财务 | 每次离职 | 总经理 | 离职当天关闭 |
| 面谈完成率 | 店长 | 每次离职 | 总经理 | 100%完成面谈 |
