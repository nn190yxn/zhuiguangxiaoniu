## 一、制度

### 1.1 体系定位

**人员管理体系解决的核心问题：谁来干、怎么来、怎么留、怎么走。**

### 1.2 核心原则

| 原则 | 说明 |
|------|------|
| 教销分离 | 教练管教学，顾问管销售，不混岗 |
| 星级激励 | 教练晋升看数据不看关系，星级直接挂钩收入 |
| 能上能下 | 做得好升级、做不好降级，不养闲人 |
| 制度先行 | 招聘/入职/晋升/离职全流程标准化 |

### 1.3 通用红线

| 红线 | 违反后果 |
|------|---------|
| 数据造假 | 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退 |
| 任何人不得超越岗位职责擅自决策 | 越权→书面警告 |
| 岗位职责明确到人，不得互相推诿 | 推诿→书面警告 |

### 1.4 与其他体系的关系

```
人员管理体系（02）
  ├── 为门店运营（01）提供合格的人
  ├── 是店长管理（03）的管理对象
  ├── 支撑服务标准（04）的落地执行
  ├── 支撑教学标准（05）的教学质量
  └── 驱动业绩管理（06）的业绩产出
```

## 二、要求

### 2.1 人员管理全流程总览

```
招聘需求 → 招聘发布 → 简历初筛 → 面试 → 录用
  ↓                                              ↓
02B                                          入职培训 02C
  ↓                                              ↓
岗位说明书 02A ←──────────────────── 试用期考核 → 转正/辞退
  ↓                                              ↓
星级晋升 02D ←─── 日常管理 ───→ 薪酬核算 02E
  ↓                                              ↓
降级/警告 ───────────────────────→ 离职管理 02F
```

### 2.2 各文件执行要求

| 文件 | 执行人 | 执行频率 |
|------|--------|---------|
| 02A 岗位说明书 | 总经理 | 岗位变动时执行 |
| 02B 招聘流程 | 店长+总经理 | 有招聘需求时执行 |
| 02C 入职培训 | 店长 | 新员工入职时执行 |
| 02D 星级晋升 | 店长+总经理 | 每季度1次 |
| 02E 薪酬结构 | 财务+总经理 | 每月核算 |
| 02F 离职管理 | 店长+总经理 | 员工离职时执行 |
| 02G 工作量管理 | 店长+督导 | 每日执行/每周抽查 |

## 五、节点管理与跟踪考核

### 5.1 节点管理

| 节点 | 时间 | 责任人 | 跟踪方式 |
|------|------|--------|---------|
| 招聘需求审批 | 岗位空缺3天内 | 店长→总经理 | 招聘需求表 |

| 薪资核算 | 每月5日前 | 财务 | 薪资明细表 |

### 5.2 考核机制

| 考核项 | 考核对象 | 考核频率 | 考核人 |
|--------|---------|---------|--------|
| 招聘时效 | 店长 | 每次招聘 | 总经理 |
| 培训完成率 | 店长 | 每月 | 总经理 |
| 星级评定准确率 | 店长 | 每季度 | 总经理 |
| 薪资核算准确率 | 财务 | 每月 | 总经理 |
