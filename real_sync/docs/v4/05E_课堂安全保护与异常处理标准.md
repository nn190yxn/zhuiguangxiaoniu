# 05E_课堂安全保护与异常处理标准

## 1. 文件定位

本标准用于统一追光小牛课堂中的安全保护要求和异常处理流程。

本文件重点解决：

1. 哪些场景必须优先看安全。
2. 哪些动作和课堂环节需要保护。
3. 发生哭闹、冲撞、跌倒、受伤、离场等情况时怎么处理。

## 2. 基本原则

1. 安全优先于进度。
2. 保护优先于教学。
3. 控风险优先于控场面。
4. 先处理人，再处理流程，再处理记录。

## 3. 适用范围

适用于：

1. 所有日常课程、体验课、展示课。
2. 所有涉及翻滚、平衡、跳跃、攀爬、器材切换的课堂。
3. 所有正式员工、实习员工、跟岗员工。

## 4. 课前安全要求

### 4.1 场地检查

1. 地面干燥、无障碍物、无明显滑倒风险。
2. 器材稳定，无破损、松动、锐角外露。
3. 高风险区域有足够缓冲垫和看护位置。
4. 学员进场前完成风险点复查。

### 4.2 人员检查

1. 主教必须清楚当堂高风险动作。
2. 助教必须清楚自己站位和保护点。
3. 实习员工不得被安排独立看守高风险动作区。
4. 学员存在身体不适、情绪异常、家长特别提醒时，必须课前确认。

## 5. 高风险场景管理

### 5.1 重点看护场景

1. 翻滚类动作。
2. 平衡木、单脚平衡等离地或易失衡动作。
3. 跳箱、跨越、连续跳跃等冲击性动作。
4. 攀爬、悬垂、快速轮换器材场景。

### 5.2 风险等级划分

为便于排班、站位和授权，本标准将课堂动作风险划分为 3 级：

1. `L1 低风险`
   - 地面基础动作。
   - 低速度、低高度、低并发。
   - 失误后通常不会造成明显撞击或跌落。
2. `L2 中风险`
   - 存在失衡、碰撞或轻度跌落可能的动作。
   - 需要明确站位、清晰口令和近身保护准备。
3. `L3 高风险`
   - 存在翻滚失控、离地跌落、冲击落地或器材失误风险。
   - 必须明确主责、保护位和并发控制，不满足条件不得开展。

### 5.3 常见动作风险参考

1. `L1 低风险` 参考
   - 原地热身。
   - 低强度爬行。
   - 低高度传接球。
2. `L2 中风险` 参考
   - 单脚站。
   - 低高度平衡木行走。
   - 基础连续跳跃。
3. `L3 高风险` 参考
   - 前滚翻、后滚翻等翻滚类动作。
   - 跳箱、跨越障碍。
   - 攀爬、悬垂、较高平衡木。

### 5.4 基本要求

1. 主教必须提前口头确认规则和保护要求。
2. 助教必须站在最容易出风险的位置。
3. 一名员工无法同时覆盖多个高风险点时，必须减并发或降低难度。
4. 安全不到位时，有权暂停动作，不得硬推进课堂。

### 5.5 不同风险等级的执行要求

1. `L1 低风险`
   - 可由主教统一组织，助教以秩序维护和掉队补位为主。
   - 可适度提高并发，但仍需保证全场可见。
2. `L2 中风险`
   - 必须明确保护位和进出顺序。
   - 不得边讲边散放学员自行尝试。
   - 新人可参与辅助，但不得作为唯一看护人。
3. `L3 高风险`
   - 必须先讲规则、做示范、清保护位、控并发。
   - 实习员工不得承担主保护位。
   - 无法满足保护条件时，必须降阶或替换动作。

## 6. 保护原则

1. 先预判，再站位，再保护。
2. 保护动作以防跌落、防撞击、防失控为主。
3. 保护人员必须关注学员重心变化和动作失衡点。
4. 对初学者、胆怯学员、情绪波动学员和特殊关注学员提高保护等级。

### 6.1 主教保护职责

1. 识别当堂高风险动作和高风险学员。
2. 决定动作是否开展、降阶或替换。
3. 在高风险动作前完成规则说明和示范。
4. 明确助教站位和保护点。

### 6.2 助教保护职责

1. 按主教安排站在主风险点。
2. 保护时不分心、不聊天、不离岗搬器材。
3. 学员状态异常时第一时间近身保护并反馈主教。

### 6.3 保护站位要求

1. 站在学员最可能失衡、跌落或偏移的方向。
2. 站位应能同时保护人和隔离周边碰撞风险。
3. 不得站在看不见起跳点、落地点或翻滚出口的位置。

### 6.4 并发控制要求

1. 同一高风险动作区的并发人数必须与看护人数匹配。
2. 保护不到位时，优先减少同时练习人数。
3. 高风险动作不得在秩序混乱、器材未清场时启动。

## 7. 异常事件处理流程

## 7.1 哭闹 / 情绪失控

1. 助教先做近身保护，避免冲撞和跟跑。
2. 主教迅速调整课堂节奏，保证其余学员有人带。
3. 先降刺激，再安抚，再判断是否继续参与。
4. 持续失控或影响群体安全时，必须带离高风险区域。

## 7.2 冲撞 / 推搡 / 课堂失序

1. 立即分开相关学员。
2. 优先控制风险位置和器材区。
3. 主教快速重建规则，助教做秩序补位。
4. 情节严重时，课后必须记录并上报。
5. 同一学员当堂多次冲撞或推搡时，应纳入重点观察名单。

## 7.3 跌倒 / 轻伤

1. 立即暂停周边动作，避免二次伤害。
2. 先观察意识、情绪、疼痛点和活动能力。
3. 由主教负责判断是否继续课堂。
4. 需要家长介入时，由主教做主说明。
5. 课后完成异常记录。
6. 同一区域当天重复出现跌倒事件时，必须复查动作安排和器材设置。

## 7.4 疑似较重受伤

1. 立即停课或局部停课。
2. 不强行拖拽、不强行复位、不让学员继续测试动作。
3. 第一时间通知店长和家长。
4. 按门店应急流程处理，并完整留痕。
5. 当堂相关动作应立即暂停，复盘前不得恢复。

## 7.5 学员擅自离场 / 走失风险

1. 离出口最近的员工立即拦截和陪同。
2. 主教必须同步掌握其余学员状态。
3. 不得出现全场跟着一个孩子跑、其余学员失管的情况。
4. 事件必须记录并复盘原因。

## 7.6 异常事件等级划分

为便于上报与复盘，异常事件建议分为 3 级：

1. `E1 一般异常`
   - 短时哭闹、轻微碰撞、可快速恢复的课堂失序。
   - 处理后可继续课堂，但应做好简要记录。
2. `E2 重点异常`
   - 多次冲撞、反复离场、明显轻伤、持续失控。
   - 必须由主教主导处理，课后上报店长并复盘。
3. `E3 严重异常`
   - 疑似较重受伤、家长现场激烈投诉、明显安全责任事件。
   - 必须立即升级，启动门店应急和管理介入。

## 8. 家长在场处理要求

1. 发生异常时先处理孩子，再处理家长情绪。
2. 由主教统一对外说明，助教补位维持课堂秩序。
3. 不在家长面前互相指责或推责。
4. 不做超出事实的承诺和判断。

### 8.1 家长沟通口径要求

1. 先说事实，再说处理，再说后续安排。
2. 不抢着下专业结论，不随意判断责任归属。
3. 对已发生异常，不得淡化、隐瞒或拖延说明。

## 9. 实习员工安全边界

实习员工不得单独承担：

1. 高风险动作保护主位。
2. 异常事件独立决策。
3. 受伤后续对家长的主沟通。
4. 高风险区域无人复核情况下的放行动作。

可承担：

1. 在正式员工指挥下做辅助保护。
2. 做秩序控制、器材整理、低风险补位。
3. 做基础异常信息记录。

### 9.1 实习员工放行前安全要求

实习员工在被允许承担更高强度课堂协助前，至少应通过以下检查：

1. 知道高风险动作区的基本站位。
2. 知道哭闹、跌倒、离场时第一反应动作。
3. 不会在高风险动作中离岗、背身或分心。
4. 清楚什么情况必须立刻叫主教。

## 10. 记录与上报

以下情况必须记录：

1. 受伤。
2. 多次哭闹或失控。
3. 冲撞、推搡、离场。
4. 家长现场投诉或明显不满。
5. 保护失位、器材问题等近失误事件。

### 10.1 上报时效要求

1. `E1 一般异常` 当堂或课后当日完成记录。
2. `E2 重点异常` 当堂通知店长，课后立即复盘并留痕。
3. `E3 严重异常` 立即通知店长和相关管理人，不得延迟。

记录至少包括：

1. 时间。
2. 班级和课程。
3. 涉及学员。
4. 事件经过。
5. 当场处理。
6. 后续交接对象。

### 10.2 近失误事件定义

以下情况虽未造成实际受伤，也应按近失误事件记录：

1. 高风险动作保护失位但未出事。
2. 器材松动、移位、破损被及时发现。
3. 实习员工被误安排到高风险主位但及时纠正。
4. 学员险些离场或险些与器材发生明显碰撞。

## 11. 复盘要求

发生安全相关异常后，门店必须复盘：

1. 是动作风险问题，还是人员站位问题。
2. 是课堂组织问题，还是学员个体风险识别不足。
3. 是制度缺口，还是执行不到位。
4. 是否需要补训、调整排班、调整授权范围。

### 11.1 复盘输出要求

每次 `E2` 及以上事件复盘，至少应输出：

1. 事件等级。
2. 直接原因。
3. 间接原因。
4. 当班人员是否存在越权、失位或漏判。
5. 后续整改动作。
6. 复查时间和责任人。

## 12. 一句话结论

本标准要求课堂任何时候都以安全为首位，宁可降难度、减并发、慢节奏，也不能让保护缺位、异常失管和责任不清成为常态。
