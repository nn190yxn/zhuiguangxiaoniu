# 09G_教学实践技能

## 1. 文件定位

本文件用于统一教练在真实课堂中的教学表达、结构设计、示范纠错和氛围营造方法。

## 2. 核心原则

1. 好教学不是讲得多，而是学员听得懂、做得出、跟得上。
2. 教学表达要短、清、准、可执行。
3. 先建秩序，再提要求，再做进阶。
4. 游戏化是为提高参与和理解，不是为了替代教学目标。

## 3. 课堂结构要求

1. 开场要收心、立规则、给目标。
2. 热身要服务主体内容，不做无关堆动作。
3. 主体训练要有主目标和清晰切换点。
4. 结尾要有回收、反馈和家长可感知的总结。

建议课堂最小闭环：

1. 开场收拢。
2. 热身导入。
3. 主体训练。
4. 游戏或应用。
5. 结尾收口。

## 4. 指令表达要求

1. 一次只说一个关键点。
2. 先说做什么，再说怎么做。
3. 重点词固定、重复、清楚。
4. 用学员能理解的话，不用复杂术语堆砌。

常用表达顺序：

1. 先示意注意。
2. 再说动作任务。
3. 再说关键要求。
4. 最后再放开始口令。

## 5. 示范与纠错要求

1. 示范前先保证全员能看到。
2. 示范要突出关键动作和关键错误点。
3. 纠错先抓最影响安全和完成度的问题。
4. 纠错时用可执行语言，不用抽象评价代替指导。

纠错顺序建议：

1. 先纠安全问题。
2. 再纠主动作问题。
3. 最后纠细节质量问题。

## 6. 氛围营造要求

1. 鼓励是为了建立投入，不是无条件夸奖。
2. 要让学员感到被看见，但不能让课堂失控。
3. 需要竞争时要控节奏，需要合作时要控规则。
4. 对情绪波动学员先稳状态，再提表现要求。

## 7. 教练现场判断重点

1. 现在该提目标，还是先稳秩序。
2. 现在该统一推进，还是先处理个别掉队。
3. 现在该继续进阶，还是先回到基础动作。
4. 现在该现场纠错，还是课后单独提示。

## 8. 常见误区

1. 讲得太多，学员反而听不懂。
2. 一边放学员做，一边临时改规则，造成混乱。
3. 纠错抓太多点，导致学员不知道先改什么。
4. 只顾氛围，不顾课堂结构。
