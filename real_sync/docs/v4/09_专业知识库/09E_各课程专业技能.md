# 09E_各课程专业技能

## 1. 文件定位

本文件用于统一不同课程线的专业目标和教学关注点，帮助教练知道“每条课程线到底在练什么、怎么解释、怎么避免串课”。

## 2. 使用原则

1. 不同课程可以共享训练方法，但不能丢掉各自主目标。
2. 课程介绍、课堂执行和家长反馈，必须围绕该课程主价值展开。
3. 课程间可以互补，但不能用一个课程的语言替代另一个课程。

## 3. 各课程线的核心定位

### 3.1 感统体操

1. 重点在基础动作模式、身体控制、平衡协调和感觉输入整合。
2. 更强调控制、适应和基础能力建立。

课堂重点：

1. 让孩子先建立敢参与、能跟随、能完成的基础课堂状态。
2. 优先观察平衡、协调、空间感和指令反应。
3. 对掉队学员优先做降阶，而不是直接拔高动作难度。

家长反馈重点：

1. 今天主要在练基础控制和感觉整合。
2. 孩子在哪些基础能力上开始稳定。
3. 下一步更适合继续稳哪一类基础动作。

### 3.2 篮球

1. 重点在运控、节奏、手眼协调、空间判断和团队规则。
2. 更强调球感、配合和场景反应。

课堂重点：

1. 先建立控球和运球节奏，再谈投篮结果。
2. 先练停、看、传、接，再拉高对抗复杂度。
3. 把规则理解和团队协同纳入课堂目标。

家长反馈重点：

1. 不只讲投进几个球，更要讲控球、节奏和规则理解。
2. 说明孩子在球感、反应或配合中的进步点。

### 3.3 跳绳

1. 重点在节奏、下肢协调、耐力和连续动作能力。
2. 更强调重复质量、节拍感和专注度。

课堂重点：

1. 先练起跳节奏和手脚配合，再追连续个数。
2. 先稳动作一致性，再做速度提高。
3. 对容易急躁的孩子先稳成功率。

家长反馈重点：

1. 除了个数，也要反馈节奏、协调和耐力。
2. 说明孩子是卡在节奏、手脚配合还是持续输出。

### 3.4 中考体训

1. 重点在明确项目成绩、体能储备和专项训练计划。
2. 更强调阶段目标、训练密度和结果跟踪。

课堂重点：

1. 每节课要让学员知道今天练什么项目能力。
2. 训练安排要围绕阶段目标，不随意跳项。
3. 结果记录、复测和训练建议必须成套出现。

家长反馈重点：

1. 当前项目状态如何。
2. 主要瓶颈在技术、力量、节奏还是耐力。
3. 下一阶段训练建议是什么。

### 3.5 搏击防身

1. 重点在反应、协调、规则意识、自我保护和力量传导。
2. 更强调控制、边界和动作规范，不鼓励失控对抗。

课堂重点：

1. 强调规则、距离、控制和停手意识。
2. 让学员先学会规范动作和保护边界。
3. 对兴奋度高的学员要优先控节奏和控场面。

家长反馈重点：

1. 重点不是“打得猛不猛”，而是规则、自控和反应能力。
2. 明确课程价值在保护意识和动作控制。

### 3.6 体态矫正

1. 重点在姿态觉察、基础稳定、肌群平衡和动作控制。
2. 更强调质量、节奏和长期坚持，不追求短期大幅变化。

课堂重点：

1. 先让孩子感知姿态和发力位置，再推进训练量。
2. 先稳基础控制，再上复杂组合。
3. 对代偿明显的动作要及时降阶和纠错。

家长反馈重点：

1. 讲当前姿态问题主要表现在哪里。
2. 讲控制能力、稳定性和动作习惯的变化。
3. 不承诺短期“纠正完成”。

### 3.7 跑酷

1. 重点在越障、支撑、落地、空间判断和勇气建立。
2. 更强调安全保护、动作分解和风险控制。

课堂重点：

1. 先做动作分解和落地控制，再做连续连接。
2. 先保证保护位和并发控制，再提高刺激性。
3. 对胆怯学员优先建信心，对冒进学员优先控风险。

家长反馈重点：

1. 不只讲闯关和刺激感，更要讲空间判断和控制能力。
2. 说明孩子是在勇气、支撑还是落地控制上有进步。

## 4. 各课程线的统一判断框架

每条课程线都建议按同一套逻辑判断：

1. 这门课的主目标是什么。
2. 孩子当前卡在哪个核心能力点。
3. 今天课堂有没有围绕这个点推进。
4. 课后能否给出清楚的下一步建议。

## 5. 课堂表达要求

1. 介绍课程时，先说课程主目标，再说训练形式。
2. 课后反馈时，围绕该课程的专业价值解释孩子进步。
3. 跨课程推荐时，要讲清为什么推荐，而不是只讲好不好玩。

## 6. 教练常见误区

1. 感统课只上成热闹游戏课。
2. 篮球课只追求投篮，不做基础运控。
3. 跳绳课只数个数，不看节奏和动作质量。
4. 体态课只讲姿势，不做控制训练。
5. 跑酷课只做刺激动作，不管保护条件。
6. 中考体训只追成绩，不讲训练逻辑和阶段节奏。
7. 搏击防身只追热血氛围，不守规则和边界。
