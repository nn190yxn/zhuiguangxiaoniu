# 09C_感觉统合理论

## 1. 文件定位

本文件用于统一教练对感觉统合的基础理解，帮助课堂正确识别常见表现并做出合适的教学调整。

## 2. 基本认知

感觉统合可以简单理解为：

1. 身体持续接收信息。
2. 大脑对信息做整合和处理。
3. 身体据此做出动作、注意和行为反应。

课堂上常见的很多“不会做”“不听指令”“总是乱动”，不一定只是态度问题，也可能与信息处理方式有关。

## 3. 课堂最常用的感觉系统理解

1. 前庭觉：关系到平衡、空间感、速度感和身体稳定。
2. 本体觉：关系到发力、控制、姿势和身体边界感。
3. 触觉：关系到对接触、材质、压力和环境变化的接受度。
4. 视觉和听觉：关系到追随、理解、定位和规则接收。

## 4. 课堂常见表现

1. 特别怕转、怕高、怕离地，可能与前庭处理有关。
2. 力量忽大忽小、动作容易撞人撞器材，可能与本体觉输入不足有关。
3. 不喜欢被碰、对排队拥挤很抗拒，可能与触觉防御有关。
4. 经常听不到重点、看不到顺序，可能与听觉或视觉处理有关。

## 5. 教学调整原则

1. 先降复杂度，再提要求。
2. 先给稳定感，再给挑战。
3. 先建立成功体验，再推进进阶动作。
4. 用清楚的顺序、固定站位和可预期节奏降低混乱。

## 6. 教练现场判断顺序

1. 先看孩子是怕、乱、慢，还是总失控。
2. 再判断更像前庭、本体、触觉，还是视觉听觉处理问题。
3. 再决定该降刺激、降难度，还是增加结构提示。
4. 最后再决定课堂里怎么放进当前训练流程。

## 7. 课堂应用做法

1. 对怕动的孩子，先从低高度、低速度、低并发开始。
2. 对控制差的孩子，多用慢节奏、重反馈、短链条动作。
3. 对容易乱的孩子，优先收拢场地、减少刺激和减少同时任务量。
4. 对容易逃避的孩子，先让其进入参与，再逐步提高动作要求。

## 8. 常见课堂场景

1. 一上平衡器材就抗拒的孩子，重点不是催，而是先降挑战并建立稳定感。
2. 总撞器材或撞人的孩子，重点不是批评，而是先补身体边界和节奏控制。
3. 一多任务就乱的孩子，重点不是加快推进，而是先减少同时输入量。
4. 一被碰就抗拒的孩子，重点不是强行适应，而是先建立可接受的接触方式。

## 9. 家长沟通重点

1. 讲清孩子当前更突出的是平衡控制、身体边界还是刺激适应问题。
2. 讲清课堂里已经做了哪些调整。
3. 讲清接下来为什么先稳基础、不急着进阶。
4. 不把专业概念讲成标签。

## 10. 沟通边界

1. 可以说孩子在平衡、控制、空间感或感知反应上有课堂表现特点。
2. 可以建议持续训练和必要时做更专业评估。
3. 不在课堂里替代医疗、康复或心理诊断。
4. 不用夸张标签给家长制造焦虑。

## 11. 常见误区

1. 把所有课堂困难都归因于感觉统合。
2. 看到“乱”就只加强管控，不做结构调整。
3. 用带标签的话替代课堂事实描述。
4. 把一次表现直接说成长期结论。
