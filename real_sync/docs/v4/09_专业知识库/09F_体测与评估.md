# 09F_体测与评估

## 1. 文件定位

本文件用于统一追光小牛对体测、课堂观察和阶段评估的理解，避免出现只测不解读、只给分不出建议的问题。

## 2. 核心原则

1. 体测不是终点，而是训练计划和家长沟通的起点。
2. 评估不是贴标签，而是帮助确定当前阶段重点。
3. 单次结果只能说明当下状态，不能代替长期判断。
4. 解释结果时，必须给出后续方向。

## 3. 三类常用评估来源

1. 基础体测：看身体素质和基础能力状态。
2. 课堂观察：看真实课堂中的理解、参与和动作表现。
3. 阶段复评：看训练后变化和下一阶段目标。

## 4. 评估时必须回答的 5 个问题

1. 当前最突出的优势是什么。
2. 当前最影响训练效果的短板是什么。
3. 这个短板更偏动作、理解还是参与。
4. 近期最该优先抓的 1-2 个方向是什么。
5. 家长听完以后，是否知道接下来怎么配合。

## 5. 输出要求

一次完整评估后，至少应输出：

1. 当前主要优势。
2. 当前主要短板。
3. 当前最该优先训练的 1-2 项方向。
4. 家长能听懂的解释口径。
5. 后续课程或训练建议。

建议输出顺序：

1. 先讲当前表现。
2. 再讲核心问题。
3. 再讲训练建议。
4. 最后讲后续观察点。

## 6. 常见评估场景

### 6.1 首次体测

1. 重点是建立基线，不追求一次说完所有问题。
2. 重点识别是否存在明显掉队项和风险项。

### 6.2 阶段复评

1. 重点看变化趋势，而不是只看单次高低。
2. 重点判断训练方向是否需要调整。

### 6.3 日常课堂观察

1. 重点看课堂真实状态下的理解、参与和完成度。
2. 重点补足单纯体测看不到的表现信息。

## 7. 课堂应用要求

1. 体测前说明流程，降低学员紧张感。
2. 体测中统一口令、统一记录口径。
3. 体测后不只报分，要结合课堂表现解释含义。
4. 阶段反馈时，把结果与课程目标对应起来。
5. 不同教练之间要尽量保持同一批项目的解释口径一致。

## 8. 家长沟通模板

建议按以下结构表达：

1. 先讲孩子当前做得好的部分。
2. 再讲最影响后续提升的关键短板。
3. 再讲我们训练会优先怎么做。
4. 最后讲家长当前最需要关注什么。

## 9. 沟通边界

1. 可以讲孩子当前在平衡、协调、力量、控制等方面的表现。
2. 可以讲建议优先改善的训练方向。
3. 不把单次测评结果讲成绝对结论。
4. 不替代医疗、康复、心理或教育诊断。

## 10. 常见误区

1. 只给分，不解释。
2. 只讲问题，不讲方向。
3. 只讲体测，不讲课堂观察。
4. 把评估结果说得过满。
