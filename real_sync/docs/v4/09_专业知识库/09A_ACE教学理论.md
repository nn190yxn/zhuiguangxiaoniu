# 09A_ACE教学理论

## 1. 文件定位

本文件用于统一追光小牛课堂对 `ACE` 的理解和使用方式。

它重点回答 3 个问题：

1. 好课堂为什么不能只看动作完成。
2. 教练在课堂里到底该观察什么。
3. 如何把 `ACE` 变成可执行的教学判断。

## 2. 核心定义

`ACE` 不是一个口号，而是课堂观察和教学设计的共同语言：

1. `A` 是 `Action`，指动作能力与身体执行。
2. `C` 是 `Cognition`，指理解、注意、判断、规则处理。
3. `E` 是 `Engagement`，指参与状态、情绪投入和课堂合作。

## 3. 基本原则

1. 不只看会不会做，还要看听不听得懂、愿不愿意参与。
2. 不把短期表现等同于长期能力。
3. 不拿单一动作成绩替代整体成长判断。
4. 不在家长面前夸大结论，必须用课堂事实说话。

## 4. ACE 的课堂判断顺序

建议教练按照以下顺序快速判断：

1. 先看 `E`，学员有没有进入课堂、有没有跟上节奏。
2. 再看 `C`，学员是否听懂规则、能否理解动作要求。
3. 最后看 `A`，学员动作完成度、控制力和质量如何。

如果 `E` 没进来，直接猛练动作通常效果差。

如果 `C` 没建立，只靠反复示范也容易越练越乱。

## 5. 教学应用要求

1. 设计活动时，必须同时考虑动作目标、认知要求和参与方式。
2. 调整课堂时，先判断问题是出在 `A`、`C` 还是 `E`。
3. 做反馈时，要能用 `ACE` 解释进步点和短板点。
4. 做阶段评估时，不能只保留动作结果，要保留课堂观察结论。

## 6. 课堂判断的最小闭环

建议教练每节课至少完成以下判断：

1. 本节课主目标主要落在 `A`、`C`、`E` 哪个维度。
2. 学员最卡的点是在动作不会、规则没懂，还是参与没进来。
3. 当堂调整是该降动作、简化规则，还是先收情绪和节奏。
4. 课后反馈时，要能说清本节课主要推进了哪个维度。

## 7. 常见课堂信号

### 6.1 A 动作信号

1. 平衡、协调、发力、节奏是否稳定。
2. 动作是否需要频繁身体辅助。
3. 同一动作是否能重复保持质量。

### 6.2 C 认知信号

1. 能否听懂一段指令。
2. 能否按顺序完成动作。
3. 能否在规则改变时及时调整。

### 6.3 E 参与信号

1. 是否愿意进场和参与。
2. 是否频繁分心、躲避或退出。
3. 是否能在课堂里保持互动与投入。

## 8. 常见课堂问题的 ACE 拆分

1. 学员一直不动，不一定只是懒，也可能是 `E` 没进来或 `C` 没听懂。
2. 学员动作乱，不一定只是能力差，也可能是 `C` 不清或节奏太快。
3. 学员动作能做但总断节奏，可能是 `E` 不稳或 `A` 稳定性不足。
4. 学员愿意参与但质量差，通常要优先处理 `A`。

## 9. 家长反馈建议结构

建议按以下顺序表达：

1. 今天孩子在哪个维度进入状态最快。
2. 当前最需要继续稳定的是哪个维度。
3. 我们下一步会先抓什么。
4. 家长在家里最适合怎么配合观察。

## 10. 对家长的解释边界

1. 可以说学员在动作、理解、参与哪个维度表现更突出。
2. 可以说当前更适合先稳参与，再提动作质量。
3. 不直接下医学诊断。
4. 不把课堂观察说成绝对能力结论。

## 11. 常见误区

1. 把 `ACE` 只理解成评估表，不用于课堂即时判断。
2. 只讲 `A`，忽略 `C` 和 `E`。
3. 看到课堂乱就一味加指令，不先判断到底卡在哪个维度。
4. 给家长反馈时只讲结果，不讲过程维度。
