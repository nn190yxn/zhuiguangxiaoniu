# 05C 学员升班考核标准

**版本：** v3.1
**适用对象：** 全体教练、教学主管、店长
**归口部门：** 教学部

---

## 一、制度

### 1.1 目的

建立科学、透明的升班考核机制，确保学员在合适的时间进入合适的级别，保障教学质量和学员发展。

### 1.2 适用范围

所有在籍学员的升班、留级、降级评估。

### 1.3 红线

1. **数据造假** → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
2. 伪造升班考核成绩 → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
3. 未经考核擅自升班/降级 → 书面警告+乐捐进成长基金（员工10元/店长20元）
4. 为完成升班率指标强制升班 → 书面警告+乐捐进成长基金（员工10元/店长20元）

## 二、要求

### 2.1 升班四大原则

| 原则 | 说明 |
|------|------|
| 以ACE评估为依据 | 升班必须基于ACE评估数据，不得凭主观判断 |
| 以学员发展为中心 | 升班是为了学员更好发展，不是为了完成指标 |
| 家长知情同意 | 升班前必须与家长充分沟通，取得家长同意 |
| 教练团队共识 | 升班决定需主教练+教学主管共同确认 |

### 2.2 升班考核周期

| 考核类型 | 频次 | 时间 |
|----------|------|------|
| 常规升班考核 | 每季度1次 | 每季度最后一周 |
| 专项升班考核 | 按需 | 教练或家长提出申请后2周内 |

### 2.3 升班基本条件

| 条件 | 要求 |
|------|------|
| ACE评估 | 最近一次评估≥75分 |
| 出勤率 | ≥80% |
| 在当前级别学习时长 | ≥3个月 |
| 教练推荐 | 主教练+教学主管同意 |

### 3.2 降级条件

| 条件 | 说明 |
|------|------|
| ACE评估连续2次<60分 | 建议降级，与家长充分沟通 |
| 出勤率连续2个月<60% | 评估是否适合当前级别 |
| 身体/心理原因 | 经家长同意后调整级别 |
| 安全风险 | 无法完成当前级别动作，存在安全隐患 |

### 3.3 升班率标准

| 等级 | 升班率 | 说明 |
|------|--------|------|
| 优秀 | ≥85% | 教学质量突出 |
| 合格 | ≥70% | 正常水平 |
| 待改进 | <70% | 需审查教学计划 |

## 四、流程SOP

### 4.1 升班考核流程

```
季度末 → 教练提交升班推荐名单（基于ACE评估+日常观察）
    → 教学主管审核 → 安排升班考核（2周内）
    → 升班考核（技能测试+课堂观察）
    → 教练+教学主管综合评定
    → 与家长沟通升班建议 → 家长同意
    → 办理升班手续 → 调整课表
    → 升班后1个月跟踪适应情况
```

### 4.2 降级处理流程

```
ACE连续2次<60分 / 出勤率连续2月<60%
    → 教练分析原因 → 与家长面谈沟通
    → 制定降级/留级方案 → 家长同意
    → 办理级别调整 → 调整课表
    → 降级后加强关注，每月复评
```

### 4.3 升班争议处理流程

```
家长不同意升班 → 了解顾虑 → 教练解释升班依据
    → 家长仍不同意 → 尊重家长意愿，暂不升班
    → 3个月后重新评估
教练与教学主管意见不一致 → 共同听课评估 → 达成共识
```

## 五、节点管理与跟踪考核

### 5.1 检查机制

| 检查项 | 频次 | 检查人 | 检查方式 |
|--------|------|--------|----------|
| 升班考核完成率 | 每季度 | 教学主管 | 考核记录表 |
| 升班率 | 每季度 | 店长 | GYMGO数据 |
| 家长沟通完成率 | 每季度 | 店长 | CRM记录 |
| 升班后适应跟踪 | 升班后1个月 | 教练 | 课堂观察+家长反馈 |
| 降级处理合规性 | 每季度 | 教学主管 | 降级记录表 |

### 5.2 考核标准

| 指标 | 达标 | 未达标处理 |
|------|------|------------|
| 升班率 | ≥70% | <70%→教学主管审查教学计划，制定改进方案 |
| 升班考核完成率 | 100% | 漏1人→口头警告 |
| 家长沟通完成率 | 100% | 未经家长同意升班→书面警告 |
| 升班后适应跟踪 | 100% | 漏1人→口头警告 |

### 5.3 数据造假处理

- 伪造升班考核成绩 → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
- 为完成指标强制升班 → 书面警告+乐捐进成长基金（员工10元/店长20元）
