# 05A ACE落地执行标准

**版本：** v2.2
**适用对象：** 全体教练、教学主管、店长
**归口部门：** 教学部

---

## 一、制度

### 1.1 目的

建立追光小牛独家ACE三维评估体系，科学评估学员发展，为教学改进和升班决策提供数据支撑。

### 1.2 ACE定义

| 维度 | 英文全称 | 中文含义 | 权重 |
|------|----------|----------|------|
| A | Athletic | 运动能力 | 30分 |
| C | Cognitive | 认知能力 | 40分 |
| E | Engagement | 参与动能 | 30分 |

**总分：100分**

### 1.3 ACE等级

| 等级 | 分数 | 说明 |
|------|------|------|
| 优秀 | ≥90分 | 表现突出，可考虑加速进阶 |
| 合格 | 70-89分 | 正常发展，按计划推进 |
| 待改进 | 60-69分 | 需要关注，制定专项提升计划 |
| 不合格 | <60分 | 需要重点干预，教练+家长面谈 |

### 1.4 红线

1. **数据造假** → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
2. 伪造ACE评估数据 → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
3. 未按规定周期完成ACE评估 → 口头警告
4. ACE评估结果未及时告知家长 → 口头警告

## 二、要求

### 2.1 评估周期

| 评估类型 | 频次 | 时间节点 |
|----------|------|----------|
| 入学评估 | 1次 | 入学第1周内 |
| 阶段评估 | 每季度1次 | 每季度最后一周 |
| 升班评估 | 按需 | 升班前1周 |
| 月度观察记录 | 每月1次 | 每月最后一周 |

### 2.2 评估要求

- 每次评估必须有具体观察记录，不得凭印象打分
- 评估完成后3天内将结果录入GYMGO系统
- 评估完成后7天内与家长沟通评估结果
- 同一学员的评估应由主教练完成，副教练可补充意见

### 2.3 E维度底线标准

E维度（参与动能）是学员持续学习的基础，以下为底线标准：

| 指标 | 底线标准 | 说明 |
|------|----------|------|
| 出勤率 | ≥80% | 低于80%需启动出勤干预 |
| 续费率 | ≥35%（Q2起步） | 低于35%需分析原因并改进 |
| 升班率 | ≥70% | 低于70%需审查教学计划 |

## 三、标准

### 3.1 A维度 — 运动能力（30分）

| 评估项 | 分值 | 评分标准 |
|--------|------|----------|
| 技能完成度 | 10分 | 10分：独立完成标准动作；7分：需少量辅助完成；4分：需大量辅助；1分：无法完成 |
| 身体素质 | 10分 | 10分：力量/速度/柔韧/协调均达标；7分：3项达标；4分：2项达标；1分：1项达标 |
| 核心痛点记录 | 10分 | 10分：完整记录痛点+进步轨迹；7分：有记录但不够详细；4分：偶尔记录；1分：无记录 |

### 3.2 C维度 — 认知能力（40分）

| 评估项 | 分值 | 评分标准 |
|--------|------|----------|
| 指令听从 | 10分 | 10分：首次指令即执行；7分：需重复1次；4分：需重复2-3次；1分：需多次重复仍难执行 |
| 专注时长 | 10分 | 10分：全程专注（45min）；7分：专注30min以上；4分：专注20min以上；1分：专注不足20min |
| 规则理解 | 10分 | 10分：完全理解并遵守；7分：基本理解偶尔违规；4分：部分理解经常违规；1分：难以理解规则 |
| 反应速度 | 10分 | 10分：反应迅速准确；7分：反应正常偶有延迟；4分：反应较慢；1分：反应明显迟缓 |

### 3.3 E维度 — 参与动能（30分）

| 评估项 | 分值 | 评分标准 |
|--------|------|----------|
| 兴趣投入度 | 10分 | 10分：主动参与、积极尝试；7分：能参与但不够主动；4分：需鼓励才参与；1分：抗拒参与 |
| 情绪状态 | 10分 | 10分：情绪稳定积极；7分：偶有情绪波动但能自我调节；4分：经常情绪波动需安抚；1分：情绪失控频繁 |
| 社交互动 | 10分 | 10分：主动与同伴互动合作；7分：能配合但不够主动；4分：被动参与互动；1分：拒绝互动 |

### 3.4 ACE评估等级与行动

| 等级 | 分数 | 行动要求 |
|------|------|----------|
| 优秀 | ≥90分 | 通报表扬，考虑加速进阶，推荐参加竞赛/展演 |
| 合格 | 70-89分 | 正常推进，持续关注弱项提升 |
| 待改进 | 60-69分 | 制定专项提升计划，增加练习频次，2周后复评 |
| 不合格 | <60分 | 教练+家长面谈，分析原因，制定1个月干预计划，每月复评 |

## 四、流程SOP

### 4.1 ACE评估执行流程

```
评估准备（查阅上次评估记录+本阶段教学目标）
    → 课堂观察（至少观察2节课）
    → 逐项评分（A/C/E各维度打分）
    → 撰写评语（亮点+需提升+建议）
    → 录入系统（3天内完成GYMGO录入）
    → 家长沟通（7天内完成面谈/电话）
    → 制定下阶段教学计划
```

### 4.2 待改进/不合格学员干预流程

```
ACE评估<70分 → 教练分析原因 → 制定专项提升计划
    → 与家长面谈沟通 → 增加练习频次/调整教学方式
    → 2周后复评 → 合格→恢复正常；仍不合格→升级干预
    → 连续2次不合格 → 教学主管+店长介入评估
```

### 4.3 E维度底线预警流程

```
出勤率<80% → 顾问+教练联系家长了解原因 → 制定出勤改善计划
续费率<35% → 店长+教练分析教学问题 → 改进教学方案
升班率<70% → 教学主管审查教学计划 → 调整教学节奏
```

## 五、节点管理与跟踪考核

### 5.1 检查机制

| 检查项 | 频次 | 检查人 | 检查方式 |
|--------|------|--------|----------|
| ACE评估完成率 | 每季度 | 教学主管 | GYMGO系统数据 |
| ACE评估及时率 | 每季度 | 教学主管 | 系统录入时间 |
| 家长沟通完成率 | 每季度 | 店长 | CRM沟通记录 |
| ACE合格率 | 每季度 | 教学主管 | GYMGO数据统计 |
| E维度底线达标率 | 每月 | 店长 | 出勤/续费/升班数据 |

### 5.2 考核标准

| 指标 | 达标 | 未达标处理 |
|------|------|------------|
| ACE评估完成率 | 100% | 漏评1人→口头警告；漏评3人→书面警告 |
| ACE评估及时率 | 100% | 超时1次→口头警告 |
| ACE合格率 | ≥85% | <85%→教学主管约谈教练，制定改进计划 |
| 家长沟通完成率 | 100% | 漏1次→口头警告 |
| E维度底线达标率 | 100% | 不达标→店长+教练专项帮扶 |

### 5.3 数据造假处理

- ACE评估分数造假 → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
- 伪造评估记录 → 首次书面警告+乐捐进成长基金（员工10元/店长20元），二次辞退
