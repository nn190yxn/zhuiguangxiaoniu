# 追光小牛销售基础专业培训门店带教追踪表

**版本：** v1.0  
**适用对象：** 店长、讲师、门店带教负责人  
**用途：** 跟踪新人和在岗销售的学习、抽测、演练、通关和实战落地情况

---

## 一、使用说明

1. 本表不是签到表，而是全过程带教记录表。
2. 每位学员建议单独建立一页或一条持续记录。
3. 每周至少更新 1 次，每月通关后必须更新 1 次。
4. 表内重点看趋势，不只看单次得分。

## 八、使用建议

1. 店长每周复盘时先看趋势最差的两项，不要平均用力。
2. 对新人，优先看低龄、高龄、协同三项能否过线。
3. 对在岗销售，优先看课程承接和异议处理是否真正改善。
4. 本表建议与门店录音、演练评分表一起归档。
