# 追光小牛销售与教练协同训练体系

**版本：** v1.0  
**定位：** 面向销售、教练、店长的联合训练体系，目标是让接待、体验、反馈、成交、首月服务与续课衔接真正形成一条线  
**适用对象：** 课程顾问、教练、店长、讲师  
**前置要求：** 销售和教练均已完成各自基础层训练

---

## 一、训练定位

这套训练不是简单地把销售和教练拉到同一教室里上课，而是要解决两个岗位在真实业务链里的断点问题。

当前门店最常见的协同问题通常有：

1. 销售前采不完整，教练体验前拿不到有效信息。
2. 教练反馈专业，但顾问接不住。
3. 顾问为了成交越位，抢教练结论。
4. 教练讲完就走，顾问谈单像重新开始一场对话。
5. 签约后交接不完整，首月服务和后续续课衔接断点。

所以这套训练的目标，不是让两边都更努力，而是让两边的动作接上。

---

## 二、训练目标

完成这套训练后，销售、教练、店长应形成五项协同结果：

1. 前采信息收集、体验观察和反馈承接成为同一条链。
2. 教练反馈后，顾问能自然承接，不冷场、不重来、不抢位。
3. 成交和首月服务之间有清楚交接，不出现签完就断层。
4. 续课、升级、重点家庭推进能形成联动动作。
5. 店长能检查这条链条哪里断、谁来补、怎么纠偏。

---

## 三、协同训练的核心原则

1. 销售和教练不是各说各话，而是说同一件事的不同部分。
2. 销售负责把家长听得懂的方向讲清，教练负责把专业判断和训练重点讲清。
3. 教练不越位新签主责，销售不越位正式评估结论。
4. 店长不替代一线动作，但必须检查断点。
5. 所有协同训练必须围绕真实门店节点，不做空泛 teamwork 培训。

---

## 四、能力模型

建议把协同训练统一为六项能力。

### 1. 前采与信息交接能力

要求：

1. 销售能收全关键信息。
2. 教练体验前能拿到有效信息。
3. 店长知道前采表单和关键信息是否完整。

### 2. 体验现场分工能力

要求：

1. 销售知道体验中自己看什么。
2. 教练知道自己讲什么、什么时候讲。
3. 双方不抢位、不掉位。

### 3. 反馈承接能力

要求：

1. 教练反馈后，顾问能做家长语言翻译。
2. 顾问能自然转入课程方案。
3. 教练讲完后不会把空间全部占死。

### 4. 成交与交接能力

要求：

1. 签约时双方知道自己的动作边界。
2. 签后教练能接到完整交接信息。
3. 家长感受到的是同一套服务，不是换人重来。

### 5. 首月服务与续课协同能力

要求：

1. 顾问知道首月要跟什么。
2. 教练知道首月要反馈什么。
3. 续课前双方知道怎么配合铺垫。

### 6. 问题家庭联合推进能力

要求：

1. 遇到犹豫家庭、风险会员、重点家庭时，双方知道谁主说、谁补充、谁跟进。
2. 店长知道什么时候要介入升级。

---

## 五、课程模块设计

建议按 6 个模块展开。

### 模块 A：完整业务链路共识训练

**培训目标**

让销售、教练、店长先看到自己其实在同一条链上，而不是各管一段。

**主要内容**

1. 新客从邀约到成交的完整链路。
2. 成交到首月服务的完整链路。
3. 首月到阶段反馈、续课与升级的链路。
4. 三岗在每一段的职责划分。

### 模块 B：前采与体验前交接训练

**培训目标**

让体验开始前，销售和教练就站在同一信息面上。

**主要内容**

1. 前采必须收哪些信息。
2. 家长目标和孩子基础情况怎么交接给教练。
3. 哪些信息不全会直接影响体验和反馈质量。
4. 店长如何检查体验前信息完整度。

### 模块 C：体验现场分工与观察训练

**培训目标**

让销售和教练知道体验中自己各自该看什么、做什么。

**主要内容**

1. 销售体验中重点观察孩子参与状态和家长反应。
2. 教练体验中重点观察动作、配合、规则、体能、表现。
3. 什么情况下销售可以补充，什么情况下必须交给教练。
4. 体验现场常见协同错误与纠偏。

### 模块 D：反馈承接与联合成交训练

**培训目标**

让教练和销售形成“专业支撑 + 成交承接”的完整闭环。

**主要内容**

1. 教练反馈标准结构。
2. 顾问承接标准结构。
3. 低龄家庭的联合反馈怎么走。
4. 高龄家庭和专项目标家庭怎么走。
5. 顾问不能抢结论，但必须会推进方案。

### 模块 E：签约交接与首月服务协同训练

**培训目标**

让签约不是终点，而是服务链的起点。

**主要内容**

1. 签约后销售要交接什么。
2. 教练首课前应知道什么。
3. 首课后、7 天、首次体测、阶段反馈的协同节点。
4. 家长在首月最容易出现什么问题，双方怎么配合处理。

### 模块 F：续课、升级与问题家庭联合推进训练

**培训目标**

让续课和重点家庭处理不再各自为战。

**主要内容**

1. 续课窗口期双方各自做什么。
2. 升级课程时，谁主讲路径、谁主讲专业变化。
3. 犹豫型家庭、风险会员、体验感差家庭如何联合推进。
4. 店长介入升级的判断标准。

---

## 六、建议训练方式

1. 必须采用双人或三人角色演练。
2. 同一场景要同时练“教练怎么讲”“顾问怎么接”。
3. 必须保留录音或现场视频复盘。
4. 店长必须参与点评，否则协同问题很难真正收口。

建议结构：

1. 15 分钟讲链路。
2. 20 分钟拆案例。
3. 35 分钟双人 / 三人演练。
4. 20 分钟点评和重演。

---

## 七、考核建议

建议重点考 4 类协同能力：

1. 前采和交接是否完整。
2. 教练反馈和顾问承接是否顺畅。
3. 签约后交接和首月服务是否连续。
4. 问题家庭是否知道如何联合推进。

建议考核形式：

1. 双人情景模拟。
2. 签约交接检查题。
3. 首月节点协同题。
4. 风险家庭升级处理题。

---

## 八、店长检查重点

店长在这套协同训练里不是替代执行，而是负责看断点。

建议重点检查：

1. 前采是否完整。
2. 体验后顾问是否接住反馈。
3. 签后交接是否完整。
4. 首月服务节点是否覆盖。
5. 续课前双方是否已有铺垫。

---

## 九、结论

这套 `销售与教练协同训练体系` 的意义，不是单独提升销售，也不是单独提升教练，而是让门店最核心的业务链从“两段话”变成“一条线”。

它解决的是：

1. 两岗之间的断点。
2. 体验和成交之间的断点。
3. 签约和服务之间的断点。
4. 续课和协同推进之间的断点。
