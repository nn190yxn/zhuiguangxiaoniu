# 09D_七大身体素质

## 1. 文件定位

本文件用于统一追光小牛对七大身体素质的定义、观察口径和训练方向，避免课堂训练只有动作名称、没有能力目标。

## 2. 七大身体素质

1. 平衡力：保持身体稳定和调整重心的能力。
2. 协调力：身体多个部位按顺序、按节奏协同完成动作的能力。
3. 柔韧性：关节活动范围和肌肉伸展能力。
4. 力量：肌肉产生和输出力量的能力。
5. 肌力与肌耐力：持续控制和重复输出的能力。
6. 灵敏性：快速变向、快速调整姿态和快速应对变化的能力。
7. 速度与反应：快速启动、快速完成动作和快速响应信号的能力。

## 3. 教学原则

1. 先基础控制，再专项表现。
2. 先质量稳定，再追求数量和强度。
3. 先动作模式正确，再追求难度升级。
4. 同一课堂可以有多个素质目标，但必须有主目标。

## 4. 课程设计时的判断顺序

1. 这节课主练哪一项身体素质。
2. 该素质在这条课程线上应该怎么体现。
3. 学员当前是缺基础、缺稳定，还是缺速度和输出。
4. 当堂安排应该先打底、先重复，还是先做变化。

## 5. 课堂观察要点

1. 平衡看能否稳住和找回重心。
2. 协调看动作是否连贯、顺序是否正确。
3. 柔韧看动作范围是否受限、代偿是否明显。
4. 力量看发力是否有效，而不是只看表面用力。
5. 肌耐力看重复执行后质量是否快速下滑。
6. 灵敏看切换是否及时、身体是否发散失控。
7. 速度与反应看启动是否迟缓、信号处理是否跟得上。

## 6. 常见课堂表现对应

1. 总站不稳、落地乱，通常先看平衡和控制。
2. 会单个动作但连不起来，通常先看协调。
3. 动作做得出来但很快塌，通常先看肌耐力。
4. 起动慢、反应慢，通常先看速度与反应。
5. 切换方向总失控，通常先看灵敏和重心控制。

## 7. 训练安排要求

1. 热身阶段优先激活与准备，不直接堆强度。
2. 主体阶段围绕一个主素质目标展开。
3. 游戏和对抗设计要服务主目标，不是只求热闹。
4. 拉伸和放松用于恢复，不用于替代主体训练。

建议训练推进顺序：

1. 先做低复杂度基础动作。
2. 再做重复稳定。
3. 再做节奏变化或速度变化。
4. 最后再放进游戏或应用场景。

## 8. 对家长的反馈口径

1. 说清今天主练的是哪一项能力。
2. 说清孩子当前更突出的问题是稳定性、控制力还是反应速度。
3. 说清下一阶段建议继续加强什么。
4. 不用模糊词替代具体表现。

## 9. 常见误区

1. 只按动作名安排训练，不按素质目标安排。
2. 只追数量，不看动作质量。
3. 只看单次高光表现，不看稳定重复能力。
4. 给家长反馈时只说“体能差”这种笼统结论。
