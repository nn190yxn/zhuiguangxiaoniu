# 追光小牛销售基础专业培训 PPT 详细内容稿

**版本：** v1.1  
**适用对象：** 讲师、店长、课件制作人员  
**定位：** 可直接用于制作正式培训课件的逐页详细内容稿  
**配套文件：** `销售基础专业培训体系总方案_正式版.md`、`销售专业知识卡片包_正式版.md`、`销售专业知识抽测题库_正式版.md`、`销售场景演练脚本_正式版.md`

---

## 一、使用说明

1. 这份文件不是简单页纲，而是逐页内容稿。
2. 每一页都包含建议标题、页面核心内容、页面排版建议、讲师备注和互动方式。
3. 课件制作时可以直接按本稿填充页面，不需要再从零整理内容。
4. PPT 仍然是带训辅助工具，不替代知识卡、题库和场景演练。
5. 整体风格建议统一为“判断 -> 翻译 -> 承接 -> 边界”。

---

## 二、整体设计方向

1. 风格上不要做成传统说教式培训课件，要更像实战训练课件。
2. 视觉上建议采用“问题场景 + 结构拆解 + 标准表达”的组合，不要整页堆大段字。
3. 页面中尽量多出现家长原话、孩子表现、课程匹配逻辑和错误说法对照。
4. 专业页必须让销售看完以后能直接知道“这句话怎么跟家长讲”。
5. 低龄与高龄必须明显分色或分标签，避免培训时讲混。

建议颜色方向：

1. 品牌和课程页：用品牌主视觉色，强调统一感。
2. 低龄页：用更轻、更柔和的色系，突出成长底子。
3. 高龄页：用更明确、更有力量感的色系，突出结果、体测和表现。
4. 演练与考核页：用结构化信息块，突出行动感。

## 三、逐页详细内容

### 第 1 页：封面

**页面标题**

追光小牛销售基础专业培训

**页面副标题**

让销售听得懂专业、说得出专业、接得住反馈、转得动课程

**页面内容建议**

1. 主标题放页面中间，副标题放标题下方。
2. 右下角放培训对象：课程顾问 / 店长 / 讲师。
3. 页面底部放日期、讲师姓名、门店或项目名称。

**页面排版建议**

1. 封面视觉不要塞太多字。
2. 可使用孩子训练中的真实动作图或门店训练场景图。
3. 视觉重点要体现“专业训练”而不是“普通销售课”。

**讲师备注**

开场直接讲三句话：

1. 这不是单纯销售技巧培训。
2. 这也不是把销售培养成教练。
3. 这是让销售在门店场景里能做基础专业表达和课程承接。

### 第 2 页：为什么门店现在必须做这套培训

**页面标题**

为什么现在必须做销售基础专业培训

**页面核心内容**

左侧放“当前常见问题”，右侧放“这套培训要解决什么”。

左侧建议写：

1. 销售会接待，但不会讲专业。
2. 体测讲完像报成绩，讲不出背后逻辑。
3. 教练反馈后，顾问接不住。
4. 课程推荐常常变成“推自己最熟的一门课”。
5. 门店不同人说法不统一，家长听到的信息不一致。

右侧建议写：

1. 统一品牌和课程口径。
2. 让销售会做基础专业翻译。
3. 让销售知道怎么和教练配合。
4. 让课程推荐更像方案，而不是卖课。
5. 让门店形成可训练、可检查、可复制的体系。

**页面排版建议**

1. 两栏对照式排版。
2. 左侧可以配“旧状态”，右侧配“目标状态”。
3. 建议加一条底部总结条：从“靠个人经验”升级为“靠统一体系”。

**互动方式**

请现场学员回答：门店当前最常见的三个表达问题是什么。

**讲师备注**

这一页要先让大家认同问题存在，否则后面知识内容容易被认为“只是多学一点东西”。

### 第 3 页：本次培训最终要做到什么

**页面标题**

这套培训最终要的不是知识，而是结果

**页面核心内容**

页面中部放 4 个结果模块：

1. 听得懂专业
2. 说得出专业
3. 接得住教练反馈
4. 转得动课程方案

每个模块下补一句解释：

1. 听得懂专业：听到体测、感统、表现时，知道背后在说什么。
2. 说得出专业：能用家长听得懂的话讲清楚，不堆术语。
3. 接得住教练反馈：教练讲完后，顾问能自然承接，不冷场、不抢位。
4. 转得动课程方案：能从孩子表现和目标出发，给到合理课程建议。

**页面排版建议**

1. 用四象限或四步箭头图。
2. 每个模块配一个关键词图标：耳朵、嘴巴、接力、路径。

**讲师备注**

强调：这套培训不是让大家会背术语，而是让家长更愿意听、听得懂、听完愿意接受方案。

### 第 4 页：销售与教练的边界必须先讲清

**页面标题**

销售能做什么，不能做什么

**页面核心内容**

页面分三块：

**销售可以做的**

1. 讲孩子当前表现。
2. 讲基础影响。
3. 讲基础训练方向。
4. 讲课程建议和成长路径。

**销售不能做的**

1. 不能给孩子下诊断。
2. 不能替代教练做正式评估结论。
3. 不能说“肯定会怎样”。
4. 不能承诺结果。

**教练主责的内容**

1. 正式评估和专业判断。
2. 训练重点和教学安排。
3. 深度反馈与训练结论。

**页面底部总结文案**

销售负责把家长听得懂的方向讲清楚，教练负责把专业判断和训练安排做扎实。

**页面排版建议**

1. 用三栏表格最合适。
2. 错误表达可用红色小标签标出来，如“感统有问题”“肯定能长高”。

**讲师备注**

这页一定要讲透，因为后面所有专业表达都建立在不越界的前提上。

### 第 5 页：品牌总口径不是卖课，是卖成长方案

**页面标题**

追光小牛卖的不是单节课，而是一套运动成长方案

**页面核心内容**

主文案建议直接上：

1. 运动成长课是每个孩子的必修课。
2. 追光小牛做的不是单节课销售，而是儿童运动成长方案。
3. 先体测、再判断、再建议课程，是因为孩子要的是适合当下阶段的路径，不是盲选课程。

补充说明建议写成三条：

1. 孩子不是只缺一门课，而是需要合适的成长入口。
2. 门店价值不只是“上课”，更在于“判断 + 承接 + 反馈 + 陪伴”。
3. 课程只是载体，成长路径才是核心。

**页面排版建议**

1. 上半页主 slogan。
2. 下半页用三条价值解释。
3. 可以加一句家长语言版本：不是今天卖你一门课，而是帮你把孩子现在该先抓什么看清楚。

**讲师备注**

这一页要帮销售从“卖单门课”的思路，切换到“做成长方案入口”的思路。

### 第 6 页：ACE 体系怎么一句话讲给家长听

**页面标题**

ACE 体系不是专业词，而是家长能听懂的价值结构

**页面核心内容**

建议页面中央做 ACE 三字母展开：

1. A = Athletic，运动能力
2. C = Cognitive，认知能力
3. E = Engagement，参与度

每个字母下建议补一行家长化表达：

1. A：孩子身体底子更好，动作更稳、更敢动、更有表现。
2. C：运动不只是练身体，也在帮孩子建立专注、反应和大脑使用效率。
3. E：孩子更愿意参与、更敢进入集体、更愿意持续做一件事。

底部建议放一句统一话术：

ACE 用家长能听懂的话来说，就是练好身体、开发大脑、爱上运动。

**页面排版建议**

1. 三栏卡片式布局。
2. 每一栏不超过三行，避免太满。

**互动方式**

请两位学员各用 20 秒讲一遍 ACE，不准照读英文。

**讲师备注**

提醒大家：ACE 不是为了显得专业，而是为了让家长快速理解品牌为什么不只是教动作。

### 第 7 页：课程全景图要讲成“入口”，不是菜单

**页面标题**

门店课程体系全景图

**页面核心内容**

页面中部建议做 5 个课程模块：

1. 快乐体操：低龄基础入口，练感统、平衡、协调、胆量、参与感。
2. 体能训练课 / 体能强化课：系统补体能底盘，适合多项基础一起补。
3. 体能跳绳课：专项目标明确，强化节奏、协调、持续控制和达标能力。
4. 增高体能课：围绕成长阶段的科学运动基础，不做结果承诺。
5. 篮球体能课：把运动兴趣、团队参与、规则意识和社交表达连起来。

底部建议补一句：

不是哪门课更好卖，而是哪门课更适合孩子现在这个阶段。

**页面排版建议**

1. 用横向路径图或卡片矩阵图。
2. 每门课只写“定位 + 适合谁 + 核心价值”三件事。

**讲师备注**

这一页的作用不是详细讲完所有课程，而是先让销售脑中有全景图。

---

### 第 8 页：课程推荐逻辑不是先推课，而是先判断

**页面标题**

课程推荐逻辑总图

**页面核心内容**

用四步流程写清：

1. 先看年龄：低龄和高龄的入口逻辑不同。
2. 再看表现：是感统、参与感、协调、底盘，还是专项目标更突出。
3. 再看家长当前目标：家长最想解决的是参与、体测、身高、社交还是专项结果。
4. 最后做课程分流：先确定入口课，而不是直接把所有课并列介绍一遍。

下方补“常见匹配逻辑”：

1. 低龄、感统、胆量、规则感弱，优先快乐体操。
2. 整体体能底子薄，优先体能训练课 / 体能强化课。
3. 跳绳目标明确，且基础能接住，优先体能跳绳课。
4. 身高方向和整体身体状态是重点，可考虑增高体能课。
5. 中大龄孩子需要团队参与和兴趣入口，可衔接篮球体能课。

**页面排版建议**

1. 上面放 4 步流程。
2. 下面放 5 条匹配逻辑。

**讲师备注**

这页是后面所有课程承接页的总入口，要讲清“先判断，后推荐”。

### 第 9 页：为什么销售必须懂七大身体素质

**页面标题**

七大身体素质不是术语，是孩子表现背后的底层能力

**页面核心内容**

建议主文案写：

1. 家长看到的是孩子现在的表现，销售要能听懂背后的基础。
2. 很多家长说的是“坐不住”“不敢玩”“体育差”“总慢半拍”，背后常常不是一句“体能差”能解释的。
3. 七大身体素质的意义，不是为了背定义，而是为了更准确地判断、翻译和承接课程。

底部可放七项关键词：

1. 力量
2. 速度
3. 灵敏
4. 协调
5. 平衡
6. 柔韧
7. 耐力

**页面排版建议**

1. 上半页写“为什么重要”。
2. 下半页做 7 个关键词标签。

**讲师备注**

这一页不展开细讲每个能力，核心是先让销售知道为什么要学它们。

### 第 10 页：力量与速度页的详细内容

**页面标题**

力量和速度，家长最容易误解的两个基础点

**页面核心内容**

左栏写“力量”：

1. 在看什么：能不能把身体撑住、推起来、带起来、稳下来。
2. 常见表现：坐姿塌、趴桌、跳不起来、支撑时间短、玩一会就累。
3. 家长能听懂的话：不是完全不会，而是发力底子不够，所以很多动作能开始但做不稳、做不久。
4. 常见承接：低龄先快乐体操，中大龄先体能训练课 / 体能强化课。

右栏写“速度”：

1. 在看什么：听到指令后，能不能快速启动、反应、完成。
2. 常见表现：起动慢、反应慢、节奏一快就掉队。
3. 家长能听懂的话：不是孩子不积极，而是从听到到做出来这一步偏慢。
4. 常见承接：低龄先练反应和参与状态，高龄练启动、切换、折返和追逐。

**页面底部统一提醒**

家长最容易把力量弱理解成“懒”，把速度弱理解成“不积极”，销售要负责把误解翻译回来。

**页面排版建议**

1. 两栏对照式最佳。
2. 每栏都放“看什么 / 表现 / 家长语言 / 课程承接”四行结构。

**讲师备注**

讲这一页时，不要只讲定义，要多讲家长平时最常说出来的话。

### 第 11 页：灵敏、协调、平衡页的详细内容

**页面标题**

灵敏、协调、平衡，很多“看着会却做不出来”的问题都在这里

**页面核心内容**

页面分三块：

**灵敏**

1. 关键词：遇到变化慢半拍。
2. 表现：变向慢、规则一变就乱、快节奏活动跟不上。
3. 家长语言：孩子不是完全不会，而是变化一来身体调整慢。

**协调**

1. 关键词：会但做不出来。
2. 表现：连续动作乱、模仿慢、组合动作不顺。
3. 家长语言：脑子知道怎么做，但身体配合还没连起来。

**平衡**

1. 关键词：不稳和不敢。
2. 表现：单脚站不稳、走窄线晃、跳落地容易散。
3. 家长语言：很多孩子不是单纯胆小，而是身体稳不住，所以更不敢把动作做出去。

底部建议补一句：

这三项常常直接影响孩子的参与感、胆量、展示意愿和集体活动状态。

**页面排版建议**

1. 三个竖向卡片。
2. 每个卡片控制 4 行核心内容。

**讲师备注**

这一页很重要，因为低龄家长最常提到的“胆小”“不敢”“笨手笨脚”，很多都能从这里找到入口。

---

### 第 12 页：柔韧与耐力页的详细内容

**页面标题**

柔韧和耐力，不只是动作问题，还影响孩子的舒服度和持续性

**页面核心内容**

左栏写“柔韧”：

1. 在看什么：身体是否紧、动作是否打开。
2. 常见表现：弯不下去、动作幅度小、做动作显得僵。
3. 家长语言：孩子身体比较紧，所以很多动作会更费劲、更放不开。

右栏写“耐力”：

1. 在看什么：能不能持续输出，不是一开始能不能做。
2. 常见表现：一动就累、后程掉线、做一会就想停。
3. 家长语言：很多孩子不是不愿坚持，而是体能承载不够，所以越做越没状态。

底部建议补一句：

柔韧影响动作舒适度，耐力影响持续参与度，这两项都会影响孩子愿不愿意继续运动。

**互动方式**

抽 3 位学员，各自用 20 秒把“柔韧弱”或“耐力弱”讲成家长语言。

**讲师备注**

强调：销售讲这两项时，千万不要只停留在体测分数上。

### 第 13 页：为什么体测一定要分龄讲

**页面标题**

同样是体测，低龄和高龄完全不是同一套讲法

**页面核心内容**

左侧写低龄逻辑：

1. 更看发展基础。
2. 更看感统、参与感、胆量、规则适应。
3. 更少讲分数，多讲底子。

右侧写高龄逻辑：

1. 更看课堂状态和体育表现。
2. 更看体测达标、任务完成、自信和同伴比较。
3. 更强调结果影响，但不能只报结果。

页面底部总结：

低龄重点是“底子有没有建起来”，高龄重点是“底子已经怎样影响结果和状态”。

**页面排版建议**

1. 左右对照式。
2. 页面中间可以加一条分割线：同样是体测，不同年龄，讲法不同。

**讲师备注**

这一页讲完，后面分龄体测页才不会混。

### 第 14 页：3~6 岁体测解读主线

**页面标题**

3~6 岁体测解读主线：不是追分数，而是看底子

**页面核心内容**

页面可用四个模块：

1. 感统基础：孩子能不能把身体和环境信息接收好、整理顺。
2. 动作建立：基础动作有没有连起来，不只是会不会单个动作。
3. 参与感与胆量：敢不敢动、愿不愿意试、进不进入集体。
4. 规则适应：排队、切换、跟随、集体节奏能不能接住。

底部建议补一句统一话术：

低龄体测不要上来就讲“差了多少”，而要讲“现在最该先把哪块底子带起来”。

**页面排版建议**

1. 四宫格排版最合适。

1. 每周至少 1 次，60 至 90 分钟。
2. 必须包含抽测、演练、点评、记录。
3. 四周轮转：品牌课程、体测解读、低龄沟通、协同异议。

下半页写“月度通关”：

1. 每月 1 次正式通关。
2. 考知识抽测、体测口述、场景演练、课程承接、协同意识。
3. 85 分以上可独立接待，70 至 84 分需继续带教，70 分以下进入复训。

底部再加“带教留痕”：

所有结果都要回到 `门店带教追踪表_正式版.md`，否则培训会变成一次性活动。

**页面排版建议**

1. 上下两层结构，最清楚。
2. 可在右侧补一个小闭环图：抽测 -> 演练 -> 通关 -> 追踪 -> 复训。

**讲师备注**

这一页是把培训从“上课”拉回“管理机制”的关键页。

### 第 32 页：结尾行动页

**页面标题**

培训结束以后，门店马上要做什么

**页面核心内容**

建议直接给行动清单：

1. 本周先完成知识卡和课程卡第一轮学习。

页面底部建议放一句收尾文案：

这套培训不是为了多一堆资料，而是为了让门店每个人都更会讲、更会接、更会转、更能配合。

**页面排版建议**

1. 行动清单式最直接。
2. 结尾页建议视觉简洁，不要太满。

**讲师备注**

最后一定把行动说具体，不要停留在“大家回去多练”。

## 四、制作补充建议

1. 所有专业页都建议加一个“家长能听懂的话”模块，避免课件重新被做成术语课。
2. 课程页建议加“适合谁 / 不要怎么推”两条，避免课件做成产品宣传页。
3. 演练页尽量加入真实家长原话，增强代入感。
4. 如后续正式制作 `.pptx`，建议每页控制在 30 至 80 字主文案，加讲师备注页做补充。
5. 如果时间只够做精简版，优先保留第 2、4、5、8、10、11、13、14、17、18、19、22、23、25、26、27、28、29、31、32 页。
