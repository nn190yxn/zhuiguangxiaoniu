# 追光小牛店长带教与经营闭环配套表单包

**版本：** v1.0  
**定位：** 作为 `店长带教与经营闭环训练体系_正式版.md` 的执行工具包，帮助店长把训练组织、过程检查、问题带教和经营复盘真正落地  
**适用对象：** 店长、储备店长、骨干带教人、讲师、培训负责人

---

## 一、使用说明

这份表单包不是额外增加复杂流程，而是把店长本来就要做的动作，收成可执行、可追踪、可复盘的格式。

建议使用原则：

1. 一张表只解决一类动作，不要把所有信息堆进同一张表里。
2. 表单填写要尽量基于真实案例、真实名单、真实数据，不写空话。
3. 所有表单都要能回收出下一步动作，而不只是留痕。
4. 店长使用表单时，重点不是写得多，而是抓得准、跟得动、复得出来。

---

## 二、表单 1：门店周训练与检查排期表

**适用场景**

用于店长安排一周内的销售训练、教练训练、协同训练和跟场检查节奏。

**建议使用频率**

每周一制定，每周六复盘。

**表单模板**

| 周次 | 本周关键目标 | 训练对象 | 训练主题 | 训练形式 | 时间安排 | 负责人 | 检查方式 | 本周达标标准 | 复盘时间 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| 第__周 | 例如：提升体验后承接质量 | 销售 / 教练 / 联合 | 例如：反馈承接 | 讲解 / 演练 / 跟场 |  |  |  |  |  |

**填写提示**

1. 本周关键目标不要超过 2 个，避免训练目标过散。
2. 训练主题必须和门店当前断点直接相关。
3. 检查方式应明确是录音复盘、现场跟场、抽测还是结果复查。

---

## 三、表单 2：门店跟场与录音复盘检查表

**适用场景**

用于店长现场看销售、教练或联合场景时，快速记录关键问题和纠偏动作。

**建议使用频率**

每周至少 2 次跟场或录音复盘。

**表单模板**

| 日期 | 员工姓名 | 岗位 | 场景节点 | 观察到的关键表现 | 做得好的地方 | 当前主要问题 | 影响结果的原因判断 | 当场纠偏动作 | 一周内复查点 | 复查时间 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
|  |  | 销售 / 教练 / 联合 | 前采 / 体验 / 反馈 / 成交 / 交接 / 首月服务 |  |  |  |  |  |  |  |

**填写提示**

1. 场景节点必须写具体，不要只写“今天表现一般”。
2. 影响原因要尽量区分是能力问题、习惯问题还是协同问题。
3. 当场纠偏动作要写成可执行句子，比如“下一次反馈先讲亮点，再讲重点问题”。

---

## 四、表单 3：问题员工诊断与1周纠偏计划表

**适用场景**

用于店长对单个问题员工做短周期、强跟踪的纠偏带教。

**建议使用频率**

发现明显短板时立即建立，连续跟 1 周。

**表单模板**

| 员工姓名 | 岗位 | 当前最突出问题 | 具体表现 | 造成影响 | 问题判断 | 本周训练目标 | 纠偏动作 1 | 纠偏动作 2 | 谁陪练 | 复查时间 | 达标标准 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
|  | 销售 / 教练 / 店长储备 |  |  |  | 能力 / 习惯 / 协同 / 态度 |  |  |  |  |  |  |

**填写提示**

1. 一周只抓 1 到 2 个关键问题，不要贪多。
2. 达标标准必须能判断，比如“能完整说出反馈骨架并完成一次稳定承接”。
3. 纠偏动作优先选择可观察动作，不写“提高积极性”这类空泛表述。

---

## 五、表单 4：重点名单与关键家庭推进会记录表

**适用场景**

用于店长主持重点新客、犹豫家庭、风险会员的推进会。

**建议使用频率**

每周至少 1 次，必要时可加开。

**表单模板**

| 会议日期 | 名单类型 | 家庭 / 会员姓名 | 当前阶段 | 当前卡点 | 目前已做动作 | 会议判断 | 主跟进人 | 配合人 | 下次触达时间 | 下次复盘时间 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
|  | 新客 / 犹豫家庭 / 风险会员 / 重点续课 |  | 体验后 / 首月 / 阶段反馈前 / 续课前 |  |  |  |  |  |  |  |

**填写提示**

1. 会议判断要写清楚“现在最可能影响推进的核心因素”。
2. 主跟进人与配合人必须明确，不要多人同时负责导致悬空。
3. 下次触达时间和复盘时间都必须有具体日期。

---

## 六、表单 5：月度带教与经营闭环复盘表

**适用场景**

用于店长月度回看训练、带教和经营结果是否真正联动。

**建议使用频率**

每月 1 次。

**表单模板**

| 月份 | 本月门店关键问题 | 本月重点训练动作 | 参与对象 | 完成情况 | 数据变化 | 仍未解决的问题 | 下月优先动作 | 负责人 | 复盘日期 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
|  |  |  | 销售 / 教练 / 联合 / 店长 |  | 邀约 / 到店 / 体验 / 成交 / 首月稳定 / 续课 |  |  |  |  |

**填写提示**

1. 数据变化不要只看结果绝对值，也要看和本月训练主题是否一致。
2. 仍未解决的问题要落到具体节点，不写“整体执行一般”。
3. 下月优先动作应和下一个月训练排期表形成衔接。

---

## 七、建议落地顺序

1. 每周先填 `门店周训练与检查排期表`。
2. 训练过程中同步使用 `门店跟场与录音复盘检查表`。
3. 遇到问题员工立即建立 `问题员工诊断与1周纠偏计划表`。
4. 每周用 `重点名单与关键家庭推进会记录表` 做推进。
5. 每月底用 `月度带教与经营闭环复盘表` 做总复盘。

---

## 八、结论

如果主文件解决的是“店长要会什么”，那么这份表单包解决的是“店长具体怎么做、怎么查、怎么跟、怎么复盘”。

这也是把店长训练真正从培训文件，推进到门店经营动作的关键一步。
