# 追光小牛销售基础专业培训体系总方案

**版本：** v1.1  
**适用对象：** 新销售、在岗销售、店长、讲师、销售与教练联合培训场景  
**适用场景：** 新人入职、在岗提升、门店统一口径、每周演练、每月通关考核  
**定位：** 建立一套面向课程顾问的基础专业能力训练体系，让销售既能讲专业，又不越过教练边界。

---

## 一、培训定位

这套培训不是单纯的销售技巧培训，也不是把销售培养成教练，而是建立一套面向门店接待与转化场景的基础专业能力体系。

培训后的销售应具备四项核心结果：

1. 听得懂专业。
2. 说得出专业。
3. 接得住教练反馈。
4. 转得动课程方案。

统一边界如下：

1. 销售可以理解儿童发展、体测项目与常见表现之间的关系。
2. 销售可以做基础解读、基础反馈和课程承接。
3. 销售不能替代教练做正式评估、深度诊断、训练结论和教学判断。
4. 所有表达必须避免医学化、恐吓化、绝对化、贴标签化。

---

## 二、培训对象

本体系同时覆盖三类对象。

### 1. 新销售入职培训

目标：快速建立品牌、课程、基础专业和接待转化的统一框架。

重点：

1. 建立基础认知。
2. 学会标准表达。
3. 学会基础承接。

### 2. 在岗销售提升培训

目标：把原本会接待、会邀约的销售升级为能做专业解读和课程推荐的顾问型销售。

重点：

1. 提升体测解读能力。
2. 提升家长痛点挖掘深度。
3. 提升销售与教练配合能力。

### 3. 门店全员统一口径培训

目标：让销售、教练、店长在接待、体验、反馈、转化中说同一套话。

重点：

1. 统一品牌口径。
2. 统一分龄解读逻辑。
3. 统一课程承接路径。

---

## 三、培训总目标

本培训体系的总目标是：让销售从“会接待、会报价、会跟进”，升级为“能做基础专业表达、能完成较完整家长反馈、能与教练协同完成课程承接与转化”的专业型顾问。

具体分成三条并重目标：

1. 专业认知建立：让销售理解品牌、课程、儿童发展和体测基础逻辑。
2. 体测后转化：让销售能把结果讲清、把问题讲明、把课程接顺。
3. 销售与教练配合：让前采、体验、反馈、成交形成统一闭环。

## 四、能力模型

建议将销售基础专业培训的核心能力统一为六项。

### 1. 品牌与课程理解能力

要求：

1. 知道追光小牛卖的不是单节课，而是一套儿童运动成长方案。
2. 知道 ACE 体系的核心意义。
3. 知道各正式课程的定位、差异和适配场景。

### 2. 儿童发展基础理解能力

要求：

1. 能理解动作发展、感觉统合、敏感期等基础逻辑。
2. 能区分低龄和高龄孩子的关注重点。
3. 能把专业概念翻译成家长听得懂的话。

### 3. 体测项目与结果解读能力

要求：

1. 知道每个项目在看什么。
2. 知道做得不好通常会有哪些表现。
3. 知道如何把结果自然转入训练方向和课程建议。

### 4. 家长需求识别与沟通能力

要求：

1. 能从孩子表现识别家长痛点。
2. 能从生活、学习、课堂、社交多个维度建立需求。
3. 能避免把沟通做成单纯灌输知识。

### 5. 课程承接与方案推荐能力

要求：

1. 能围绕孩子年龄、体测表现和家长目标推荐课程。
2. 能说清课程为什么适合现在这个阶段。
3. 能把体验和体测结果自然衔接到正式课程。

### 6. 销售与教练协同配合能力

要求：

1. 知道前采怎么收信息。
2. 知道体验中自己看什么。
3. 知道体验后如何承接教练反馈。
4. 知道哪些话自己能说，哪些必须交给教练说。

## 五、培训内容结构

整套培训建议按六大模块展开。

### 模块 A：品牌与业务底层认知

培训目标：让销售知道自己卖的是什么，为什么先体测、再建议课程。

主要内容：

1. 追光小牛品牌定位与核心价值。
2. ACE 体系基础表达。
3. 五周期训练规划。
4. 年龄分层训练思路。
5. 课程体系总览。
6. 门店服务链路与体测角色。

培训结果：

1. 能讲清品牌做什么。
2. 能讲清课程差异。
3. 能讲清体测在成交链路中的作用。

### 模块 B：儿童发展基础认知

培训目标：让销售理解孩子表现背后的发展基础，而不是只会说“体能差”。

主要内容：

1. 七大身体素质。
2. 动作发展基础。
3. 感统基础。
4. 敏感期基础。
5. 常见低龄问题与高龄问题的差异。
6. 家长常见误区与表达边界。

分龄重点：

1. `3~6岁`：感统、参与感、协调、平衡、规则、胆量、动作建立。
2. `7~12岁`：课堂状态、任务完成、体测结果、运动表现、自信与社交。

### 模块 C：体测项目与结果解读

培训目标：让销售真正会看体测、会讲体测、会承接体测。

主要内容：

1. 各项目在看什么。
2. 不同年龄段项目差异。
3. 常见表现与背后能力。
4. 对学习、生活、性格、社交和运动参与的影响。
5. 标准家长表达。
6. 课程承接话术。
7. 沟通边界与禁区。

体测模块必须拆成两组：

1. `3~6岁体测项目知识卡组`
2. `7~12岁体测项目知识卡组`

### 模块 D：家长沟通与需求挖掘

培训目标：让销售把专业内容转成需求，而不是只把知识讲给家长听。

主要内容：

1. 家长常见关注点分类。
2. 低龄家长痛点挖掘。
3. 高龄家长痛点挖掘。
4. 从生活表现切入。
5. 从学习表现切入。
6. 从课堂表现切入。
7. 异议处理与边界口径。

### 模块 E：销售与教练配合

培训目标：让销售和教练在接待、体验、反馈、转化上形成一条线。

主要内容：

1. 前采信息收集标准。
2. 体验前信息交接。
3. 体验中销售观察重点。
4. 体验后教练反馈承接方法。
5. 销售与教练各自边界。
6. 现场协同成交逻辑。

### 模块 F：实战演练与转化

培训目标：把知识变成真实门店表达和真实成交能力。

主要内容：

1. 接待场景演练。
2. 体测后反馈演练。
3. 教练反馈承接演练。
4. 课程推荐演练。
5. 异议处理演练。
6. 低龄家长场景演练。
7. 高龄家长场景演练。

## 六、体测模块专项设计

### 1. 结构原则

体测知识卡不再采用“同一张卡内部分龄表达”的旧方式，直接拆成两个独立卡组。

原因如下：

1. 项目本身不同。
2. 家长痛点不同。
3. 解读重点不同。
4. 课程承接逻辑不同。

### 2. `3~6岁体测卡组` 的训练重点

统一聚焦：

1. 感统基础。
2. 动作建立。
3. 参与感与胆量。
4. 平衡、协调、节奏、控制。
5. 规则意识与集体适应。

统一表达方向：

1. 不强调分数焦虑。
2. 更多讲发展基础和动作底子。
3. 更多讲家长在生活中能观察到的表现。

### 3. `7~12岁体测卡组` 的训练重点

统一聚焦：

1. 课堂状态。
2. 学习任务完成度。
3. 体育表现与体测达标。
4. 自信、同伴比较与持续参与。
5. 运动技能与体能底子。

统一表达方向：

1. 不简单报结果，要讲实际影响。
2. 更强调课堂、体育课、运动参与和任务表现。
3. 课程承接更偏专项补底和结果改善。

### 4. 体测解读统一骨架

每张体测卡统一采用以下骨架：

1. 这个项目在看什么。
2. 做得不好常见会有什么表现。
3. 对学习、生活、性格与社交有什么影响。
4. 销售怎么讲。
5. 课程怎么接。
6. 沟通边界提醒。

## 八、实施机制

建议采用“日常抽查 + 每周演练 + 每月通关”的结构推进。

### 1. 日常机制

建议动作：

1. 晨会抽背知识卡。
2. 快问快答体测项目。
3. 抽查课程承接口径。

目标：

1. 保持记忆热度。
2. 形成统一口径。
3. 及时发现表达偏差。

### 2. 每周演练

频率：每周至少 1 次，60 至 90 分钟。

建议轮转主题：

1. 第 1 周：品牌与课程表达演练。
2. 第 2 周：体测解读演练。
3. 第 3 周：家长痛点挖掘与课程承接演练。
4. 第 4 周：销售与教练协同演练。

每周演练应形成三项记录：

1. 演练评分。
2. 共性问题。
3. 下周纠偏重点。

### 3. 每月通关考核

频率：每月 1 次。

建议包含三部分：

1. 知识抽测。
2. 口头抽背。
3. 场景模拟。

考核目标：

1. 不只考知道。
2. 还要考会说。
3. 更要考会用。

---

## 九、考核模型

建议采用 100 分制。

1. 专业知识抽测：30 分。
2. 体测项目解读：25 分。
3. 家长沟通表达：20 分。
4. 课程承接能力：15 分。
5. 销售与教练协同意识：10 分。

建议通关线：

1. 85 分以上：可独立接待与基础反馈。
2. 70 至 84 分：可上岗，但需带教跟进。
3. 70 分以下：继续复训，不建议独立承担关键转化场景。

## 十、知识卡在体系中的定位

知识卡不是全部，但它是整个培训体系的底座。

知识卡主要承担五个角色：

1. 新人快速上手工具。
2. 门店统一口径工具。
3. 晨会抽背工具。
4. 体测后表达训练工具。
5. 复训与纠偏工具。

因此后续知识卡必须满足以下标准：

1. 一张卡拿起来就能背。
2. 每张卡都要有完整逐字稿。
3. 每张卡都要有课程承接话术。
4. 低龄与高龄体测卡必须独立成组。
5. 卡片内容既要能讲专业，也要能直接转化。

## 十五、内测落地建议

建议先按一个完整月跑第一轮内测。

### 第 1 周

1. 讲师按带训大纲完成第 1 到第 2 场。
2. 门店开始使用知识卡和题库做晨会抽测。
3. 店长开始建立个人带教追踪表。

### 第 2 周

1. 讲师完成第 3 到第 4 场。
2. 门店按每周演练方案开始低龄、高龄分场景演练。
3. 重点记录知识会不会转成家长语言。

### 第 3 周

1. 完成第 5 场协同训练。
2. 开始跑顾问-教练双人配合演练。
3. 重点记录谁会抢教练结论，谁接不住反馈。

### 第 4 周

1. 完成第 6 场异议处理与通关预演。
2. 组织一次月度通关考核。
3. 根据结果拉出复训名单和下月重点。

内测阶段重点不要追求一步到位，而是优先看四件事：

1. 文件是否真的有人在用。
2. 销售是否真的能开口。
3. 店长是否真的能持续追踪。
4. 顾问和教练是否真的形成了更顺的配合。

## 十六、后续优化原则

后续进入内测后，统一按以下原则优化：

1. 不在单个板块反复深挖过久。
2. 优先根据真实门店使用反馈调整。
3. 先修最影响落地的表达、流程和评分问题。
4. 不轻易新增大批平行文件，优先在现有闭环内迭代。
