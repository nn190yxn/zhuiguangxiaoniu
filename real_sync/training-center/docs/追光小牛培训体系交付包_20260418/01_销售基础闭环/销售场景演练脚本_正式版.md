# 追光小牛销售场景演练脚本

**版本：** v1.0  
**适用对象：** 课程顾问、店长、讲师、销售与教练联合演练  
**配套文件：** `销售专业知识卡片包_正式版.md`、`销售专业知识抽测题库_正式版.md`、`销售基础专业培训体系总方案_正式版.md`  
**使用场景：** 岗前带训、每周演练、店长复盘、月度通关前模拟

---

## 一、使用原则

1. 每次演练优先练“判断 + 翻译 + 承接”，不是比谁说得更长。
2. 演练必须分低龄场景和高龄场景，不能混着练。
3. 每轮演练都要有一个明确目标，如低龄平衡承接、高龄跳绳承接、教练反馈承接。
4. 所有表达必须避免医学化、恐吓化、绝对化、贴标签化。
5. 演练结束后必须留下三项结果：得分、共性问题、下次纠偏点。

---

## 二、通用演练结构

1. 讲师导入场景：2 分钟。
2. 学员单人或双人演练：3 至 5 分钟。
3. 讲师追问：1 至 2 分钟。
4. 店长或讲师点评：2 分钟。
5. 当场重说一遍修正版：1 分钟。

建议评分统一看五项：

1. 有没有先判断孩子当前表现。
2. 有没有翻译成家长听得懂的话。
3. 有没有讲到学习、生活、参与感、自信或社交影响。
4. 有没有把课程承接自然接出来。
5. 有没有踩边界。

---

## 三、单人演练脚本

### 场景 1：低龄平衡与胆量

**场景设定**

孩子 `4岁半`，到店时不敢上平衡木，走窄线时会停，家长说孩子从小就胆小。

**家长原话**

`他是不是就是胆子太小，所以什么都不敢玩？`

**顾问目标**

1. 不能直接把原因归成性格。
2. 要连接平衡、身体控制和安全感。
3. 要自然承接快乐体操。

**标准演练脚本**

`很多低龄孩子看起来像胆子小，背后不一定只是性格，更多时候是身体控制和安全感这块还没建立起来。像平衡、身体稳定、敢不敢把动作做出去，这些底子如果还没跟上，孩子就会更保守。这个阶段我们更建议先把平衡、参与感和动作底子带起来，所以会更适合从快乐体操这类基础课开始。`

**讲师追问**

1. 如果家长继续说“回家多带他去公园就行了吧”，你怎么接？
2. 你这段话里哪一句最容易说重？

### 场景 2：低龄感统与坐不住

**场景设定**

孩子 `5岁`，体验课中切换项目慢，排队会乱动，家长担心孩子是不是感统有问题。

**家长原话**

`他是不是感统有问题？`

**顾问目标**

1. 不能直接下判断。
2. 要用“信息输入到动作输出”的逻辑解释。
3. 要保留课程价值。

**标准演练脚本**

`我们现场不会直接给孩子下这种结论。更容易理解的说法是，孩子这块基础可能还需要重点带一带，也就是身体和环境里的信息进来以后，能不能接收好、整理顺，再把动作和状态调出来。现在看他在排队、切换和规则跟随上会有点乱，说明这块底子还值得早点带。像快乐体操这种课，会把平衡、协调、规则感和参与状态一起往上带。`

**讲师追问**

1. 如果家长追问“那是不是越早练越好”，你怎么说才不制造焦虑？
2. 这里最不能出现的两个词是什么？

### 场景 3：高龄跳绳目标明确

**场景设定**

孩子 `8岁`，跳绳成绩不稳定，家长希望尽快提分。

**家长原话**

`我们就想先把跳绳提上去，别的先不管。`

**顾问目标**

1. 先认可目标，不要直接反驳。
2. 要判断是专项问题还是底子问题。
3. 要讲清体能跳绳课与基础体能课的承接区别。

**标准演练脚本**

`跳绳目标很明确，这个方向我们可以一起抓。只是我们通常会先看一下，孩子现在是单纯跳绳专项动作没练顺，还是节奏、协调、持续控制和体能底子也要一起补。如果基础已经有了，体能跳绳课会更直接；如果现在一快就乱、一累就掉，可能要先把体能训练课 / 体能强化课打底，再去拉专项，会更稳。`

**讲师追问**

1. 什么情况下你可以直接推荐体能跳绳课？
2. 如果家长只比较“哪门课提分更快”，你怎么回应？

### 场景 4：高龄体育成绩差

**场景设定**

孩子 `10岁`，折返跑和台阶测试偏弱，家长认为孩子就是不愿意练。

**家长原话**

`他体育差就是因为不肯练。`

**顾问目标**

1. 不能把责任简单归到孩子态度上。
2. 至少讲到一项体能基础和一项课堂或体育表现影响。
3. 承接到体能训练课 / 体能强化课。

**标准演练脚本**

`有些孩子看起来像不愿意练，其实是练起来真的很吃力。像折返跑这类项目，不只是跑得快不快，它还看启动、变向、制动和再次加速；台阶测试偏弱，也常说明体能承载不足。孩子一做就累、总跟不上，成就感会越来越低，久了就更容易显得抗拒。像这种情况，我们一般会先建议把体能底子补起来，再去看专项结果。`

**讲师追问**

1. 这里为什么不能说“孩子自律性差”？
2. 如果家长问多久能改善，你该怎么答？

## 四、双人演练脚本

### 场景 5：销售与教练反馈承接

**场景设定**

孩子 `6岁`，教练刚完成体验反馈，提到平衡、协调和规则跟随需要一起带。

**角色分工**

1. 教练负责专业反馈，不做谈单推进。
2. 顾问负责把反馈翻译成家长可理解的成长路径和课程方案。

**教练示范表达**

`今天体验下来，孩子在平衡、连续动作配合和规则切换上都还有提升空间。不是单项动作不会，而是身体组织和参与状态还没完全连起来。`

**顾问承接脚本**

`您可以把它理解成，孩子现在不是某一个动作不会，而是整个人的稳定、配合和参与状态还没真正带顺。所以我们不建议只盯一个动作去练，而是先把底子补起来。结合孩子年龄，现阶段更适合从快乐体操先入，让平衡、协调、规则感和参与感一起往上走。`

**讲师点评重点**

1. 顾问有没有重复教练原话，而不是做家长语言翻译。
2. 顾问有没有越位说结论。
3. 顾问有没有自然接出课程。

### 场景 6：高龄体验后方案承接

**场景设定**

孩子 `9岁`，教练反馈跳绳、折返跑和核心稳定都偏弱，家长重点关注学校体测。

**教练示范表达**

`孩子今天主要暴露的是节奏、折返切换和中段稳定不够，所以跳绳一快就乱，变向一急就容易散。`

**顾问承接脚本**

`您关心学校体测，这个方向是对的。只是孩子现在不是只差一个跳绳动作，而是节奏、切换和中段稳定一起影响了结果。所以课程上我们更建议先看底子够不够。如果底子明显薄，就先体能训练课 / 体能强化课打底；如果基础已经有了，再衔接体能跳绳课，会更容易把结果稳稳提起来。`

**讲师点评重点**

1. 有没有先接住家长目标。
2. 有没有讲清“不是只差一个动作”。
3. 有没有给出清楚的推荐分流逻辑。

## 五、异议处理演练脚本

### 场景 7：孩子还小

**家长原话**

`孩子还小，先不着急吧。`

**标准脚本**

`完全可以理解。只是低龄阶段的重点本来就不是赶结果，而是趁孩子现在更适合建立底子的时候，把平衡、协调、参与感和规则感这些基础先带起来。不是说晚了就不行，而是现在顺着规律做，通常会更省力。`

### 场景 8：先回家练练

**家长原话**

`我们先回家练练，不行再说。`

**标准脚本**

`回家练当然可以。只是很多家庭真正难的不是知道做什么，而是不知道先练什么、怎么持续练、练偏了怎么调整。门店课程的价值不是多几个动作，而是有体测、有反馈、有阶段调整，能让孩子少走弯路。`

### 场景 9：我们再看看

**家长原话**

`我们再看看。`

**标准脚本**

`可以，您回去也可以再比较一下。只是建议您重点看三件事：能不能讲清孩子现在的表现，能不能讲清为什么这么安排，后面有没有持续反馈和成长路径。这样您比较的就不只是价格，而是这个方案到底适不适合孩子现在这个阶段。`

---

## 六、讲师使用建议

1. 每次带训优先选 3 个场景，不要一节课把所有脚本都过完。
2. 同一场景至少练两轮，第二轮必须按点评后重说。
3. 低龄场景优先查“有没有把发展基础讲清”，高龄场景优先查“有没有把结果影响和课程分流讲清”。
4. 双人演练时，必须单独点评顾问和教练两边的边界。
5. 所有演练记录建议同步进 `门店带教追踪表_正式版.md`。
