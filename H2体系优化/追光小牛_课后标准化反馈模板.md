# 追光小牛 · 课后标准化反馈模板

> 版本：v1.0 | 2026年7月起执行
> 适用对象：全体教练
> 用途：每节课后60分钟内，在GYMGO系统 + 家庭群发送
> 核心语言：ACE成长体系（Athleticism · Cognition · Engagement）

---

## 一、为什么要做课后反馈？

课后反馈不是"交作业"，是你和家长的**第二次连接**。

第一次连接是上课时家长在门外看到的——孩子在动、在笑、在练。
第二次连接就是这条反馈——让家长知道**今天学了什么、进步在哪里、回家怎么配合**。

**好反馈 → 家长感知专业 → 续费意愿强 → 转介绍自然来**

---

## 二、ACE家长反馈结构

每一次反馈，按这个顺序写：

```
① A（运动能力）→ 今天学了什么动作/技能？
② C（认知能力）→ 今天在专注/反应/规则方面表现如何？
③ E（参与动能）→ 今天参与状态怎么样？亮点是什么？
④ 家庭建议 → 回家可以练什么？
```

**不用写很长，但四个点都要覆盖。**

## 三、标准模板（填空即用）

### 模板A：常规课反馈（每日用）

> **追光小牛运动馆【学员姓名】今日课堂反馈🌸**
>
> **A运动能力：** 今天学习了_______（动作/技能），完成情况_______。
> （例：今天学习了前滚翻，能在保护下独立完成，团身不够紧需要继续练）
>
> **C认知能力：** 今天_______（专注/反应/协调/规则理解）方面_______。
> （例：今天指令反应很快，能听懂两步口令；排队轮换时比上周更自觉）
>
> **E参与动能：** 今天课堂状态_______，亮点是_______。
> （例：今天非常投入，游戏环节主动举手当队长，带领队友完成了接力赛）
>
> **📝 家庭建议：** 回家可以_______，每天_______分钟。
> （例：回家可以练"原地拍球"50次，每天5分钟，下周我们继续练行进间运球）
>
> 教练：_______  感谢您的信任，我们一起见证孩子的成长💪

---

### 模板B：暑假班反馈（密集训练用）

> **追光小牛【暑假班·学员姓名】今日训练反馈🔥**
>
> **今日训练主题：** _______
>
> **A运动能力：** 今天训练了_______（核心动作/技能），进步_______。
> （例：跳箱落地控制比开班时稳了很多，膝盖弯曲角度到位了）
>
> **C认知能力：** 今天在_______方面有提升。
> （例：现在能自己数节奏跳绳，不需要教练喊口令了——这是自我节奏感建立的好信号）
>
> **E参与动能：** 今天的状态_______。
> （例：全程情绪高涨，虽然最后一组有点累了但还是坚持完成了）
>
> **📝 家庭建议：** _______
>
> 教练：_______  坚持就是胜利💪

### 模板C：阶段体测/评估后反馈（季度用）

> **追光小牛【学员姓名】阶段成长反馈📊**
>
> **阶段对比：**
> - 上次评估（____月）：A___分  C___分  E___分
> - 本次评估（____月）：A___分  C___分  E___分
>
> **进步亮点：** _______
> （例：A维度从65分提升到78分——跳跃落地稳定性进步明显）
>
> **当前重点：** _______
> （例：C维度仍需加强——复杂动作中的身体协调性，我们会放入每节课的热身环节练习）
>
> **下阶段计划：** _______
> （例：下季度重点推进跳绳+跑酷课程，继续提升运动能力的同时锻炼反应速度）
>
> **📝 家庭配合：** _______
>
> 教练：_______  下一阶段，我们一起努力🚀

## 四、ACE话术库（写反馈时参考）

不知道怎么夸？不知道怎么说"问题"？从下面选：

### A（运动能力）话术

| 场景 | 话术 |
|:---|:---|
| 动作完成好 | "动作完成度很高，身体控制力明显进步" |
| 动作在进步 | "比上次稳定了很多，还需要继续打磨动作细节" |
| 动作待加强 | "动作模式还在建立中，我们下周会继续巩固" |
| 体能好 | "耐力很好，能跟完整个课程" |
| 体能偏弱 | "后半程体力有点跟不上了，建议增加日常运动量" |

### C（认知能力）话术

| 场景 | 话术 |
|:---|:---|
| 指令感好 | "指令感很好，一说就明白，能立即执行" |
| 专注力好 | "整节课注意力集中，不容易受外界干扰" |
| 专注力偏弱 | "偶有分心，容易被旁边的事物吸引，我们会在课中加强对他的引导" |
| 规则意识好 | "能快速理解规则并遵守，示范能力强" |
| 反应速度快 | "反应速度很快，在游戏环节中表现突出" |
| 协调性在进步 | "身体双侧协调性在提升，左右动作的衔接越来越流畅" |

### E（参与动能）话术

| 场景 | 话术 |
|:---|:---|
| 非常投入 | "情绪高涨，全程积极参与，课堂氛围带动者" |
| 稳步参与 | "今天愿意参与所有环节，虽然没有特别兴奋但很稳定" |
| 比以前勇敢 | "比上次更勇敢了，敢于尝试之前不敢做的动作" |
| 社交进步 | "今天主动和队友配合，团队意识在萌芽" |
| 挑战意愿强 | "遇到困难不轻易放弃，愿意反复尝试直到成功" |
| 略羞涩 | "今天有点害羞，需要教练引导才能加入，但我们给了足够的耐心和鼓励" |

## 五、反馈的4个坑（别踩）

| 坑 | 错误示范 | 正确做法 |
|:---|:---|---|
| 只讲A不讲C和E | "今天练了前滚翻" | "今天前滚翻完成度不错（A），指令反应也很快（C），整节课情绪很好（E）" |
| 只说问题不说方向 | "跳绳节奏感不好" | "跳绳节奏感还在建立中，我们下周会用节拍练习帮他找到感觉" |
| 说的太专业听不懂 | "核心力量不足，髋屈肌群紧张" | "腰腹力量还需要加强，回家可以多练平板支撑" |
| 只说"挺好的"没有细节 | "今天表现不错" | "今天上课很专注，能独立完成连续拍球20次，比上周多了5次" |

## 七、一句话原则

> **让家长看到的不是"今天练了什么"，而是"孩子在哪个维度进步了"。**
