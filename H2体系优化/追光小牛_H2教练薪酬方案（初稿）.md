## 一、教练收入总公式

```
月收入 = V4底薪（按星级）
        + V4课时费（按星级×出勤人数）× 出勤系数
        + 业绩提成（新签+续费合并，2%-6%阶梯）
        + 续费激励（续费金额×比例）
        + 消课达标奖（门店超额才有）
```

## 三、H2新增：出勤系数

### 3.1 为什么加

教练不管会员出勤——家长请假说"好的" → 出勤率低 → 耗课慢 → 续费窗口打不开。

**课时费不再固定发放，而是和教练的出勤管理质量挂钩。**

### 3.2 出勤系数怎么算

```
出勤系数 =（排课到课率系数 × 50%）+（人均耗课系数 × 50%）
```

**两个维度并行，互相制衡，防止作弊。**

### 3.3 排课到课率

| 定义 | 数据来源 |
|:---|:---|
| 该教练当月所有课的 总实到人次 ÷ 总应到人次 | GYMGO排课系统 |
| 应到=排课时登记的学员数，实到=教练点名标记的数 | |

### 3.4 人均耗课

| 定义 | 数据来源 |
|:---|:---|
| 当月教练名下所有会员的耗课总和 ÷ 名下会员总数 | GYMGO+店长月初核定名单 |
| 月初由店长核定教练名下会员名单和人均耗课目标 | 店长确认 |

> **防作弊逻辑**：教练把长期不来的会员从排课里删了 → 排课到课率上去了，但人均耗课会降下来 → 两头堵。

### 3.5 系数分档标准

| 等级 | 排课到课率 | 人均耗课达成率 | 对应系数 | 教练感知 |
|:---:|:---:|:---:|:---:|:---:|
| ⚪ 好 | ≥80% | ≥100% | **1.10** | 涨10% |
| 🟢 达标 | 70-79% | 80-99% | **1.00** | 持平 |
| 🟡 一般 | 60-69% | 60-79% | **0.90** | 降10% |
| 🔴 差 | <60% | <60% | **0.80** | 降20% |

### 3.6 举例

**教练A（真正管出勤）：** 排课到课率82%→1.10，人均耗课达成95%→1.00

```
出勤系数 =（1.10 × 50%）+（1.00 × 50%）= 1.05
一星8人班：80元 × 1.05 = 84元/节（比V4涨了）
```

**教练B（作弊删会员）：** 排课到课率85%→1.10，人均耗课只有60%→0.80

```
出勤系数 =（1.10 × 50%）+（0.80 × 50%）= 0.95
一星8人班：80元 × 0.95 = 76元/节（比V4降了）
```

**教练C（不管出勤）：** 排课到课率65%→0.90，人均耗课70%→0.90

```
出勤系数 =（0.90 × 50%）+（0.90 × 50%）= 0.90
一星8人班：80元 × 0.90 = 72元/节（降了）
```

## 四、H2新增：续费激励

### 4.1 规则

```
续费激励 = 教练名下会员的续费金额 × 2%
不设门槛，续一单拿一单
```

### 4.2 举例

教练月续4单 × 均价4,500元 = 18,000元续费流水

```
续费激励 = 18,000 × 2% = 360元
```

---

## 五、H2新增：消课达标奖

### 5.1 规则

```
门店月耗课目标 = 该店待消课金额 ÷ 10（即10个月周转）

当月实际耗课 - 门店目标 = 超额部分

超额≤10% → 超额×3%
超额10-20% → 超额×5%
超额>20% → 超额×8%

全店奖金池分配：店长30% / 教练团队60%（按耗课贡献分） / 销售10%
```

### 5.2 举例

小河店：月目标13.2万，实际17万（超额28%）

```
超额部分 = 3.8万
奖金池 = 3.8万 × 8% = 3,040元
店长30% = 912元
教练团队60% = 1,824元（按个人耗课分）
销售10% = 304元
```
