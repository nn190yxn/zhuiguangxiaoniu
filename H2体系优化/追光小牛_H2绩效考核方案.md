# 追光小牛 · H2绩效考核方案

> 版本：v1.0 | 2026年7月起执行
> 适用：全员考核，逐岗设定KPI

---

## 一、考核总则

### 1.1 考核目的

考核不是为了扣钱，是为了让每个人清楚：
- **H2的重点是什么**（续费/消课/教案/休眠激活）
- **自己做到什么程度算合格**
- **做得好有什么回报，做不好有什么后果**

### 1.2 考核频率

| 岗位 | 考核频率 | 考核人 |
|:---|:---:|:---|
| 总经理 | 季度 | 创始人 |
| 督导 | 月度 | 总经理 |
| 教学主管 | 月度 | 总经理 |
| 店长 | 月度 | 督导+总经理 |
| 教练 | 季度（星级评定） | 店长+教学主管→总经理审批 |
| 销售 | 月度 | 店长→总经理 |

## 八、考核结果应用

### 8.1 星级与薪酬挂钩

| 星级 | 底薪 | 课时费（按出勤人数定） | 业绩提成 |
|:---|:---:|:---:|:---:|
| 实习 | 2100 | 按V4标准 | 新签+续费合并→2%-6%阶梯 |
| 新星 | 2500 | 按V4标准 | 同上 |
| 一星 | 2500 | 按V4标准 | 同上 |
| 二星 | 2700 | 按V4标准 | 同上 |
| 三星 | 3000 | 按V4标准 | 同上 |

> 注：课时费标准、提成阶梯详见 `02E_薪酬结构.md`。H2新增的出勤系数、消课达标奖详见《H2教练薪酬方案》。

### 8.2 不达标处理

| 情况 | 处理 |
|:---|:---|
| 单月店长KPI<60% | 督导约谈，制定改善计划 |
| 连续2月店长KPI<60% | 总经理约谈+帮扶 |
| 教练连续2月星级不达标 | 降级警告→改善期1个月→仍不达标→降级 |
| 销售连续2月新签<60% | 店长约谈→帮扶→仍不达标→调岗/优化 |
