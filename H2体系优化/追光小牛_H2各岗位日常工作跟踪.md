# 追光小牛 · H2 各岗位日常工作跟踪

> 每个岗位每天看什么、做什么、管什么。一张表定频次。

---

## 一、教练

| 频次 | 事项 | 标准/工具 | 没做到怎么办 |
|:---:|:---|---|:---|
| **日** | 提交教案（上课前） | 按教案审核清单提交 | 未提交→不能上课 |
| **日** | 按教案上课 | 当日教案在手 | 巡课发现不符→书面提醒 |
| **日** | 课后反馈（系统+家庭群） | ACE话术模板，60分钟内 | 系统抽查，缺失扣星级分 |
| **日** | 点名标记出勤（课后30min内） | GYMG0系统 | 店长日检查 |
| **周** | 查看名下会员出勤数据 | GYMG0系统 | 店长抽查 |
| **周** | 休眠/窗口期会员触达 | 店长分配的名单 | 周会汇报进度 |
| **月** | 续费产值统计 | GYMG0系统 | 关联星级评定 |
| **季** | 体测+阶段评估报告 | AI工具+给家长 | 季度必做，关联星级 |

## 二、店长

| 频次 | 事项 | 标准/工具 | 没做到怎么办 |
|:---:|:---|---|:---|
| **日** | **看数据：今日线索数+邀约数+到店数+成交数+业绩** | GYMG0数据仪表盘 | 督导检查 |
| **日** | **盯资源：今日各渠道新增线索数（抖音/美团/转介绍）** | 抖音/美团后台 | 线索下降→查原因 |
| **日** | **主动拓资源：今日做了什么动作找新资源？（异业/社区/商场活动/转介绍推动）** | 店长工作记录 | 日会汇报 |
| **日** | **盯邀约成交：今天预约到店的客户是谁、跟进了没有** | 销售跟进记录 | 店长晨会过 |
| **日** | 看日控6勾 | GYMG0数据 → 看数→开会→派工→巡店→升级→帮扶 | 督导检查 |
| **日** | 检查教练教案提交 | 审核清单，初审打勾 | 未交→找教练 |
| **日** | 检查教练出勤标记完成率 | GYMG0系统 | 未标→提醒 |
| **日** | 巡店（安全和卫生） | 巡店清单 | 巡店记录 |
| **周** | **本周业绩目标+完成进度** | 周目标vs实际 | 周会复盘 |
| **周** | **主动拓资源：本周做了几个异业触达/社区活动/转介绍推动** | 店长工作记录 | 周会汇报 |
| **周** | **资源量复盘：本周线索量+邀约率+到访成交率** | GYMG0漏斗数据 | 周会分析 |
| **周** | 分配续费清单（≤20节+休眠） | GYMG0导出名单→分配教练 | 周会检查 |
| **周** | 检查教练续费触达进度 | 教练汇报+系统确认 | 周会复盘 |
| **周** | 随堂听课≥1节 | 听课评分表 | 记录交教学主管 |
| **月** | 月初核定教练名下会员名单 | 确认人均耗课目标 | 督导检查 |
| **月** | **月业绩复盘：新签完成率/续费完成率/资源转化率** | 月目标vs实际 | 店长会汇报 |
| **季** | 星级评定数据准备 | GYMG0拉数据 | 配合教学主管 |
