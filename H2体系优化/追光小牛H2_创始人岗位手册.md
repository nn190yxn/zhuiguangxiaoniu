# 追光小牛 · 创始人岗位手册（H2版）

> 版本：v1.0 | 适用期间：2026年7月-12月
> 定位：定标准、定方向、看红线

---

## 一、岗位定位

**创始人不做日常运营，不做最贵的店长。**

创始人 H2 唯一的工作重心：
> **确保体系在跑，而不是自己下场跑。**

具体就是三件事：
1. **定标准** —— 星级制度、薪酬方案、教学服务承诺的最终拍板
2. **看方向** —— H2 战略节奏把控，不做细节纠偏
3. **看红线** —— 只看连续红色预警，不审日常

---

## 二、H2 核心职责（不超过6条）

| 序号 | 职责 | 说明 |
|:---:|:---|---|
| 1 | **星级评定体系定稿与执行监督** | 9月首轮评定前完成终审，确保公平执行 |
| 2 | **薪酬改革方案定稿** | V4不动+新增三块：续费激励/出勤系数/消课达标奖 |
| 3 | **教学服务承诺底线守护** | 5条承诺出现重大违约时介入，不审日常执行 |
| 4 | **红线监督** | 体罚/数据造假/安全等10条红线，有举报即查 |
| 5 | **月度品质报告审阅** | 只看连续红色预警（连续2月不达标→约谈） |
| 6 | **H2节奏调整决策** | 7-8月/9-10月/11-12月，每阶段末做一次方向校准 |

## 三、日常工作节奏

### 每日（5分钟）
```
□ 不看日控数据（那是总经理/店长的事）
□ 只接收：财务系统自动推送的"红色红灯清单"
  → 无红灯 → 过
  → 有红灯 → 转发给总经理："这个看一下"
```

### 每周（30分钟）
```
□ 周一：看上周总经理提交的"一周异常摘要"
  → 无异常 → 过
  → 有连续异常 → 标记关注
□ 不参加周例会（避免陷入日常细节）
```

### 每月（2小时）
```
□ 月度品质报告审阅（总经理提交）
  → 核心指标：各店续费率/消课完成率/库存周转/星级分布
  → 只看连续2个月红色预警的店/人
  → 连续红色 → 约谈店长/教练
  → 3个月连续红色 → 考虑人事调整
```

### 每季（4小时）
```
□ 星级评定结果终审（9月/12月）
  → 确认评定结果公平性
  → 审批星级调整生效
□ 季度战略复盘
  → 当前进度 vs H2目标
  → 是否需要调整节奏
```

| 时间 | 决策事项 | 参与程度 |

## 五、红线（碰了就要你介入的）

| 红线 | 处理方式 |
|:---|---:|
| 体罚/变相体罚学员 | 立即辞退，创始人直接处理 |
| 数据造假（业绩/教案/出勤） | 首次警告+乐捐，二次辞退 |
| 私自收费/收受红包 | 立即辞退 |
| 重大安全事故 | 创始人亲自处理 |
| 连续3个月绿色指标变红 | 调整责任人 |

## 六、沟通汇报线（H2版）

```
        创始人
      （定标准/看红线/定方向）
           │
      ┌────┴────┐
      │         │
    总经理     督导
  （日常运营  （总经理品质执行臂
   品质控制）   事实记录+上报）
      │
  ┌──┴──┬──┬──┐
 店长1  店2  店3 ...
```

**关键原则：**
- 创始人不对店长/教练直接下指令（全部通过总经理）
- 创始人不参加周例会/日会
- 创始人只看红色，不看绿色
