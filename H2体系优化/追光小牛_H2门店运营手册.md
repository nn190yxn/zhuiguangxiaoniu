## 一、店长角色定位（H2版）

### 1.1 一句话定位

> **店长只看三件事：系统在不在跑、人在不在做、红灯有没有亮。**

- 系统在不在跑 → 日控6勾每天有没有人打勾
- 人在不在做 → 续费/休眠激活/教案审核的动作有没有执行
- 红灯有没有亮 → 数据异常一出现就要处理

### 1.2 H2角色边界（调整重点）

| 角色 | 新签 | 续费（H2调整） | 休眠激活（H2新增） | 教案审核（H2新增） |
|:---|:---:|:---:|:---:|:---:|
| **你（店长）** | 盯进度+派工 | **盯流程+分配+升级** | **盯触达率+分配** | **检查提交+初审** |
| 教练 | — | 信任传递（评估+反馈+体测） | 回访+邀约 | 写教案+课后反思 |
| 销售 | 主责 | **成交关门** | 协助 | — |
| 教学主管 | — | — | — | 终审+质量评分 |

### 1.3 H2核心指标（店长级）

| 指标 | H2目标 | 你每天/周要看的 |
|:---|---:|:---|
| 月耗课 | **55.9万（10个月周转）** | 日耗课是否达标 |
| 续费窗口覆盖率 | **100%** | 周排查是否有遗漏 |
| 休眠触达率 | **≥80%（7月）** | 周触达进度 |
| 教案提交率 | **100%** | 日课前检查 |
| 会员出勤率 | **≥75%** | 出勤系数挂钩薪酬 |
| 课后反馈及时率 | **≥90%** | 日抽查 |
| 消课达标奖 | **超额阶梯3%-5%-8%** | 月核算 |

### 4.2 H2新增/调整周动作详解

#### 周三：续费窗口排查（H2调整）

> 原流程：排查→分配教练跟进→教练全责续费
> H2流程：排查→分配教练做信任传递→移交意向家长→**销售做成交关门**

```
Step 1 → 打开GYMGO→筛选未来30天到期会员
Step 2 → 逐个确认续费意向和状态
Step 3 → 分配动作：
   ├─ 有意向但还没确认 → 教练做阶段评估+面谈（信任传递）
   ├─ 教练评估后 "可以续了" → 移交给销售跟进成交（H2新增）
   └─ 无意向/卡点 → 店长升级介入
Step 4 → 形成续费跟进名单（含责任人+状态+预计成交日）
Step 5 → 班前会通报本周续费进度
```

#### 周五：教案审核统计（H2新增）

```
本周教案数据统计：
  ☐ 教案提交率：提交数÷应提交数 = __%
  ☐ 审核一次通过率：一次通过数÷提交数 = __%
  ☐ 退修次数：__次

异常处理：
  - 提交率<100% → 约谈未提交教练
  - 连续2次退修 → 主管辅导
```

#### 周五：消课+出勤数据统计（H2新增）

```
本周消课数据：
  ☐ 本周实际耗课：____元
  ☐ 本月累计耗课：____元
  ☐ 本月目标耗课：____元（门店月目标）
  ☐ 进度：__%

本周出勤数据：
  ☐ 本周平均出勤率：__%
  ☐ 出勤率<75%的会员：__人（需重点关注）
```
