# 追光小牛 · 教案审核打分表

> 版本：v3.0 | 2026年7月起执行
> 适用对象：教学主管/店长/三星教练
> 用途：审核教练提交的教案是否达标，评分结果纳入星级考核
> 核心语言：ACE成长体系（Athleticism · Cognition · Engagement）

---

## 一、一票否决项（任一项不满足→直接退回，不评分，不得上课）

| 序号 | 否决项 | 检查标准 |
|:---:|:---|---|
| 1 | 无安全预案 | 高风险动作未标注/无保护站位/无并发控制 → 三者缺一即否决 |
| 2 | 课程流程不完整 | 热身/技能教学/游戏互动/放松拉伸 四个环节缺少任意一个 |
| 3 | **无ACE三维目标** | A（运动能力）/C（认知能力）/E（参与动能）三个维度缺任意一个 |
| 4 | **无主攻维度选择** | 教案未标注本课主攻维度（☐A ☐C ☐E） |
| 5 | 无器材清单 | 未列出本课所需的器材名称和数量 |

---

## 二、评分表（满分100分）

### （一）基本信息 + ACE目标（15分）

| 检查项 | 分值 | 达标标准 | 得分 |
|:---|---:|:---|---:|
| ACE三维目标完整 | 6分 | A（运动能力）/C（认知能力）/E（参与动能）各写了一句话 | /6 |
| 主攻维度明确 | 3分 | 圈选了本课主攻维度（☐A ☐C ☐E） | /3 |
| 课程线类型标注 | 3分 | 标注了本课是功能性训练还是技能性训练 | /3 |
| 基本信息齐全 | 3分 | 课程名称、班级、日期、主教、助教、学员人数全部填写 | /3 |
| **小计** | **15分** | | **/15** |

### （二）课程流程（45分）

#### ① 热身环节（10分）—— 主要看E激活

| 检查项 | 分值 | 达标标准 | 得分 |
|:---|---:|:---|---:|
| E要点（如何让学员进入状态） | 4分 | 写了热身中的趣味性/E激活方案 | /4 |
| A要点（热身动作写具体） | 3分 | 写了具体的动作名称和顺序 | /3 |
| C要点（热身中建立的认知/规则） | 3分 | 写了热身环节的口令/规则引导 | /3 |
| **小计** | **10分** | | **/10** |

#### ② 技能教学环节（20分）—— 主要看A+C

| 检查项 | 分值 | 达标标准 | 得分 |
|:---|---:|:---|---:|
| 核心技能明确 | 3分 | 写清楚了本节课教的1-2个核心运动技能 | /3 |
| 教学步骤完整 | 4分 | 写了示范→练习→纠正→再练习的完整步骤 | /4 |
| **ACE分层设计** | **7分** | A层（运动能力：基础版/进阶版）+ C层（认知能力：指令简化/完整）+ E层（参与动能：胆怯/兴奋学员的引导方式）——三个维度至少写了两个 | **/7** |
| C维度体现 | 4分 | 教学中体现了认知能力的训练（指令理解/规则意识/身体协调） | /4 |
| 主教/助教分工 | 2分 | 写了主教和助教的课堂分工 | /2 |
| **小计** | **20分** | | **/20** |

#### ③ 游戏互动环节（10分）—— 主要看E拉动

| 检查项 | 分值 | 达标标准 | 得分 |
|:---|---:|:---|---:|
| 有游戏名称 | 2分 | 写了具体的游戏名称 | /2 |
| ACE关联明确 | 4分 | A关联（练了什么运动能力）/ C关联（规则是否可懂）/ E关联（全员参与+激励方式） | /4 |
| 有音乐设定 | 2分 | 写了音乐名称/类型，或标注"无音乐" | /2 |
| 有激励方式 | 2分 | 写了激励方式（积分/勋章/称号/击掌） | /2 |
| **小计** | **10分** | | **/10** |

#### ④ 放松拉伸环节（5分）—— 主要看ACE总结

| 检查项 | 分值 | 达标标准 | 得分 |
|:---|---:|:---|---:|
| A总结（运动能力） | 2分 | 写了拉伸动作+运动能力的总结 | /2 |
| C总结（认知能力） | 1分 | 写了规则/方法方面的总结 | /1 |
| E总结（参与动能） | 2分 | 写了学员状态/进步表扬 | /2 |
| **小计** | **5分** | | **/5** |

### （三）安全与纪律（25分）

| 检查项 | 分值 | 达标标准 | 得分 |
|:---|---:|:---|---:|
| 身体安全（A） | 8分 | 高风险动作标注、保护站位明确、并发控制写清 | /8 |
| 心理安全（E） | 5分 | 写了胆怯学员/情绪波动学员的处理预案 | /5 |
| 认知安全（C） | 4分 | 写了高风险动作前的规则讲解方式 | /4 |
| 器材清单完整 | 5分 | 器材名称+数量+规格都写清楚 | /5 |
| 课程线对齐 | 3分 | 本课内容与课程线专业目标一致（不跑题串课） | /3 |
| **小计** | **25分** | | **/25** |

### （四）课后执行确认（课后勾选，5分）

| 检查项 | 分值 | 达标标准 | 得分 |
|:---|---:|:---|---:|
| 按教案执行了教学 | 3分 | 教案流程与课堂实际一致 | /3 |
| 课后反馈按时发了 | 2分 | 60分钟内系统+家庭群已发 | /2 |
| **小计** | **5分** | | **/5** |

---

## 三、总分与等级判定

| 总分 | 等级 | 处理 |
|:---:|:---:|:---|
| **80-90分** | **A（优秀）** | 通过，纳入优秀教案库，月度分享，星级加分 |
| **68-79分** | **B（合格）** | 通过 |
| **50-67分** | **C（需修改）** | 退回修改，24小时内重提交 |
| **<50分** | **D（不合格）** | 退回重写，48小时内重提交；连续2次D→约谈 |

## 五、打分表与指引对照关系

| 指引（教练写教案时看） | 打分表（主管审核时用） |
|:---|---|
| "A（运动能力）/C（认知能力）/E（参与动能）各写一句目标" | (一) ACE三维目标完整：6分 |
| "选一个主攻维度" | (一) 主攻维度明确：3分 |
| "功能性训练 vs 技能性训练" | (一) 课程线类型标注：3分 |
| "热身写A/C/E要点" | ① A要点3分 + C要点3分 + E要点4分 |
| "技能教学写ACE三层分层" | ② ACE分层设计：7分 |
| "游戏写ACE关联" | ③ ACE关联：4分 |
| "放松写A/C/E三个总结" | ④ A总结2分 + C总结1分 + E总结2分 |
| "安全写A身体/E心理/C认知" | (三) 身体8分 + 心理5分 + 认知4分 |
| "课后反思完成ACE四问" | (四) ACE课堂判断四问：5分 |

> **教练拿着指引写 = 对着打分表上的得分点写**
> **主管拿着打分表审 = 对着指引上的标准审**
> **同一套标准，上下一致。**
