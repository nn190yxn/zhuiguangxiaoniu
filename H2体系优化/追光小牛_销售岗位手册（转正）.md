## 一、岗位定位

> **新签主责，续费关门，转介绍主动要。**

销售是现金流的发动机。H2核心变化：续费从"教练全责"调整为"教练信任传递+销售成交关门"，销售不仅要做好新签，还要接住续费的最后一脚。

---

## 二、核心职责（7条）

| # | 职责 | 具体内容 |
|:---:|:---|:---|
| 1 | **新签成交** | 线索跟进、邀约到店、体验成交，完成个人新签目标 |
| 2 | **续费成交** | 接教练移交的续费客户→做成交关门 |
| 3 | **转介绍** | 每笔成交/续费后主动开口要转介绍 |
| 4 | **资源获取** | 五条线拉资源：抖音/美团/转介绍/异业/商场活动 |
| 5 | **抖音运营** | 每天1条校区视频，线索30分钟内回复 |
| 6 | **美团运营** | 经营分≥80分，回复率100% |
| 7 | **成交交接** | 每次签单后5项信息录入GYMGO+同步教练 |

## 三、日常工作流

### 每天

| 时间 | 事项 | 标准 |
|:---:|:---|---:|
| 早 | 查看今日邀约到店名单+预约时间 | 今日到店表，晨会过 |
| 随时 | **线上线索30min内响应** | 抖音/美团/转介绍线索，超时书面警告 |
| 随时 | 跟进已邀约未到店客户（催到店） | 跟进记录 |
| 随时 | 邀约到店≥X组（按店目标） | GYMGO系统 |
| 随时 | **每成交一笔，主动开口要转介绍** | 店长抽查话术 |
| 下午 | 接教练移交的续费客户→24h内联系 | 续费交接流程 |
| 下班前 | **今日业绩+明日邀约计划** | 日报提交店长确认 |

### 每周

| 事项 | 标准 |
|:---|---:|
| **本周新签目标+续费成交目标复盘** | 周目标vs实际 |
| **本周资源量统计（抖音/美团/转介绍各多少条）** | 渠道来源统计 |
| **本周邀约率+到访成交率复盘** | 漏斗数据分析 |
| 续费成交跟进 | 续费清单周会汇报 |
| 转介绍开口执行自查 | 本周成交了几笔、要了几个转介绍 |

### 每月

| 事项 | 标准 |
|:---|---:|
| **月业绩复盘（新签金额+续费金额+转介绍单数）** | 月目标vs实际，月会汇报 |
| **月资源转化分析（各渠道来了多少、转化了多少）** | GYMGO漏斗，月会汇报 |

## 四、考核指标

| 维度 | 权重 | 指标 | 达标 |
|:---|:---:|:---|---:|
| 新签业绩 | 40% | 个人月新签目标达成率 | ≥80% |
| 续费成交 | 15% | 接教练移交续费case成交率 | ≥50% |
| 转介绍 | 15% | 月转介绍线索量 | 个人目标 |
| 邀约率 | 10% | 线索→到店率 | ≥45% |
| 到访成交率 | 10% | 到店→成交率 | ≥45% |
| 抖音/美团运营 | 10% | ①每天1条视频 ②美团经营分≥80 ③线索30min内回复 | 全部达标 |

## 五、薪酬结构

```
月收入 = 底薪（3,000元/月）
        + 业绩提成（新签+续费合并，2%-6%阶梯）
        + 续费激励（续费金额×2%，不设门槛）
        + 消课达标奖（门店超额才有 → 店长30%/教练60%/销售10%）
        + 带教奖金（带新人转正后：300元/人，提前转正500元/人）
```

**业绩提成阶梯**：

| 个人月业绩 | 提成比例 |
|:---:|:---:|
| 0 - 5,000 | 2.0% |
| 5,001 - 15,000 | 2.5% |
| 15,001 - 25,000 | 3.0% |
| 25,001 - 35,000 | 3.5% |
| 35,001 - 45,000 | 4.0% |
| 45,001 - 55,000 | 4.5% |
| 55,001 - 65,000 | 5.0% |
| 65,001 - 75,000 | 5.5% |
| 75,001以上 | 6.0% |
