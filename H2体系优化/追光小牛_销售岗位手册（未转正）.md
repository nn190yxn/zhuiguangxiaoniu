## 一、岗位定位

> **先跟听 → 再协助 → 最后才能独立谈单。**

未转正销售处于学习和成长阶段，核心任务是完成试岗培训、掌握接待全流程、通过销售演练考核，达到转正标准。

---

## 二、核心职责（转正前）

| # | 职责 | 具体内容 |
|:---:|:---|:---|
| 1 | **完成试岗培训** | Day1-3试岗培训+Day4-6跟岗实习，Day7通过试岗考核 |
| 2 | **做好销售助理工作** | 协助正式顾问接待、邀约、系统录入、视频拍摄 |
| 3 | **学习接待全流程** | 掌握首次到店8步法+异议处理话术 |
| 4 | **通过销售演练考核** | 8步法+异议处理≥B级，通过后方可独立签单 |
| 5 | **积累有效电话** | 转正前累计≥60通有效邀约电话 |

## 三、日常工作流

### 试岗期（第1-7天）

| 天数 | 内容 | 考核方式 |
|:---:|:---|---:|
| Day1 | 品牌文化+组织架构+课程体系+GYMGO操作 | 口头+实操 |
| Day2 | 首次接待8步法+签单话术+模拟接待 | 模拟考核 |
| Day3 | 续费话术+家长沟通+投诉处理 | 情景模拟 |
| Day4-6 | 跟岗实习：全程跟带教，旁听≥3组接待+电话回访 | 跟岗记录 |
| Day7 | 试岗考核：接待流程模拟+系统操作+课程体系问答 | 店长评分 |

### 销售助理期（每天）

| 时间 | 事项 | 标准 |
|:---:|:---|---:|
| 上午 | 检查今日邀约名单+协助备单 | 帮正式顾问提醒到店客户 |
| 随时 | 线上线索协助处理（回复+录入） | 2h内 |
| 随时 | 电话邀约已留资未到访客户 | 每天≥10通有效电话 |
| 到店客户 | 协助接待（倒水/引导/准备体测表） | 主动配合 |
| 午/晚 | 每天拍摄1条校区视频 | 手机拍摄，带教审核后发 |
| 下午 | 整理客户跟进记录 | 带教审核 |
| 下班前 | 学习话术+复盘当天遇到的情况 | 学习记录 |

### 通过演练考核后

通过销售演练考核（8步法+异议处理≥B级）后，可独立签单：

| 事项 | 标准 |
|:---|---:|
| 独立线索跟进+邀约 | 按8步法执行 |
| 独立接待+成交 | 按流程执行，拿2%-6%提成 |
| 每成交一笔主动要转介绍 | 带教抽查 |
| 每日业绩+明日计划报告长 | 日报 |

## 四、转正标准

| 维度 | 标准 |
|:---|:---:|
| 入职天数 | ≤48天 |
| 销售流程考核 | 8步法完整通过 |
| GYMGO录入准确率 | ≥95% |
| 试岗期考核 | 全部通过 |
| 跟岗实习确认 | 带教确认全部执行合格 |
| 有效电话累计 | ≥60通 |
| 接待实操 | 带教监督下完成≥3组完整接待 |
| 美团/抖音操作 | 能独立完成核销+回复+评价管理 |
| 培训考核 | 课程体系+销售话术+异议处理均达标 |
| 家长投诉 | 零投诉 |
| 个人出勤 | 100% |

**业绩转正标准（满足任一即可）**：

| 方案 | 周期 | 累计业绩 | 说明 |
|:---:|:---:|:---:|:---|
| 方案A | **首月转正** | **5万元** | 能力强当月达标当月转 |
| 方案B | **2个月转正** | **8万元** | 月均4万，稳定型 |
| 方案C | **3个月转正** | **9万元** | 月均3万，常规型 |

> 业绩含新签+续费合并计算。到考核期未达标→延长1个月，延长期满仍未达标→辞退。

## 六、带教管理

- 带教师傅负责全程指导和考核
- 每一份客户跟进记录需带教审核
- 通过销售演练考核后方可独立签单
- 未通过演练考核前不得独立报价和成交
- 试岗期未通过→结束试岗；试用期未通过→延长1个月或辞退

**带教激励**：你顺利转正后，带教师傅可获得300元奖金（提前转正500元）。
