## 二、H2 战略目标

### 2.1 战略定位

> **现金流是发动机，消课是传动轴，续费是方向盘。**

### 2.2 四大战略目标

```
🥇 现金流：H2业绩442万，全年完成率80%+
   ├─ 续费：追100% = 225万 ← 核心增长引擎
   └─ 新签：稳住 = 217万 ← 转介绍+新渠道降成本

🥈 消课：月均55.9万，库存10个月周转
   ├─ 暑假班+休眠激活 → 打开续费窗口
   └─ 出勤系数挂钩薪酬 → 倒逼教练管出勤

🥉 续费能力系统化（为2027年续费50%+打底）
   ├─ ≤20节预警+教练信任传递+销售关门
   └─ 续费激励到位 → 复利增长

㊃ 教练激活+H2过渡方案落地
   ├─ 薪酬改革+星级评定+教案自主化
   └─ 末位淘汰配合招聘旺季
```

## 四、定事儿：行动计划

> 按优先级逐块展开，持续补充中。

### 4.1 续费225万行动计划

**核心逻辑：**
```
教练做信任积累（反馈+评估+体测）→ 不说成交
销售做成交关门（续费最后一脚）
续费激励×2% → 续了就拿钱
```

#### 关键动作清单

| # | 动作 | 责任人 | 时间 | 说明 |
|:---:|---|---|:---:|---|
| 1 | ≤20节预警周报 | 店长 | 每周 | 从系统导出清单，分配至对应教练 |
| 2 | 课后标准化反馈 | 教练 | 每次课后 | 含学员表现+动作要点+家庭建议 |
| 3 | 阶段体测+规划报告 | 教练 | 结合暑假班 | AI已支持，仅需录入+给家长 |
| 4 | 教练说"可以续了" | 教练 | 窗口期 | 仅做信任传递，不负责成交 |
| 5 | 销售接手成交 | 销售 | 教练移交后 | 谈价格/方案/付款 |
| 6 | 续费激励核算 | 财务 | 月度 | 续费金额×2%，随工资发放 |

#### 拟细化内容（待补充）
- 休眠激活具体节奏（7月集中触达80%→电话回访→体测邀约→暑假班回归）
- 教练反馈模板设计
- 销售与教练的交接流程SOP

### 4.2 新签217万+资源获取行动计划

**核心判断：**
> 当前月均线索仅约200条，月均新签天花板约18万。
> H2月均需新签36.2万 → 资源量必须翻倍。

#### 渠道扩源方案

| 渠道 | 现状 | H2目标/月 | 关键动作 |
|:---|:---:|:---:|:---|
| 抖音 | 约175条/月 | **≥200条** | 提升内容质量；尝试每周直播（哪怕一次） |
| 美团 | 约125条/月 | **≥150条** | 优化店铺页面+评价维护+活动更新 |
| 转介绍 | 约50-70条/月 | **≥100条** | 满意度提升→触发机制（老带新激励） |
| 异业合作 | 0 | **30-50条** | 幼儿园/学校体测导流、社区合作 |
| 商场活动 | 0 | **20-30条** | 周末体能挑战赛→留资 |
| 私域裂变 | 0 | **20-30条** | 现有会员朋友圈集赞送课 |

#### 蓄水节奏

```
7-8月（暑假班）= 蓄水期
  暑假班家长 → 满意度高 → 转介绍来源
  休眠激活回归 → 转化来源

9月（开学季）= 放水期 🔥
  砸资源转化 → 冲100万

10月（续费窗口）= 二次蓄水
  续费会员 → 口碑传播 → 新的转介绍

11月（双十一）= 二次放水 🔥
  续费+转介绍双驱动 → 冲92万
```
