# 追光小牛 · 教练教案制作指引

> 版本：v3.0 | 2026年7月起执行
> 适用对象：全体教练
> 用途：教练写教案时对照这份指引，确保每一份教案达到标准
> 核心语言：ACE成长体系（Athleticism · Cognition · Engagement）

---

## 一、为什么要有教案？

教案不是写给公司看的，是写给**你自己**看的。

一份好教案能让你：
- **上课前想清楚**：这节课要解决学员A（运动能力）、C（认知功能）、E（参与动能）哪个维度的卡点
- **上课时有节奏**：不慌不忙，知道下一步做什么
- **下课后有依据**：课后反馈可以基于教案写，不用临时想

**教案质量 = 课堂质量 = 家长感知的专业度 = 续费意愿**

---

## 二、追光小牛核心教学语言：ACE成长体系

ACE不是口号，是追光小牛课堂观察、教案设计和家长沟通的共同语言。

### 2.1 ACE三维定义

| 维度 | 英文 | 中文 | 教练要关注什么 |
|:---|:---|---:|:---|
| **A** | **Athleticism** | **运动能力** | 身体素质（力量/速度/灵敏/耐力）+ 运动技能掌握 |
| **C** | **Cognition** | **认知能力** | **2026升级重点**——专注力、反应速度、身体协调性、规则意识、指令听从。通过运动优化大脑功能。 |
| **E** | **Engagement** | **参与动能** | 内驱动力、兴趣、自信、社交互动、挑战意愿。让孩子真正爱上运动。 |

**ACE的核心逻辑：**
- 孩子不仅要练得强壮（A）
- 还要通过运动优化大脑功能（C）← 这是追光小牛区别于其他体能馆的核心差异
- 更要让孩子发自内心喜欢运动（E）

### 2.2 课堂判断顺序

教练在课堂上判断学员，必须按这个顺序：

> **①先看E：孩子愿不愿意参与？情绪状态怎么样？**
> **②再看C：孩子听不听得懂指令？专注力跟不跟得上？**
> **③最后看A：孩子的运动能力完成度如何？**

**为什么？**
- **E没进来**，直接练A效果打折扣，孩子还可能产生抵触
- **C没建立**，指令听不懂，动作完成质量不会高
- **E和C到位了**，A的提升才是可持续的

### 2.3 每节课的ACE最小判断

每节课完成4个判断：

```
判断1：本课主攻ACE哪个维度？
判断2：学员最卡的点在哪里？（动作做不了A / 指令听不懂C / 不愿参与E）
判断3：当堂调整方向？（降动作难度A / 简化指令C / 先稳情绪E）
判断4：课后反馈时，能用ACE语言告诉家长孩子哪里进步了？
```

### 2.4 功能性训练 vs 技能性训练

追光小牛的课程线分为两大类，写教案时必须清楚自己的课属于哪一类：

| 类型 | 课程线 | ACE侧重 |
|:---|---:|:---|
| **功能性训练** | 感统体操、体态矫正、跑酷 | **E和C先行**——先在参与动能和认知功能上打好基础，再抓运动能力 |
| **技能性训练** | 篮球、跳绳、中考体训、搏击防身 | **A和C并重**——运动技能和认知功能同步推进 |

## 三、教案六大原则

### 原则一：安全性 ⭐（一票否决）

**ACE视角：安全不只是身体安全（A），还包括心理保护（E）和认知清晰（C）。**

| 检查项 | ACE | 合格标准 |
|:---|:---|---:|
| 身体安全 | A | 高风险动作标注、保护站位明确、并发人数控制 |
| 心理安全 | E | 没有会让学员恐惧/尴尬的动作设计；胆怯学员有逐步适应方案 |
| 认知安全 | C | 高风险动作前必须写清楚：先讲规则→再示范→再执行 |
| 应急预案 | A+E | 写了本课常见意外及第一反应动作 |

### 原则二：渐进性

**ACE视角：渐进不只是动作难度递增（A），还包括认知复杂度递进（C）和参与深度递进（E）。**

| 检查项 | ACE | 合格标准 |
|:---|:---|---:|
| 动作递进 | A | 热身→基础→进阶→挑战，难度逐步提升 |
| 认知递进 | C | 指令从简单到复杂：先单一口令→再组合口令→再条件变化 |
| 参与递进 | E | 从个体练习→双人配合→小组合作→全班对抗 |
| 降阶/升阶 | A+C | 每个核心动作写出降阶方案和升阶方向 |

### 原则三：系统性

**不允许跑题串课。ACE目标必须对齐课程主线。**

| 检查项 | 合格标准 |
|:---|---:|
| 课程线对齐 | 篮球课不能上成体能课 |
| 与月度主题对齐 | 教案标注本月教学主题 |
| 周间衔接 | 标注本课与上周的递进关系 |

### 原则四：趣味性（主攻E维度）

**ACE视角：趣味性直接服务于E维度——孩子的内在驱动力。**

| 检查项 | 合格标准 |
|:---|---:|
| 情境设计 | 至少1个环节有情景带入（故事、角色、挑战任务），让孩子"有理由参与" |
| 游戏化 | 至少1个环节包含游戏/竞赛/闯关 |
| 音乐 | 写清楚音乐名称或类型 |
| 激励方式 | 积分/勋章/称号/击掌鼓励 |
| E观察 | 教案注明本课重点观察哪类学员的E状态（慢热的/兴奋过度的/容易放弃的） |

### 原则五：个性化（ACE三层分层）

**ACE视角：分层不只分动作难度（A），还要分认知支持（C）和参与引导（E）。**

| 检查项 | ACE | 合格标准 |
|:---|:---|---:|
| A层分层 | A | 核心环节至少2个难度层级（基础版/进阶版） |
| C层分层 | C | 对理解慢的学员用短指令+示范，对理解快的学员用完整指令 |
| E层分层 | E | 参与方式分层：观望→跟随→主导 |

### 原则六：可执行性

| 检查项 | 合格标准 |
|:---|---:|
| 时间分配 | 热身10+技能20+游戏15+放松10 = 55min，留5min过渡 |
| 器材可行 | 器材校区已有，数量和规格明确 |
| 人员可行 | 主教/助教分工清楚 |

## 四、教案制作步骤

### 第一步：填基本信息 + ACE三维目标

```
课程名称：___________    课程线类型：☐功能性训练 ☐技能性训练
班级/级别：___________
上课日期：___________    主教：___________ 助教：___________
学员人数：___________    课程时长：___________
本月教学主题：___________

【本课ACE目标】（三个维度各写一句）
A目标（运动能力——身体素质/技能掌握）：_______________
C目标（认知能力——专注/反应/协调/规则意识）：_______________
E目标（参与动能——兴趣/自信/社交/挑战意愿）：_______________
★ 本课主攻维度：☐A  ☐C  ☐E
```

**要点：**
- 三个维度各写一句，不能不写或少写
- 功能性训练优先选E或C为主攻；技能性训练优先选A或C为主攻

---

### 第二步：写课程流程（四环节）

#### 环节一：热身（10分钟）

**ACE为什么做热身？**
- **E（参与动能）**：把孩子从"静止"切换到"运动"状态，建立参与感
- **A（运动能力）**：身体预热，防止受伤
- **C（认知能力）**：让孩子知道"要上课了"，收心，建立专注

**教案里写什么：**

```
热身动作：写具体动作名称和顺序（不可以只写"热身"）

A要点：热身动作与今天主练的运动能力有无关联？
  例：本课练跳跃→热身中加入"原地纵跳+落地缓冲"

C要点：热身中建立了什么认知/规则？
  例：本课需要排队轮换→热身中就练习"听哨声换动作"

E要点：如何让学员进入状态？
  例：动物模仿热身，教练带头玩起来
```

---

#### 环节二：技能教学（20分钟）

**ACE为什么做技能教学？**
- **A（运动能力）**：这是家长付钱的核心原因——孩子要学到运动技能
- **C（认知能力）**：运动中的指令理解、规则意识、身体协调——这些都是C的训练
- **E（参与动能）**：让孩子在"学会了"的过程中获得成就感

**教案里写什么：**

```
核心技能：这节课教什么动作/运动能力？

教学步骤：
  Step 1 示范——教练做标准动作，边做边讲要点（A+C）
  Step 2 练习——学员分组练习，逐个纠正（A）
  Step 3 纠正——共性问题集中纠正，个别单独提示（A+C）
  Step 4 再练习——纠正后再巩固（A）

ACE分层：
  A层（运动能力）：基础版（降阶） / 进阶版（升阶）
  C层（认知能力）：指令简化版（慢） / 完整版（正常）
  E层（参与动能）：胆怯学员怎么引导 / 兴奋过度怎么控制
```

---

#### 环节三：游戏互动（15分钟）

**ACE为什么做游戏？**
- **E（参与动能）**：孩子投入了才会真的爱上运动——这是续费的情感基础
- **A（运动能力）**：在游戏中巩固练习的运动技能
- **C（认知能力）**：游戏规则本身就是认知训练——专注、理解、反应

**教案里写什么：**

```
游戏名称：必须有名（不是"做游戏"）

规则：怎么玩？

ACE关联：
  A关联：这个游戏练了什么运动能力？
  C关联：游戏规则孩子能听懂吗？需要先演示吗？
  E关联：全班都能参与吗？有激励方式吗？

音乐：写清楚音乐名称或类型

激励方式：积分/勋章/称号/击掌
```

---

#### 环节四：放松拉伸（10分钟）

**ACE为什么做放松？**
- **A（运动能力）**：身体恢复，拉伸放松，防止运动后不适
- **E（参与动能）**：用正面的总结收尾，让孩子期待下次来
- **C（认知能力）**：帮孩子回顾"今天学了什么"，加深记忆

**教案里写什么：**

```
拉伸动作：写具体（不可以只写"拉伸"）

ACE课堂总结：
  A总结：今天练了什么运动能力？
  C总结：今天学习了什么规则/方法？
  E总结：今天谁进步了？哪位学员值得表扬？
```

---

### 第三步：写安全与器材

```
安全注意事项：
  ☐ 身体安全（A）：高风险动作标注+保护站位+并发控制
  ☐ 心理安全（E）：胆怯学员怎么处理
  ☐ 认知安全（C）：高风险动作前的规则讲解方式

器材清单：
  ☐ 器材名称+数量+规格
```

## 七、常见问题自查

| 问题 | 级别 | 怎么改 |
|:---|:---:|:---|
| 没写ACE三个维度目标 | ✗ 不合格 | A/C/E各写一句 |
| 没选主攻维度 | ✗ 不合格 | 圈出本课重点推A/C/E哪个 |
| "热身"两个字就完了 | ✗ 不合格 | 写具体动作 + A/C/E要点 |
| 技能教学没写ACE分层 | ✗ 不合格 | A层/C层/E层至少写两层 |
| 没写游戏名 | ✗ 不合格 | 想一个名字 |
| 没写安全保护 | ✗ 一票否决 | 不能上课 |
| 没写器材清单 | ✗ 不合格 | 写清楚器材+数量 |
| 没写音乐 | 🟡 建议加 | 有比没有好 |
| 没写ACE教学反思 | 🟡 须补填 | 四个判断点必须完成 |
