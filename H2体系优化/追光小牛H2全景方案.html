🏃 追光小牛 2026 H2 全景方案

战略目标 → 核心问题 → 启动线 → 薪酬 → 星级 → 节奏 → 关键指标

🏆 战略目标
激活教练专业投入度
续费能力系统化 → 为2027年续费占比50%+打底
从"连而不锁"到"真连锁"
落地标准：家长走进任何一家店，体验是一样的
  • 教案格式一致 / 续费流程一致 / 服务承诺一致
🔴 核心问题
课时费偏高 → 教练舒适区
  • 课时费仅占消课13%，但月入6-7k已满足 → 不续费、不管出勤
库存周转 15.6 个月 目标 8-10 个月
业绩完成 17% 时间已过 33%
🎯 7-8月三大启动线
① 教学服务承诺 7/1执行，全员已签
  • 5条承诺 → 违约赔课/赔钱
  • 教练"敢承诺=敢跟家长开口"
② 暑假班6大课程 全天可排
  • 体能觉醒营 / 跑酷突击营 / 感统超能营
  • 跳绳满分营 / 决胜满分营 / 篮球小将探索营
  • ✅ 休眠会员回归的钩子 / ✅ 教练密集练兵 / ✅ 消课加速器
③ 休眠会员激活 目标触达率80%
  • 412人90天未上课 → 第一责任人：有会员的教练
  • 实习/新星教练：辅助跟学，不单独承担
  • 暑假班作为"回来"的理由 → 激活后自然入班
💰 薪酬方案 V4不动 + 新增三块
🟢 V4原封不动
  • 底薪按星级（实习2100 → 三星3000）
  • 课时费按出勤人数（V4阶梯表）
  • 业绩提成 2%-6%（新签+续费合并计算）
🟡 新增①：续费激励 续一单拿一单
  • 续费金额 × 固定比例 1.5%-3%（按星级递增）
  • 不设续费率门槛，续了就有
  • 和V4业绩提成叠加（续费被算两次）
🟡 新增②：出勤系数
  • 课时费 × 会员出勤系数
  • 出勤≥85%→×1.10 / 75-85%→×1.00 / 65-75%→×0.90 / <65%→×0.80
  • 不关心出勤的教练 → 课时费自然减少
🟡 新增③：消课达标奖 全店奖金池
  • 目标：待消课 ÷ 10 = 月耗课目标（10个月周转）
  • 超额部分阶梯发奖：超10%以内3% / 超10-20% 5% / 超20%+ 8%
  • 全店分：店长30% / 教练团队60% / 销售10%
  • 按超额部分算，不是按全金额算
⭐ 星级评定 H2过渡版
等级体系（5级不变）
  • 实习 → 新星 → 一星 → 二星 → 三星 → 四星/五星（明年逐步开放）
首轮9月评定·降低版门槛
实习→新星
  • 续费≥15% / 耗课≥100节 / 自身出勤达标
新星→一星
  • 续费≥20% / 耗课≥150节 / 学员出勤率≥60%
一星→二星
  • 续费≥30% / 耗课≥200节 / 月耗课额≥2万
二星→三星
  • 续费≥40% / 耗课≥250节 / 月业绩≥3万
第二轮12月评定
  • 门槛上浮 5-10%
  • 正式开放四星（续费≥50% / 耗课≥260节 / 带教反哺）
  • 班均人次≥5人开始纳入考核
教案审核分工（H2过渡版）
  • 实习/一星教案 → 三星教练初审 → 教学主管终审
  • 二星教案 → 教学主管直接审
  • 三星教案 → 季度抽查
  • 三星教练：每店1-2人，每周审3-4份，约20-30分钟/周
各星级考核重心
实习
  • 核心命题：能不能留下 → 自身出勤+教案按时交+学习态度
新星/一星
  • 核心命题：会不会上课 → 耗课量+教案通过率+学员出勤
二星
  • 核心命题：会员粘不粘 → 会员出勤率+耗课量+续费起步
三星
  • 核心命题：能不能续费+带教 → 续费率+消课达成+审核低星教案
