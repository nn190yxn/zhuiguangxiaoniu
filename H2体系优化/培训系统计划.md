# H2 培训系统实施计划

## 文件
`training-system.html` — 合并培训自学 + 宣讲演示

## 双模式架构

```
首页：选择入口
  ├── 「自学培训」— 按角色分路径 + 考核
  └── 「宣讲演示」— 按主题投屏讲解

───────────────────────────────────

模式一：自学培训（按角色分路径）
  选择角色 → 专属学习路径 → 逐模块解锁 → 测验通过 → 情景考核

模式二：宣讲演示（按主题投屏）
  选择主题 → 选择受众(全员/某岗位) → 投屏讲解页
```

---

## 模式一：自学培训

### 角色学习路径（38 个模块）

#### 教练 (7 个模块)
1. H2与我有什么关系 — 续费角色变化、收入公式概览
2. 怎么写教案 — ACE模板、审核流程、一次通过率
3. 课后反馈怎么做 — ACE话术、60分钟规则、家庭群双发
4. 体测与阶段评估 — 五类体测时机、报告输出
5. 续费信任传递 — 7个节点中教练的3个、面谈话术
6. 休眠会员激活 — 412人名单、触达话术、回归暑假班
7. 薪酬与星级怎么算 — 出勤系数、续费激励、升星门槛、收入举例

#### 销售 (6 个模块)
1. H2与上半年有什么不同 — 续费关门角色、资源缺口
2. 新签全流程 — 线索→邀约→到店→成交，漏斗标准
3. 续费成交关门 — 接教练移交、24h响应、关门话术
4. 转介绍系统化 — 5个触发点、话术、激励方案
5. 五条线拉资源 — 抖音/美团/转介绍/异业/社区
6. 薪酬考核与我 — 新签KPI、续费成交率、日周清单

#### 店长 (8 个模块)
1. H2店长角色变化 — 从等线索到主动拓资源、教案检查新增
2. 日控6勾详解 — 每勾具体看什么、怎么记、怎么跟
3. 周控5查详解 — 续费排查、资源复盘、转介绍、教案、消课
4. 月核4报详解 — 消课奖核算、出勤系数、教案汇总、经营复盘
5. 续费管理全链条 — 分配→检查→升级→移交→销售跟进
6. 教案检查与课堂巡检 — 初审标准、安全审核、随堂听课
7. 团队管理 — 教练1对1沟通、工作量核查、排课逻辑
8. 店长KPI与考核 — 10项KPI权重、不达标处理

#### 教学主管 (6 个模块)
1. H2教学主管定位 — 与督导平级分工、管教练质量
2. 教案终审标准 — 90分制打分表、退回反馈、D级辅导
3. 课堂抽查与ACE匹配度 — 每月≥2节、评分表、归档
4. ACE培训体系 — 新教练考核≥90%、全员ACE掌握度考核
5. 教练帮扶 — 连续2次D级的处理流程
6. 星级评定教学维度 — 40分教学质量数据准备、9月+12月

#### 督导 (5 个模块)
1. H2督导定位 — 与教学主管平级分工、管店长执行
2. 数据巡检与巡店 — GYMG0标记异常、巡店记录表、数据真实性抽查
3. 续费流程检查 — 7个节点各店覆盖、店长KPI评分
4. 新媒体运营 — 抖音分配核销、视频发布检查、美团经营分≥80
5. 转介绍与休眠检查 — 推动店长执行、7月专项触达率

#### 创始人/总经理 (6 个模块)
1. H2战略全景 — 442万、四大引擎、组织链条
2. 四大引擎运作机制 — 交付/续费/运营/销售链怎么闭合
3. 7-12月作战节奏 — 月度业绩、蓄水逻辑、关键节点
4. 薪酬驱动逻辑 — 五层结构、出勤系数防作弊、做与不做月差2090
5. 绩效考核体系 — 各岗频率、KPI权重、不达标处理
6. 跟进节奏与复盘 — 日会/周会/月会、8月预告

### 测验
- 每模块 3-5 题（选择+情景判断），总分100，≥80通过
- 最后模块为情景综合考核
- localStorage 记录进度，断点续学，逐模块解锁

### 宣讲页设计
- 投屏优化：大字号标题、高对比度
- 每页一个要点 + 支撑数据
- 左右键翻页，也可以在顶部切换主题
- 顶部显示当前主题、受众标签
- 底部页码

## 数据量

- 模式一：38 个模块 × 约 200 字内容 + 4 题/模块 ≈ 152 题
- 模式二：10 个主题 × 约 8-12 页/主题
- 预估单文件 5000-6000 行
