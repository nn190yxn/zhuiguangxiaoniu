# 追光小牛 · 教练岗位手册（未转正）

> 版本：v1.0 | 2026年7月起执行 | 实习教练人手一册

---

## 一、岗位定位

> **先观察 → 再协助 → 最后才能独立带课。**

未转正教练处于学习和成长阶段，核心任务是完成试岗培训、掌握助教工作标准、积累实操经验，达到转正条件。

---

## 二、核心职责（转正前）

| # | 职责 | 具体内容 |
|:---:|:---|:---|
| 1 | **完成试岗培训** | Day1-3试岗培训+Day4-6跟岗实习，Day7通过试岗考核 |
| 2 | **做好助教工作** | 按助教标准执行（抓安全+抓秩序+抓个别+抓器材+抓补位） |
| 3 | **完成助教课时** | 累计完成≥100孩子人次助教服务 |
| 4 | **学习写教案** | 按ACE模板写教案，每份带教审核通过 |
| 5 | **写观课记录** | 转正前完成≥8篇，交给带教审核签字 |
| 6 | **完成独立主课** | 在带教监督下完成≥8节完整主课 |

---

## 三、日常工作流

### 试岗期（第1-7天）

见《实习教练工作标准与转正标准》→ 按Day1-Day7计划执行。

### 助教期（第2周起，每天）

| 时间 | 事项 | 标准 |
|:---:|:---|---:|
| 上课前 | 主动与主教沟通教案，协助准备器材 | 提前到馆 |
| 上课前 | 熟悉本堂学员名字和性格 | 提前看名单 |
| 课中 | 按助教标准站位+执行 | 抓安全/抓秩序/抓个别/抓器材/抓补位 |
| 课后 | 整理教室和器材 | 10分钟内完成 |
| 课后 | 主动询问主教助教表现和提升点 | 记录反馈 |
| 课后 | 写观课记录 | 按模板，交给带教签字 |

### 每周

| 事项 | 标准 |
|:---|---:|
| 向带教汇报本周学习情况 | 主动约时间 |
| 检查助教课时累计进度 | 系统数据+带教确认 |
| 按时提交教案 | 至少1份/周，带教审核通过 |

### 入职工资

| 阶段 | 底薪 |
|:---|:---:|
| 实习教练 | 2,100元/月 |
| 转正后（新星） | 2,500元/月+课时费+业绩提成+续费激励 |

## 四、转正标准

| 维度 | 标准 |
|:---|:---:|
| 入职天数 | ≤48天 |
| 试讲评分 | ≥B级 |
| 安全知识测试 | 满分 |
| 助教工作标准 | 带教确认全部执行合格 |
| **助教课时** | **≥100孩子人次**（系统点到数据） |
| **独立主课时数** | **在带教监督下完成≥8节完整主课** |
| 观课记录 | ≥8篇，已审核通过 |
| 教案 | ≥4份完整教案已审核通过 |
| 学员出勤率 | ≥60% |
| 家长投诉 | 零投诉 |
| 个人出勤 | 100%（无迟到/旷工） |

**转正流程**：
```
提交申请 → 店长初审 → 独立带一节完整课（实操考核）+ 体测解读
  → 店长评分 → 总经理审批 → 转正生效
```

---

## 五、带教管理

- 带教师傅负责全程指导和考核
- 观课记录交给带教师傅审核签字，店长每月抽查
- 每份教案师傅审核通过后方可上课
- 转正前不得独立承担主教行为
- 不得因排班压力事实性独立顶岗

**带教激励**：你顺利转正后，带教师傅可获得300元奖金（提前转正500元）。
