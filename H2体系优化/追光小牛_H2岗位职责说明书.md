# 追光小牛 · H2岗位职责说明书

> 版本：v1.1 | 2026年7月起执行
> 核心变化：H2调整后的各岗位定位、职责、协作关系

---

## 一、岗位总览

### 1.1 组织架构（H2版）

```
                     创始人
                       │
                     总经理
                    /      \
                 督导     教学主管
                  │          │
            ┌─────┼─────┐    教练团队
          店长1 店长2 店长3~5  （教学质量）
           / \     / \     / \
        教练 销售 教练 销售 教练 销售
```

**两条线并行：**
- **督导线**（管执行）：受总经理授权，检查评估店长运营执行情况
- **教学主管线**（管质量）：负责教案审核、课堂质量、ACE培训、星级评定教学维度

### 1.2 H2核心分工原则

| 角色 | 新签 | 续费（H2调整） | 教学 | 管理 |
|:---|:---:|:---:|:---:|:---:|
| **创始人** | — | 定方向 | 定标准 | 终审 |
| **总经理** | 盯目标 | 盯数据 | 审核体系 | 管人 |
| **督导** | 查数据 | 查续费流程 | — | 查店长执行 |
| **教学主管** | — | — | **审教案+查课堂+评星级+做培训** | 管教练质量 |
| **店长** | 盯进度+派工 | **分配+升级+检查** | 初审教案+巡课 | 日周月运行 |
| **教练** | — | **信任传递+反馈+体测** | 写教案+上课+反馈 | — |
| **销售** | **主责** | **成交关门（接教练移交）** | — | — |

> ⚠️ H2最大变化：续费从"教练全责"拆分为"教练信任传递→销售成交关门"
> ⚠️ 督导与教学主管平级，一个管运营执行，一个管教学质量，互不隶属

## 四、督导岗位职责

> **查店面执行，验数据真实，管新媒体运营，偏执行层。**

### 4.2 与教学主管的分工

| 对比 | 督导 | 教学主管 |
|:---|:---|:---:|
| 管什么 | **运营执行+新媒体**——店长执行+线上运营 | **教学质量**——教练的教案、课堂、评估是否达标 |

## 五、教学主管岗位职责

### 5.1 一句话定位

> **管教案质量，抓课堂执行，做教练培训，评星级水准。**

### 5.2 核心职责

| # | 职责 | 具体内容 |
|:---:|:---|:---|
| 1 | **教案终审** | **H2新增**：对教练提交的教案做终审质量评分（按ACE审核打分表） |
| 2 | **课堂抽查** | 每月随机听每位教练≥2节课，评课堂匹配度 |
| 3 | **ACE培训** | 组织ACE全员培训、新教练ACE考核、ACE掌握度考核 |
| 4 | **星级评定教学维度** | 提供教案评分、课堂匹配度、ACE评估合格率等数据 |
| 5 | **月度教案数据汇总** | 统计提交率、通过率、退修率、平均评分 |
| 6 | **教练帮扶** | 对教案评分低/课堂匹配度低的教练做专项辅导 |
| 7 | **ACE教学标准更新** | 根据实际执行情况，迭代教案模板和审核标准 |

### 5.3 督导与教学主管协作

| 场景 | 督导 | 教学主管 |
|:---|:---|:---:|
| 店长未督促教练交教案 | 检查→问责店长 | — |
| 教练教案质量差 | — | 审核→退回→辅导 |
| 课堂安全出问题 | 巡店发现→上报 | 审核教案安全预案是否缺失 |
| 教练星级评定 | 提供出勤+续费数据 | 提供教案评分+课堂匹配度+ACE数据 |

## 七、教练岗位职责

### 7.1 一句话定位

> **教好课，做反馈，出体测，续费做信任传递。**

### 7.2 核心职责

| # | 职责 | 具体内容 |
|:---:|:---|:---|
| 1 | 写教案 | 按ACE模板写教案，上课前提交+通过审核 |
| 2 | 按教案上课 | 执行标准化教学，安全保护到位 |
| 3 | 课后反馈 | 每节课后60分钟内发ACE反馈（系统+家庭群） |
| 4 | 阶段体测 | 每季度+暑假班前后完成体测并输出报告 |
| 5 | 续费信任传递 | H2调整：窗口期会员→做阶段评估+体测+面谈→"建议继续"→移交给销售 |
| 6 | 休眠激活 | H2新增：7月集中回访所负责的休眠会员 |
| 7 | 学员出勤管理 | 关注会员出勤率，低出勤会员主动联系家长 |

### 7.3 续费分工说明（H2过渡期）

```
长期目标：教练承担续费全部责任
H2过渡期：鼓励有意愿的教练先做起来，销售做backup
          不想做的教练 → 信任传递后移交销售
          愿意做的教练 → 自己谈成交，拿续费提成
```

### 7.4 H2教练禁止行为

- ❌ 不允许不交教案就上课 → 一票否决
- ❌ 不允许不发课后反馈 → 不结算课时费
- ❌ 不允许不做安全预案 → 一票否决

## 九、H2关键流程协作关系

### 9.1 续费流程分工（H2过渡期）

```
教练想做续费成交 →
  自己做（拿续费提成），销售不做干扰

教练不想做或忙不过来 →
  做信任传递（评估+体测+面谈）
  → 移交销售 → 销售做成交
  → 店长检查流程是否跑通

长期目标：教练承担续费全部责任，H2先让意愿者先富起来
```

### 9.2 教案审核流程分工

```
教练写 → 店长/三星初审 → 教学主管终审 → 通过→上课
                        退回→教练改→再审
```

### 9.3 休眠激活流程分工

```
教练回访 → 暑假班邀约 → 回归上课 → 体测+反馈 → 续费流程
店长检查进度+调配人手
教学主管：回归后的ACE评估
```
