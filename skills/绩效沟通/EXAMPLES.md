# 脱敏案例集

以下案例为模拟/脱敏处理，仅用于展示 Skill 输出效果。

## 案例：绩效沟通双向诊断

### 输入对话

```markdown
【上级】：小王，这个月你的业绩没达标，怎么回事？
【员工】：嗯...最近家里事有点多，我也在尽力推进，就是市场环境不太好，同事那边配合也有点慢。
【上级】：家里的事我理解，但业绩是硬指标。别人也能完成，你怎么就完成不了？是不是得反思一下自己的问题？
【员工】：好的领导，我知道了，我会再想想办法的。
【上级】：那就这样吧，下个月必须达标，不然考核会很被动。
【员工】：嗯...好的。
【上级】：对了，这个月迟到3次，怎么回事？
【员工】：有时候送孩子上兴趣班，偶尔会晚一点。
【上级】：制度就是制度，迟到就是迟到，没有任何借口。
【员工】：...（沉默）
```

### 诊断输出（节选）

```markdown
【对话概览】
- 总字数：312字
- 对话轮次：8轮
- 说话人身份：上级-员工

【潜台词解码（节选）】

员工："家里事有点多，我也在尽力推进"
→ 真实意图：隐性回避，用外部原因掩盖绩效压力
→ 原文证据：三重外部归因（家里事+市场环境+同事配合）

员工："好的领导，我知道了"
→ 真实意图：口服心不服的敷衍式回应
→ 原文证据：重复"好的"，回应长度递减

员工：[沉默]
→ 真实意图：防御性沉默，已关闭真实表达通道
→ 原文证据：上级强调"没有任何借口"后直接沉默

【上级沟通行为评估（节选）】

⚠️ 维度2：给下属留面子的表达空间逻辑
- 不当行为：直接戳破托词，审讯式追问
- 原文证据："是不是得反思一下自己的问题？"
- 对员工影响：员工从解释转为敷衍式回应

⚠️ 维度5：公开表扬私下纠错的反馈逻辑
- 不当行为：在绩效谈话中提及迟到作为"证据"
- 原文证据："迟到就是迟到，没有任何借口"
- 对员工影响：员工从解释直接切换为沉默

【核心诊断】

BEM归因：
- 上级管理行为：40%（主因）
- 员工自身动力：30%
- 员工自身能力：20%
- 环境资源：10%

流失风险等级：高

触发原因：
1. 公平性感知低（"别人也能完成，你怎么就完成不了"）
2. 尊重需求未满足（被当众追问和否定）
3. 支持性需求未满足（只压责任不给方法）

【上级自我优化建议】

1. 把"别人也能完成，你怎么就完成不了"换成"我们来一起看看卡在哪里了"
2. 把"是不是得反思一下"换成"我想听听你对这件事的分析"
3. 建立"先听再说"的习惯，绩效谈话中先让员工充分表达
```
