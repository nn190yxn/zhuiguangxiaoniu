# AI Skills Collection | 企业诊断与管理沟通工具箱

精心打磨的企业级 AI Skills，专注解决中小企业最关键的诊断与管理难题。

---

## 为什么做这个

中小企业不缺执行力，缺的是**精准找到问题**的能力。

- 增长遇到瓶颈，到底是获客问题还是盈利模式问题？
- 员工绩效不理想，是能力问题还是动力问题，或是管理方式的问题？
- 绩效沟通时，员工的"我知道了"到底是真理解还是敷衍？

这两个 skill 试图用结构化的诊断框架，回答这些问题。

---

## Skills

### 1. 企业增长诊断顾问

**一句话**：找到中小企业增长的真实瓶颈。

**诊断框架**：行业诊断 → 创始人能力评估 → 快速扫描 → 深度诊断 → 方案输出

**适用场景**：
- 业务增长停滞，不知道问题在哪
- 团队扩张后管理失控
- 想复制成功模式但不知道怎么标准化
- 创始人疲惫，找不到方向

**诊断维度**：获客能力 / 盈利效率 / 复购与推荐 / 复制能力 / 组织能力 / 战略清晰

**规模适配**：1-200人，支持基础验证期 / 模式打磨期 / 效率优化期 / 战略升级期

**输出**：结构化诊断报告 + 问题优先级排序 + 短/中/长期改进路径

---

### 2. 绩效沟通双向诊断系统

**一句话**：解码中国职场潜台词，诊断上下级沟通的真实问题。

**诊断框架**：高语境文本解码 → 双向问题归因 → 精准落地建议 → 流失风险预警

**适用场景**：
- 绩效面谈后员工离职
- 沟通感觉不对但说不清问题在哪
- 想提升一对一沟通质量
- HRBP/管理者需要客观评估沟通效果

**核心亮点**：
- 中国高语境沟通规则：字面意思 ≠ 真实意图
- 上下级双向诊断：不单向归责
- 吉尔伯特行为工程模型：区分能力问题 / 动力问题 / 管理问题 / 环境问题
- 流失风险预警：基于双因素理论的早期预警

**输出**：潜台词解码报告 / 员工能力-动力诊断 / 上级沟通行为评估 / 落地优化建议 / 风险干预预案

## 使用示例

### 企业增长诊断示例

```
用户：/增长诊断
AI：您的生意，客户是个人消费者、企业客户，还是渠道经销商？

用户：主要是个人消费者，家长带孩子来上课
AI：明白，个人消费者模式。这类业务的关键是获客成本和口碑传播。
    您的平均客单价在什么区间？100元以下/100-1000/1000-1万/1万以上？
...
AI：【行业画像摘要】
■ 基本信息
- 行业：教育培训/个人消费者
- 客单价：100-1000元
- 决策周期：当场的客户

■ 核心瓶颈
- 主诉痛点：获客难
- 建议重点模块：模块A（获客）+ 体验课转化优化
```

### 绩效沟通诊断示例

```
用户：[上传一段绩效面谈对话文本]

AI：【对话概览】
- 总字数：856字
- 对话轮次：6轮
- 说话人身份：上级-员工

【潜台词解码（节选）】
员工："家里事有点多，我也在尽力推进"
→ 真实意图：隐性回避，用外部原因掩盖绩效压力

【上级沟通行为评估】
⚠️ 维度3：权责对等
- 不当行为：只问责不提供支持
- 原文证据："别人也能完成，你是不是得反思一下自己的问题？"

【核心诊断】
保健因素不满（公平性感知低）→ 流失风险等级：高
```
